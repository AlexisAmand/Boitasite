 Recherche tabulateur pour les liens

Les navigateurs plus modernes, permettent grâce à la touche tabulateur du clavier de passer d'un lien à l'autre dans un fichier HTML (avec la touche "Entrée", le lien peut alors être exécuté).

En principe, il est passé d'un lien à l'autre dans l'ordre dans lequel ils sont définis dans le fichier. Vous pouvez cependant en HTML spécifier un autre ordre, par exemple de proposer d'abord les liens du projet et ensuite seulement les liens à d'autres pages Web.

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Strict//EN"
       "http://www.w3.org/TR/html4/strict.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body>

<h1>Parcourez donc avec [Tab] </h1>

<p>
<a href="http://french.imdb.com/" tabindex="4">Base de donnï¿½es de films sur Internet</a><br>
<a href="http://www.chart-lyrics.com/" tabindex="2">Chart Lyrics</a><br>
<a href="http://www.maville.fr/" tabindex="1">Ma ville</a><br>
<a href="http://www.infonie.fr/" tabindex="3">Infonie</a>
</p>

</body>
</html>

Explication:

Avec l'attribut tabindex= dans le repère d'ouverture du lien, vous pouvez faire mention de l'ordre de recherche par tabulateur. Notez la mention pour tous les liens et attribuez lui un nombre. En passant d'un lien à l'autre avec la touche de tabulation, le premier lien sera celui avec le plus petit numéro de tabindex, puis le deuxième plus petit etc... et à la fin celui avec le plus grand numéro tabindex. Dans l'exemple ci-dessus le tabulateur passera d'abord au troisième lien, puis au deuxième, puis au quatrième et enfin au premier.

Les nombres compris entre 0 et 32767 sont permis.

Attention:

Les tabindex se rapportent toujours au fichier affiché complet. Y seront inclus également les  formulaires, le  passage de lien dans les graphiques composés de liens et les  objets. Si, dans votre fichier, en plus de liens normaux, vous avez aussi de tels éléments, vous devez spécifier l'ordre de recherche par tabulateur pour tous les éléments. voir aussi  recherche par tabulateur, raccourcis clavier et mise en grisé.

  

 Raccourcis clavier pour les liens

Vous pouvez proposer à l'utilisateur de passer à un lien déterminé en appuyant sur une touche. Le lien est ainsi choisi - il peut ensuite être exécuté avec la touche "Entrée".

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Strict//EN"
       "http://www.w3.org/TR/html4/strict.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body>

<h1>Mettez vous sur la touche!</h1>

<p>
<a href="http://www.selfhtml/selfhtml.com.fr/" accesskey="t">Selfhtml</a> [Alt]+[t]<br>
<a href="http://www.ac-toulouse.fr/philosophie/listesdediscussion.htm" accesskey="p">Liste des discussions en philosophie</a> [Alt]+[p]<br>
<a href="http://www.willy-online.de/" accesskey="w">Willy online</a> [Alt]+[w]
</p>

</body>
</html>

Explication:

Avec l'attribut accesskey= dans le repère d'ouverture du lien vous pouvez déterminer un signe sur le clavier que l'utilisateur doit appuyer pour passer directement au lien choisi (accesskey = touche d'accès). Chez la plupart des navigateurs, tout est réglé de façon à ce que vous puissiez mentionner une lettre pour accesskey=, lettre qui sera ensuite actionnée en même temps que la touche [Alt] pour accéder au lien.

Attention:

Utilisez les touches qui peuvent être trouvées sur tous les claviers. Les touches de fonction par exemple telles qu'on les trouve sur les claviers MF II pour MS Windows, ne peuvent être choisies avec cette mention.

Attirez l'attention de l'utilisateur dans le texte sur la possibilité d'un raccourci clavier. Utilisez ensuite des touches d'accès homogènes, par exemple sur chaque page la touche h pour revenir à la page d'accueil.

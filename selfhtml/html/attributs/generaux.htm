 Noter des attributs universels

Par "attributs universels" on entend ici les attributs qui sont admis dans presque tous les repères HTML sans y être traités comme attribut individuel. De tels attributs ont des tâches diverses. Il y a par exemple des attributs pour attribuer un nom individuel à un marquage HTML déterminé. D'autres attributs universels comportent des données importantes pour la langue utilisée.

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Strict//EN"
       "http://www.w3.org/TR/html4/strict.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body>

<h1 style="color:black" lang="de">Willkommen!</h1>
<h1 style="color:blue" lang="en">Welcome!</h1>
<h1 style="color:red" lang="fr">Bienvenue!</h1>
<h1 style="color:green" lang="it">Benvenuto!</h1>
<h1 style="color:olive" lang="pt">Boa vinda!</h1>

<p title="Le texte se trouve ici, dans le titre!">Mais oï¿½ est le texte?</p>

</body>
</html>

Explication:

Notez les attributs universels comme des attributs courants associés à des repères dans le repère d'ouverture HTML. Dans l'exemple sont notés plusieurs titres. Tous ont un attribut style=, qui permet le formatage CSS pour l'élément HTML correspondant. En outre, tous les titres ont un attribut lang=, qui désigne la langue du contenu. Le paragraphe de texte en bas de l'exemple a un attribut title=, qui est conçu pour les données Meta de l'élément. Les navigateurs récents affichent ce genre d'information dans une "info-bulle", quand l'utilisateur passe la souris sur le contenu de l'élément.

  

 Liste des attributs universels

Le tableau suivant récapitule les attributs universels généraux.

Attribut Signification Exemple permis dans les repères HTML suivants
class= spécifie que l'élément HTML fait partie d'une certaine classe de feuille de style. L'élément adopte le format défini globalement pour cette classe. Plus de détails dans  Définir des formats pour des classes. <h1 class="nouvellepage"> voir sommaire de référence HTML:
 attributs universels
dir= donne le sens de lecture de la langue utilisée dans l'élément HTML. Intéressant par exemple pour des langues comme l'arabe ou l'hébreu dont le sens de lecture est de droite à gauche. Sont permises les mentions ltr (gauche à droite donc normal) et rtl (droite à gauche). <td dir="rtl"> voir sommaire de référence HTML:
 attributs universels
id= Un nom explicite valable dans tout le fichier pour un élément - quand vous marquez un élément avec cet attribut n'attribuez pas de nom id dans le fichier HTML plus d'une fois. L'attribut id est intéressant par exemple en relation avec le JavaScript récent d'après la syntaxe DOM où la méthode objet  getElementById() existe. Même pour les feuilles de style CSS cet attribut joue un rôle, à savoir quand vous  définissez des formats individuels globaux. Veillez en attribuant le nom aux majuscules et minuscules! Pour un langage tel que JavaScript des noms tels que Exemple et exemple sont différents. <div id="introduction"> voir sommaire de référence HTML:
 attributs universels
lang= donne la langue utilisée à l'intérieur d'un élément HTML. intéressant pour des fichiers composés de plusieurs langues - mais aussi pour les moteurs de recherche sur Internet. Comme mention est permise une abréviation à deux lettres pour le pays par exemple de pour l'allemand, en pour l'anglais fr pour le français, it pour l'italien ou es pour l'espagnol. Vous trouverez des détails à ce sujet ainsi qu'un récapitulatif complet dans la partie traitant des  abréviations de langues. <p lang="it"> voir sommaire de référence HTML:
 attributs universels
style= permet les définitions de formatage individuel d'un élément HTML avec les feuilles de style CSS ou bien avec le  langage par défaut pour feuilles de style mentionné dans l'entête de fichier. Plus de détails à ce sujet sont développés dans le chapitre sur les CSS dans la partie expliquant comment  formater directement des éléments HTML. <p style="color:red"> voir sommaire de référence HTML:
 attributs universels
title= permet de titrer les éléments HTML avec un commentaire ou de lui donner une mention Meta. Le commentaire devrait apparaître dans une info-bulle quand l'utilisateur passe avec la souris dans l'espace d'affichage de l'élément HTML . <p title="ironique!"> voir sommaire de référence HTML:

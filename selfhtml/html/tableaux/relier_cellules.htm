 Relier des cellules dans une rangée sur plusieurs colonnes

Vous pouvez relier plusieurs cellules dans une rangée de façon à ce qu'une cellule de cette rangée s'étende sur plusieurs colonnes.

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Strict//EN"
       "http://www.w3.org/TR/html4/strict.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body>

<h1>Zoologie</h1>

<table border="1">
 <tr>
  <th colspan="2">L'humanit� est compos�e de</th>
 </tr><tr>
  <td>�nes</td>
  <td>singes</td>
 </tr>
</table>

</body>
</html>

Explication:

Avec l'attribut colspan= [nombre de colonnes ] vous obtenez qu'une cellule s'étende sur plusieurs colonnes (colspan = column span = tendre des colonnes). La mention n'est efficace que si le tableau comprend au moins autant de colonnes qu'il est mentionné.

Attention:

Si vous reliez dans une rangée de tableau par exemple des cellules sur quatre colonnes, vous devez définir dans les autres rangées du tableau non pas une mais quatre cellules distinctes. La somme des colonnes dans chaque rangée du tableau doit être la même.

  

 Relier des cellules dans une colonne sur plusieurs rangées

Vous pouvez relier plusieurs cellules dans une colonne de façon à ce qu'une cellule de cette colonne s'étende sur plusieurs rangées.

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Strict//EN"
       "http://www.w3.org/TR/html4/strict.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body>

<h1>Zoologie (II)</h1>

<table border="1">
 <tr>
  <th rowspan="2">Le monde des ânes comprend</th>
  <td>les vrais ânes</td>
 </tr><tr>
  <td>les ânes déguisés (hommes)</td>
  </tr><tr>
  <th rowspan="2">Le monde des singes comprend</th>
  <td>les vrais singes</td>
 </tr><tr>
  <td>les singes déguisés (hommes)</td>
  </tr>
</table>

</body>
</html>

Explication:

Par l'attribut rowspan= [nombre de rangées] vous obtenez qu'une cellule dans une colonne s'étende sur plusieurs rangées (rowspan = tendre des rangées). La mention n'est efficace que si le tableau comprend au moins autant de rangées qu'il est mentionné.

Attention:

Si vous reliez par exemple dans une colonne de tableau une cellule sur quatre rangées, vous devez définir dans la même rangée le contenu de la première cellule à côté de la cellule agrandie. Dans les trois rangées suivantes les trois autres cellules voisines de la cellule agrandie sont également à définir.

  

 Relier des cellules sur plusieurs rangées et plusieurs colonnes

Vous pouvez définir des cellules qui s'étendent à la fois sur plusieurs rangées et sur plusieurs colonnes.

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Strict//EN"
       "http://www.w3.org/TR/html4/strict.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body>

<h1>Zoologie (III)</h1>

<table border="1">
 <tr>
  <th colspan="2" rowspan="2">L'homme est...</th>
  <td>en partie un âne</td>
 </tr><tr>
  <td>d'autre part un singe</td>
 </tr><tr>
  <td>ni un vrai âne</td>
  <td>ni un vrai singe</td>
  <td><b>humain, un homme</b></td>
 </tr>
</table>

</body>
</html>

Explication:

Vous pouvez combiner les attributs colspan= [nombre de colonnes ] et rowspan= [nombre de rangées ] dans une cellule. Cela vous permet d'étendre la cellule sur plusieurs rangées et sur plusieurs colonnes.

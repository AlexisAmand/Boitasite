 Définir des paragraphes de texte

Les paragraphes servent à l'articulation optique d'un texte. Pour l'établissement de fichiers HTML il ne suffit pas d'insérer un brutal passage à la ligne dans l'éditeur. Les navigateurs WWW ignorent de tels passages à la ligne (voir aussi  Règles pour éditer en HTML).

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Strict//EN"
       "http://www.w3.org/TR/html4/strict.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body>
<h1>d&eacute;finir des paragraphes de texte </h1>
<p>Ici commence un paragraphe et ici il se termine.</p>
<p>Ici commence un nouveau paragraphe et ici il se termine.</p>
</body>
</html>

Explication:

<p> (p = paragraph = paragraphe) commence un paragraphe . </p> termine le paragraphe et se trouve à la fin du paragraphe.

Attention:

Les repères <p> tels qu'ils étaient autorisés dans le temps, ne sont entre-temps plus conformes au standard HTML. Notez toujours les paragraphes avec un repère d'ouverture et un repère de fermeture.

L'élément <p> ne peut contenir d'autres éléments générant eux mêmes des blocs comme par exemple des titres, des paragraphes de texte, des listes, des citations ou des adresses.

Pour le texte du paragraphes, les règles pour les  alphabets, caractères spéciaux et caractères HTML spécifiques sont à respecter.

  

 Aligner les paragraphes de texte

Par défaut les paragraphes sont alignés à gauche, sans mention contraire de votre part. Vous pouvez aligner un paragraphe à droite ou le centrer. Une justification est elle aussi possible.

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
       "http://www.w3.org/TR/html4/loose.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body>
<h1>Aligner les paragraphes de texte</h1>
<p align="center">Ceci est un paragraphe centr�.</p>
<p align="right">Ceci est un paragraphe align� � droite.</p>
</body>
</html>

Explication:

Par la commande align="center" dans le repère d'ouverture <p> vous centrez le paragraphe qui suit (align = alignement, center = centré). Par la commande align="right" le paragraphe sera aligné à droite (right = droite). Par la commande align="justify" vous forcez la justification du paragraphe qui suit (justify = justifier). Avec align="left" vous pouvez mentionner le réglage par défaut (alignement à gauche).

Attention:

Tous les navigateurs ne maîtrisent pas la justification.

align est classé dans le standard HTML 4 comme en cours d'abandon (deprecated). Il est recommandé d'utiliser à la place les feuilles de style CSS, par exemple: <p style="text-align:center">...</p>. Autres remarques dans le paragraphe suivant.

  

 Formater des paragraphes de texte avec les CSS

Comment un paragraphe sera représenté dans le navigateur, ne peut guère être modifié par HTML. Les navigateurs utilisent des formatages par défaut pour représenter le texte. Avec les  feuilles de style CSS vous pouvez cependant formater vos paragraphes comme vous le désirez. Pour employer des feuilles de style CSS, il vous faut d'abord savoir comment  définir des formats CSS. Ensuite, vous serez en mesure d'appliquer les propriétés CSS. En relation avec ce qui vient d'être décrit les données de feuilles de style suivantes sont aussi intéressantes:
 formatage de la police
 alignement et paragraphes
 espaces marges extérieures
 espace intérieur
 cadres
 couleurs et images d'arrière plan

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Strict//EN"
       "http://www.w3.org/TR/html4/strict.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body>
<h1>Formater des paragraphes</h1>
<p style="font-family:Arial,sans-serif; font-size:18px; color:blue">un paragraphe format�.</p>
<p style="background-color:yellow">un autre paragraphe format�.</p>
</body>
</html>

Explication:

Dans l'exemple deux paragraphes sont définis. Pour le premier a été décidée une représentation dans la police Arial ou autre police sans serif, d'une taille de 18 pixel et de couleur bleue. Le deuxième paragraphe a reçu une couleur d'arrière plan jaune.

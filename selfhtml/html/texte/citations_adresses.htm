 Définir des citations

Vous pouvez mettre en valeur des citations d'auteur en les plaçant dans un paragraphe séparé formaté différemment (le plus souvent en retrait).

Il s'agit ici d'un marquage logique du contenu. Comment ces paragraphes sont formatés exactement dépend de votre navigateur Web. Le but est cependant d'afficher de tels paragraphes en les mettant en valeur et en les différenciant du reste du texte.

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Strict//EN"
       "http://www.w3.org/TR/html4/strict.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body>
<h1>Franz Kafka</h1>

<p>Franz Kafka �crit sur les corneilles et le ciel :</p>

<blockquote>
Les corneilles pr�tendent qu'une seule corneille pourrait d�truire le ciel ; c'est certain mais ne prouve pourtant rien contre le ciel car le ciel signifie justement: impossibilit� pour les corneilles.
</blockquote>

</body>
</html>

Explication:

<blockquote> introduit un paragraphe séparé pour une citation(blockquote = citation justifiée). </blockquote> termine le paragraphe.

Attention:

L'élément blockquote est volontiers employé pour réaliser des retraits. Étant donné cependant qu'il s'agit d'une commande logique qui n'est pas liée à un formatage déterminé, l'effet de retrait n'est pas garanti.

  

 Citations avec mention de la source URI

Si une citation est disponible sur Internet, vous pouvez mentionner l'adresse URI de la source.

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Strict//EN"
       "http://www.w3.org/TR/html4/strict.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body>
<h1>L'�nergie de comprendre</h1>

<p>N'abandonnez pas car vous avez:</p>
<blockquote cite="http://selfhtml.selfhtml.com.fr/">
l'�nergie de comprendre
</blockquote>

</body>
</html>

Explication:

Avec l'attribut cite= dans le repère d'ouverture <blockquote> vous pouvez mentionner l'adresse URI de la source citée (cite = citation).

Attention:

Netscape 4.x et l' Explorer Internet MS 4.x ne visualisent pas cette mention. Netscape 4.x connaît à la place une mention type="cite", qui provoque l'affichage d'une barre de couleur dans la marge gauche. Cette mention ne fait de toutes façons pas partie du standard HTML.

  

 Définir des adresses

Vous pouvez mettre en valeur des adresses Internet de personnes ou de fichiers dans un paragraphe séparé formaté différemment (le plus souvent représenté en italique et en retrait) . Là aussi il s'agit d'une commande logique qui n'est liée à aucun formatage déterminé.

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Strict//EN"
       "http://www.w3.org/TR/html4/strict.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body>
<h1>Si vous voulez le savoir exactement</h1>

<p>L'�l�ment HTML pour les adresses est � peine utilis�.
Si vous voulez le savoir exactement adressez-vous �:</p>

<address>Dave Raggett, dsr@w3.org</address>

<p>Dave Raggett fait partie du consortium W3 et publie beaucoup de choses sur HTML sur le serveur du W3.</p>

</body>
</html>

Explication:

<address> introduit un nouveau paragraphe pour des adresses Internet. </address> termine le paragraphe.

Attention:

L'élément <address> ne peut contenir aucun autre élément générateur de bloc tels que par exemple les titres, les paragraphes, les listes, les citations ou les adresses.

L'élément <address> est également judicieux en relation avec les  liens à une adresse électronique.

  

 Formater les citations et adresses avec les feuilles de style CSS

Vous n'avez aucune influence avec HTML sur la représentation exacte des citations et adresses . Les navigateurs utilisent des formatages par défaut. Avec les  feuilles de style CSS vous pouvez pourtant formater de tels éléments comme vous le désirez. Pour utiliser des feuilles de style CSS, il vous faut d'abord savoir comment  définir des formats CSS. Ensuite, vous serez en mesure d'appliquer les propriétés CSS. En relation avec ce qui vient d'être décrit les données de feuilles de style suivantes sont aussi intéressantes:
 formatage de la police
 espaces marges extérieures
 espace intérieur
 cadres
 couleurs et images d'arrière plan

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Strict//EN"
       "http://www.w3.org/TR/html4/strict.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body>
<h1>Citation et adresse</h1>

<blockquote style="padding:10px; border:thin solid blue;">
La clart� n'est si on veut, que la variante la plus �vidente de la v�rit�.
</blockquote>

<address style="color:blue;">
Stefan M&uuml;nz, selfhtml@selfhtml.com.fr
</address>

</body>
</html>

Explication:

L'élément blockquote dans l'exemple reçoit une fine bordure bleue continue et le texte qui s'y trouve est séparé de la bordure par un espace de 10 pixels. L'élément address se voit affecter pour sa part une couleur de police bleue.

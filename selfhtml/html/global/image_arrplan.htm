 Incorporer une image d'arrière plan

Vous pouvez déterminer une image d'arrière plan pour l'affichage d'un fichier HTML. Cette image sera toujours répétée sur toute la fenêtre d'affichage de façon à donner un effet de papier peint (Wallpaper). Particulièrement bien adaptés pour des effets de papier peint: les graphiques relativement petits (guère plus grands que la taille d'une icône) et qui représentent quelque motif abstrait. Vous trouverez des exemples typiques dans le chapitre traitant des graphiques dans la partie sur les  papiers peints (images d'arrière plan)

Le graphique d'arrière plan doit être disponible comme fichier graphique au  format GIF ou au  format JPEG.

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
       "http://www.w3.org/TR/html4/loose.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body background="arrplan.jpg" text="#990000" link="#0000CC" vlink="#000066" alink="#000000">
<h1>Text</h1>
<a href="http://fr.yahoo.com/"><b>lien ï¿½  Yahoo</b></a>
</body>
</html>

Explication:

La commande pour incorporer une image d'arrière plan se fait dans le repère d'ouverture <body> du fichier HTML. En ajoutant l'attribut background= vous déterminez un fichier graphique comme image d'arrière plan (background = arrière plan). La totalité du fichier sera affiché sur l'image d'arrière plan définie ici.

Attention:

L'attribut background= est classé en cours d'abandon et doit être évité à l'avenir. Vous obtenez également le même effet à l'aide des  feuilles de style CSS , par exemple comme ceci:
<body style="background-image:url(arrplan.jpg)">

Pour l'adressage de l'image d'arrière plan s'appliquent les règles pour  référencer en HTML.

L'image d'arrière plan n'est valable que pour le fichier dans lequel elle est définie. Si vous établissez un document constitué de plusieurs fichiers HTML qui doivent tous avoir le même fond il vous faut répéter la commande dans chaque fichier HTML.

Si vous définissez une image d'arrière plan vous devez aussi définir des  couleurs pour le texte et les liens .

Utilisez pour les pages WWW orientées texte des images d'arrière plan discrètes qui ne nuisent pas à la lisibilité du texte. Vous ne devriez utiliser des images d'arrière plan qui tapent à l'œil que si toute la page WWW est orientée sur le graphisme et à condition que les graphiques de premier plan suivent avec l'arrière plan tape à l'œil.

Netscape à partir de la version 4.x et l'Explorer Internet MS à partir de la version 4.x interprètent également les  graphiques GIF animés comme image d'arrière-plan.

L'Explorer Internet connaît un attribut supplémentaire bgproperties="fixed" dans le repère d'introduction <body>, à l'aide de laquelle vous créez une image d'arrière plan qui ne bouge pas lors du déroulement (écran suivant/précédent). Cet attribut n'a cependant jamais fait partie du standard HTML et peut aussi être remplacé par une mention de feuilles de style, par exemple comme ceci:
<body style="background-image:url(arrplan.jpg); background-attachment:fixed;">

  

Autres possibilités

Pour l'utilisation de feuilles de style, il vous faut d'abord savoir comment  définir des formats CSS Ensuite, vous serez en mesure d'utiliser les propriétés CSS. En relation avec ce qui vient d'être décrit les données de feuilles de style suivantes sont aussi intéressantes:
 background-image
 background-repeat
 background-attachment
 background-position
 background
 color
 :link, :visited, :hover, :active, :focus (pseudo formats pour les liens)

Le repère<body> peut également contenir des  attributs universels .

 Définir les marges de page

Veuillez n'utiliser les commandes décrites ici qu'exceptionnellement et pour un motif fondé - elles ne sont expliquées que par souci d'être complet. Ces attributs n'ont jamais fait partie du standard HTML et sont un exemple typique du désordre qui s'ensuit quand les navigateurs n'en font qu'à leur tête. Utilisez dans tous les cas les feuilles de style CSS pour définir les marges de pages.

Exemple:

 Exemple d'affichage: aperçu

<html>
<head>
<title>Texte du titre</title>
</head>
<body marginwidth="50" marginheight="100" topmargin="100" leftmargin="50">
<h1>Texte</h1>
beaucoup de texte etc.
</body>
</html>

Explication:

La mention pour les marges de page se fait dans le repère d'ouverture <body> du fichier HTML. En ajoutant leftmargin= vous déterminez l'espace entre le bord gauche voire droit de la fenêtre et le contenu du fichier (leftmargin = marge gauche) pour l'Explorer Internet. Avec topmargin= vous déterminez l'espace entre le bord haut voire bas de la fenêtre et le contenu du fichier (topmargin = marge haute) pour l'Explorer Internet. Avec marginwidth= et marginheight= vous déterminez les espaces correspondants pour Netscape. Toutes les mentions se font en pixels. La marge gauche est toujours aussi grande que la marge droite de même que la marge haute et la marge basse.

Attention:

Les marges de pages définies ne sont valables que pour le fichier HTML dans lequel elles sont définies. Si vous établissez un document constitué de plusieurs fichiers HTML et que ce document doit présenter toujours les mêmes marges de page vous serez obligés de redéfinir les marges dans chaque fichier HTML. Vous pouvez naturellement également définir des marges de page différentes pour chaque fichier.

Marges de page avec feuilles de style CSS:

Voici encore un exemple comment vous pouvez obtenir les mêmes marges de page que dans l'exemple ci-dessus à l'aide de feuilles de style:

Exemple:

<head>
<style type="text/css">
body { margin-left:50px; margin-right:50px; margin-top:100px; margin-bottom:100px }
</style>
</head>

Explication:

Avec les mentions ci-dessus dans l'entête vous parvenez au même résultat qu'avec les commandes HTML décrites auparavant. Ces mentions sont des mentions de feuilles de style CSS qui sont interprétées par Netscape depuis la version 4.0 et par l' Explorer Internet MS depuis la version 3.0. De plus ces mentions font partie du standard du consortium W3.

Pour utiliser les feuilles de style CSS, il vous faut d'abord savoir comment  définir des formats CSS. Ensuite vous serez en mesure d'utiliser les propriétés CSS:
 margin-top
 margin-bottom
 margin-left
 margin-right
 margin

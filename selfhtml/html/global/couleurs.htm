 Couleur pour l'arrière plan

Vous pouvez préciser une couleur pour l'arrière plan de la fenêtre d'affichage. Tout le fichier HTML aura cette couleur comme arrière-plan.

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
       "http://www.w3.org/TR/html4/loose.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body bgcolor="#CCFFFF">
<h1>texte, liens, r�f�rences de graphiques etc...</h1>
</body>
</html>

Explication:

La mention de la couleur de fond se fait grâce au repère HTML d'ouverture <body> du fichier. Avec l'attribut bgcolor= vous déterminez la couleur pour l'arrière-plan de l'écran (bgcolor = background color = couleur d'arrière-plan). Tout le fichier HTML sera affiché sur la couleur d'arrière plan définie ici. Pour le choix des couleurs s'appliquent les règles pour  définir des couleurs en HTML.

Attention:

L'attribut bgcolor= est classé en cours d'abandon et doit être évité à l'avenir. Vous obtenez le même effet à l'aide de  feuilles de style CSS , par exemple ainsi:
<body style="background-color:#CCFFFF">

Si vous établissez un document comprenant plusieurs fichiers HTML qui doit être uniforme tant en couleur de fond qu'en couleur de texte il vous faut redéfinir la couleur d'arrière plan dans chaque fichier. Vous pouvez aussi, bien sûr, définir des couleurs différentes pour chaque fichier.

Si vous définissez une couleur d'arrière plan vous devriez aussi définir une couleur d'avant plan qui convienne pour le texte (suffisamment contrastée) (à ce sujet voir le paragraphe suivant).

  

 Couleurs pour le texte et les liens

Vous pouvez définir les couleurs pour tout le fichier:

  • pour le texte (valable pour tous les éléments - titres, texte, listes etc.-),
  • pour les  liens à des endroits qui n'ont pas encore été visités,
  • pour les liens à des endroits qui ont déjà été visités,
  • pour des liens pendant qu'ils sont cliqués.

Exemple:

 Exemple d'affichage: aperçu

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
       "http://www.w3.org/TR/html4/loose.dtd">
<html>
<head>
<title>Texte du titre</title>
</head>
<body bgcolor="#663333" text="#FFCC99" link="#FF9966" vlink="#FF9900" alink="#FFFFFF">
<h1>texte</h1>
<a href="http://fr.yahoo.com/">lien � Yahoo</a>
</body>
</html>

Explication:

Les mentions se font dans le repère d'ouverture du fichier HTML <body>. Pour le choix des couleurs s'appliquent les règles pour  définir les couleurs en HTML.

Avec text= vous définissez une couleur pour le texte.
Avec link= vous définissez une couleur pour les liens à des fichiers non encore visités (link = lien).
Avec vlink= vous définissez une couleur pour les liens à des fichiers déjà visités (vlink = visited link = lien visité).
Avec alink= vous définissez une couleur pour les liens que l'utilisateur est juste en train de cliquer (alink = activated link = lien activé).

Attention:

Tous ces attributs sont classés en cours d'abandon et doivent être évités à l'avenir. Vous obtenez le même effet à l'aide des  feuilles de style CSS , par exemple dans l'entête de fichier entre <head> et </head>avec le code source suivant:
<style type="text/css">
body { background-color:#663333; color:#FFCC99; }
a:link { color:#FF9966; }
a:visited { color:#FF9900; }
a:active { color:#FFFFFF; }
</style>

Les couleurs de premier plan définies pour le texte doivent contraster avec la couleur définie pour l'arrière plan. Si par exemple vous définissez une couleur d'arrière plan sombre vous devriez choisir des couleurs claires pour le texte (par exemple blanc jaune vert clair et bleu ciel).

  

Autres possibilités

Pour l'utilisation des feuilles de style CSS, il vous faut d'abord savoir comment  définir des formats CSS. Ensuite, vous serez en mesure d'utiliser les propriétés de feuilles de style. En relation avec ce qui vient d'être décrit les données de feuilles de style suivantes sont aussi intéressantes:
 background-color
 color
 Pseudo-formats pour les liens

Le repère <body> peut aussi contenir des  attributs universels.

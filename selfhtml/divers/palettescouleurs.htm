Types de palettes de couleurs et noms de couleur

À différents endroits, il vous est possible de spécifier des couleurs pour des pages Web:

  • En HTML par exemple lors de la définition de  couleurs pour tout le fichier pour l'arrière plan du document, le texte et les liens ou bien pour les  couleurs d'arrière-plan pour les tableaux,
  • Dans les CSS par exemple pour les propriétés  color et  background-color,
  • Pour les types de graphiques orientés sur les palettes comme le  format GIF.

Par les possibilités syntaxiques offertes en HTML et dans les CSS, vous pouvez codifier jusqu'à 16,7 millions de tonalités de couleurs différentes. Il existe cependant des palettes standard qui utilisent certaines tonalités bien précises parmi ces 16,7 millions de possibilités. Il n'est certes pas obligatoire de s'en tenir à ces palettes, cependant quand rien dans la mise en forme ne s'y oppose ces palettes donnent un certain ordre d'idées. En tout cas, ces palettes ont un avantage technique: la chance que les couleurs aient le même aspect sur différents types de matériels et sous différents systèmes d'exploitation, est plus grande que pour les couleurs avec des tonalités "quelconques".

La palette VGA (16 couleurs):

 Exemple d'affichage: aperçu (avec les noms de couleurs)

La palette VGA est le plus petit dénominateur commun de toutes les cartes graphiques et des écrans VGA compatibles. Pour les 16 couleurs de la palette VGA, les noms prédéfinis sont aussi permis en HTML et dans les CSS.

La palette Netscape (216 couleurs):

 Exemple d'affichage: aperçu

Cette palette de couleurs a été introduite à l'époque par Netscape. Tous les navigateurs Netscape veillent à ce que les couleurs de cette palette soient représentées correctement quelle que soit la plate forme.

Les noms de couleur Netscape (120 couleurs):

 Exemple d'affichage: aperçu

Ce n'est pas à vrai dire une palette au sens propre du terme, mais une collection de couleurs en vrac avec certaines tonalités de couleurs. La collection a été introduite en son temps par Netscape pour proposer en plus de la palette de 216 couleurs qui suit un schéma algorithmique, encore quelques tons prédéfinis pour les tonalités intermédiaires et les couleurs pastel. Ces noms de couleurs sont interprétés par la majorité des navigateurs modernes pour les mentions de couleurs en HTML ou dans les CSS.

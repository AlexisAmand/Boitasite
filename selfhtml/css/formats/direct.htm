 Définitions de format avec l'attribut style

Vous pouvez formater des repères HTML distincts dans le corps visible du document HTML à l'aide de propriétés de feuilles de style CSS . Les mentions de formatage s'appliquent alors pour le passage de validité du repère HTML concerné, et conformément aux règles de l'héritage naturel de format également pour les éléments enfants de l'élément concerné. Cette possibilité est indiquée, soit si vous renoncez autrement aux CSS et n'en avez besoin que pour certaines exceptions, ou bien si vous utilisez des  formats globaux, mais désirez mettre en forme certains éléments autrement qu'avec les définitions de format qui y sont énoncées.

Exemple:

 Exemple d'affichage: aperçu

<html>
<head>
<title>Titre du fichier</title>
<style type="text/css">
<!--
body { background-color:#000000; color:#E0E0E0 }
-->
</style>
</head>
<body>
<h1>La page avec l'&eacute;l&eacute;ment particulier</h1>
<p>Ici il existe bien des d&eacute;finitions de formats globaux,
mais l'&eacute;l&eacute;ment qui suit doit �tre quelque chose de particulier:</p>

<p style="background-color:#808040; color:#D8FD02;
font-family:'Century Schoolbook',serif; font-size:24pt; letter-spacing:3px;
padding:40px; border:double #D8FD02 4px;"
title="Citation de Francis Picabia">

Explication:

Vous pouvez définir des formats pour un élément HTML, en notant dans le repère d'ouverture HTML l'attribut style= suivi de la définition de format désirée. Dans la définition de format les  propriétés CSS sont permises. Ici s'applique la même syntaxe que dans les parenthèses accolades pour les formats globaux. Chaque propriété comprend un nom par exemple color (couleur de police) et une valeur par exemple #D8FD02, séparés par deux points. Fermez chaque définition de format par un point virgule. Il n'y a que pour la dernière définition de format que le point virgule peut être omis.

Dans l'exemple ci-dessus des mentions de format sont utilisées sur un simple élément p. Il reçoit à l'aide des propriétés CSS pour la couleur d'arrière-plan (background-color), la couleur de police (color), le genre de police (font-family), la taille de police (font-size), l'espacement des caractères (letter-spacing), l'espace intérieur séparant du bord (padding) et la bordure (border) une apparence qui ne fait plus du tout penser à un "paragraphe".

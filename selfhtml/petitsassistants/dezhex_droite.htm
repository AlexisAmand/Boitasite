Traducteur décimal/hexadécimal

Entrez dans le champ de saisie soit une valeur hexadécimale que vous voulez traduire en valeur décimale ou une valeur décimale que vous voulez traduire en valeur hexadécimale. Choisissez ensuite le sens de calcul. Le résultat est alors affiché immédiatement dans le champ de saisie. Avec le bouton "effacer l'entrée" vous remettez le formulaire dans son état initial.

Les chiffres hexadécimaux sont, outre les chiffres de 0 à 9, également les lettres:
A/a (=10),
B/b (=11),
C/c (=12),
D/d (=13),
E/e (=14),
F/f (=15).

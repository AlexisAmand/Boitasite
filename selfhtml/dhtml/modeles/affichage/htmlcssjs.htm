HTML est le langage pour structurer des projets hypertexte. une bénédiction pour toutes les pages Web. Sans doute le format de fichier le plus populaire de l'histoire de l'informatique. CSS est le langage pour le formatage de pages Web. s'enclenche sans heurt dans HTML et obtient que les structures deviennent couleurs et formes. JavaScript est le langage pour les pages Web dynamiques.s'enclenche sans heurt en HTML et aussi toujours plus dans les CSS. ce qui est d'ailleurs le plus difficile à comprendre. HTML CSS JavaScript

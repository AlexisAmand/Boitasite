Boutons (surfaces pour déclencher)

Vous pouvez utiliser ces graphiques sur vos pages d'accueil. Avec la plupart des navigateurs Web, il vous suffit de cliquer avec la touche droite de la souris sur le graphique souhaité pour le sauvegarder dans un répertoire de votre choix.

Chaque boputon est représenté trois fois l'un sous l'autre pour démontrer par exemple quel effet il a dans une barre de navigation. Tous les boutons sont vides et ont la pleine intensité de couleurs de 16,7 millions de couleurs afin que vous puissiez y porter les inscriptions de votre choix. Pour ce faire, créez autant de copies du graphique de bouton vierge que vous en avez besoin. Portez ensuite dans un programme graphique les inscriptions sur les boutons du même moule.

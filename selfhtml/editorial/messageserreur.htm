L'erreur est humaine

SELFHTML comprend plus de 1500 pages de livres. Pour de telles quantités de données, il est tout à fait normal que de nombreuses petites erreurs, parfois cachées, se trouvent dans le texte ou dans les exemples. La rédaction de SELFHTML s'est efforcée d'en éliminer un maximum. Mais une absence absolue d'erreurs pour un projet de cette importance dépasse malheureusement le cadre des possibilités humaines.

Pourtant, il est possible de corriger les erreurs. Afin que toutes les corrections aient une forme homogène qui facilite leur exploitation, les annonces d'erreur dans SELFHTML seront traitées grâce au formulaire de cette page. Les erreurs signalées sont rassemblées pour être traitées dans un  Errata SELFHTML (Le lien mène à l'offre en ligne de SELFHTML actuel).

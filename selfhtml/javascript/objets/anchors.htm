anchors: généralités sur l'utilisation

Avec l'objet anchors, qui se trouve sous l'objet  document dans la hiérarchie des objets JavaScript, vous avez accès à des ancres de liens définies dans un fichier HTML. Une ancre en HTML, c'est par exemple:
<a name="top">Ici commence la page</a>.
En JavaScript 1.0 il est possible finalement de lire le nombre d'ancres dans un fichier. En JavaScript 1.2 pourtant, d'autres propriétés ont été implémentées.

Les façons suivantes d'adresser une certaine ancre avec JavaScript existent:

Schéma 1 / exemple 1:

document.anchors[#].propri�t�

document.anchors[0].name

Schéma 2 / exemple 2:

document.anchors["nomde_lancre"].propri�t�

document.anchors["haut"].text

Explication:

Vous pouvez adresser une ancre de lien de deux façons:

  • avec un numéro d'index (comme dans le Schéma 1 / exemple 1)
    Lors de l'utilisation de numéros index, mentionnez document.anchors suivi entre crochets, du numéro d'ordre dans le fichier de l'ancre à laquelle vous pensez. Ne perdez pas de vue que le comptage commence à 0, c'est à dire que vous adressez la première ancre avec anchors[0], la deuxième ancre avec anchors[1] etc... Pour le comptage, prévaut l'ordre dans lequel sont notées les ancres dans le fichier.
  • avec le nom de l'ancre comme nom d'index (comme dans le Schéma 2 / exemple 2)
    Ici, vous notez comme pour l'adressage avec un numéro d'index des crochets derrière document.anchors. Dans ces crochets, notez entre guillemets le nom que vous avez mentionné pour la définition de l'ancre dans le repère d'ouverture <a>-Tag avec l'attribut name=.

Attention:

Dans l'Explorer Internet MS il n'est pas possible d'accéder à une ancre de lien avec le schéma 2 . Vous pouvez toutefois accéder à toutes les ancres et à leurs propriétés par l'objet  all.

Explication:

Avec document.anchors[0].name vous pouvez rechercher le nom d'une ancre de lien dans un fichier HTML. L'exemple contient d'abord un texte HTML et un lien. Lorsqu'on le clique, le nom du lien est sorti avec  alert().

Attention:

Actuellement, il n'est possible qu'avec Netscape 6.1 de modifier le nom d'une ancre.

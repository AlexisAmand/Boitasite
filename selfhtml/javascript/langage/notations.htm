 Caractères de commande pour les chaînes de caractères

Pour les variables caractères, il est possible d'insérer des caractères de commande dans la valeur de la variable.

Exemple:

<script type="text/javascript">
<!--
  var Variable1 = "Ici il y a un\npassage � la ligne";
  var Variable2 = "Ici il y a un\fretour du chariot";
  var Variable3 = "Ici il y a un\bretour en arri�re";
  var Variable4 = "Ici il y a un\rpassage � la ligne extra-DOS";
  var Variable5 = "Ici il y a une\ttabulation";
  var Variable6 = "Ici il y a un\"guillemet";
// -->
</script>

Explication:

Les caractères de commande de cette nature sont introduits par le signe \ Derrière suit une lettre qui marque le caractère de commande.

La chaîne de caractères \n est appropriée par exemple dans des messages  alert pour insérer dans le texte sorti un passage à la ligne. .
La chaîne de caractères \t est appropriée par exemple dans des messages alert(...) pour une information sous forme de tableau.
La chaîne de caractères \r peut être noté en plus du caractère de commande \n si celle-ci \n ne fonctionne pas seule.
La chaîne de caractères \" doit être notée si vous voulez noter des guillemets dans une chaîne de caractères. (masquer le signe qui englobe la chaîne de caractères).

  

 Notation de valeurs numériques

Vous pouvez noter des nombres tout à fait normalement. Veillez simplement à ce que pour les chiffres décimaux, un point soit noté à la place de la virgule. Ainsi le nombre PI par exemple s'écrira 3.1415 Pour des chiffres très élevés ou très bas et pour des nombres décimaux complexes il existe aussi d'autres possibilités de notation.

Exemple:

<script language="JavaScript">
<!--
  var a = 1E1;
  var b = 1.2345E4;
  var c = 2e-3;
// -->
</script>

Explication:

Avec e ou bien E vous spécifiez le nombre de zéros qui suivent le chiffre précédant le e ou le E.

Le premier nombre de l'exemple, 1E1, est un 1 suivi d'un 0 donc 10.
Le deuxième nombre de l'exemple, 1.2345E4, est une autre façon d'écrire 12345. Le point décimal est simplement repoussé vers la droite d'autant de positions qu'il est mentionné derrière le caractère E.
Le troisième nombre dans l'exemple, 2e-3, est une autre façon d'écrire 0.002. Le point décimal est simplement repoussé vers la gauche d'autant de positions qu'il est mentionné derrière le caractère E. La direction contraire est provoquée par le signe moins derrière le e.

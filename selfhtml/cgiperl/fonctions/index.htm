Fonctions Perl

Connaissances préalables conseillées pour la compréhension:  Introduction en CGI et Perl et  Éléments de langage Perl.
Cette partie comprend une référence orientée par thèmes de toutes les fonctions importantes dont dispose Perl. Les différentes fonctions sont chacune présentées avec un exemple.
La partie comprend les pages suivantes:

 Fonctions pour les chaînes de caractères
 Fonctions pour les listes et les hashes
 Fonctions pour les calculs
 Fonctions pour le contrôle du script en cours
 Fonctions pour les date et heure
 Fonctions pour l'entrée et la sortie et la lecture et l'écriture de données
 Fonctions pour la gestion de fichiers et de répertoires
 Fonctions pour les appels du système d'exploitation
 Fonctions pour les informations tirées des fichiers de configuration
 Fonctions pour les modules et progiciels (packages)

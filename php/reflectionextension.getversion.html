ReflectionExtension::getVersion

(PHP 5, PHP 7)

ReflectionExtension::getVersion — Récupère la version de l'extension

Description

public ReflectionExtension::getVersion ( void ) : string

Récupère la version de l'extension.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

La version de l'extension.

Exemples

Exemple #1 Exemple avec ReflectionExtension::getVersion()

<?php
$ext = new ReflectionExtension('mysqli');
var_dump($ext->getVersion());
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

string(3) "0.1"

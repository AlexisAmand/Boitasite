EvLoop::idle

(PECL ev >= 0.2.0)

EvLoop::idle — Crée un objet EvIdle watcher associé avec l'instance de la boucle de l'événement courant

Description

final public EvLoop::idle ( callable $callback [, mixed $data = NULL [, int $priority = 0 ]] ) : EvIdle

Crée un objet EvIdle watcher associé avec l'instance de la boucle de l'événement courant

Liste de paramètres

Tous les paramètres ont la même signification que ceux de la méthode EvIdle::__construct()

Valeurs de retour

Retourne un objet EvIdle en cas de succès.

xhprof_sample_enable

(PECL xhprof >= 0.9.0)

xhprof_sample_enable — Démarre le profilage en mode échantillonnage

Description

xhprof_sample_enable ( void ) : void

Démarre le profilage en mode échantillonnage, ce qui correspond à une version plus légère de la fonction xhprof_enable(). L'intervalle d'échantillonnage est de 0.1 seconde, et les échantillons enregistrent la pile complète des fonctions appelées. La principale utilité de cette fonction est lorsque l'on souhaite diagnostiquer et surveiller les performances sans pour autant consommer trop de ressources.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

NULL

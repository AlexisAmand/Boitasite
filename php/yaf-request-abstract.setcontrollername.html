Yaf_Request_Abstract::setControllerName

(Yaf >=1.0.0)

Yaf_Request_Abstract::setControllerName — Définie le nom du contrôleur

Description

public Yaf_Request_Abstract::setControllerName ( string $controller [, bool $format_name = true ] ) : void

Définie le nom du contrôleur dans la requête ; habituellement utilisé par les routeurs personnalisées pour définir le nom du contrôleur de la route résultante.

Liste de paramètres

controller

chaîne de caractères ; nom du contrôleur, en sryle camel, comme "Index" ou "Foo_Bar"

format_name

Introduit en Yaf 3.2.0 ; par défaut, Yaf formattera le nom en mode camel ; si définit à FALSE, Yaf définira le nom original dans la requête.

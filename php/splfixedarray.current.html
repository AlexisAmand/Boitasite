SplFixedArray::current

(PHP 5 >= 5.3.0, PHP 7)

SplFixedArray::current — Lit l'élément courant du tableau à taille fixe

Description

public SplFixedArray::current ( void ) : mixed

Lit l'élément courant du tableau à taille fixe.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

La valeur de l'élément courant.

Erreurs / Exceptions

Lance une exception RuntimeException lorsque le pointeur interne du tableau pointe vers un index invalide ou s'il est supérieur à la taille de ce dernier.

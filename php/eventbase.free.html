EventBase::free

(PECL event >= 1.10.0)

EventBase::free — Libère les ressources allouées pour l'événement de base

Description

public EventBase::free ( void ) : void

Désalloue les ressources allouées par libevent pour l'objet EventBase.

Avertissement

La méthode EventBase::free() ne détruit pas l'objet lui-même. Pour détruire l'objet complètement, appelez la fonction unset() ou assignez-lui la valeur NULL.

Cette méthode ne désalloue, ni ne détache aucun événement qui serait actuellement associé avec l'objet EventBase, ni ne ferme aucun de leur socket ; gardez cela en tête.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

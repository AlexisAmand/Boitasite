UI\run

(UI 2.0.0)

UI\run — Entre dans la boucle UI

Description

UI\run ([ int $flags ] ) : void

Doit pousser PHP à entrer dans la boucle principale. Par défaut, le contrôle ne sera pas retourné à la fonction appelante.

Liste de paramètres

flags

Mettre UI\Loop pour retourner le contrôle et UI\Wait pour retourner le contrôle après l'attente.

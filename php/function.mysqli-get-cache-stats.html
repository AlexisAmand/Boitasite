mysqli_get_cache_stats

(PHP 5 >= 5.3.0 and < 5.4.0)

mysqli_get_cache_stats — Retourne les statistiques sur le cache du client Zval

Avertissement

Cette fonction a été SUPPRIMÉE à partir de PHP 5.4.0.

Description

mysqli_get_cache_stats ( void ) : array
Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Retourne tableau vide. Disponible uniquement avec mysqlnd.

Liste de paramètres

Valeurs de retour

Retourne un tableau vide en cas de succès, FALSE sinon.

Throwable::__toString

(PHP 7)

Throwable::__toString — Récupère la représentation sous forme de chaîne de charactère de l'objet jeté

Description

abstract public Throwable::__toString ( void ) : string

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la représentation sous forme de chaîne de caractères de l'objet jeté.

Yaf_Route_Regex::route

(Yaf >=1.0.0)

Yaf_Route_Regex::route — Le but de route

Description

public Yaf_Route_Regex::route ( Yaf_Request_Abstract $request ) : bool

Route une requête entrante.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

request

Valeurs de retour

Si le masque fourni par le troisième paramètre de la méthode Yaf_Route_Regex::_construct() correspond avec la requête URI, TRUE sera retourné, sinon FALSE.

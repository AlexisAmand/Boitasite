ArrayIterator::offsetGet

(PHP 5, PHP 7)

ArrayIterator::offsetGet — Récupère la valeur pour une position

Description

public ArrayIterator::offsetGet ( mixed $index ) : mixed

Récupère la valeur pour une position donnée.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

index

La position dont on doit récupérer la valeur.

Valeurs de retour

La valeur à la position index.

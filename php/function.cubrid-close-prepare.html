cubrid_close_prepare

(PECL CUBRID >= 8.3.0)

cubrid_close_prepare — Ferme le gestionnaire de requête

Description

cubrid_close_prepare ( resource $req_identifier ) : bool

La fonction cubrid_close_prepare() ferme le gestionnaire de requête fourni par l'argument req_identifier et libère la région mémoire associée. C'est un alias de la fonction cubrid_close_request().

Liste de paramètres

req_identifier

Identifiant de requête.

Valeurs de retour

TRUE en cas de succès.

Exemples

Exemple #1 Exemple avec cubrid_close_prepare()

<?php
$con = cubrid_connect ("localhost", 33000, "demodb", "dba", "");
if ($con) {
   echo "Connexion réalisée avec succès";
   $req = cubrid_execute ( $con, "select * from members", 
                           CUBRID_INCLUDE_OID | CUBRID_ASYNC);
   if ($req) {
      while ( list ($id, $name) = cubrid_fetch ($req) ){
         echo $id;
         echo $name;
      } 
      cubrid_close_prepare($req); // vous pouvez aussi utiliser cubrid_close_request($req)
   }
   cubrid_disconnect($con);
}
?>

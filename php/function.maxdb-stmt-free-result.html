maxdb_stmt_free_result

maxdb_stmt::free_result

(PECL maxdb >= 1.0)

maxdb_stmt_free_result -- maxdb_stmt::free_result — Libère la mémoire des résultats stockés pour un gestionnaire de requête donné

Description

Style procédural

maxdb_stmt_free_result ( resource $stmt ) : void

Style orienté objet

maxdb_stmt::free_result ( void ) : void

maxdb_stmt_free_result() libère la mémoire des résultats associés avec la requête représentée par le paramètre stmt, qui a été alloué avec la fonction maxdb_stmt_store_result().

Valeurs de retour

Cette fonction ne retourne aucune valeur.

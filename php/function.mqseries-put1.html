mqseries_put1

(PECL mqseries >= 0.10.0)

mqseries_put1 — MQSeries MQPUT1

Description

mqseries_put1 ( resource $hconn , resource &$objDesc , resource &$msgDesc , resource &$pmo , string $buffer , resource &$compCode , resource &$reason ) : void

L'appel de mqseries_put1() (MQPUT1) ajoute un message à une file d'attente. La file d'attente n'a pas besoin d'être ouverte.

Vous pouvez utiliser à la fois des appels à mqseries_put() ainsi qu'à mqseries_put1() pour ajouter des messages à une file d'attente ; quel appel à utiliser dépend des circonstances. calls to put messages on a queue; which call to use depends on the circumstances. Utiliser l'appel à mqseries_put() (MQPUT) pour ajouter plusieurs messages à la même file d'attente. Utiliser l'appel à mqseries_put1() (MQPUT1) pour ajouter un seul message à une file d'attente. Cet appel englobe les appels à MQOPEN, MQPUT, et MQCLOSE dans un seul appel, minimisant le nombre d'appels à émettre.

Liste de paramètres

hConn

Gestionnaire de connexion.

Ce gestionnaire représente la connexion au gestionaire de files d'attente.

objDesc

Descripteur de l'objet. (MQOD)

Cette structure identifie la file d'attente dans laquelle le message sera ajouté.

msgDesc

Descripteur du message (MQMD).

pmo

Options d'ajout du message (MQPMO).

compCode

Code de complétion.

reason

La raison qui qualifie le compCode.

Valeurs de retour

Aucune valeur n'est retournée.

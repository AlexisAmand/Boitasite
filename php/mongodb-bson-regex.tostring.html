MongoDB\BSON\Regex::__toString

(mongodb >=1.0.0)

MongoDB\BSON\Regex::__toString — Retourne la chaîne représentant cette REGEX

Description

final public MongoDB\BSON\Regex::__toString ( void ) : string

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la chaîne représentant cette REGEX.

Exemples

Exemple #1 Exemple avec BSON\Regex::__toString()

<?php

$regex = new MongoDB\BSON\Regex('regex', 'i');
var_dump((string) $regex);

?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

string(8) "/regex/i"

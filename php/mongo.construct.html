Mongo::__construct

(PECL mongo >=0.9.0)

Mongo::__construct — Le but de __construct

Description

public Mongo::__construct ([ string $server [, array $options ]] )

Cette méthode écrase le constructeur MongoClient et désactive les écritures reconnues.

Reportez-vous à la documentation de MongoClient::__construct() pour une description des paramètres.

Valeurs de retour

Avertissement

L'instanciation de cette classe émet une alerte de niveau E_DEPRECATED et désactive les écritures reconnues.

Veuillez utiliser la classe MongoClient à la place.

Gmagick::nextimage

(PECL gmagick >= Unknown)

Gmagick::nextimage — Se déplace sur l'image suivante

Description

public Gmagick::nextimage ( void ) : bool

Associe l'image suivante dans la liste des images avec un objet Gmagick.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Erreurs / Exceptions

Émet une exception GmagickException en cas d'erreur.

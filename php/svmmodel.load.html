SVMModel::load

(PECL svm >= 0.1.00.1.0)

SVMModel::load — Charge un modèle SVM sauvegardé

Description

public SVMModel::load ( string $filename ) : bool

Charge un fichier de modèle prêt pour une classification ou une régression.

Liste de paramètres

filename

Le nom de fichier du modèle.

Valeurs de retour

Lance une exception SVMException en cas d'erreur. Retourne TRUE en cas de succès.

fann_set_activation_steepness_hidden

(PECL fann >= 1.0.0)

fann_set_activation_steepness_hidden — Défini la raideur de la pente d'activation pour tous les neurones des couches cachées

Description

fann_set_activation_steepness_hidden ( resource $ann , float $activation_steepness ) : bool

Défini la raideur de la pente d'activation pour tous les neurones des couches cachées.

Liste de paramètres

ann

Ressource de réseau de neurones.

activation_steepness

La pente d'activation.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

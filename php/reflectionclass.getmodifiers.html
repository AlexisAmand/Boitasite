ReflectionClass::getModifiers

(PHP 5, PHP 7)

ReflectionClass::getModifiers — Récupère les modificateurs de classe

Description

public ReflectionClass::getModifiers ( void ) : int

Retourne un champ de bit des modificateurs d'accès pour cette classe.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un masque de bits des constantes de modificateurs.

XMLDiff\Memory::diff

(PECL xmldiff >= 0.8.0)

XMLDiff\Memory::diff — Différence entre deux documents XML

Description

public XMLDiff\Memory::diff ( string $from , string $to ) : string

Effectue la différence de deux chaînes contenant des documents XML et produit les informations de différence.

Liste de paramètres

from

Document XML source.

to

Document XML cible.

Valeurs de retour

Une chaîne contenant la différence entre les deux documents XML ou NULL.

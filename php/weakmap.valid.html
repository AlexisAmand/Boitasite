WeakMap::valid

(PECL weakref >= 0.2.0)

WeakMap::valid — Vérifie si l'itérateur est toujours sur un élément map valide

Description

public WeakMap::valid ( void ) : bool

Vérifie si l'itérateur est toujours sur un élément map valide.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE si l'itérateur est sur un élément valide qui peut être accédé, FALSE sinon.

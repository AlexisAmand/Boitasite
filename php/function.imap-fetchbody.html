imap_fetchbody

(PHP 4, PHP 5, PHP 7)

imap_fetchbody — Retourne une section extraite du corps d'un message

Description

imap_fetchbody ( resource $imap_stream , int $msg_number , string $section [, int $options = 0 ] ) : string

Récupère une section particulière du corps des messages spécifiés. Les parties du corps ne sont pas décodées par cette fonction.

Liste de paramètres

imap_stream

Un flux IMAP retourné par la fonction imap_open().

msg_number

Le numéro du message

section

Le numéro de la section. C'est une chaîne d'entiers, délimités par une virgule qui correspondent aux index dans la liste des sections du message, tel que prévu par la spécification IMAP4.

options

L'option imap_fetchbody() est un masque qui peut contenir les valeurs suivantes :

  • FT_UID - msg_number est un UID
  • FT_PEEK - Ne pas lever le drapeau \Seen (Message lu) s'il n'est pas déjà levé.
  • FT_INTERNAL - La chaîne renvoyée est au format interne, et ne va pas canoniser les CRLF.

Valeurs de retour

Retourne une section particulière du corps des messages spécifiés, sous la forme d'une chaîne de caractères.

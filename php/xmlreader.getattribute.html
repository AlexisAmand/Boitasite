XMLReader::getAttribute

(PHP 5 >= 5.1.0, PHP 7)

XMLReader::getAttribute — Récupère la valeur d'un attribut par nom

Description

public XMLReader::getAttribute ( string $name ) : string

Retourne la valeur d'un attribut par nom ou NULL si l'attribut n'existe pas ou n'est pas positionné sur le noeud.

Liste de paramètres

name

Le nom de l'attribut.

Valeurs de retour

La valeur de l'attribut ou NULL si l'attribut avec le name donné n'existe pas ou n'est pas positionné sur le nœud d'un élément.

Collectable::isGarbage

(PECL pthreads >= 2.0.8)

Collectable::isGarbage — Détermine si un objet a été marqué comme obsolète

Description

public Collectable::isGarbage ( void ) : bool

Peut être appelé dans Pool::collect() pour déterminir si cet objet est obsolète.

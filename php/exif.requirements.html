Pré-requis

Votre PHP doit être compilé avec l'option --enable-exif. Pour activer la prise en charge des tags EXIF multibyte l'extension mbstring doit être activée lors de la compilation avec --enable-mbstring. PHP n'a besoin d'aucune bibliothèque additionnelle pour faire fonctionner le module exif.

Windows seulement : L'extension mbstring est toujour activée. Notez que l'extension mbstring doit être chargée avant EXIF dans php.ini.

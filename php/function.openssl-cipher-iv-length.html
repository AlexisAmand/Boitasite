openssl_cipher_iv_length

(PHP 5 >= 5.3.3, PHP 7)

openssl_cipher_iv_length — Récupère la longueur cipher iv

Description

openssl_cipher_iv_length ( string $method ) : int

Récupère la longueur cipher du vecteur d'initialisation.

Liste de paramètres

method

La méthode cipher, voir la fonction openssl_get_cipher_methods() pour une liste de valeurs potentielles.

Valeurs de retour

Retourne la longueur cipher en cas de succès, ou FALSE si une erreur survient.

Erreurs / Exceptions

Émet une erreur de niveau E_WARNING lorsque l'algorithme cipher est inconnu.

Exemples

Exemple #1 Exemple avec openssl_cipher_iv_length()

<?php
$method = 'AES-128-CBC';
$ivlen = openssl_cipher_iv_length($method);

echo $ivlen;
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

16

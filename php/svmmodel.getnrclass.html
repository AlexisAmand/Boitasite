SVMModel::getNrClass

(PECL svm >= 0.1.5)

SVMModel::getNrClass — Retourne le nombre de classes utilisées pour entraîner le modèle

Description

public SVMModel::getNrClass ( void ) : int

Retourne le nombre de classes utilisées pour entraîner le modèle ; retournera 2 pour une seule classe et pour les modèles de régression.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne le nombre de classes, sous la forme d'un entier.

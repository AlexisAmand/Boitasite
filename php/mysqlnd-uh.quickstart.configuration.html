Configuration

Le plugin est implémenté comme une extension PHP. Voir aussi les instructions d'installation pour installer l'extension » PECL/mysqlnd_uh. Puis, chargez l'extension dans PHP et activez le plugin dans le fichier de configuration de PHP en utilisant la directive de configuration PHP mysqlnd_uh.enable. L'exemple ci-dessous montre les configurations par défaut de l'extension.

Exemple #1 Activation du plugin (php.ini)

mysqlnd_uh.enable=1
mysqlnd_uh.report_wrong_types=1

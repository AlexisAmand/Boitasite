socket_clear_error

(PHP 4 >= 4.2.0, PHP 5, PHP 7)

socket_clear_error — Efface toutes les erreurs précédemment générées par un socket

Description

socket_clear_error ([ resource $socket ] ) : void

Efface tous les codes d'erreur qui ont été enregistrés pour le socket socket, ou bien pour le socket général.

socket_clear_error() permet de remettre à zéro les codes d'erreur d'un socket ou du socket global. Cela peut être utile pour détecter l'avènement d'une erreur durant une partie de l'application.

Liste de paramètres

socket

Une ressource de socket valide, créée par la fonction socket_create().

Valeurs de retour

Aucune valeur n'est retournée.

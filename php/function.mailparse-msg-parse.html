mailparse_msg_parse

(PECL mailparse >= 0.9.0)

mailparse_msg_parse — Analyse incrémentalement des données dans un buffer

Description

mailparse_msg_parse ( resource $mimemail , string $data ) : bool

Analyse incrémentalement des données dans une ressource MIME fournie.

Cette fonction vous permet d'envoyer sous forme de flux des portions d'un fichier à la fois, plutôt que de tout lire et de tout analyser en une seule fois.

Liste de paramètres

mimemail

Une ressource MIME valide.

data

Note:

Le bloc final d'un data est supposé être un caractère de nouvelle ligne (CRLF) ; sinon, la dernière ligne du message se sera pas analysée.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

ReflectionGenerator::getFunction

(PHP 7)

ReflectionGenerator::getFunction — Récupère le nom de fonction du générateur

Description

public ReflectionGenerator::getFunction ( void ) : ReflectionFunctionAbstract

Permet l'obtention du nom de fonction du générateur en retournant une classe dérivé depuis ReflectionFunctionAbstract.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne une classe ReflectionFunctionAbstract. Ceci sera ReflectionFunction pour les fonctions, ou ReflectionMethod pour les méthodes.

Exemples

Exemple #1 Exemple avec ReflectionGenerator::getFunction()

<?php

function gen()
{
    yield 1;
}

$gen = gen();

$reflectionGen = new ReflectionGenerator($gen);

var_dump($reflectionGen->getFunction());

L'exemple ci-dessus va afficher quelque chose de similaire à :

object(ReflectionFunction)#3 (1) {
  ["name"]=>
  string(3) "gen"
}

Error::__clone — Cloner l'erreur

Description

final private Error::__clone ( void ) : void

Une erreur ne peut pas être cloner, cette méthode produit donc une erreur fatale.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

Erreurs / Exceptions

Les erreurs ne sont pas clonable.

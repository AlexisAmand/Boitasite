La classe MongoDB\BSON\MinKey

(mongodb >=1.0.0)

Introduction

Type BSON spécial qui compare la plus petite valeur possible de toutes les autres valeurs d'éléments BSON possibles.

Note: Il s'agit d'un type interne de MongoDB utilisé pour l'indexation et la défragmentation.

La classe MongoDB\Driver\WriteConcernError

(mongodb >=1.0.0)

Introduction

La classe MongoDB\Driver\WriteConcernError contient des informations relative à une erreur concernant l'écriture et peut être retournée par MongoDB\Driver\WriteResult::getWriteConcernError().

/* Méthodes */
final public getCode ( void ) : int
final public getInfo ( void ) : object|null
final public getMessage ( void ) : string

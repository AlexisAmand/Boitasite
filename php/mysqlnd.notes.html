Notes

Cette section fournit une compilation de notes diverses sur l'utilisation du pilote natif MySQL.

  • Utiliser mysqlnd signifie utiliser les flux PHP pour la connexion sous-jascente. Avec mysqlnd, la documentation des flux PHP (Flux) devrait être consultée en ce qui concerne les notions comme les timeouts et non pas la documentation de la librairie cliente MySQL.

SplFileInfo::isWritable

(PHP 5 >= 5.1.2, PHP 7)

SplFileInfo::isWritable — Indique si on peut écrire dans un fichier

Description

public SplFileInfo::isWritable ( void ) : bool

Vérifie si le fichier a les autorisations en écriture.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE si le fichier peut être écrit, et FALSE sinon;

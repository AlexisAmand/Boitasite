SNMP::close

(PHP 5 >= 5.4.0, PHP 7)

SNMP::close — Ferme une session SNMP

Description

public SNMP::close ( void ) : bool

Libère les objets de la session SNMP précédemment alloués.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Exemples

Exemple #1 Exemple avec SNMP::close()

<?php
  $session = new SNMP(SNMP_VERSION_1, "127.0.0.1", "public");
  # ...
  # get, walk, etc vont ici !
  # ...
  $session->close();
?>

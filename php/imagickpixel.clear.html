ImagickPixel::clear

(PECL imagick 2.0.0)

ImagickPixel::clear — Supprime toutes les ressources associées avec l'objet

Description

ImagickPixel::clear ( void ) : bool
Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Supprime l'objet ImagickPixel, le laissant dans un statut initial. Supprime également toutes les couleurs associées avec l'objet.

Valeurs de retour

Retourne TRUE en cas de succès.

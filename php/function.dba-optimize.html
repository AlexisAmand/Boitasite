dba_optimize

(PHP 4, PHP 5, PHP 7)

dba_optimize — Optimise une base DBA

Description

dba_optimize ( resource $handle ) : bool

dba_optimize() optimise la base de données identifiée par handle.

Liste de paramètres

handle

Le gestionnaire de base de données, retourné par la fonction dba_open() ou la fonction dba_popen().

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

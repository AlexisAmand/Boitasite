zip_entry_read

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PECL zip >= 1.0.0)

zip_entry_read — Lit le contenu d'un fichier dans un dossier

Description

zip_entry_read ( resource $zip_entry [, int $length = 1024 ] ) : string

zip_entry_read() lit dans un dossier d'archives ouvert.

Liste de paramètres

zip_entry

Un dossier d'archives retourné par la fonction zip_read().

length

Le nombre d'octets à retourner.

Note:

Ceci doit être la taille non-compressée que vous voulez lire.

Valeurs de retour

Retourne les données lues, une chaîne vide si l'on est à la fin du fichier, ou FALSE si une erreur survient.

Historique

Version Description
8.0.0 Cette fonction est obsolète en faveur de l'API orientée objet, voir ZipArchive::getFromIndex().

MongoLog::getCallback

(PECL mongo >= 1.3.0)

MongoLog::getCallback — Récupère la précédente fonction de rappel définie

Description

public static MongoLog::getCallback ( void ) : callable

Récupère la fonction de rappel.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la fonction de rappel, ou FALSE si aucune fonction de rappel n'a été définie.

Notes

Attention

Cette fonction n'est disponible qu'avec PHP 5.3.0 et suivant.

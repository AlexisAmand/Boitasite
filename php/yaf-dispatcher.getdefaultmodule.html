Yaf_Dispatcher::getDefaultModule

(Yaf >=3.2.0)

Yaf_Dispatcher::getDefaultModule — Récupère le nom du module par défaut

Description

public Yaf_Dispatcher::getDefaultModule ( void ) : string

Récupère le nom du module par défaut.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Le nom du module par défaut, sous la forme d'une chaîne de caractères ; par défaut, vaut "Index"

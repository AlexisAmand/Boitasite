ReflectionClass::getTraitNames

(PHP 5 >= 5.4.0, PHP 7)

ReflectionClass::getTraitNames — Retourne un tableau de noms des traits utilisés par cette classe

Description

public ReflectionClass::getTraitNames ( void ) : array

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un tableau de noms des traits en valeurs. Retourne NULL en cas d'erreur.

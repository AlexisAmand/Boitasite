eio_grp_cancel

(PECL eio >= 0.0.1dev)

eio_grp_cancel — Annule un groupe de requêtes

Description

eio_grp_cancel ( resource $grp ) : void

eio_grp_cancel() annule un groupe de requêtes spécifié par la ressource de groupe grp.

Liste de paramètres

grp

La ressource du groupe de requêtes retournée par la fonction eio_grp().

Valeurs de retour

Aucune valeur n'est retournée.

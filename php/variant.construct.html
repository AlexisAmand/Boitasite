variant::__construct

(PHP 4 >= 4.1.0, PHP 5, PHP 7)

variant::__construct — Constructeur de la classe variant

Description

variant::__construct ([ mixed $value = NULL [, int $type = VT_EMPTY [, int $codepage = CP_ACP ]]] )

Construit un nouvel objet variant.

Liste de paramètres

value
Valeur initiale. Si omit, ou défini à NULL un objet VT_EMPTY vide est crée.
type
Spécifie le type de contenue de l'objet variant. Les valeurs possibles sont l'une des constantes VT_XXX Constantes pré-définies. PHP peut détecter les paramètres passé par référence automatiquement ; Ils n'ont pas besoin d'être passé an tant qu'objets variant. Consultez la bibliothèque MSDN pour des informations additionnelles sur le type VARIANT.
codepage
Spécifie la codepage qui est utilisé pour convertir les chaîne de caractères vers unicode. Voir le paramètre avec le même nom dans la classe com pour plus d'informations.

EvLoop::child

(PECL ev >= 0.2.0)

EvLoop::child — Crée un objet EvChild associé avec la boucle d'événements courante

Description

final public EvLoop::child ( string $pid , string $trace , string $callback [, string $data [, string $priority ]] ) : EvChild

Crée un objet EvChild associé avec la boucle d'événements courante.

Liste de paramètres

Tous les paramètres ont la même signification que pour la méthode EvChild::__construct().

Valeurs de retour

Retourne un objet EvChild en cas de succès.

Configuration à l'exécution

Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.

Options de configuration Mysqlnd_uh
Nom Défaut Modifiable Historique
mysqlnd_uh.enable 1 PHP_INI_SYSTEM
mysqlnd_uh.report_wrong_types 1 PHP_INI_ALL

Voici un éclaircissement sur l'utilisation des directives de configuration.

mysqlnd_uh.enable integer

Active ou désactive le plugin. Si désactivé, l'extension ne va pas autoriser les utilisateurs à le connexer à mysqlnd pour interception les appels à mysqlnd.

mysqlnd_uh.report_wrong_types integer

Si l'on doit rapporter des types de valeurs retournées erronées comme des erreurs de niveau E_WARNING. Il est vivement recommandé d'utiliser cette configuration pour détecter les erreurs.

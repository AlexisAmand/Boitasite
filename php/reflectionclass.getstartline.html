ReflectionClass::getStartLine

(PHP 5, PHP 7)

ReflectionClass::getStartLine — Récupère le numéro de ligne de départ

Description

public ReflectionClass::getStartLine ( void ) : int

Récupère le numéro de ligne de départ.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Le numéro de ligne de départ, sous la forme d'un entier.

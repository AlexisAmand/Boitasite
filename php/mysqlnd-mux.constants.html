Constantes pré-définies

Ces constantes sont définies par cette extension, et ne sont disponibles que si cette extension a été compilée avec PHP, ou bien chargée au moment de l'exécution.

Other

Le numéro de version du plugin peut être obtenu en utilisant la constante MYSQLND_MUX_VERSION ou la constante MYSQLND_MUX_VERSION_ID. La constante MYSQLND_MUX_VERSION est la représentation sous forme de chaîne de caractères du numéro de version numérique MYSQLND_MUX_VERSION_ID, qui est un entier comme par exemple, 10000. Les développeurs peuvent calculer le numéro de version comme ceci.

Version (partie) Exemple
Majeur*10000 1*10000 = 10000
Mineur*100 0*100 = 0
Patch 0 = 0
MYSQLND_MUX_VERSION_ID 10000

MYSQLND_MUX_VERSION (chaîne de caractères)
Chaîne de caractère représentant la version du plugin, par exemple 1.0.0-prototype.
MYSQLND_MUX_VERSION_ID (entier)
Numéro de version du plugin, par exemple, 10000.

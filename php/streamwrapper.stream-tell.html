streamWrapper::stream_tell

(PHP 4 >= 4.3.2, PHP 5, PHP 7)

streamWrapper::stream_tell — Lit la position courante dans un flux

Description

public streamWrapper::stream_tell ( void ) : int

Cette méthode est appelée en réponse à ftell() pour déterminer la position courante.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Doit retourner la position courante dans le flux.

eio_grp_limit

(PECL eio >= 0.0.1dev)

eio_grp_limit — Définit la limite d'un groupe

Description

eio_grp_limit ( resource $grp , int $limit ) : void

Limite le nombre de requêtes dans un groupe de requêtes.

Liste de paramètres

grp

La ressource du groupe de requêtes.

limit

Nombre de requêtes maximales dans le groupe.

Valeurs de retour

Aucune valeur n'est retournée.

TokyoTyrantIterator::rewind

(PECL tokyo_tyrant >= 0.2.0)

TokyoTyrantIterator::rewind — Réinitialise l'itérateur

Description

public TokyoTyrantIterator::rewind ( void ) : void

Réinitialise l'itérateur pour une nouvelle itération. Cette méthode est appelée automatiquement au début d'une boucle foreach.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Lance une exception TokyoTyrantException si l'initialisation de l'itérateur échoue.

Throwable::getCode

(PHP 7)

Throwable::getCode — Récupère le code de l'exception

Description

abstract public Throwable::getCode ( void ) : int

Retourne le code d'erreur associé avec l'objet jeté.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne le code de l'exception en tant que entier dans Exception mais peut être un autre type dans les descendant de Exception (par exemple chaîne de caractères dans PDOException).

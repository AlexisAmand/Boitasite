Yaf_Dispatcher::registerPlugin

(Yaf >=1.0.0)

Yaf_Dispatcher::registerPlugin — Enregistre un plugin

Description

public Yaf_Dispatcher::registerPlugin ( Yaf_Plugin_Abstract $plugin ) : Yaf_Dispatcher

Enregistre une plugin (voir Yaf_Plugin_Abstract). Générallement, l'enregistrement d'un plugin se situe dans le bootstrap (voir Yaf_Bootstrap_Abstract).

Liste de paramètres

plugin

Valeurs de retour

Exemples

Exemple #1 Exemple avec Yaf_Dispatcher::registerPlugin()

<?php
class Bootstrap extends Yaf_Bootstrap_Abstract {
  public function _initPlugin(Yaf_Dispatcher $dispatcher) {
    /**
    * Yaf suppose que les scripts de plugin se trouvent dans [application.directory] .  "/plugins" 
    * dans ce cas, il serait :
    * [application.directory] . "/plugins/" . "User" . [application.ext]
    */ 
    $user = new UserPlugin();
    $dispatcher->registerPlugin($user);
  }
}
?>

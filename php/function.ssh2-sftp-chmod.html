ssh2_sftp_chmod

(PECL ssh2 >= 0.12)

ssh2_sftp_chmod — Modifie le mode d'un fichier

Description

ssh2_sftp_chmod ( resource $sftp , string $filename , int $mode ) : bool

Tente de modifier le mode du fichier spécifié, en utilisant le mode fourni.

Liste de paramètres

sftp

Une ressource SSH2 SFTP, ouverte avec la fonction ssh2_sftp().

filename

Chemin vers le fichier.

mode

Permissions sur le fichier. Voir la fonction chmod() pour plus de détails concernant ce paramètre.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Exemples

Exemple #1 Changement du mode du fichier sur le serveur distant

<?php
$connection = ssh2_connect('shell.example.com', 22);
ssh2_auth_password($connection, 'nomUtilisateur', 'MotDePasse');
$sftp = ssh2_sftp($connection);

ssh2_sftp_chmod($sftp, '/dossier/fichier', 0755);
?>

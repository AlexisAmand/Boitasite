ReflectionFunction::invoke

(PHP 5, PHP 7)

ReflectionFunction::invoke — Invoque une fonction

Description

public ReflectionFunction::invoke ([ mixed $... ] ) : mixed

Invoque une fonction reflétée.

Liste de paramètres

...

La liste d'arguments à passer à la fonction. Il est possible de passer un nombre variable d'arguments à la fonction, comme pour la fonction call_user_func().

Valeurs de retour

Retourne le résultat de la fonction invoquée.

Exemples

Exemple #1 Exemple avec ReflectionFunction::invoke()

<?php
function title($title, $name)
{
   return sprintf("%s. %s\r\n", $title, $name);
}

$function = new ReflectionFunction('title');

echo $function->invoke('Dr', 'Phil');
?>

L'exemple ci-dessus va afficher :

Dr. Phil

Notes

Note:

Si la fonction a des arguments qui ont besoin d'être des références, alors ils doivent être passés par références dans la liste des arguments.

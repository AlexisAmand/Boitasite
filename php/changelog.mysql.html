Historique des modifications

Les modifications suivantes ont été apportées aux classes/fonctions/méthodes de cette extension.

Historique des modifications général pour l'extension ext/mysql

Cet historique des modifications fait référence à l'extension ext/mysql.

Modifications globales sur ext/mysql

La liste suivante présente les modifications sur l'extension entière ext/mysql.

Version Description
7.0.0

Cette extension a été supprimée de PHP. Pour plus de détails, voir Choisir une API.

5.5.0

Cette extension est devenue obsolète. La connexion à une base de données MySQL via les fonctions mysql_connect(), mysql_pconnect() ou une connexion implicite via toute autre fonction mysql_* génèrera une alerte de niveau E_DEPRECATED.

5.5.0

Toutes les anciennes fonctions et alias obsolètes émettent maintenant une alerte de niveau E_DEPRECATED. Ces fonctions sont :

mysql(), mysql_fieldname(), mysql_fieldtable(), mysql_fieldlen(), mysql_fieldtype(), mysql_fieldflags(), mysql_selectdb(), mysql_createdb(), mysql_dropdb(), mysql_freeresult(), mysql_numfields(), mysql_numrows(), mysql_listdbs(), mysql_listtables(), mysql_listfields(), mysql_db_name(), mysql_dbname(), mysql_tablename(), et mysql_table_name().

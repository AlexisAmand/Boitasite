pcntl_strerror

(PHP 5 >= 5.3.4, PHP 7)

pcntl_strerror — Récupère le message d'erreur système associé avec l'errno fourni

Description

pcntl_strerror ( int $errno ) : string

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

errno

Valeurs de retour

Retourne la description de l'erreur en cas de succès ou FALSE si une erreur survient.

Constantes pré-définies

Ces constantes sont définies par cette extension, et ne sont disponibles que si cette extension a été compilée avec PHP, ou bien chargée au moment de l'exécution.

PTHREADS_INHERIT_ALL (integer)
Les options par défaut pour tous les threads, faisant que les pthreads copient les environnements lorsque les nouveaux threads sont démarrés
PTHREADS_INHERIT_NONE (integer)
N'hérite de rien lorsque les nouveaux threads sont démarrés
PTHREADS_INHERIT_INI (integer)
Hérite des entrées INI lorsque les nouveaux threads sont démarrés
PTHREADS_INHERIT_CONSTANTS (integer)
Hérite des constantes déclarées par l'utilisateur lorsque les nouveaux threads sont démarrés
PTHREADS_INHERIT_CLASSES (integer)
Hérite des classes déclarées par l'utilisateur lorsque les nouveaux threads sont démarrés
PTHREADS_INHERIT_FUNCTIONS (integer)
Hérite des fonctions déclarées par l'utilisateur lorsque les nouveaux threads sont démarrés
PTHREADS_INHERIT_INCLUDES (integer)
L'héritage inclus l'information de fichier lorsque les nouveaux threads sont démarrés
PTHREADS_INHERIT_COMMENTS (integer)
Hérite de tous les commentaires lorsque les nouveaux threads sont démarrés
PTHREADS_ALLOW_HEADERS (integer)
Autorise les nouveaux threads à envoyer les en-têtes à la sortie standart (normalement non autorisé)

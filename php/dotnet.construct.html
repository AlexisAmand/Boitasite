dotnet::__construct

(PHP 4 >= 4.1.0, PHP 5, PHP 7)

dotnet::__construct — Constructeur de la classe dotnet

Description

dotnet::__construct ( string $assembly_name , string $class_name [, int $codepage = CP_ACP ] )

Construit un nouvel objet dotnet.

Liste de paramètres

assembly_name
Spécifie l'assemblage qui doit être chargé.
class_name
Spécifie la classe dans cet assemblage à instancier.
codepage
Codepage à utiliser pour les transformations de chaîne de caractères unicode ; voir la classe com pour plus de détails sur les codepages.

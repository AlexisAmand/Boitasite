QuickHashIntStringHash::saveToString

(PECL quickhash >= Unknown)

QuickHashIntStringHash::saveToString — Retourne une version linéarisée de la table de hachage

Description

public QuickHashIntStringHash::saveToString ( void ) : string

Cette méthode retourne une version linéarisée de la table de hachage dans un format que la fonction loadFromString() peut lire.

Valeurs de retour

Cette méthode retourne une chaîne contenant un format linéarisé de la table de hachage. Chaque élément est stocké sous la forme d'une valeur sous 4 octets en Endianness que le système courant utilise.

Exemples

Exemple #1 Exemple avec QuickHashIntStringHash::saveToString()

<?php
$hash = new QuickHashIntStringHash( 1024 );
var_dump( $hash->exists( 4 ) );
var_dump( $hash->add( 4, "thirty four" ) );
var_dump( $hash->exists( 4 ) );
var_dump( $hash->add( 5, "fifty five" ) );

var_dump( $hash->saveToString() );
?>

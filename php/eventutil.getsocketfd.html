EventUtil::getSocketFd

(PECL event >= 1.7.0)

EventUtil::getSocketFd — Retourne le descripteur de fichier d'un socket ou d'un flux

Description

public static EventUtil::getSocketFd ( mixed $socket ) : int

Retourne le descripteur numérique de fichier d'un socket ou d'un flux spécifié par le paramètre socket, comme le fait l'extension Event en interne pour toutes les méthodes acceptant un socket ou un flux.

Liste de paramètres

socket

Socket ou flux.

Valeurs de retour

Retourne descripteur de fichier numérique en cas de succès, FALSE sinon. EventUtil::getSocketFd() retourne FALSE dans le cas où le type du fichier n'a pu être reconnu, ou bien lorsque le descripteur du fichier associé avec le paramètre socket n'est pas valide.

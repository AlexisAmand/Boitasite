SyncReaderWriter::readunlock

(PECL sync >= 1.0.0)

SyncReaderWriter::readunlock — Libère un verrou de lecture

Description

public SyncReaderWriter::readunlock ( void ) : bool

Libère un verrou de lecture sur un objet SyncReaderWriter.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

TRUE si la libération du verrou a été un succès, FALSE sinon.

Exemples

Exemple #1 Exemple avec SyncReaderWriter::readunlock()

<?php
$readwrite = new SyncReaderWriter("FileCacheLock");
$readwrite->readlock();
/* ... */
$readwrite->readunlock();
?>

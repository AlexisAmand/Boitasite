Imagick::getImageExtrema

(PECL imagick 2.0.0)

Imagick::getImageExtrema — Lit les extrêmes d'une image

Avertissement

Cette fonction est OBSOLÈTE à partir de Imagick 3.4.4. Dépendre de cette fonctionnalité est fortement déconseillé.

Description

Imagick::getImageExtrema ( void ) : array

Lit les extrêmes d'une image. Retourne un tableau associatif, avec les clés "min" et "max".

Valeurs de retour

Retourne un tableau associatif, avec les clés "min" et "max".

Erreurs / Exceptions

Lance une exception ImagickException si une erreur survient.

Ev::iteration

(PECL ev >= 0.2.0)

Ev::iteration — Retourne le nombre de fois que la boucle d'événement par défaut a été sollicitée pour un nouvel événement

Description

final public static Ev::iteration ( void ) : int

Retourne le nombre de fois que la boucle d'événement par défaut a été sollicitée pour un nouvel événement. Peut être utile comme compteur de génération.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne le nombre de sollicitation de la boucle d'événement par défaut.

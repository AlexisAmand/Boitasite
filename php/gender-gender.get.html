Gender\Gender::get

(PECL gender >= 0.6.0)

Gender\Gender::get — Récupère le sexe d'un prénom

Description

public Gender\Gender::get ( string $name [, int $country ] ) : int

Récupère le sexe d'un prénom pour un pays spécifique.

Liste de paramètres

name

Le prénom à vérifier.

country

L'identifiant du pays (une constante de la classe Gender).

Valeurs de retour

Retourne le sexe du prénom.

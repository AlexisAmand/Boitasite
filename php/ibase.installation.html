Installation

Cette extension a été déplacée dans le module » PECL et ne sera plus intégrée dans PHP à partir de PHP 7.4.0

Des informations sur l'installation de ces extensions PECL peuvent être trouvées dans le chapitre du manuel intitulé Installation des extensions PECL. D'autres informations comme les notes sur les nouvelles versions, les téléchargements, les sources des fichiers, les informations concernant les mainteneurs ainsi qu'un CHANGELOG, peuvent être trouvées ici : » https://pecl.php.net/package/ibase.

Pour activer le support de Firebird/InterBase, vous devez compiler PHP avec l'option --with-interbase[=DIR], où DIR est le dossier d'installation de Firebird/InterBase (qui est par défaut, /usr).

Note: Note aux utilisateurs Win32/Win64

Afin de faire fonctionner cette extension, quelques bibliothèques DLL doivent être disponibles via le PATH du système Windows. Lisez la F.A.Q intitulée "Comment ajouter mon dossier PHP à mon PATH Windows" pour plus d'informations. Le fait de copier les bibliothèques DLL depuis le dossier PHP dans le dossier système Windows fonctionne également (car le dossier système est par défaut dans le PATH système), mais cette méthode n'est pas recommandée. Cette extension nécessite que les fichiers suivants se trouvent dans le PATH : fbclient.dll,gds32.dll

Si vous installez un serveur Firebird/InterBase sur la même machine que celle faisant fonctionner PHP, vous aurez déjà cette bibliothèque et fbclient.dll,gds32.dll (gds32.dll est généré depuis l'installeur) devrait déjà être dans votre PATH.

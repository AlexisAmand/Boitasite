phpdbg_break_method

(PHP 5 >= 5.6.3, PHP 7)

phpdbg_break_method — Insère un point d'arrêt à l'entrée d'une méthode

Description

phpdbg_break_method ( string $class , string $method ) : void

Insère un point d'arrêt à l'entrée de la méthode method de la classe class.

Liste de paramètres

class

Le nom de la classe.

method

Le nom de la méthode.

Valeurs de retour

Aucune valeur n'est retournée.

Installation

Il n'y pas d'installation nécessaire pour utiliser ces fonctions, elles font parties du coeur de PHP.

Pour activer le hachage Argon2 de mot de passe, néanmoins, PHP doit être compilé avec le support de libargon2 en utilisant l'option de configuration --with-password-argon2[=DIR].

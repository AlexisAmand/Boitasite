SVMModel::predict

(PECL svm >= 0.1.0)

SVMModel::predict — Prédit une valeur pour des précédentes données non vues

Description

public SVMModel::predict ( array $data ) : float

Cette fonction accepte un tableau de données et tente de prédire la valeur de la classe ou de la régression, suivant le modèle extrait depuis les données précédemment entraînées.

Liste de paramètres

data

Le tableau à classifier. Ce peut être une série de paires clé => valeur dans l'ordre croissant des clés, mais non nécessairement continue.

Valeurs de retour

Retourne la valeur prédite. Ce peut être un libellé de classe dans le cas d'une classification, une valeur réelle dans le cas d'une régression. Lance une exception SVMException si une erreur survient.

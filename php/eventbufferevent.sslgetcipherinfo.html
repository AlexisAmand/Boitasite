EventBufferEvent::sslGetCipherInfo

(PECL event >= 1.10.0)

EventBufferEvent::sslGetCipherInfo — Retourne une description textuelle d'un cipher

Description

public EventBufferEvent::sslGetCipherInfo ( void ) : string

Récupère la description du cipher courant via la fonction SSL_CIPHER_description de l'API SSL (voir la page man de SSL_CIPHER_get_name(3)).

Note:

Cette fonction n'est disponible que si Event a été compilé avec le support OpenSSL.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne une description textuelle du cipher en cas de succès, ou bien FALSE si une erreur survient.

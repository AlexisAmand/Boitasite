gmp_binomial

(PHP 7 >= 7.3.0)

gmp_binomial — Calcul le coefficient binomial

Description

gmp_binomial ( mixed $n , int $k ) : GMP

Calcul le coefficient binomial C(n, k).

Liste de paramètres

n

Il peut être soit une ressource GMP en PHP 5.5 et antérieurs, soit un objet GMP en PHP 5.6 et suivants, soit une chaîne numérique qu'il est possible de convertir plus tard en un nombre.

k

Valeurs de retour

Retourne le coefficient binomial C(n, k), ou FALSE si une erreur survient.

Erreurs / Exceptions

Émet E_WARNING si k est négatif.

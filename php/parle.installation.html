Installation

Cette extension » PECL n'est pas intégrée à PHP.

Des informations sur l'installation de ces extensions PECL peuvent être trouvées dans le chapitre du manuel intitulé Installation des extensions PECL. D'autres informations comme les notes sur les nouvelles versions, les téléchargements, les sources des fichiers, les informations concernant les mainteneurs ainsi qu'un CHANGELOG, peuvent être trouvées ici : » https://pecl.php.net/package/parle.

Une DLL pour cette extension PECL est disponible ici : » http://windows.php.net/downloads/pecl/releases/parle/.

Options de configuration disponibles
OptionDescription
--enable-parleActivation de l'extension parle.
--enable-parle-utf32Activation du support interne UTF-32. Disponible depuis parle 0.7.2.

Fragmentation

Pour se connecter à un cluster fragmenté, vous devez spécifier une ou plusieurs instances mongos dans la chaîne de connexion. Les serveurs doivent être séparés par une virgule.

<?php

// Utilisation d'un serveur depuis la liste
$m = new MongoClient("mongodb://mongos1.example.com:27017");

// Utilisation de plusieurs serveurs depuis la liste
$m = new MongoClient("mongodb://mongos1.example.com:27017,mongos2.example.com:27017");

?>

Que le cluster fragmenté soit un serveur mongod simple ou un jeu complet de réplication, le processus de connexion du driver reste le même. Toutes les communications à la base de données seront dirigées vers mongos.

Pour plus d'informations sur la fragmentation avec MongoDB, reportez-vous à cette » documentation.

ResourceBundle::create

resourcebundle_create

ResourceBundle::__construct

(PHP 5 >= 5.3.2, PHP 7, PECL intl >= 2.0.0)

ResourceBundle::create -- resourcebundle_create -- ResourceBundle::__construct — Crée une ressource représentant un faisceau

Description

Style orienté objet (méthode)

public static ResourceBundle::create ( string $locale , string $bundlename [, bool $fallback ] ) : ResourceBundle

Style procédural

resourcebundle_create ( string $locale , string $bundlename [, bool $fallback ] ) : ResourceBundle

Style orienté objet (constructeur) :

public ResourceBundle::__construct ( string $locale , string $bundlename [, bool $fallback ] )

Crée une ressource représentant un faisceau.

Liste de paramètres

locale

Locale depuis laquelle les ressources doivent être chargées (nom de la locale, i.e. en_CA).

bundlename

Le dossier où les données sont stockées ou le nom du fichier .dat.

fallback

Si la locale doit correspondre exactement ou si le fallback vers la locale parent est autorisé.

Valeurs de retour

Retourne un objet ResourceBundle ou NULL si une erreur survient.

Exemples

Exemple #1 Exemple avec resourcebundle_create()

<?php
$r = resourcebundle_create( 'es', "/usr/share/data/myapp");
echo $r['teststring'];
?>

Exemple #2 Exemple avec ResourceBundle::create()

<?php
$r = ResourceBundle::create( 'es', "/usr/share/data/myapp");
echo $r['teststring'];
?>

L'exemple ci-dessus va afficher :

¡Hola, mundo!

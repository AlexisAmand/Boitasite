ldap_add_ext

(PHP 7 >= 7.3.0)

ldap_add_ext — Ajoute une entrée dans un dossier LDAP

Description

ldap_add_ext ( resource $link_identifier , string $dn , array $entry [, array $serverctrls = array() ] ) : resource

Réalise la même chose que la fonction ldap_add() mais retourne comme résultat une ressource LDAP à analyser avec la fonction ldap_parse_result().

Liste de paramètres

Voir la fonction ldap_add()

Valeurs de retour

Retourne un identifiant de ressource LDAP ou FALSE si une erreur survient.

Notes

Note: Cette fonction gère les chaînes binaires.

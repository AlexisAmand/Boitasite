imagecreatefrombmp

(PHP 7 >= 7.2.0)

imagecreatefrombmp — Crée une nouvelle image depuis un fichier ou une URL

Description

imagecreatefrombmp ( string $filename ) : resource

imagecreatefrombmp() retourne un identifiant d'image représentant l'image obtenue à partir du nom de fichier fournit.

Astuce

Vous pouvez utiliser une URL comme nom de fichier avec cette fonction, si le gestionnaire fopen a été activé. Voyez fopen() pour plus de détails sur la façon de spécifier le nom du fichier. Reportez-vous aux Liste des protocoles et des gestionnaires supportés pour plus d'informations sur les capacités des différents gestionnaires, les notes sur leur utilisation, ainsi que les informations sur les variables prédéfinies qu'elles fournissent.

Liste de paramètres

filename

Chemin de l'image BMP.

Valeurs de retour

Retourne un identifiant de ressource d'image en cas de succès, ou FALSE si une erreur survient.

Exemples

Exemple #1 Convertit une image BMP en image PNG en utilisant imagecreatefrombmp()

<?php
// Load the BMP file
$im = imagecreatefrombmp('./example.bmp');

// Le convertit en fichier PNG avec les paramètres par défault
imagepng($im, './example.png');
imagedestroy($im);
?>

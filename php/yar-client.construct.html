Yar_Client::__construct

(PECL yar >= 1.0.0)

Yar_Client::__construct — Crée un client

Description

final public Yar_Client::__construct ( string $url [, array $options ] )

Crée un Yar_Client sur un serveur Yar_Server.

Liste de paramètres

url

URL du serveur Yar.

Valeurs de retour

Une instance Yar_Client.

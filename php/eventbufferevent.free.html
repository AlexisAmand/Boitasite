EventBufferEvent::free

(PECL event >= 1.2.6-beta)

EventBufferEvent::free — Libère un événement de tampon

Description

public EventBufferEvent::free ( void ) : void

Libère toutes les ressources allouées par un événement de tampon.

Habituellement, il n'est pas nécessaire d'appeler cette méthode, sachant que normalement, la libération totale des ressources associées s'effectue dans les destructeurs internes de l'objet. Cependant, il se peut qu'un long script alloue beaucoup d'instances, ou bien qu'un script utilise énormément de mémoire ; dans ces cas, il peut être nécessaire de libérer les ressources aussi vite que possible. Aussi, l'utilisation de la méthode EventBufferEvent::free() peut être utile pour protéger l'exécution du script d'atteindre la valeur de memory_limit.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

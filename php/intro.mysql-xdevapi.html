Introduction

Cette extension fournit un accès au Document Store de MySQL à travers la DevAPI X. La DevAPI X est une API commune fournit par plusieurs MySQL Connectors fournissant un accès facile aux tables relationelles aussi bien qu'aux collections de documents, qui sont représentés en JSON, depuis une API avec des opérations de style CRUD.

La DevAPI de X utilise le protocole X, le protocole nouvelle génération client-serveur de MySQL serveur 8.0.

Pour des informations générales sur MySQL Document Store, référez-vous au chapitre » MySQL Document Store du manuel MySQL.

MongoDB\BSON\Int64::__construct

(mongodb >=1.5.0)

MongoDB\BSON\Int64::__construct — Construit un nouveau Int64 (inutilisé)

Description

final private MongoDB\BSON\Int64::__construct ( void )

Les objets MongoDB\BSON\Int64 sont crées à travers une conversion depuis un type BSON entier 64-bit sur une platforme 32-bit et ne peuvent pas être construit directement.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

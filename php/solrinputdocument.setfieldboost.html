SolrInputDocument::setFieldBoost

(PECL solr >= 0.9.2)

SolrInputDocument::setFieldBoost — Définit la valeur de boost pour un champ

Description

public SolrInputDocument::setFieldBoost ( string $fieldName , float $fieldBoostValue ) : bool

Définit la valeur de boost pour un champ. Ceci remplacera la valeur de boost courante pour ce champ.

Liste de paramètres

fieldName

Le nom du champ.

fieldBoostValue

La valeur de boost.

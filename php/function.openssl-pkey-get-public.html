openssl_pkey_get_public

(PHP 4 >= 4.2.0, PHP 5, PHP 7)

openssl_pkey_get_public — Extrait une clé publique d'un certificat, et la prépare

Description

openssl_pkey_get_public ( mixed $certificate ) : resource

openssl_pkey_get_public() extrait la clé publique du certificat certificate et la prépare pour être utilisée par les autres fonctions.

Liste de paramètres

certificate

certificate peut avoir l'une des valeurs suivantes :

  1. Une ressource d'un certificat X.509.
  2. Une chaîne au format file://path/to/file.pem. Le fichier ainsi désigné doit contenir une clé publique ou un certificat au format PEM (éventuellement les deux).
  3. Une clé publique au format PEM.

Valeurs de retour

Retourne une ressource positive, représentant une clé en cas de succès, ou FALSE si une erreur survient.

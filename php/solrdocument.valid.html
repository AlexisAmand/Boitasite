SolrDocument::valid

(PECL solr >= 0.9.2)

SolrDocument::valid — Vérifie si la position courant interne est toujours valide

Description

public SolrDocument::valid ( void ) : bool

Vérifie si la position courant interne est toujours valide. Utilisée lors d'opérations foreach.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE en cas de succès, ou FALSE si la position courante n'est plus valide.

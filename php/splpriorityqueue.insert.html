SplPriorityQueue::insert

(PHP 5 >= 5.3.0, PHP 7)

SplPriorityQueue::insert — Insère un élément dans la file d'attente

Description

public SplPriorityQueue::insert ( mixed $value , mixed $priority ) : bool

Insère la valeur value avec la priorité priority dans la file d'attente.

Liste de paramètres

value

La valeur à insérer.

priority

La priorité associée.

Valeurs de retour

Retourne TRUE.

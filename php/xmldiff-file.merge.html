XMLDiff\File::merge

(PECL xmldiff >= 0.8.0)

XMLDiff\File::merge — Fusionne un document XML

Description

public XMLDiff\File::merge ( string $src , string $diff ) : string

Crée un document XML basé sur des différences et sur le document source.

Liste de paramètres

src

Chemin vers le document XML source.

diff

Chemin vers le document XML contenant les informations de différence.

Valeurs de retour

Une chaîne de caractères représentant le nouveau document XML ou NULL.

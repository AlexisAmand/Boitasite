Comment sont générées les documentations

Ce manuel est écrit en XML, en utilisant la » DTD de DocBook XML, et » PhD (Le moteur [PH]P d'affichage [D]ocBook) pour la maintenance et le formatage.

Grâce au format XML comme source, nous avons la possibilité de générer de nombreux formats tout en ayant une seule source pour tous les formats. L'outil utilisé pour formater le manuel en ligne est » PhD. Nous utilisons » Microsoft HTML Help Workshop pour générer le format Windows HTML Help du manuel, et, bien sûr, PHP lui-même pour effectuer des conversions et du formatage.

Le manuel PHP est généré en de nombreux langages et formats, lisez » https://www.php.net/docs.php pour plus d'informations. Le code source XML peut être téléchargé depuis le SVN et lu sur » https://svn.php.net/viewvc/. La documentation est stockée dans le module phpdoc.

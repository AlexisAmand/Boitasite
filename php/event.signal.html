Event::signal

(PECL event >= 1.2.6-beta)

Event::signal — Construit un objet d'événement signal

Description

public static Event::signal ( EventBase $base , int $signum , callable $cb [, mixed $arg ] ) : Event

Construit un objet d'événement signal. C'est une méthode simple pour créer un événement de signal. Notez que la méthode générique Event::__construct() peut construire aussi un objet d'événement de signal.

Liste de paramètres

base

L'objet de base d'événement associé.

signum

Le numéro du signal.

cb

La fonction de rappel de l'événement signal. Voir les fonctions de rappel d'événements.

arg

Données personnalisées. Si spécifié, elles seront passées à la fonction de rappel lorsque l'événement lancera les triggers.

Valeurs de retour

Retourne un objet d'événement en cas de succès, FALSE sinon.

ibase_gen_id

(PHP 5, PHP 7 < 7.4.0)

ibase_gen_id — Incrémente le générateur donné et retourne sa nouvelle valeur

Description

ibase_gen_id ( string $generator [, int $increment = 1 [, resource $link_identifier = NULL ]] ) : mixed
Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Valeurs de retour

Retourne une nouvelle valeur générée sous la forme d'un entier ou d'une chaîne de caractères si la valeur est trop grosse.

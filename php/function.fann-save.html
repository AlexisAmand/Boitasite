fann_save

(PECL fann >= 1.0.0)

fann_save — Sauvegarde le réseau complet dans un fichier de configuration

Description

fann_save ( resource $ann , string $configuration_file ) : bool

Sauvegarde le réseau complet dans un fichier de configuration.

Le fichier de configuration contient toutes les informations sur le réseau neuronal, et active la fonction fann_create_from_file() pour créer une copie exacte du réseau neuronal, et tous les paramètres associés au réseau neuronal.

Ces trois paramètres (fann_set_callback(), fann_set_error_log(), fann_set_user_data()) ne sont PAS sauvegardés dans le fichier car ils ne peuvent pas être portés en toute sécurité dans un endroit différent. De même, les paramètres temporaires générés pendant l'entrainement comme fann_get_MSE() ne sont pas sauvegardés.

Liste de paramètres

ann

Ressource de réseau de neurones.

configuration_file

Le chemin vers le fichier de configuration.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

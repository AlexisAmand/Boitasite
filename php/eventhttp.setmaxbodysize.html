EventHttp::setMaxBodySize

(PECL event >= 1.4.0-beta)

EventHttp::setMaxBodySize — Définit la taille maximale du corps de la demande

Description

public EventHttp::setMaxBodySize ( int $value ) : void

Définit la taille maximale du corps de la demande.

Liste de paramètres

value

La taille du corps, en octets.

Valeurs de retour

Aucune valeur n'est retournée.

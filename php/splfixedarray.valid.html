SplFixedArray::valid

(PHP 5 >= 5.3.0, PHP 7)

SplFixedArray::valid — Vérifie si le tableau contient encore un élément

Description

public SplFixedArray::valid ( void ) : bool

Vérifie si le tableau contient encore un élément.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE si le tableau contient encore un élément, FALSE sinon.

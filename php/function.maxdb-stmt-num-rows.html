maxdb_stmt_num_rows

maxdb_stmt::num_rows

(PECL maxdb >= 1.0)

maxdb_stmt_num_rows -- maxdb_stmt::num_rows — Retourne le nombre de lignes dans le jeu de résultats

Description

Style procédural

maxdb_stmt_num_rows ( resource $stmt ) : int

Style orienté objet

int $maxdb_stmt->num_rows;

Retourne le nombre de lignes dans le jeu de résultats.

Valeurs de retour

Un entier représentant le nombre de lignes dans le jeu de résultats.

Exemples

Exemple #1 Style orienté objet

<?php
/* Ouvre une connexion */
$maxdb = new maxdb("localhost", "MONA", "RED", "DEMODB");

/* Vérification de la connexion */
if (maxdb_connect_errno()) {
    printf("Echec de la connexion : %s\n", maxdb_connect_error());
    exit();
}

$query = "SELECT zip, name FROM hotel.city ORDER BY name";
if ($stmt = $maxdb->prepare($query)) {

    /* Exécute une requête */
    $stmt->execute();

    /* Stock le résultat */
    $stmt->store_result();

    printf("Nombre de lignes : %d.\n", $stmt->num_rows);

    /* Fermeture de la requête */
    $stmt->close();
}

/* Fermeture de la connexion */
$maxdb->close();
?>

Exemple #2 Style procédural

<?php
/* Ouvre une connexion */
$link = maxdb_connect("localhost", "MONA", "RED", "DEMODB");

/* Vérification de la connexion */
if (maxdb_connect_errno()) {
    printf("Echec de la connexion : %s\n", maxdb_connect_error());
    exit();
}

$query = "SELECT zip, name FROM hotel.city ORDER BY name";
if ($stmt = maxdb_prepare($link, $query)) {

    /* Exécute une requête */
    maxdb_stmt_execute($stmt);

    /* Stock le résultat */
    maxdb_stmt_store_result($stmt);

    printf("Nombre de lignes : %d.\n", maxdb_stmt_num_rows($stmt));

    /* Fermeture de la requête */
    maxdb_stmt_close($stmt);
}

/* Fermeture de la connexion */
maxdb_close($link);
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

Nombre de lignes : 25.

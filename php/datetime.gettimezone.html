DateTime::getTimezone

DateTimeImmutable::getTimezone

DateTimeInterface::getTimezone

date_timezone_get

(PHP 5 >= 5.2.1, PHP 7)

DateTime::getTimezone -- DateTimeImmutable::getTimezone -- DateTimeInterface::getTimezone -- date_timezone_get — Retourne le fuseau horaire relatif au DateTime fourni

Description

Style orienté objet

public DateTime::getTimezone ( void ) : DateTimeZone
public DateTimeImmutable::getTimezone ( void ) : DateTimeZone
public DateTimeInterface::getTimezone ( void ) : DateTimeZone

Style procédural

date_timezone_get ( DateTimeInterface $object ) : DateTimeZone

Retourne le fuseau horaire relatif au DateTime fourni.

Liste de paramètres

object

Seulement en style procédural : un objet DateTime retourné par date_create()

Valeurs de retour

Retourne un objet DateTimeZone en cas de succès ou FALSE si une erreur survient.

Exemples

Exemple #1 Exemple avec DateTime::getTimezone()

Style orienté objet

<?php
$date = new DateTime(null, new DateTimeZone('Europe/London'));
$tz = $date->getTimezone();
echo $tz->getName();
?>

Style procédural

<?php
$date = date_create(null, timezone_open('Europe/London'));
$tz = date_timezone_get($date);
echo timezone_name_get($tz);
?>

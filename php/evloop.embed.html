EvLoop::embed

(PECL ev >= 0.2.0)

EvLoop::embed — Crée une instance de l'observateur EvEmbed associé avec l'objet de l'EvLoop courant

Description

final public EvLoop::embed ( string $other [, string $callback [, string $data [, string $priority ]]] ) : EvEmbed

Crée une instance de l'observateur EvEmbed associé avec l'objet EvLoop courant.

Liste de paramètres

Tous les paramètres ont la même signification que pour la méthode EvEmbed::__construct().

Valeurs de retour

Retourne l'objet EvEmbed en cas de succès.

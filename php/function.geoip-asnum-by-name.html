geoip_asnum_by_name

(PECL geoip >= 1.1.0)

geoip_asnum_by_name — Récupère l'ASN (Autonomous System Numbers)

Description

geoip_asnum_by_name ( string $hostname ) : string

La fonction geoip_asnum_by_name() va récupérer l'ASN (Autonomous System Numbers) associé avec l'adresse IP.

Liste de paramètres

hostname

Le nom d'hôte ou l'adresse IP.

Valeurs de retour

Retourne l'ASN en cas de succès, ou FALSE si l'adresse ne peut être trouvée dans la base de données.

Exemples

Exemple #1 Exemple avec geoip_asnum_by_name()

Cet exemple va afficher l'ASN pour l'hôte www.example.com.

<?php
$asn = geoip_asnum_by_name('www.example.com');

if ($asn) {
    echo 'L\'ASN est : ' . $asn;
}
?>

L'exemple ci-dessus va afficher :

L'ASN est : AS15133 EdgeCast Networks, Inc

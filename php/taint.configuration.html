Configuration à l'exécution

Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.

Taint Options de configuration
Nom Défaut Modifiable Historique
taint.enable 0 PHP_INI_SYS
taint.error_level E_WARNING PHP_INI_ALL

Voici un éclaircissement sur l'utilisation des directives de configuration.

taint.enable integer

Si l'on doit activer taint.

taint.error_level integer

le type d'erreur que taint va reporter lorsqu'il trouvera une chaîne non propre.

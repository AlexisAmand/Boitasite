ReflectionFunctionAbstract::returnsReference

(PHP 5 >= 5.2.0, PHP 7)

ReflectionFunctionAbstract::returnsReference — Vérifie si la fonction retourne une référence

Description

public ReflectionFunctionAbstract::returnsReference ( void ) : bool

Vérifie si la fonction retourne une référence.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

TRUE si la fonction retourne une référence, FALSE sinon.

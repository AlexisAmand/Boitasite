Configuration à l'exécution

Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.

Options de configuration Mysqlnd_mux
Nom Défaut Modifiable Historique
mysqlnd_mux.enable 0 PHP_INI_SYSTEM

Voici un éclaircissement sur l'utilisation des directives de configuration.

mysqlnd_mux.enable integer

Active ou désactive le plugin. Si désactivé, l'extension ne se raccordera pas à mysqlnd pour se comporter comme proxy des appels API C mysqlnd.

EvLoop::suspend

(PECL ev >= 0.2.0)

EvLoop::suspend — Suspend la boucle

Description

public EvLoop::suspend ( void ) : void

Les méthodes EvLoop::suspend() et EvLoop::resume() suspendent et reprennent une boucle.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

Voir aussi

  • EvLoop::resume() - Reprend une boucle d'évément précédemment stoppée
  • Ev::suspend() - Suspend la boucle par défaut d'événements

ldap_first_entry

(PHP 4, PHP 5, PHP 7)

ldap_first_entry — Retourne la première entrée

Description

ldap_first_entry ( resource $link_identifier , resource $result_identifier ) : resource

Retourne la première entrée du résultat. Celles qui suivent seront lues grâce à la fonction ldap_next_entry(), en l'appelant aussi souvent que nécessaire.

Les entrées d'un résultat LDAP sont lues séquentiellement avec les fonctions ldap_first_entry() et ldap_next_entry().

Liste de paramètres

link_identifier

Un identifiant de lien LDAP, retourné par la fonction ldap_connect().

result_identifier

Valeurs de retour

Retourne la première entrée en cas de succès, FALSE si une erreur survient.

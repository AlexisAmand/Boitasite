mb_ord

(PHP 7 >= 7.2.0)

mb_ord — Récupère le point de code d'un charactère

Description

mb_ord ( string $str [, string $encoding ] ) : int

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

str

encoding

Valeurs de retour

Retourne le point de code d'un charactère ou FALSE si une erreur survient.

  • ord() - Convertit le premier octet d'une chaîne en une valeur entre 0 et 255

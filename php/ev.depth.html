Ev::depth

(PECL ev >= 0.2.0)

Ev::depth — Retourne la profondeur de récursion

Description

final public static Ev::depth ( void ) : int

Le nombre de fois où la méthode Ev::run() est entrée, moins le nombre de fois où la méthode Ev::run() est sortie normalement, en d'autres termes, la profondeur de récursion. En dehors de Ev::run(), ce nombre vaut 0. Dans une fonction de rappel, ce nombre vaut 1, tant que la méthode Ev::run() est invoquée récursivement (ou depuis un autre thread), auquel cas, il sera supérieur.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

ev_depth() retourne la profondeur de récursion de la boucle par défaut.

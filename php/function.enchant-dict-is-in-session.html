enchant_dict_is_in_session

(PHP 5 >= 5.3.0, PHP 7, PECL enchant >= 0.1.0 )

enchant_dict_is_in_session — Vérifie si un mot existe dans une session de vérification

Description

enchant_dict_is_in_session ( resource $dict , string $word ) : bool

Vérifie si un mot existe ou non dans la session courante.

Liste de paramètres

dict

Ressource de dictionnaire

word

Le mot à vérifier

Valeurs de retour

Retourne TRUE si le mot existe, FALSE sinon.

ReflectionClassConstant::__construct

(PHP 7 >= 7.1.0)

ReflectionClassConstant::__construct — Construit une ReflectionClassConstant

Description

public ReflectionClassConstant::__construct ( mixed $class , string $name )

Construit un nouvel objet ReflectionClassConstant.

Liste de paramètres

class

Soit une chaîne de caractères contenant le nom de la classe a refléter, soit un objet.

name

Le nom de la constante de classe.

Valeurs de retour

Retourne une instance construite de ReflectionClassConstant.

Erreurs / Exceptions

Lance une Exception dans le cas où la constante de classe fournit n'existe pas.

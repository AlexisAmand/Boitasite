pcntl_wtermsig

(PHP 4 >= 4.1.0, PHP 5, PHP 7)

pcntl_wtermsig — Retourne le signal qui a provoqué la fin du processus fils

Description

pcntl_wtermsig ( int $status ) : int

Retourne le numéro du signal qui a provoqué la fin du processus fils. Cette fonction est uniquement utile si pcntl_wifsignaled() retourne TRUE.

Liste de paramètres

status

Le paramètre status est le paramètre status passé à un appel de pcntl_waitpid() ayant réussi.

Valeurs de retour

Retourne le numéro du signal, sous la forme d'un entier.

Voir aussi

  • pcntl_waitpid() - Attend la fin de l'exécution d'un processus fils
  • pcntl_signal() - Installe un gestionnaire de signaux

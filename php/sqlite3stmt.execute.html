SQLite3Stmt::execute

(PHP 5 >= 5.3.0, PHP 7)

SQLite3Stmt::execute — Exécute une requête préparée

Description

public SQLite3Stmt::execute ( void ) : SQLite3Result

Exécute une requête préparée et retourne un objet représentant le jeu de résultats.

Attention

Les objets de jeu de résultats récupérés par l'appel de cette méthode sur le même objet d'instruction ne sont pas indépendants, mais partagent plutôt la même structure sous-jacente. Par conséquent, il est recommandé d'appeler SQLite3Result::finalize(), avant d'appeler de nouveau SQLite3Stmt::execute() sur le même objet d'instruction.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un objet SQLite3Result si la requête préparée a été exécutée avec succès, ou FALSE si une erreur survient.

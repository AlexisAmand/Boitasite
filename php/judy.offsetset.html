Judy::offsetSet

(PECL judy >= 0.1.1)

Judy::offsetSet — Position à assigner

Description

public Judy::offsetSet ( mixed $offset , mixed $value ) : bool

Assigne une valeur à une position donnée.

Liste de paramètres

offset

La position à laquelle assigner une valeur.

value

La valeur à assigner.

Valeurs de retour

Aucune valeur n'est retournée.

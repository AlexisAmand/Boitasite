ZMQSocket::unbind

(PECL zmq >= 1.0.4)

ZMQSocket::unbind — Dé-lie le socket

Description

public ZMQSocket::unbind ( string $dsn ) : ZMQSocket

Dé-lie le socket du point final. Le point final est défini au format transport://address où le transport peut être une des valeurs suivantes : inproc, ipc, tcp, pgm or epgm.

Liste de paramètres

dsn

Le DSN précédemment lié, par exemple : transport://address.

Valeurs de retour

Retourne l'objet courant. Lance une exception ZMQSocketException en cas d'erreur.

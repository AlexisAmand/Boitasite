Constantes pré-définies

Ces constantes sont définies par cette extension, et ne sont disponibles que si cette extension a été compilée avec PHP, ou bien chargée au moment de l'exécution.

DBASE_VERSION (string)
La version de l'extension (Disponible à partir de dbase 7.0.0)
DBASE_RDONLY (int)
Ouvre une base de donnée en lecture seule. Utilisé avec dbase_open(). (Disponible à partir de dbase 7.0.0)
DBASE_RDWR (int)
Ouvre une base de donnée en lecture et écriture. Utilisé avec dbase_open(). (Disponible à partir de dbase 7.0.0)
DBASE_TYPE_DBASE (int)
Créer une base de donnée style dBASE. Utilisé avec dbase_create(). (Disponible à partir de dbase 7.0.0)
DBASE_TYPE_FOXPRO (int)
Créer une base de donnée style FoxPro. Utilisé avec dbase_create(). (Disponible à partir de dbase 7.0.0)

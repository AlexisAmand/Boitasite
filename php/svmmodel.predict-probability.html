SVMModel::predict_probability

(PECL svm >= 0.1.4)

SVMModel::predict_probability — Retourne les probabilités pour les données précédentes non présentées

Description

public SVMModel::predict_probability ( array $data ) : float

Cette fonction accepte une tableau de données et tente de prédire la classe, comme avec les fonctions de prédiction. De plus, cependant, cette fonction retourne un tableau de probabilités, une par classe du modèle, qui représente la chanse estimée que les données fournies soient un membre de cette classe. Nécessite que le modèle à utiliser soit traité avec le paramètre de probabilité fixé à TRUE.

Liste de paramètres

data

Le tableau à classifier. Ce doit être une série de paires clé=>valeur dont les clés sont en ordre croissant, mais pas nécessairement continue.

probabilities

La valeur fournie sera complétée avec les probabilités. Ce peut être soit NULL dans le cas où le modèle ne contient aucune information de probabiblité, ou un tableau dont les index sont les noms de classe, et les valeurs, la probabilité.

Valeurs de retour

Retourne la valeur prédie. Ce sera un libel de classe dans le cas d'une classification, une valeur réelle dans le cas d'une régression. Lance une exception de type SVMException si une erreur survient.

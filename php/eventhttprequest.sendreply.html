EventHttpRequest::sendReply

(PECL event >= 1.4.0-beta)

EventHttpRequest::sendReply — Envoi une réponse HTML au client

Description

public EventHttpRequest::sendReply ( int $code , string $reason [, EventBuffer $buf ] ) : void

Envoi une réponse HTML au client. Le corps de la réponse contient les données du paramètre optionnel buf.

Liste de paramètres

code

Le code réponse HTTP à envoyer.

reason

Un bref message à envoyer avec le code réponse.

buf

Le corps de la réponse.

Valeurs de retour

Aucune valeur n'est retournée.

Ev::stop

(PECL ev >= 0.2.0)

Ev::stop — Arrête la boucle d'événements par défaut

Description

final public static Ev::stop ([ int $how ] ) : void

Arrête la boucle d'événements par défaut.

Liste de paramètres

how

Une constants parmie les constantes Ev::BREAK_*.

Valeurs de retour

Aucune valeur n'est retournée.

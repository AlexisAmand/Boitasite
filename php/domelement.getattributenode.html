DOMElement::getAttributeNode

(PHP 5, PHP 7)

DOMElement::getAttributeNode — Retourne le noeud d'attribut

Description

public DOMElement::getAttributeNode ( string $name ) : DOMAttr

Retourne le noeud d'attribut avec le nom name pour l'élément courant.

Liste de paramètres

name

Le nom de l'attribut.

Valeurs de retour

Le noeud d'attribut. Notez que pour la déclaration de namespace XML (les attributs xmlns et xmlns:*) une instance de DOMNameSpaceNode est retourné au lieu de DOMAttr.

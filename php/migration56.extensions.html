Autres modifications dans les extensions

cURL

Un certain nombre de constantes de la bibliothèque cURL, précédemment marqué comme obsolète, ont été supprimés :

  • CURLOPT_CLOSEPOLICY
  • CURLCLOSEPOLICY_CALLBACK
  • CURLCLOSEPOLICY_LEAST_RECENTLY_USED
  • CURLCLOSEPOLICY_LEAST_TRAFFIC
  • CURLCLOSEPOLICY_OLDEST
  • CURLCLOSEPOLICY_SLOWEST

OCI8

  • Le support des jeux de résultats implicites pour Oracle Database 12c a été ajouté via la nouvelle fonction oci_get_implicit_resultset().
  • Utiliser oci_execute($s, OCI_NO_AUTO_COMMIT) pour un SELECT ne lance plus nécessairement un ROLLBACK interne lors la fermeture de la connexion.
  • Ajout de sondes DTrace contrôlées par l'option de configuration --enable-dtrace.
  • oci_internal_debug() n'a plus d'effet.
  • Le format de sortie de phpinfo() pour OCI8 a changé.

Zip

Une option de configuration --with-libzip est ajoutée pour utiliser une installation système de libzip. La version 0.11 de libzip est requise, mais il est recommandé d'utiliser une version 0.11.2 ou plus récente.

MySQLi

Une nouvelle option mysqli.rollback_on_cached_plink a été ajoutée, elle contrôle le comportement de restauration des connexions persistantes.

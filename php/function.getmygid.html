getmygid

(PHP 4 >= 4.1.0, PHP 5, PHP 7)

getmygid — Retourne le GID du propriétaire du script

Description

getmygid ( void ) : int

Retourne le GID du propriétaire du script.

Valeurs de retour

Retourne le GID du propriétaire du script courant, ou FALSE si une erreur survient.

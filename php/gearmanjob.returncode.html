GearmanJob::returnCode

(PECL gearman >= 0.5.0)

GearmanJob::returnCode — Récupère le dernier code retourné

Description

public GearmanJob::returnCode ( void ) : int

Retourne le dernier code retourné par le serveur de travaux.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Un code de retour valide Gearman.

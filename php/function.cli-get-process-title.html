cli_get_process_title

(PHP 5 >= 5.5.0, PHP 7)

cli_get_process_title — Retourne le titre du processus courant

Description

cli_get_process_title ( void ) : string

Retourne le titre du processus courant, tel que défini par la fonction cli_set_process_title(). Notez que ce titre peut être légèrement différent de celui affiché via les commandes ps et top, suivant le système sous-jacent.

Cette fonction n'est disponible qu'en mode CLI.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne le titre du processus courant, sous la forme d'une chaîne de caractères, ou NULL si une erreur survient.

Erreurs / Exceptions

Une alerte de niveau E_WARNING sera générée si le système sous-jacent n'est pas supporté.

Exemples

Exemple #1 Exemple avec cli_get_process_title()

<?php
echo "Titre du processus : " . cli_get_process_title() . "\n";
?>

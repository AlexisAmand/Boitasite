EvLoop::verify

(PECL ev >= 0.2.0)

EvLoop::verify — Effectue des vérifications de consistence interne (pour le débogage)

Description

public EvLoop::verify ( void ) : void

Effectue des vérifications de consistence interne (pour le débogage de libev) et stoppe le programme si des structures de données corrompues sont trouvées.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

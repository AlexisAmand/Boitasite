curl_share_strerror

(PHP 7 >= 7.1.0)

curl_share_strerror — Retourne une chaîne qui décrit le code d'erreur fournit

Description

curl_share_strerror ( int $errornum ) : string

Retourne un message d'erreur textuel décrivant le code d'erreur fournit.

Liste de paramètres

errornum

L'une des constantes des » codes d'erreurs cURL.

Valeurs de retour

Retourne une description de l'erreur ou NULL pour les codes d'erreurs invalides.

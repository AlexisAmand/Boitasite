imagesetclip

(PHP 7 >= 7.2.0)

imagesetclip — Définit le rectangle de coupure

Description

imagesetclip ( resource $im , int $x1 , int $y1 , int $x2 , int $y2 ) : bool

imagesetclip() définit le rectangle de coupure actuel, c'est à dire la zone au-delà de laquelle aucun pixel ne sera dessiné.

Liste de paramètres

image

Une ressource d'image, retournée par une des fonctions de création d'images, comme imagecreatetruecolor().

x1

La coordonée x du coin supérieur gauche.

y1

La coordonée y du coin supérieur gauche.

x2

La coordonée x du coin inférieur droite.

y2

La coordonée y du coin inférieur droite.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

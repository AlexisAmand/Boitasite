Yaf_Response_Abstract::prependBody

(Yaf >=1.0.0)

Yaf_Response_Abstract::prependBody — Le but de prependBody

Description

public Yaf_Response_Abstract::prependBody ( string $content [, string $key ] ) : bool

Ajout un contenu à un bloc de contenu existant.

Liste de paramètres

body

Le contenu.

key

La clé du contenu ; vous pouvez spécifier un contenu avec une clé ; si vous n'en spécifiez pas, alors Yaf_Response_Abstract::DEFAULT_BODY sera utilisé.

Note:

Ce paramètre a été introduit en version 2.2.0.

Valeurs de retour

Exemples

Exemple #1 Exemple avec Yaf_Response_Abstract::prependBody()

<?php
$response = new Yaf_Response_Http();

$response->setBody("World")->prependBody("Hello ");

echo $response;
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

Hello World

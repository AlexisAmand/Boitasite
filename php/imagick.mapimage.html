Imagick::mapImage

(PECL imagick 2.0.0)

Imagick::mapImage — Remplace la couleur d'une image par celle d'une image de référence

Avertissement

Cette fonction est OBSOLÈTE à partir de Imagick 3.4.4. Dépendre de cette fonctionnalité est fortement déconseillé.

Description

Imagick::mapImage ( Imagick $map , bool $dither ) : bool

Liste de paramètres

map

dither

Valeurs de retour

Retourne TRUE en cas de succès.

Erreurs / Exceptions

Lance une exception ImagickException si une erreur survient.

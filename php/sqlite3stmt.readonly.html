SQLite3Stmt::readOnly

(PHP 5 >= 5.3.6, PHP 7)

SQLite3Stmt::readOnly — Détermine si une déclaration est uniquement en lecture seule

Description

public SQLite3Stmt::readOnly ( void ) : bool

Détermine si une déclaration est uniquement en lecture seule. Une déclaration est considéré uniquement en lecture seule, si elle ne fait aucun changement direct au contenue du fichier de base de donnée. Noter que les fonctions SQL définit par l'utilisateur peuvent indirectement changer la base de données en tant qu'effet secondaire.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE si la déclaration est uniquement en lecture seule, FALSE sinon.

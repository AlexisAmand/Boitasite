ZipArchive::setEncryptionIndex

(PHP >= 7.2.0, PECL zip >= 1.14.0)

ZipArchive::setEncryptionIndex — Définit la méthode de chiffrement d'une entrée définit par son index

Description

public ZipArchive::setEncryptionIndex ( int $index , string $method [, string $password ] ) : bool

Définit la méthode de chiffrement d'une entrée définit par son index.

Liste de paramètres

index

Index de l'entrée.

method

La méthode de chiffrement définit par l'une des constantes ZipArchive::EM_.

password

Mot de passe optionel, valeur par défaut utilisé si absent.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Notes

Note:

Cette fonction est seulement disponible quand compilé contre libzip ≥ 1.2.0.

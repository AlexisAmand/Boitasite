Exemple d'utilisation de VarnishStat

Cet exemple montre comme récupérer les statistiques Varnish depuis la mémoire partagée.

Exemple #1 Récupération de statistiques

<?php

$vs = new VarnishStat;

try {
    $data = $vs->getSnapshot();
} catch (VarnishException $e) {
    echo $e->getMessage();
    exit(3);
}

exit(0);
?>

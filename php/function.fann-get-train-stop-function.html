fann_get_train_stop_function

(PECL fann >= 1.0.0)

fann_get_train_stop_function — Retourne la fonction d'arrêt utilisée pendant l'entrainement

Description

fann_get_train_stop_function ( resource $ann ) : int

Retourne la fonction d'arrêt utilisée pendant l'entrainement.

Les fonctions d'arrêt sont décrites dans les constantes des fonctions d'arrêt.

La fonction d'arrêt par défaut est FANN_STOPFUNC_MSE.

Liste de paramètres

ann

Ressource de réseau de neurones.

Valeurs de retour

La constante de fonction d'arrêt, ou FALSE si une erreur survient.

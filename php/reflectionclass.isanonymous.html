ReflectionClass::isAnonymous

(PHP 7)

ReflectionClass::isAnonymous — Vérifie si la classe est anonyme

Description

public ReflectionClass::isAnonymous ( void ) : bool

Vérifie si une classe est une classe anonyme.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Exemples

Exemple #1 Exemple avec ReflectionClass::isAnonymous()

<?php
class TestClass {}
$anonClass = new class {};

$normalClass = new ReflectionClass('TestClass');
$anonClass  = new ReflectionClass($anonClass);

var_dump($normalClass->isAnonymous());
var_dump($anonClass->isAnonymous());

?>

L'exemple ci-dessus va afficher :

bool(false)
bool(true)

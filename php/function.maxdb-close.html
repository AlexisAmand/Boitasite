maxdb_close

maxdb::close

(PECL maxdb >= 1.0)

maxdb_close -- maxdb::close — Ferme une connexion à une base de données

Description

Style procédural

maxdb_close ( resource $link ) : bool

Style orienté objet

maxdb::close ( void ) : bool

maxdb_close() ferme une connexion à une base de données spécifiée par le paramètre link.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

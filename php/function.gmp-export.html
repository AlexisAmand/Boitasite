gmp_export

(PHP 5 >= 5.6.1, PHP 7)

gmp_export — Exportation vers une chaîne binaire

Description

gmp_export ( GMP $gmpnumber [, int $word_size = 1 [, int $options = GMP_MSW_FIRST | GMP_NATIVE_ENDIAN ]] ) : string

Exporte un nombre GMP vers une chaîne binaire.

Liste de paramètres

gmpnumber

Le nombre GMP à exporter.

word_size

La valeur par défaut est 1. Le nombre d'octets dans chaque partie de données binaires. Principalement utilisé avec le paramètre options.

options

La valeur par défaut est GMP_MSW_FIRST | GMP_NATIVE_ENDIAN.

Valeurs de retour

Retourne une chaîne de caractères ou FALSE si une erreur survient.

Exemples

Exemple #1 Exemple avec gmp_export()

<?php
$number = gmp_init(16705);
echo gmp_export($number) . "\n";
?>

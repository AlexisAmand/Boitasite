Configuration à l'exécution

Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.

IMAP Options de configuration
Nom Défaut Modifiable Historique
imap.enable_insecure_rsh "0" PHP_INI_SYSTEM Disponible à partir de PHP 7.1.25, 7.2.13 et 7.3.0. Auparavant, c'était activé implicitement.
Pour plus de détails sur les modes PHP_INI_*, reportez-vous à Où une directive de configuration peut être modifiée.

Voici un éclaircissement sur l'utilisation des directives de configuration.

imap.enable_insecure_rsh boolean

Établir une connecion à un serveur peut invoquer des commandes rsh ou ssh, sauf si cette option php.ini est désactivé.

Avertissement

Ni PHP ni la librairy IMAP filtre les noms des boîtes au lettres avant de les passer aux commandes rsh ou ssh, par conséquent passer des données non fiable à cette fonction sans désactiver cette option php.ini est dangereux.

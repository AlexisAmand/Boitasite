SyncReaderWriter::writelock

(PECL sync >= 1.0.0)

SyncReaderWriter::writelock — Obtiens un verrou d'écriture exclusif

Description

public SyncReaderWriter::writelock ([ int $wait = -1 ] ) : bool

Obtiens un verrou d'écriture exclusif sur un objet SyncReaderWriter.

Liste de paramètres

wait

Le nombre de millisecondes à attendre le verrou. Une valeur de -1 signifie que l'on attend indéfiniement.

Valeurs de retour

TRUE si le verrou a été obtenu, FALSE sinon.

Exemples

Exemple #1 Exemple avec SyncReaderWriter::writelock()

<?php
$readwrite = new SyncReaderWriter("FileCacheLock");
$readwrite->writelock();
/* ... */
$readwrite->writeunlock();
?>

gmp_sign

(PHP 4 >= 4.0.4, PHP 5, PHP 7)

gmp_sign — Signe du nombre GMP

Description

gmp_sign ( GMP $a ) : int

Vérifie le signe d'un nombre.

Liste de paramètres

a

Soit une ressource de nombre GMP en PHP 5.5 et antérieur, un objet GMP dans PHP 5.6 et ultérieur, ou une chaîne numérique à condition qu'il soit possible de convertir ce dernier en un entier.

Valeurs de retour

Retourne le signe de a : 1 si a est positif, -1 s'il est négatif et 0 si a est égal à zéro.

Exemples

Exemple #1 Exemple avec gmp_sign()

<?php
// positif
echo gmp_sign("500") . "\n";

// negatif
echo gmp_sign("-500") . "\n";

// zéro
echo gmp_sign("0") . "\n";
?>

L'exemple ci-dessus va afficher :

1
-1
0

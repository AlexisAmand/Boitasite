SoapClient::__getLastRequestHeaders

(PHP 5, PHP 7)

SoapClient::__getLastRequestHeaders — Retourne les en-têtes de la dernière requête SOAP

Description

public SoapClient::__getLastRequestHeaders ( void ) : string

Retourne les en-têtes de la dernière requête SOAP

Note:

Cette fonction n'est disponible que si l'objet SoapClient a été créé avec l'option trace à TRUE

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Les dernieres entêtes SOAP.

Exemples

Exemple #1 SoapClient::__getLastRequestHeaders() example

<?php
$client = SoapClient("some.wsdl", array('trace' => 1));
$result = $client->SomeFunction();
echo "REQUEST HEADERS:\n" . $client->__getLastRequestHeaders() . "\n";
?>

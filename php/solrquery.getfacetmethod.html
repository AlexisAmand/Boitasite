SolrQuery::getFacetMethod

(PECL solr >= 0.9.2)

SolrQuery::getFacetMethod — Récupère la valeur du paramètre facet.method

Description

public SolrQuery::getFacetMethod ([ string $field_override ] ) : string

Récupère la valeur du paramètre facet.method. La méthode accepte un champ à y substituer.

Liste de paramètres

field_override

Le nom du champ.

Valeurs de retour

Retourne une chaîne de caractères en cas de succès, et NULL s'il n'est pas défini.

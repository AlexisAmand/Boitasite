enchant_broker_free_dict

(PHP 5 >= 5.3.0, PHP 7, PECL enchant >= 0.1.0 )

enchant_broker_free_dict — Libère une ressource de dictionnaire

Description

enchant_broker_free_dict ( resource $dict ) : bool

Libère une ressource de dictionnaire.

Liste de paramètres

dict

Ressource de dictionnaire

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

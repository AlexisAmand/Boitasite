dba_sync

(PHP 4, PHP 5, PHP 7)

dba_sync — Synchronise une base de données DBA

Description

dba_sync ( resource $handle ) : bool

dba_sync() synchronise la base de données spécifiée par handle. Si accepté, cela va probablement lancer une opération de réécriture physique du fichier.

Liste de paramètres

handle

Le gestionnaire de base de données, retourné par la fonction dba_open() ou la fonction dba_popen().

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

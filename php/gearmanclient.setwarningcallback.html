GearmanClient::setWarningCallback

(PECL gearman >= 0.5.0)

GearmanClient::setWarningCallback — Définit une fonction de rappel lors de l'émission d'alerte depuis l'agent

Description

public GearmanClient::setWarningCallback ( callable $callback ) : bool

Définit une fonction à appeler lorsqu'un agent lance une alerte. La fonction de rappel doit accepter un seul argument, un objet GearmanTask.

Liste de paramètres

callback

Une fonction à appeler.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

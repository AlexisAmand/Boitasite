fann_create_train

(PECL fann >= 1.0.0)

fann_create_train — Crée une structure vide de données d'entrainement

Description

fann_create_train ( int $num_data , int $num_input , int $num_output ) : resource

Crée une structure vide de données d'entrainement.

Liste de paramètres

num_data

Le nombre de données d'entrainement

num_input

Le nombre d'entrées par données d'entrainement

num_output

Le nombre de sorties par données d'entrainement

Valeurs de retour

Retourne une ressource de données d'entrainement en cas de succès, ou FALSE si une erreur survient.

Notes

Note:

Cette fonction est maintenant disponible si l'extension fann a été compilée avec libfann >= 2.2.

  • fann_destroy_train() - Détruit les données d'entrainement

fann_get_rprop_delta_max

(PECL fann >= 1.0.0)

fann_get_rprop_delta_max — Retourne la taille maximale de l'étape

Description

fann_get_rprop_delta_max ( resource $ann ) : float

La taille maximale de l'étape est un nombre positif déterminant combien la taille de l'étape peut être au maximum.

Le delta maximal par défaut est de 50.0.

Liste de paramètres

ann

Ressource de réseau de neurones.

Valeurs de retour

La taille maximale de l'étape, ou FALSE si une erreur survient.

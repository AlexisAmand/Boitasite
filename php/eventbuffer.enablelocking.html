EventBuffer::enableLocking

(PECL event >= 1.2.6-beta)

EventBuffer::enableLocking —

Description

public EventBuffer::enableLocking ( void ) : void

Active le verrouillage de l'EventBuffer, lui permettant ainsi de pouvoir être utilisé de façon sécurisé par plusieurs threads à la fois. Lorsque le verrouillage est activé, le verrou sera placé lorsque les fonctions de rappel seront appelées. Ce comportement peut conduire à des verrous morts si vous n'êtes pas attentif.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

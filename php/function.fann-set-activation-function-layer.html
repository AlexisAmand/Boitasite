fann_set_activation_function_layer

(PECL fann >= 1.0.0)

fann_set_activation_function_layer — Défini la fonction d'activation pour tous les neurones de la couche spécifiée

Description

fann_set_activation_function_layer ( resource $ann , int $activation_function , int $layer ) : bool

Défini la fonction d'activation pour tous les neurones de la couche dont le numéro est layer, en comptant la couche d'entrée comme étant la couche numéro 0.

Il n'est pas possible de définir les fonctions d'activation pour les neurones de la couche d'entrée.

Liste de paramètres

ann

Ressource de réseau de neurones.

activation_function

La constante de fonctions d'activation.

layer

Le numéro de la couche.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

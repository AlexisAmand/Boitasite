enchant_broker_free

(PHP 5 >= 5.3.0, PHP 7, PECL enchant >= 0.1.0 )

enchant_broker_free — Libère la ressource de sponsor ainsi que ses dictionnaires

Description

enchant_broker_free ( resource $broker ) : bool

Libère une ressource de sponsor ainsi que tous ses dictionnaires.

Liste de paramètres

broker

Ressource de sponsor

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

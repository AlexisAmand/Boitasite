EvWatcher::feed

(PECL ev >= 0.2.0)

EvWatcher::feed — Alimente les revents fournis dans la boucle d'événements

Description

public EvWatcher::feed ( int $revents ) : void

Alimente les revents fournis dans la boucle d'événements, comme si l'événement spécifié était survenu dans l'observateur.

Liste de paramètres

revents

Masque de bits de l'observateur de réception d'événements.

Valeurs de retour

Aucune valeur n'est retournée.

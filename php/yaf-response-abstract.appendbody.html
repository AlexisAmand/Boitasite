Yaf_Response_Abstract::appendBody

(Yaf >=1.0.0)

Yaf_Response_Abstract::appendBody — Ajoute le corps de la réponse

Description

public Yaf_Response_Abstract::appendBody ( string $content [, string $key ] ) : bool

Ajoute le contenu dans un bloc de contenu existant.

Liste de paramètres

body

Le contenu.

key

La clé du contenu ; vous pouvez définir un contenu avec sa clé ; si vous n'en spécifiez pas, alors Yaf_Response_Abstract::DEFAULT_BODY sera utilisé.

Note:

ce paramètre a été introduit en version 2.2.0.

Valeurs de retour

Un booléen.

Exemples

Exemple #1 Exemple avec Yaf_Response_Abstract::appendBody()

<?php
$response = new Yaf_Response_Http();

$response->setBody("Hello")->prependBody(" World");

echo $response;
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

Hello World

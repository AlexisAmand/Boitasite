Installation

Pour utiliser cURL depuis les scripts PHP, vous devez compiler celui-ci avec l'option --with-curl[=DIR] où DIR est le chemin jusqu'au dossier contenant les dossiers lib et include. Dans le dossier include, il doit se trouver un dossier appelé curl, qui contient notamment les fichiers easy.h et curl.h. Il doit aussi se trouver un fichier nommé libcurl.a dans le dossier lib. Antérieur à PHP 5.5.0 il était possible de configurer PHP pour utiliser cURL pour les flux d'URL --with-curlwrappers.

Note: Note aux utilisateurs Win32
Afin d'activer ce module dans l'environnement Windows, libeay32.dll et ssleay32.dll, ou, depuis OpenSSL 1.1 libcrypto-*.dll et libssl-*.dll, doivent être présents dans votre PATH. libssh2.dll doit aussi être présent dans votre PATH. Vous n'avez pas besoin de libcurl.dll du site cURL.

tidy::cleanRepair

tidy_clean_repair

(PHP 5, PHP 7, PECL tidy >= 0.5.2)

tidy::cleanRepair -- tidy_clean_repair — Effectue les opérations de nettoyage et de réparation préparées pour un fichier HTML

Description

Style orienté objet

public tidy::cleanRepair ( void ) : bool

Style procédural

tidy_clean_repair ( tidy $object ) : bool

Nettoie et répare l'objet Tidy object passé en argument.

Liste de paramètres

object

L'objet Tidy

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Exemples

Exemple #1 Exemple avec tidy::cleanrepair()

<?php
$html = '<p>test</I>';

$tidy = tidy_parse_string($html);
$tidy->cleanRepair();

echo $tidy;
?>

L'exemple ci-dessus va afficher :

<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 3.2//EN">
<html>
 <head>
  <title></title>
 </head>
 <body>
  <p>test</p>
 </body>
</html>

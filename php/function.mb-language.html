mb_language

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

mb_language — Définit/Récupère le langage courant

Description

mb_language ( string $language ) : bool
mb_language ( void ) : string

Définit/Récupère le langage courant.

Liste de paramètres

language

Utilisé pour encoder les courriels électroniques. Les langages valides sont énuméré dans le tableau suivant. mb_send_mail() utilise cette option pour encoder les e-mails.

Language Charset Encoding Notes
Allemand/de ISO-8859-15 Quoted-Printable  
Anglais/en ISO-8859-1 Quoted-Printable  
Arménien/hy ArmSCII-8 Quoted-Printable  
Japonais/ja ISO-2022-JP BASE64  
Coréen/ko ISO-2022-KR BASE64  
neutre UTF-8 BASE64  
Russe/ru KOI8-R Quoted-Printable  
Turc/tr ISO-8859-9 Quoted-Printable Disponible à partir de PHP 5.2.0
Ukrainien/ua KOI8-U Quoted-Printable Disponible à partir de PHP 5.3.0
uni UTF-8 BASE64  
Chinois Simplifié/zh-cn HZ BASE64  
Chinois Traditionnel/zh-tw BIG-5 BASE64  

Valeurs de retour

Si language est fourni et language est valide, retourne TRUE. Sinon, retourne FALSE. Lorsque language est omit, retourne le nom du langage courant, en tant que chaîne de caractères.

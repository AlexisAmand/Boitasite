MongoDB::setSlaveOkay

(PECL mongo >=1.1.0)

MongoDB::setSlaveOkay — Modifie la configuration de slaveOkay pour cette base de données

Description

public MongoDB::setSlaveOkay ([ bool $ok = TRUE ] ) : bool

Voir la section requête de ce manuel pour plus d'informations sur la distribution de la lecture pour les secondaires.

Liste de paramètres

ok

Si la lecture doit être envoyée au second membre d'un jeu de réplication pour toutes les requêtes possibles utilisant l'instance MongoDB.

Valeurs de retour

Retourne la valeur de slaveOkay pour cette instance.

Historique

Version Description
1.2.11 Lance une alerte de niveau E_DEPRECATED lorsqu'utilisé.

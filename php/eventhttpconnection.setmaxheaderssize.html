EventHttpConnection::setMaxHeadersSize

(PECL event >= 1.2.6-beta)

EventHttpConnection::setMaxHeadersSize — Définit la taille maximale des en-têtes

Description

public EventHttpConnection::setMaxHeadersSize ( string $max_size ) : void

Définit la taille maximale des en-têtes pour la connexion.

Liste de paramètres

max_size

La taille maximale des en-têtes, en octets.

Valeurs de retour

Aucune valeur n'est retournée.

EvLoop::stop

(PECL ev >= 0.2.0)

EvLoop::stop — Stoppe la boucle d'événement

Description

public EvLoop::stop ([ int $how ] ) : void

Stoppe la boucle d'événement.

Liste de paramètres

how

Une constante Ev::BREAK_*.

Valeurs de retour

Aucune valeur n'est retournée.

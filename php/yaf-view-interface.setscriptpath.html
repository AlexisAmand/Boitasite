Yaf_View_Interface::setScriptPath

(Yaf >=1.0.0)

Yaf_View_Interface::setScriptPath — Le but de setScriptPath

Description

abstract public Yaf_View_Interface::setScriptPath ( string $template_dir ) : void

Définit le dossier de base contenant les templates. Habituellement, ce dossier est défini en utilisant la classe Yaf_Dispatcher.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

template_dir

Un chemin absolu vers le dossier contenant les templates ; Par défaut, Yaf_Dispatcher utilise application.directory . "/views".

Valeurs de retour

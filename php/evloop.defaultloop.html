EvLoop::defaultLoop

(PECL ev >= 0.2.0)

EvLoop::defaultLoop — Retourne ou crée la boucle d'événements par défaut

Description

public static EvLoop::defaultLoop ([ int $flags = Ev::FLAG_AUTO [, mixed $data = NULL [, float $io_interval = 0. [, float $timeout_interval = 0. ]]]] ) : EvLoop

Si la boucle d'événements par défaut n'est pas créée, la méthode EvLoop::defaultLoop() la crée avec les paramètres spécifiés. Sinon, elle retournera l'instance de l'objet correspondant précédemment créé en ignorant tous les paramètres.

Liste de paramètres

flags

Un des drapeaux de boucle d'évéments

data

Données personnalisées à associer avec la boucle.

io_collect_interval

Voir la fonction io_interval

timeout_collect_interval

Voir la fonction timeout_interval

Valeurs de retour

Retourne l'objet EvLoop en cas de succès.

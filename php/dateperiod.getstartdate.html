DatePeriod::getStartDate

(PHP 5 >= 5.6.5, PHP 7)

DatePeriod::getStartDate — Obtient la date de début

Description

Style orienté objet

public DatePeriod::getStartDate ( void ) : DateTimeInterface

Obtient la date de début d'une période.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Renvoi un object DateTimeImmutable quand la DatePeriod est initialisée avec un object DateTimeImmutable comme paramètre start.

Renvoi un object DateTime dans les autres cas.

Exemples

Exemple #1 Exemple DatePeriod::getStartDate()

<?php
$period = new DatePeriod('R7/2016-05-16T00:00:00Z/P1D');
$start = $period->getStartDate();
echo $start->format(DateTime::ISO8601);
?>

L'exemple ci-dessus va afficher :

2016-05-16T00:00:00+0000

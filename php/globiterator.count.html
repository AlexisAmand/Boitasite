GlobIterator::count

(PHP 5 >= 5.3.0, PHP 7)

GlobIterator::count — Lit le nombre de dossiers et fichiers

Description

public GlobIterator::count ( void ) : int

Lit le nombre de dossiers et fichiers trouvés par l'expression Glob.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Le nombre de dossiers et fichiers est retourné sous forme d'entier.

Exemples

Exemple #1 Exemple avec GlobIterator::count()

<?php
$iterator = new GlobIterator('*.xml');

printf("Trouvé %d élément(s)\r\n", $iterator->count());
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

Matched 8 item(s)

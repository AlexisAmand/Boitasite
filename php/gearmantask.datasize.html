GearmanTask::dataSize

(PECL gearman >= 0.5.0)

GearmanTask::dataSize — Récupère la taille des données retournées

Description

public GearmanTask::dataSize ( void ) : int

Retourne la taille des données retournées d'une tâche.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

La taille des données, ou FALSE s'il n'y a pas de données.

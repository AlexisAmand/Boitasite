odbc_field_len

(PHP 4, PHP 5, PHP 7)

odbc_field_len — Lit la longueur d'un champ

Description

odbc_field_len ( resource $result_id , int $field_number ) : int

Lit la longueur du champ identifié par son numéro.

Liste de paramètres

result_id

L'identifiant de résultat.

field_number

Le numéro du champ. La numérotation commence à 1.

Valeurs de retour

Retourne la longueur du champ ou FALSE si une erreur survient.

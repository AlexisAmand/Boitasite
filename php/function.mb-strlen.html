mb_strlen

(PHP 4 >= 4.0.6, PHP 5, PHP 7)

mb_strlen — Retourne la taille d'une chaîne

Description

mb_strlen ( string $str [, string $encoding = mb_internal_encoding() ] ) : int

Récupère la taille de la chaîne fournie.

Liste de paramètres

str

La chaîne à analyser.

encoding

Le paramètre encoding est l'encodage des caractères. S'il est omis, l'encodage de caractères interne sera utilisé.

Valeurs de retour

Retourne le nombre de caractères dans la chaîne str, avec l'encodage encoding. Un caractère multi-octets est alors compté pour 1.

Erreurs / Exceptions

Si l'encodage est inconnue, une erreur de niveau E_WARNING est générée.

GearmanWorker::setOptions

(PECL gearman >= 0.5.0)

GearmanWorker::setOptions — Définit les options de l'agent

Description

public GearmanWorker::setOptions ( int $option ) : bool

Définit une ou plusieurs options de l'agent.

Liste de paramètres

option

Les options à définir.

Valeurs de retour

Retourne toujours TRUE.

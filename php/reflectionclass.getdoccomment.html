ReflectionClass::getDocComment

(PHP 5, PHP 7)

ReflectionClass::getDocComment — Récupère les commentaires de documentation

Description

public ReflectionClass::getDocComment ( void ) : string

Récupère les commentaires de documentation depuis une classe. Les commentaires de documentation commencent avec /**. S'il y a plusieurs commentaires de documentation au-dessus de la définition de la classe, celle qui est la plus proche de la classe sera prise.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Le commentaire de documentation, s'il existe, FALSE sinon.

Exemples

Exemple #1 Exemple avec ReflectionClass::getDocComment()

<?php
/** 
* Une classe de test
*
* @param  foo bar
* @return baz
*/
class TestClass { }

$rc = new ReflectionClass('TestClass'); 
var_dump($rc->getDocComment())
?>

L'exemple ci-dessus va afficher :

string(55) "/** 
* Une classe de test
*
* @param  foo bar
* @return baz
*/"

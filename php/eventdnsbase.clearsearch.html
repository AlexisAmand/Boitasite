EventDnsBase::clearSearch

(PECL event >= 1.2.6-beta)

EventDnsBase::clearSearch — Supprime tous les suffixes courants de recherche

Description

public EventDnsBase::clearSearch ( void ) : void

Supprime tous les suffixes courants de recherche depuis la base DNS ; la méthode EventDnsBase::addSearch() ajoute un suffixe.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

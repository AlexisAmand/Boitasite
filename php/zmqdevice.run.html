ZMQDevice::run — Exécute le nouveau périphérique

Description

public ZMQDevice::run ( void ) : void

Exécute le périphérique.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

L'appel à cette méthode est bloquante tant que le périphérique est en fonctionnement. Il n'est pas recommandé d'utiliser les périphériques depuis des scripts interactifs. En cas d'échec, cette méthode a lancé une exception ZMQDeviceException.

ZMQSocket::recvMulti

(PECL zmq >= 0.8.0)

ZMQSocket::recvMulti — Recoit un message multipart

Description

public ZMQSocket::recvMulti ([ int $mode = 0 ] ) : array

Recoit un tableau contenant le message multipart depuis un socket. Par défaut, la réception va être bloquante tant qu'un message est disponible à moins que ZMQ::MODE_NOBLOCK ne soit utilisé.

Liste de paramètres

mode

Drapeaux de mode pour recevoir les messages multipart, ou faire que l'opération ne soit pas bloquante. Voir les constantes ZMQ::MODE_*.

Valeurs de retour

Retourne le tableau contenant les parties du message. Lance une exception ZMQSocketException en cas d'erreur. Si ZMQ::MODE_NOBLOCK est utilisé et que l'opération est bloquante, FALSE sera retourné.

gmp_random_range

(PHP 5 >= 5.6.3, PHP 7)

gmp_random_range — Génère un nombre aléatoire

Description

gmp_random_range ( GMP $min , GMP $max ) : GMP

Génère un nombre aléatoire. Le nombre sera dans l'intervalle min et max.

min et max peuvent être tous les deux négatifs, mais min doit toujours être plus petit que max.

Liste de paramètres

min

Un nombre GMP représentant la borne inférieure pour le nombre aléatoire.

max

Un nombre GMP représentant la borne supérieure pour le nombre aléatoire.

Valeurs de retour

Un nombre GMP aléatoire.

Exemples

Exemple #1 Exemple avec gmp_random_range()

<?php
$rand1 = gmp_random_range(0, 100);    // nombre aléatoire entre 0 et 100
$rand2 = gmp_random_range(-100, -10); // nombre aléatoire entre -100 et -10

echo gmp_strval($rand1) . "\n";
echo gmp_strval($rand2) . "\n";
?>

L'exemple ci-dessus va afficher :

42
-67

eio_set_max_parallel

(PECL eio >= 0.0.1dev)

eio_set_max_parallel — Définit le nombre maximal de threads en parallèle

Description

eio_set_max_parallel ( int $nthreads ) : void

Liste de paramètres

nthreads

Nombre de threads en parallèle

Valeurs de retour

Aucune valeur n'est retournée.

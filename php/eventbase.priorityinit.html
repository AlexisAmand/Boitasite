EventBase::priorityInit

(PECL event >= 1.2.6-beta)

EventBase::priorityInit — Défini le nombre de propriétés par événement de base

Description

public EventBase::priorityInit ( int $n_priorities ) : bool

Défini le nombre de propriétés par événement de base.

Liste de paramètres

n_priorities

Le nombre de propriétés par événement de base.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

La classe MongoInt64

(PECL mongo >=1.0.9)

Avertissement

Cette extension, qui définie cette classe est obsolète. Veuillez utiliser l'extension MongoDB à la place. Il n'y a pas d'équivalent à cette classe dans la nouvelle extension.

A la place, la nouvelle extension choisit le type approprié de base de données suivant une valeur entière.

Introduction

la classe peut être utilisée pour sauvegarder les entiers 64 bits dans la base de données sur un système 32 bits.

Synopsis de la classe

/* Champs */

}

Fields

value
C'est la valeur textuelle du nombre 64-bit. Actuellement, la 123ème valeur sera "123".

Nouveaux paramètres

Certaines fonctions ont reçu de nouveaux paramètres optionnels en PHP 5.2.x :

Coeur de PHP :

  • htmlentities() : ajout du paramètre double_encode en PHP 5.2.3.
  • htmlspecialchars() : ajout du paramètre double_encode en PHP 5.2.3.
  • base64_decode() : ajout du paramètre strict
  • setcookie() : ajout du paramètre httponly
  • setrawcookie() : ajout du paramètre httponly
  • session_set_cookie_params() : ajout du paramètre httponly
  • memory_get_usage() : ajout du paramètre real_usage
  • get_loaded_extensions() : ajout du paramètre zend_extensions en PHP 5.2.4

curl:

  • curl_multi_info_read() : ajout du paramètre msgs_in_queue

datetime

  • date() : ajout du caractère de format "u" (millisecondes) en PHP 5.2.2

imap:

  • imap_open() : ajout du paramètre n_retries
  • imap_reopen() : ajout du paramètre n_retries

mbstring:

  • mb_strrpos() : ajout du paramètre offset
    Avertissement

    Le paramètre offset a été ajouté à la place du paramètre encoding. La compatibilité ascendante a été assurée en permettant au paramètre encoding d'être précisé comme troisième paramètre. L'utilisation du mode de compatibilité n'est pas recommandé, car il sera supprimé dans une version ultérieure.

ming:

  • SWFMovie::streamMP3() : ajout du paramètre skip en PHP 5.2.1

openssl:

  • openssl_verify() : ajout du paramètre signature_algo

pgsql:

  • pg_escape_bytea() : ajout du paramètre connection
  • pg_escape_string() : ajout du paramètre connection

simplexml:

  • SimpleXMLElement::__construct() : ajout du paramètre is_prefix
  • SimpleXMLElement::attributes() : ajout du paramètre is_prefix
  • SimpleXMLElement::children() : ajout du paramètre is_prefix
  • simplexml_load_file() : ajout du paramètre is_prefix
  • simplexml_load_string() : ajout du paramètre is_prefix

spl:

  • array iterator_to_array(Traversable it [, bool use_keys = true]) : ajout du paramètre use_keys en PHP 5.2.1

xmlreader:

  • XMLReader::open() : ajout du paramètre encoding et options
  • XMLReader::XML() : ajout du paramètre encoding et du paramètre options

XMLWriter:

  • XMLWriter::WriteElement() : le paramètre content est devenu optionnel en PHP 5.2.3
  • XMLWriter::WriteElementNs() : le paramètre content est devenu optionnel en PHP 5.2.3

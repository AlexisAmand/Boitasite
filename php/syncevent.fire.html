SyncEvent::fire

(PECL sync >= 1.0.0)

SyncEvent::fire — Lance/défini l'événement

Description

public SyncEvent::fire ( void ) : bool

Lance/défini un objet SyncEvent. Conserve plusieurs threads en attente si l'objet d'événement a été créé avec une valeur manuelle à TRUE.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

TRUE si l'événement a été lancé, FALSE sinon.

Exemples

Exemple #1 Exemple avec SyncEvent::fire()

<?php
// Dans une application web :
$event = new SyncEvent("GetAppReport");
$event->fire();

// Dans un cron :
$event = new SyncEvent("GetAppReport");
$event->wait();
?>

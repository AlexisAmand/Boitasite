wddx_deserialize

(PHP 4, PHP 5, PHP 7)

wddx_deserialize — Délinéarise un paquet WDDX

Avertissement

Cette fonction est a été SUPPRIMÉE à partir de PHP 7.4.0.

Description

wddx_deserialize ( string $packet ) : mixed

Délinéarise un paquet packet WDDX.

Avertissement

Ne passez pas d'entrée utilisateur non vérifiée à wddx_deserialize(). La délinéarisation peut faire que le code soit chargé, et exécuté lors de l'instanciation et l'autochargement de l'objet, et un utilisateur mal intentionné peut être capable d'y loger un exploit. Utilisez une format, sûr, d'échange de données standardisé comme JSON (à l'aide de json_decode() et json_encode()) si vous devez passer des données linéarisé à l'utilisateur.

Liste de paramètres

packet

Un paquet WDDX, sous la forme d'une chaîne de caractères ou d'un flux.

Valeurs de retour

Retourne la valeur délinéarisée, pouvant être une chaîne de caractères, un nombre ou un tableau. Notez que les structures sont délinéarisées dans des tableaux associatifs.

(PECL gearman >= 0.6.0)

GearmanWorker::removeOptions — Supprime des options de l'agent

Description

public GearmanWorker::removeOptions ( int $option ) : bool

Supprime une ou plusieurs options de l'agent.

Liste de paramètres

option

Les options à supprimer

Valeurs de retour

Retourne toujours TRUE.

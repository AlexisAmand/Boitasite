imap_unsubscribe

(PHP 4, PHP 5, PHP 7)

imap_unsubscribe — Termine la souscription à une boîte aux lettres

Description

imap_unsubscribe ( resource $imap_stream , string $mailbox ) : bool

Termine la souscription à la boîte aux lettres mailbox.

Liste de paramètres

imap_stream

Un flux IMAP retourné par la fonction imap_open().

mailbox

Le nom de la boîte aux lettres, voir la documentation de la fonction imap_open() pour plus de détails.

Avertissement

Passer des données qui ne sont pas digne de confiance à ce paramètre est dangereux, sauf si, imap.enable_insecure_rsh est désactivé.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

mysqli_connect

(PHP 5, PHP 7)

mysqli_connect — Alias de mysqli::__construct()

Description

Cette fonction est un alias de : mysqli::__construct()

Bien que la documentation de mysqli::__construct() inclus aussi des exemples procéduraux qui utilisent la fonction mysqli_connect(), voici un court exemple :

Exemples

Exemple #1 Exemple avec mysqli_connect()

<?php
$link = mysqli_connect("127.0.0.1", "my_user", "my_password", "my_db");

if (!$link) {
    echo "Erreur : Impossible de se connecter à MySQL." . PHP_EOL;
    echo "Errno de débogage : " . mysqli_connect_errno() . PHP_EOL;
    echo "Erreur de débogage : " . mysqli_connect_error() . PHP_EOL;
    exit;
}

echo "Succès : Une connexion correcte à MySQL a été faite! La base de donnée my_db est géniale." . PHP_EOL;
echo "Information d'hôte : " . mysqli_get_host_info($link) . PHP_EOL;

mysqli_close($link);
?>

Les exemples ci-dessus vont afficher quelque chose de similaire à :

Succès : Une connexion correcte à MySQL a été faite! La base de donnée my_db est génial.
Information d'hôte : localhost via TCP/IP

imap_mutf7_to_utf8

(PHP 5 >= 5.3.0, PHP 7)

imap_mutf7_to_utf8 — Décode une chaîne de charactère UTF-7 modifié en UTF-8

Description

imap_mutf7_to_utf8 ( string $in ) : string

Décode une chaîne de caractères UTF-7 modifié (comme spécifié dans la RFC 2060, section 5.1.3) en UTF-8.

Note:

Cette fonction est seulement disponible, si libcclient exporte utf8_to_mutf7().

Liste de paramètres

in

Une chaîne de caractères encodé en UTF-7 modifié.

Valeurs de retour

Retourne in convertit en UTF-8, ou FALSE si une erreur survient.

PDF_add_pdflink

(PHP 4, PECL pdflib >= 1.0.0)

PDF_add_pdflink — [Obsolète] Ajoute une annotation sur un lien vers un fichier dans la page PDF courante

Description

PDF_add_pdflink ( resource $pdfdoc , float $bottom_left_x , float $bottom_left_y , float $up_right_x , float $up_right_y , string $filename , int $page , string $dest ) : bool

PDF_add_pdflink() ajoute une annotation sur un lien vers un fichier dans la page courante. Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Cette fonction est obsolète depuis la version 6 de la PDFlib, utilisez la fonction PDF_create_action() avec le paramètre type=GoToR et la fonction PDF_create_annotation() avec le paramètre type=Link à la place.

Event::__construct

(PECL event >= 1.2.6-beta)

Event::__construct — Construit un objet Event

Description

public Event::__construct ( EventBase $base , mixed $fd , int $what , callable $cb [, mixed $arg = NULL ] )

Construit un objet Event.

Liste de paramètres

base

La base d'événement à associer.

fd

Ressource de flux, ressource de socket, ou descripteur numérique de fichier. Pour les événements timer, passez comme valeur -1. Pour les événements de type signal, passez le numéro du signal, i.e. SIGHUP.

what

Drapeaux d'événements. Voir les drapeaux d'événements pour plus de détails.

cb

La fonction de rappel de l'événement. Voir les fonctions de rappel d'événements pour plus de détails.

arg

Données personnalisées. Si spécifié, elles seront passées à la fonction de rappel lorsque l'événement aura lancé les triggers.

Valeurs de retour

Retourne l'objet Event.

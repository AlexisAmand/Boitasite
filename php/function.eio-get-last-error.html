eio_get_last_error

(PECL eio >= 1.0.0)

eio_get_last_error — Retourne une chaîne décrivant la dernière erreur associée avec la ressource demandée

Description

eio_get_last_error ( resource $req ) : string

eio_get_last_error() retourne une chaîn edécrivant la dernière erreur associée avec le paramètre req.

Liste de paramètres

req

La ressource demandée

Valeurs de retour

eio_get_last_error() retourne une chaîne décrivant la dernière erreur avec la ressource demandée spécifiée par le paramètre req.

Avertissement

Cette fonction est EXPERIMENTALE. Le comportement de cette fonction, son nom, et toute la documentation autour de cette fonction peut changer sans préavis dans une prochaine version de PHP. Cette fonction doit être utilisée à vos risques et périls.

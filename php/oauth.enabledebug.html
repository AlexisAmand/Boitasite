OAuth::enableDebug

(PECL OAuth >= 0.99.3)

OAuth::enableDebug — Active les messages de débogage

Description

public OAuth::enableDebug ( void ) : bool

Active les messages de débogage, pratiques en développement, mais inutiles en production.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

TRUE

Historique

Version Description
0.99.8 La propriété debug a été ajoutée

rrd_version

(PECL rrd >= 1.0.0)

rrd_version — Récupère les informations de version de la bibliothèque rrdtool

Description

rrd_version ( void ) : string

Récupère les informations de version de la bibliothèque rrdtool.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Une chaîne de caractères contenant le numéro de version de la bibliothèque rrdtool i.e. "1.4.3".

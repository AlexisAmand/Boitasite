SolrUtils::escapeQueryChars

(PECL solr >= 0.9.2)

SolrUtils::escapeQueryChars — Protège une requête

Description

public static SolrUtils::escapeQueryChars ( string $str ) : string

Le support Lucene protège les caractères spéciaux faisant partis de la syntaxe de la requête.

La liste actuelle des caractères spéciaux est : 

+ - && || ! ( ) { } [ ] ^ " ~ * ? : \ /

Ces caractères font partis de la syntaxe de la requête et doivent être échappés.

Liste de paramètres

str

Chaîne à protéger.

Valeurs de retour

Retourne la chaîne protégée ou FALSE si une erreur survient.

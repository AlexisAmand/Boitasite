ldap_control_paged_result_response

(PHP 5 >= 5.4.0, PHP 7)

ldap_control_paged_result_response — Récupère le cookie de pagination LDAP

Avertissement

Cette fonction est OBSOLÈTE à partir de PHP 7.4.0. Dépendre de cette fonction est fortement déconseillé.

Description

ldap_control_paged_result_response ( resource $link , resource $result [, string &$cookie [, int &$estimated ]] ) : bool

Récupère les informations de pagination envoyées par le serveur.

Liste de paramètres

link

Un identifiant de lien LDAP, retourné par la fonction ldap_connect().

result

cookie

Une structure opaque envoyée par le serveur.

estimated

Le nombre estimé d'entrées à récupérer.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Historique

Version Description
7.4.0 Cette fonction est devenue obsolète.

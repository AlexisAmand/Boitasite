EventListener::getSocketName

(PECL event >= 1.5.0)

EventListener::getSocketName — Récupère l'adresse courant pour laquelle le socket d'écoute est lié

Description

public static EventListener::getSocketName ( string &$address [, mixed &$port ] ) : bool

Récupère l'adresse courant pour laquelle le socket d'écoute est lié.

Liste de paramètres

address

Paramètre de sortie. L'adresse IP suivant la famille d'adresse du socket.

port

Paramètre de sortie. Le port sur lequel le socket est lié.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

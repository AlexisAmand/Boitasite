pow

(PHP 4, PHP 5, PHP 7)

pow — Expression exponentielle

Description

pow ( number $base , number $exp ) : number

Retourne base élevé à la puissance exp.

Note:

Depuis PHP 5.6, vous pourriez préférer utiliser l'opérateur **.

Liste de paramètres

base

La base à utiliser

exp

L'exponentielle

Valeurs de retour

base élevé à la puissance exp. Si les arguments ne sont pas des entiers négatifs, et que le résultat peut être représenté sous la forme d'un entier, le résultat sera retourné sous la forme d'un entier, sinon, si il sera retourné sous la forme d'un nombre à virgule flottante.

Exemples

Exemple #1 Exemple avec pow()

<?php

var_dump(pow(2, 8)); // int(256)
echo pow(-1, 20); // 1
echo pow(0, 0); // 1
echo pow(10, -1); // 0.1

echo pow(-1, 5.5); // PHP >=5.2.2: NAN
echo pow(-1, 5.5); // PHP <5.2.2: -NAN
?>

Notes

Note:

Cette fonction convertira toutes les entrées en un nombre, y compris pour les valeurs non-scalaires, ce qui peut amener à des résultats imprévisibles.

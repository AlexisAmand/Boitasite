trader_stochrsi

(PECL trader >= 0.2.0)

trader_stochrsi — Index stochastique relatif fort

Description

trader_stochrsi ( array $real [, int $timePeriod [, int $fastK_Period [, int $fastD_Period [, int $fastD_MAType ]]]] ) : array

Liste de paramètres

real

Tableau de valeurs réelles.

timePeriod

Nombre de période. Intervalle valide : 2 à 100000.

fastK_Period

Période de temps pour construire la ligne Fast-K. Intervalle valide : 1 à 100000.

fastD_Period

Lissage pour réaliser la ligne Fast-D. Intervalle valide : 1 à 100000, habituellement défini à 3.

fastD_MAType

Type de moyenne mobile pour Fast-D. Une constante de la série TRADER_MA_TYPE_* doit être utilisée.

Valeurs de retour

Retourne un tableau contenant les données calculées, ou FALSE si une erreur survient.

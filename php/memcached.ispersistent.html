Memcached::isPersistent

(PECL memcached >= 2.0.0)

Memcached::isPersistent — Vérifie si une connexion persistente vers memcache est en cours d'utilisation

Description

public Memcached::isPersistent ( void ) : bool

Memcached::isPersistent() vérifie si les connexions aux serveurs memcache sont des connexions persistentes.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE si l'instance Memcache utilise une connexion persistente, FALSE sinon.

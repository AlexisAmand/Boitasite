phpdbg_exec

(PHP 5 >= 5.6.0, PHP 7)

phpdbg_exec — Tente de définir le contexte d'exécution

Description

phpdbg_exec ([ string $context ] ) : mixed

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

context

Valeurs de retour

Si le contexte d'exécution a été défini avec succès, ce contexte sera retourné. Si le contexte d'exécution n'a pas été défini, TRUE sera retourné. Si une erreur est survenue pendant la définition du contexte, FALSE sera retourné et un erreur de niveau E_WARNING sera émise.

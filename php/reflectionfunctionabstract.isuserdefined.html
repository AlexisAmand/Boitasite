ReflectionFunctionAbstract::isUserDefined

(PHP 5 >= 5.2.0, PHP 7)

ReflectionFunctionAbstract::isUserDefined — Vérifie si la fonction est définie dans l'espace utilisateur

Description

public ReflectionFunctionAbstract::isUserDefined ( void ) : bool

Vérifie si la fonction est définie dans l'espace utilisateur, au lieu d'être une fonction interne.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

TRUE si la fonction a été définie dans l'espace utilisateur, FALSE sinon.

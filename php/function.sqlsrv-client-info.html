sqlsrv_client_info — Retourne les informations sur le client ainsi que sur la connexion spécifiée

Description

sqlsrv_client_info ( resource $conn ) : array

Retourne des informations sur le client ainsi que sur la connexion spécifiée.

Liste de paramètres

conn

La connexion sur laquelle on souhaite des informations.

Valeurs de retour

Retourne un tableau associatif dont les clés sont décrites dans la table ci-dessous. Retourne FALSE sinon.

Tableau retourné par sqlsrv_client_info
Clé Description
DriverDllName SQLNCLI10.DLL
DriverODBCVer version ODBC (xx.yy)
DriverVer version de la bibliothèque DLL native client du serveur (10.5.xxx)
ExtensionVer version de la bibliothèque php_sqlsrv.dll (2.0.xxx.x)

Exemples

Exemple #1 Exemple avec sqlsrv_client_info()

<?php
$serverName = "serverName\sqlexpress";
$connOptions = array("UID"=>"username", "PWD"=>"password");
$conn = sqlsrv_connect( $serverName, $connOptions );

if( $conn === false ) {
    die( print_r( sqlsrv_errors(), true));
}

if( $client_info = sqlsrv_client_info( $conn)) {
    foreach( $client_info as $key => $value) {
        echo $key.": ".$value."<br />";
    }
} else {
    echo "Erreur lors de la récupération des informations clientes.<br />";
}
?>

eio_set_max_poll_time

(PECL eio >= 0.0.1dev)

eio_set_max_poll_time — Définit la durée maximale de la file d'attente

Description

eio_set_max_poll_time ( float $nseconds ) : void

La file d'attente s'interrompt si elle prend plus de nseconds secondes.

Liste de paramètres

nseconds

Nombre de secondes

Valeurs de retour

Aucune valeur n'est retournée.

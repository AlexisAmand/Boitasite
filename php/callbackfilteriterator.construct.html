CallbackFilterIterator::__construct

(PHP 5 >= 5.4.0, PHP 7)

CallbackFilterIterator::__construct — Crée un itérateur filtré depuis un autre itérateur

Description

public CallbackFilterIterator::__construct ( Iterator $iterator , callable $callback )

Crée un itérateur filtré en utilisant la fonction de rappel callback pour déterminer quels éléments sont acceptés ou rejetés.

Liste de paramètres

iterator

L'itérateur à filtrer.

callback

La fonction de rappel, qui doit retourner TRUE pour accepter l'élément courant, FALSE sinon. Voir aussi les exemples.

Peut être n'importe quelle valeur vaide de type callable.

Valeurs de retour

Aucune valeur n'est retournée.

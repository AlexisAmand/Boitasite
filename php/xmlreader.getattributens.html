XMLReader::getAttributeNs

(PHP 5 >= 5.1.0, PHP 7)

XMLReader::getAttributeNs — Récupère la valeur d'un attribut par nom local et URI

Description

public XMLReader::getAttributeNs ( string $localName , string $namespaceURI ) : string

Retourne la valeur d'un attribut par nom et espace de nom URI ou une chaîne de caractères vide si l'attribut n'existe pas ou n'est pas positionné sur le noeud.

Liste de paramètres

localName

Le nom local.

namespaceURI

L'espace de nom URI.

Valeurs de retour

La valeur de l'attribut, une chaîne de caractères vide (avant PHP 5.6), ou NULL (depuis PHP 5.6) si aucun attribut avec le localName et namespaceURI donnés n'est trouvé ou positionné sur l'élément.

Historique

Version Description
5.6.0 XMLReader::getAttributeNS() retourne maintenant NULL si l'attribut n'existe pas.

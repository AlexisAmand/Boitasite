SolrQuery::getFacetFields

(PECL solr >= 0.9.2)

SolrQuery::getFacetFields — Récupère toutes les facettes des champs

Description

public SolrQuery::getFacetFields ( void ) : array

Récupère toutes les facettes des champs.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un tableau contenant tous les champs et NULL si aucun n'a été défini.

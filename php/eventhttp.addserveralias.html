EventHttp::addServerAlias

(PECL event >= 1.4.0-beta)

EventHttp::addServerAlias — Ajoute un alias du serveur pour l'objet serveur HTTP

Description

public EventHttp::addServerAlias ( string $alias ) : bool

Ajoute un alias du serveur pour l'objet serveur HTTP.

Liste de paramètres

alias

L'alias à ajouter.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

Exemples

Exemple #1 Exemple avec EventHttp::addServerAlias()

<?php
$base = new EventBase();
$http = new EventHttp($base);

$socket = socket_create(AF_INET, SOCK_STREAM, SOL_TCP);

if (!$http->bind("127.0.0.1", 8088)) {
    exit("bind(1) a échoué\n");
};

if (!$http->addServerAlias("local.net")) {
    exit("Impossible d'ajouter l'alias du serveur\n");
}

$http->setCallback("/about", function($req) {
    echo "URI : ", $req->getUri(), PHP_EOL;
    $req->sendReply(200, "OK");
});
$base->dispatch();
?>

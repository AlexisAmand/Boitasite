Pré-requis

Ce module utilise les fonctions » zlib créées par Jean-loup Gailly et Mark Adler. Vous devez utiliser la bibliothèque zlib version >= 1.0.9 avec ce module. Depuis PHP 5.4.0 vous devez utiliser la bibliothèque zlib version >= 1.2.0.4.

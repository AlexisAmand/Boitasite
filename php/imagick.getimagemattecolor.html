Imagick::getImageMatteColor

(PECL imagick 2.0.0)

Imagick::getImageMatteColor — Retourne la couleur mate de l'image

Avertissement

Cette fonction est OBSOLÈTE à partir de Imagick 3.4.4. Dépendre de cette fonctionnalité est fortement déconseillé.

Description

Imagick::getImageMatteColor ( void ) : ImagickPixel

Retourne la couleur mate de l'image.

Valeurs de retour

Retourne un objet ImagickPixel en cas de succès.

Erreurs / Exceptions

Lance une exception ImagickException si une erreur survient.

intdiv

(PHP 7)

intdiv — Division d'Entier

Description

intdiv ( int $dividend , int $divisor ) : int

Retourne le quotient entier de la divison de dividend par divisor.

Liste de paramètres

dividend

Nombre à diviser.

divisor

Nombre qui divise dividend.

Valeurs de retour

Le quotient entier de la division de dividend par divisor.

Erreurs / Exceptions

Si divisor est 0, une exception DivisionByZeroError sera émise. Si dividend est PHP_INT_MIN et que divisor est -1, dans ce cas là une exception ArithmeticError est émise.

Exemples

Exemple #1 Exemple de intdiv()

<?php
var_dump(intdiv(3, 2));
var_dump(intdiv(-3, 2));
var_dump(intdiv(3, -2));
var_dump(intdiv(-3, -2));
var_dump(intdiv(PHP_INT_MAX, PHP_INT_MAX));
var_dump(intdiv(PHP_INT_MIN, PHP_INT_MIN));
var_dump(intdiv(PHP_INT_MIN, -1));
var_dump(intdiv(1, 0));
?> 
int(1)
int(-1)
int(-1)
int(1)
int(1)
int(1)

curl_multi_select

(PHP 5, PHP 7)

curl_multi_select — Attend une activité sur n'importe quelle connexion curl_multi

Description

curl_multi_select ( resource $mh [, float $timeout = 1.0 ] ) : int

Patiente tant qu'il y a de l'activité sur une des connexions curl_multi.

Liste de paramètres

mh

Un gestionnaire cURL multiple retourné par la fonction curl_multi_init().

timeout

Durée maximale, en secondes, à attendre une réponse.

Valeurs de retour

Retourne le nombre de sockets contenues dans le jeu de sockets, en cas de succès. Cela peut être 0 s'il n'y a eu aucune activité sur aucun des sockets. En cas d'échec, la fonction retournera -1 sur un échec de sélection ou lors d'un dépassement du délai d'attente (depuis l'appel du sous système de sélection).

SplPriorityQueue::isCorrupted

(PHP 5 >= 5.3.0, PHP 7)

SplPriorityQueue::isCorrupted — Informe si la file prioritaire est dans un état corrompue

Description

public SplPriorityQueue::isCorrupted ( void ) : bool

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE si la file prioritaire est corrompue, FALSE sinon.

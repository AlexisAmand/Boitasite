La classe DOMNodeList

(PHP 5, PHP 7)

Synopsis de la classe

DOMNodeList implements Traversable , Countable {
/* Propriétés */
public readonly int $length ;
/* Méthodes */
public count ( void ) : int
item ( int $index ) : DOMNode
}

Propriétés

length

Le nombre de noeuds dans la liste. L'intervalle valide des indices des noeuds fils est 0 à length - 1, inclus.

Historique

Version Description
7.2.0 L'interface Countable est implémentée et la valeur de la propriété length est retournée.

GearmanJob::functionName

(PECL gearman >= 0.5.0)

GearmanJob::functionName — Récupère le nom de la fonction

Description

public GearmanJob::functionName ( void ) : string

Retourne le nom de la fonction pour ce travail. C'est la fonction exécutée par l'agent pour exécuter le travail.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Le nom de la fonction.

EventHttpRequest::getInputHeaders

(PECL event >= 1.4.0-beta)

EventHttpRequest::getInputHeaders — Retourne le tableau associatif contenant les en-têtes d'entrée

Description

public EventHttpRequest::getInputHeaders ( void ) : array

Retourne le tableau associatif contenant les en-têtes d'entrée.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne le tableau associatif contenant les en-têtes d'entrée.

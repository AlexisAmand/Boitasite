Constantes pré-définies

Ces constantes sont définies par cette extension, et ne sont disponibles que si cette extension a été compilée avec PHP, ou bien chargée au moment de l'exécution.

CAL_EASTER_DEFAULT (entier)
Pour easter_days() : calcule Pâques pour les années avant 1753 selon le calendrier julien, et pour les années ultérieures selon le calendrier grégorien.
CAL_EASTER_ROMAN (entier)
Pour easter_days() : calcule Pâques pour les années avant 1583 selon le calendrier julien, et pour les années ultérieures selon le calendrier grégorien.
CAL_EASTER_ALWAYS_GREGORIAN (entier)
Pour easter_days() : calcule Pâques selon le calendrier grégorien proleptique.
CAL_EASTER_ALWAYS_JULIAN (entier)
Pour easter_days() : calcule Pâques selon le calendrier julien.
CAL_GREGORIAN (entier)
Pour cal_days_in_month(), cal_from_jd(), cal_info() et cal_to_jd() : utilise le calendrier grégorien proleptique.
CAL_JULIAN (entier)
Pour cal_days_in_month(), cal_from_jd(), cal_info() et cal_to_jd() : utilise le calendrier julien.
CAL_JEWISH (entier)
Pour cal_days_in_month(), cal_from_jd(), cal_info() et cal_to_jd() : utilise le calendrier hébraïque.
CAL_FRENCH (entier)
Pour cal_days_in_month(), cal_from_jd(), cal_info() et cal_to_jd() : utilise le calendrier républicain.
CAL_NUM_CALS (entier)
Le nombre de calendriers disponible.
CAL_JEWISH_ADD_ALAFIM_GERESH (entier)
Pour jdtojewish() : ajoute un symbole geresh (qui ressemble à une apostrophe) en tant que séparateur de milliers au numéro de l'année.
CAL_JEWISH_ADD_ALAFIM (entier)
Pour jdtojewish() : ajoute le mot alafim en tant que séparateur de milliers au numéro de l'année.
CAL_JEWISH_ADD_GERESHAYIM (entier)
Pour jdtojewish() : ajoute un symbole gershayim (qui ressemble à un guillemet) avant la lettre finale des nombre de jour et d'année.
CAL_DOW_DAYNO (entier)
Pour jddayofweek() : le jour de la semaine en tant que entier, où 0 signifie Dimanche et 6 signifie Samedi.
CAL_DOW_SHORT (entier)
Pour jddayofweek() : le nom anglais abrégé du jour de la semaine.
CAL_DOW_LONG (entier)
Pour jddayofweek() : le nom anglais du jour de la semaine.
CAL_MONTH_GREGORIAN_SHORT (entier)
Pour jdmonthname() : le nom de mois grégorien abrégé.
CAL_MONTH_GREGORIAN_LONG (entier)
Pour jdmonthname() : le nom de mois grégorien.
CAL_MONTH_JULIAN_SHORT (entier)
Pour jdmonthname() : le nom de mois julien abrégé.
CAL_MONTH_JULIAN_LONG (entier)
Pour jdmonthname() : le nom de mois julien.
CAL_MONTH_JEWISH (entier)
Pour jdmonthname() : le nom de mois hébraïque.
CAL_MONTH_FRENCH (entier)
Pour jdmonthname() : le nom de mois républicain.

Lancer les tests du pilote

Le paquet PECL n'est pas inclus dans les tests, mais sont disponibles sur » Github.

Les tests .phpt

Les tests .phpt sont la façon standard de tester des extensions PHP. Après compilation du driver MongoDG avec la commande "make", vous pouvez ensuite utiliser la commande "make test" pour exécuter les tests. Ces tests nécessitent le bon fonctionnement de MongoDB et donc, vous devez configurer la suite de test en tant que tel.

Reportez-vous à » GitHub pour plus d'informations à propos de ces tests.

Rapport des erreurs

Veuillez rapporter tous les échecs ou les erreurs dans » l'outil de suivi des bogues. Il se peut que des tests ne soient pas exécutés, c'est normal et vous pouvez les ignorer.

Les nouveaux tests sont toujours les bienvenus ! Vous pouvez contribuer en proposant des nouveaux tests sur toutes les fonctionnalités.

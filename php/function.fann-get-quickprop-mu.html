fann_get_quickprop_mu

(PECL fann >= 1.0.0)

fann_get_quickprop_mu — Retourne le facteur mu

Description

fann_get_quickprop_mu ( resource $ann ) : float

Le facteur mu est utilisé pour accroître et décroitre la taille de l'étape durant l'entrainement quickprop. Le facteur mu doit toujours être inférieur à 1, sachant que sinon, il va décroitre la taille de l'étape alors qu'il est supposé l'accroitre.

Le facteur mu par défaut est 1.75.

Liste de paramètres

ann

Ressource de réseau de neurones.

Valeurs de retour

Le facteur mu, ou FALSE si une erreur survient.

posix_ttyname

(PHP 4, PHP 5, PHP 7)

posix_ttyname — Retourne le nom de device du terminal

Description

posix_ttyname ( mixed $fd ) : string

Retourne une chaîne de caractères pour le chemin absolu du terminal courant qui est ouvert sur le pointeur de fichier. fd.

Liste de paramètres

fd

Le descripteur de fichier, qui doit être soit une ressource de fichier, soit un entier. Un entier est supposé être un descripteur de fichier qui peut être passé directement à l'appel système sous-jacent.

Dans tous les autres cas, vous devez fournir une ressource de fichier.

Valeurs de retour

En cas de succès, retourne une chaîne de caractères correspondant au chemin absolu de fd. En cas d'erreur, retourne FALSE.

Event::setTimer

(PECL event >= 1.2.6-beta)

Event::setTimer — Re-configure un événement timer

Description

public Event::setTimer ( EventBase $base , callable $cb [, mixed $arg ] ) : bool

Re-configure un événement timer. Notez que cette méthode n'appelle pas la fonction obsolète event_set de libevent. Elle appelle plutôt la fonction event_assign.

Liste de paramètres

base

La base d'événement à associer.

cb

La fonction de rappel de l'événement timer. Voir les fonctions de rappel d'événements.

arg

Données personnalisées. Si spécifié, elles seront passées à la fonction de rappel lorsque l'événement lancera les triggers.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

enchant_broker_get_error

(PHP 5 >= 5.3.0, PHP 7, PECL enchant >= 0.1.0 )

enchant_broker_get_error — Retourne la dernière erreur d'un sponsor

Description

enchant_broker_get_error ( resource $broker ) : string

Retourne la dernière erreur survenue pour ce sponsor.

Liste de paramètres

broker

Ressource de sponsor

Valeurs de retour

Retourne le message si une erreur a été trouvée, ou FALSE sinon.

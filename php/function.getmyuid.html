getmyuid

(PHP 4, PHP 5, PHP 7)

getmyuid — Retourne l'UID du propriétaire du script actuel

Description

getmyuid ( void ) : int

Retourne l'UID du propriétaire du script actuel.

Valeurs de retour

Retourne l'identifiant du propriétaire du script actuel (UID) ou FALSE en cas d'erreur.

Thread::globally

(PECL pthreads < 3.0.0)

Thread::globally — Exécution

Avertissement

Cette méthode a été supprimée dans pthreads v3.

Description

public static Thread::globally ( void ) : mixed

Exécute une fonction de rappel dans le scope global

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

La valeur retournée par la fonction de rappel.

Exemples

Exemple #1 Exécution dans le scope global

<?php
class My extends Thread {
    public function run() {
        global $std;
        
        Thread::globally(function(){
            $std = new stdClass;
        });
        
        var_dump($std);
    }
}
$my = new My();
$my->start();
?>

L'exemple ci-dessus va afficher :

object(stdClass)#3 (0) {
}

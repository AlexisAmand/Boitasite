long2ip

(PHP 4, PHP 5, PHP 7)

long2ip — Convertit une adresse entier long (IPv4) en sa notation décimale à point

Description

long2ip ( int $proper_address ) : string

La fonction long2ip() génère une adresse Internet en notation décimale à point (format aaa.bbb.ccc.ddd) à partir de sa représentation entier long.

Liste de paramètres

proper_address

Une propre représentation d'une adresse sous forme entier long

Valeurs de retour

Retourne l'adresse IP Internet, sous la forme d'une chaîne de caractères.

Historique

Version Description
7.1.0 La type du paramètre proper_address a été modifié de string en integer.

Notes

Note:

Sur les architectures 32-bits, transtyper la représention d'adresses IP de string à integer ne donnera pas de résultats correct pour les nombres qui dépassent PHP_INT_MAX.

SVMModel::getSvrProbability

(PECL svm >= 0.1.5)

SVMModel::getSvrProbability — Récupère la valeur sigma pour les types de régression

Description

public SVMModel::getSvrProbability ( void ) : float

Pour les modèles de régression, retourne une valeur sigma; S'il n'y a aucune information de probabilité ou le modèle n'est pas SVR, 0 sera retourné.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne une valeur sigma.

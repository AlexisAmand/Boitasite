getprotobyname

(PHP 4, PHP 5, PHP 7)

getprotobyname — Retourne le numéro de protocole associé à un nom de protocole

Description

getprotobyname ( string $name ) : int

getprotobyname() retourne le numéro de protocole associé avec le nom de protocole name, comme dans /etc/protocols.

Liste de paramètres

name

Le nom du protocole.

Valeurs de retour

Retourne le numéro du protocole, ou FALSE si une erreur survient.

Exemples

Exemple #1 Exemple avec getprotobyname()

<?php
$protocol = 'tcp';
$get_prot = getprotobyname($protocol);
if ($get_prot === FALSE) {
    echo 'Protocole invalide';
} else {
    echo 'Protocole #' . $get_prot;
}
?>

EventBufferEvent::getDnsErrorString

(PECL event >= 1.2.6-beta)

EventBufferEvent::getDnsErrorString — Retourne une chaîne décrivant la dernière erreur DNS

Description

public EventBufferEvent::getDnsErrorString ( void ) : string

Retourne une chaîne décrivant la dernière erreur DNS lors de l'exécution de la méthode EventBufferEvent::connectHost() ou une chaîne vide s'il n'y a aucune erreur DNS de détectée.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne une chaîne décrivant l'erreur DNS, ou une chaîne vide s'il n'y a aucune erreur.

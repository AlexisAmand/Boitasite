Introduction

Les fonctions PDF en PHP peuvent créer des fichiers PDF en utilisant la bibliothèque PDFlib depuis PDFlib GmbH (» www.pdflib.com). Une version restreinte, appelée PDFlib Lite 7 est disponible gratuitement mais n'est plus maintenue depuis 2010. L'intégration de PDFlib pour PHP est toujours activement maintenue par PDFlib GmbH. Elle est principalement destinée aux versions commerciales de PDFlib, mais peut également être utilisée pour l'ancienne version restreinte de PDFlib.

Si vous souhaitez utiliser PDFlib, nous vous conseillons de commencer par les exemples inclus dans le paquet PDFlib, et de vous référer aux manuels au format PDF s'y trouvant. Au minimum, regardez l'exemple hello.php pour une utilisation basique de PDFlib.

Note:

Si vous êtes intéressé par un générateur de PDF libre qui n'utilisent pas de bibliothèques externes PDF, reportez-vous à cette section de la foire aux question.

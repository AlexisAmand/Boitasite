imap_bodystruct

(PHP 4, PHP 5, PHP 7)

imap_bodystruct — Lit la structure d'une section du corps d'un mail

Description

imap_bodystruct ( resource $imap_stream , int $msg_number , string $section ) : object

Lit la structure d'une section spécifié du corps d'un mail.

Liste de paramètres

imap_stream

Un flux IMAP retourné par la fonction imap_open().

msg_number

Le numéro du message

section

La section du corps à lire

Valeurs de retour

Retourne les informations dans un objet ; pour une description détaillée de la structure de l'objet ainsi que de ses propriétés, reportez-vous à la fonction imap_fetchstructure().

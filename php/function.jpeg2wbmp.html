jpeg2wbmp

(PHP 4 >= 4.0.5, PHP 5, PHP 7)

jpeg2wbmp — Convertit une image JPEG en image WBMP

Avertissement

jpeg2wbmp() est devenu obsolète depuis PHP 7.2.0, et sera supprimé en PHP 8.0.0. Utilisez la fonction imagecreatefromjpeg() et la fonction imagewbmp() à la place.

Description

jpeg2wbmp ( string $jpegname , string $wbmpname , int $dest_height , int $dest_width , int $threshold ) : bool

Convertit une image JPEG en image WBMP.

Liste de paramètres

jpegname

Le chemin vers le fichier JPEG.

wbmpname

Chemin vers le fichier final WBMP.

dest_height

Hauteur de l'image de destination.

dest_width

Largeur de l'image de destination.

threshold

Valeur du seuil, entre 0 et 8 inclus.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Attention

Toutefois, si libgd ne parvient pas à produire l'image, cette fonction retourne TRUE.

Exemples

Exemple #1 Exemple avec jpeg2wbmp()

<?php
// Chemin vers la cible jpeg
$path = './test.jpg';

// Récupération de la taille de l'image
$image = getimagesize($path);

// Conversion de l'image
jpeg2wbmp($path, './test.wbmp', $image[1], $image[0], 5);
?>

EventBufferEvent::setTimeouts

(PECL event >= 1.2.6-beta)

EventBufferEvent::setTimeouts — Défini le mode lecture et écriture pour le délai d'attente maximal d'un tampon d'événement

Description

public EventBufferEvent::setTimeouts ( float $timeout_read , float $timeout_write ) : bool

Défini le mode lecture et écriture pour le délai d'attente maximal d'un tampon d'événement.

Liste de paramètres

timeout_read

La lecture du délai d'attente maximal

timeout_write

L'écriture du délai d'attente maximal

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

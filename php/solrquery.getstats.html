SolrQuery::getStats — Vérifie si les statistiques sont actives ou non

Description

public SolrQuery::getStats ( void ) : bool

Vérifie si les statistiques sont actives ou non.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un booléen en cas de succès, et NULL si indéfini.

streamWrapper::unlink

(PHP 5, PHP 7)

streamWrapper::unlink — Efface un fichier

Description

public streamWrapper::unlink ( string $path ) : bool

Cette méthode est appelé en réponse à unlink().

Note:

Afin que le message d'erreur approprié soit retourné, cette méthode doit NE PAS être définie, si le flux ne supporte pas l'effacement de fichiers.

Liste de paramètres

path

L'URL du fichier qui doit être effacé.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Erreurs / Exceptions

Émet une alerte E_WARNING si l'appel à cette méthode échoue (i.e. pas implémenté).

Notes

Note:

La propriété streamWrapper::$context est mise à jour si un contexte valide est passé à la fonction.

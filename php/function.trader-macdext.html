trader_macdext

(PECL trader >= 0.2.0)

trader_macdext — MACD avec un type MA réglable

Description

trader_macdext ( array $real [, int $fastPeriod [, int $fastMAType [, int $slowPeriod [, int $slowMAType [, int $signalPeriod [, int $signalMAType ]]]]]] ) : array

Liste de paramètres

real

Tableau de valeurs réelles.

fastPeriod

Nombre de période pour le fast MA. Intervalle valide : 2 à 100000.

fastMAType

Type de moyenne mobile pour fast MA. Une constante de la série TRADER_MA_TYPE_* doit être utilisée.

slowPeriod

Nombre de période pour le MA. Intervalle valide : 2 à 100000.

slowMAType

Type de moyenne mobile pour slow MA. Une constante de la série TRADER_MA_TYPE_* doit être utilisée.

signalPeriod

Lissage de la ligne du signal (nombre de période). Intervalle valide : 1 à 100000.

signalMAType

Type de moyenne mobile pour la ligne du signal. Une constante de la série TRADER_MA_TYPE_* doit être utilisée.

Valeurs de retour

Retourne un tableau contenant les données calculées, ou FALSE si une erreur survient.

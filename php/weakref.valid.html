Weakref::valid

(PECL weakref >= 0.1.0)

Weakref::valid — Vérifie si l'objet référencé existe toujours

Description

public Weakref::valid ( void ) : bool

Vérifie si l'objet référencé existe toujours.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE si l'objet existe toujours et ainsi, est toujours accessibe via la méthode Weakref::get(), FALSE sinon.

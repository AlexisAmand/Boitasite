DOMNamedNodeMap::getNamedItem

(PHP 5, PHP 7)

DOMNamedNodeMap::getNamedItem — Retourne un noeud spécifié par son nom

Description

DOMNamedNodeMap::getNamedItem ( string $name ) : DOMNode

Récupère un noeud spécifié par son nodeName.

Liste de paramètres

name

Le nodeName du noeud à récupérer.

Valeurs de retour

Un noeud (de n'importe quel type) avec un nodeName spécifié, ou NULL si aucun noeud n'est trouvé.

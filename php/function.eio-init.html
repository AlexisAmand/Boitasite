eio_init

(PECL eio = 1.0.0)

eio_init — (Re-)initialise Eio

Description

eio_init ( void ) : void

eio_init() (re-)initialise Eio. Elle alloue la mémoire pour les structures internes de la bibliothèque libeio et Eio lui-même. Vous devez appeler la fonction eio_init() avant d'utiliser les fonctions Eio. Sinon, cette fonction sera appelée en interne dès lors qu'une fonction Eio sera appelée.

Avertissement

Cette fonctionnalité obsolète sera certainement supprimée dans le futur. Depuis Eio 1.1.0, la fonction eio_init() est obsolète. En Eio 1.0.0, en raison des » restrictions de la bibliothèque libeio, vous devez appeler la fonction eio_init() depuis un processus fils. Vous ne devez pas utiliser Eio depuis un processus parent, si vous l'utilisez dans des fils.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

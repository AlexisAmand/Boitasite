Options de contexte MongoDB

Options de contexte MongoDB — Liste des options de contexte MongoDB

Description

Options de contexte pour les transports mongodb://.

Options

log_cmd_insert callable

Une fonction de rappel appelée lors de l'insertion d'un document ; voir log_cmd_insert().

log_cmd_delete callable

Une fonction de rappel appelée lors de la suppression d'un document ; voir log_cmd_delete().

log_cmd_update callable

Une fonction de rappel appelée lors de la mise à jour d'un document ; voir log_cmd_update().

log_write_batch callable

Une fonction de rappel appelée lors de l'exécution d'un lot d'écriture ; voir log_write_batch().

log_reply callable

Une fonction de rappel appelée lors de la lecture d'une réponse de MongoDB ; voir log_reply().

log_getmore callable

Une fonction de rappel appelée lors de la récupération de plus de résultats depuis un curseur MongoDB ; voir log_getmore().

log_killcursor callable

Une fonction de rappel appelée de l'exécution d'un opcode killcursor ; voir log_killcursor().

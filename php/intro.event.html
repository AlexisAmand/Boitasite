Introduction

Ceci est une extension pour planifier efficacement les I/O, les temps et les signaux basés sur les événements en utilisant le meilleur mécanisme I/O disponible sur une plateforme spécifique. C'est un portage de libevent pour l'infrastructure PHP.

Note:

Notez que le support Windows a été introduit en event-1.9.0.

La version 1.0.0 introduit une nouvelle API orientée objet (cassant ainsi la compatibilité ascendante), et ainsi que le support de libevent 2+, incluant les écoutes HTTP, DNS, OpenSSL et des événements.

Note:

Notez que event-1.0.0 et supérieur n'est pas compatible avec les versions précédentes.

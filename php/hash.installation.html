Installation

À partir de PHP 5.1.2, l'extension Hash est livrée et compilée dans PHP par défaut.

Elle peut être explicitement désactivé par l'utilisation du switch --disable-hash pour configure. Pour les versions antérieuses, il est possible d'inclure l'extension Hash en installant le » module PECL.

À partir de PHP 7.4.0, l'extension Hash est une extension du cœur de PHP, donc elle est toujours activée.

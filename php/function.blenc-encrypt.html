blenc_encrypt

(PECL blenc >= 5)

blenc_encrypt — Chiffre un script PHP avec BLENC

Description

blenc_encrypt ( string $plaintext , string $encodedfile [, string $encryption_key ] ) : string

Chiffre le contenu plaintext et l'écrit dans le fichier encodedfile.

Liste de paramètres

plaintext

Le code source à crypter. Inutile d'avoir les balises ouvrantes/fermantes PHP.

encodedfile

Le nom du fichier où BLENC va sauvegarder le code source crypté.

encryption_key

La clé que BLENC va utiliser pour crypter le contenu. Si non fourni, BLENC va créer une clé valide.

Valeurs de retour

BLENC va retourner la clé redistribuable qui doit être sauvegardée dans key_file : le chemin de key_file est spécifié au moment de l'exécution avec l'option blenc.key_file

Exemples

Exemple #1 Exemple avec blenc_encrypt()

<?php

/* lit le code source PHP */
$source_code = file_get_contents("my_source_to_protect.php");

/* crée la version cryptée */
$redistributable_key = blenc_encrypt($source_code, "my_source_encoded.php");

/* lit le contenu de l'option key_file */
$key_file = ini_get('blenc.key_file');

/* enregistre la clé redistribuable */
file_put_contents($key_file, $redistributable_key, FILE_APPEND);
?>

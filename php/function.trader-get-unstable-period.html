trader_get_unstable_period

(PECL trader >= 0.2.2)

trader_get_unstable_period — Récupère la période d'instabilité

Description

trader_get_unstable_period ( int $functionId ) : int

Récupère le facteur d'instabilité de la période pour une fonction en particulier.

Liste de paramètres

functionId

Identifiant de la fonction du facteur à lire. Les constantes de la série TRADER_FUNC_UNST_* doivent être utilisées.

Valeurs de retour

Retourne le facteur d'instabilité de la période pour la fonction correspondante.

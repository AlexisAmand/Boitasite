L'interface Collectable

(PECL pthreads >= 2.0.8)

Introduction

Attention

Collectables était auparavant une classe (dans pthreads v2 et antérieur). Maintenant, il s'agit d'une interface dans pthreads v3 qui est implémenté par la classe Threaded.

Représente un objet de collecte de données obsolètes.

Synopsis de la classe

Collectable {
/* Méthodes */
public isGarbage ( void ) : bool
public setGarbage ( void ) : void
}

Sommaire

  • Collectable::isGarbage — Détermine si un objet a été marqué comme obsolète
  • Collectable::setGarbage — Marque un objet comme obsolète

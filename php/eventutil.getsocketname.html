EventUtil::getSocketName

(PECL event >= 1.5.0)

EventUtil::getSocketName — Récupère l'adresse courant liée au socket

Description

public static EventUtil::getSocketName ( mixed $socket , string &$address [, mixed &$port ] ) : bool

Récupère l'adresse courant liée au socket.

Liste de paramètres

socket

Ressource de socket, flux, ou descripteur de fichier de socket.

address

Paramètre de sortie. Adresse IP ou le chemin du domaine UNIX du socket, suivant la famille d'adresse du socket.

port

Paramètre de sortie. Le port lié au socket. N'a pas de sens pour les sockets de domaine UNIX.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

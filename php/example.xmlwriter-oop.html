Travailler avec l'API OO

Cet exemple montre comment travailler avec l'API orienté objet de XMLWriter.

Exemple #1 Travailler avec l'API OO

<?php

$xw = new XMLWriter();
$xw->openMemory();
$xw->startDocument("1.0");
$xw->startElement("book");
$xw->text("example");
$xw->endElement();
$xw->endDocument();
echo $xw->outputMemory();

L'exemple ci-dessus va afficher :

<?xml version="1.0"?>
<book>example</book>

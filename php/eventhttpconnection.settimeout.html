EventHttpConnection::setTimeout

(PECL event >= 1.2.6-beta)

EventHttpConnection::setTimeout — Définit le délai d'attente maximal pour la connexion

Description

public EventHttpConnection::setTimeout ( int $timeout ) : void

Définit le délai d'attente maximal pour la connexion.

Liste de paramètres

timeout

Le délai d'attente maximal, en secondes.

Valeurs de retour

Aucune valeur n'est retournée.

fann_test

(PECL fann >= 1.0.0)

fann_test — Effectue un test avec un jeu d'entrées et un jeu de sorties désirées

Description

fann_test ( resource $ann , array $input , array $desired_output ) : array

Effectue un test avec un jeu d'entrées et un jeu de sorties désirées. Cette opération met à jour l'erreur quadratique moyenne, mais ne modifie en aucune façon le réseau.

Liste de paramètres

ann

Ressource de réseau de neurones.

input

Un tableau d'entrées. Ce tableau doit être d'une longueur exacte de fann_get_num_input().

desired_output

Un tableau de sorties désirées. Ce tableau doit être d'une longueur exacte fann_get_num_output().

Valeurs de retour

Retourne les sorties de test en cas de succès, ou FALSE en cas d'erreur.

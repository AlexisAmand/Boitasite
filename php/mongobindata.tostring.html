MongoBinData::__toString

(PECL mongo >= 0.8.1)

MongoBinData::__toString — La représentation en chaîne de l'objet binaire

Cette extension, qui définie cette méthode est obsolète. Veuillez utiliser l'extension MongoDB à la place. Les alertnatives à cette méthode sont :

  • MongoDB\BSON\Binary::getData()

Description

public MongoBinData::__toString ( void ) : string

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la chaîne "<Mongo Binary Data>". Pour accéder au contenu d'un MongoBinData, utilisez le champ bin.

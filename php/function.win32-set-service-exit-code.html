win32_set_service_exit_code

(PECL win32service >=0.4.0)

win32_set_service_exit_code — Définir ou renvoyer le code de sortie pour le service en cours d'exécution

Description

win32_set_service_exit_code ([ int $exitCode = 1 ] ) : int

Change ou retourne le code de sortie. Le code de sortie est utilisé uniquement si le mode de sortie n'est pas gracieux. Si la valeur n'est pas zéro, la configuration de récupération peut être utilisée après l'échec du service. Voir » les codes d'erreur système Microsoft pour plus de détails

Attention

Cette fonction fonctionne uniquement dans SAPI "cli". Sur d'autres SAPI, cette fonction est désactivée.

Liste de paramètres

exitCode

Le code de retour utilisé à la sortie.

Valeurs de retour

Retourne le code de sortie actuel ou l'ancien.

Erreurs / Exceptions

Si cette fonction est utilisé en dehors du SAPI "cli", une erreur E_ERROR sera émise.

phpdbg_break_file

(PHP 5 >= 5.6.3, PHP 7)

phpdbg_break_file — Insère un point d'arrêt dans une ligne d'un fichier

Description

phpdbg_break_file ( string $file , int $line ) : void

Insère un point d'arrêt à la ligne line dans le fichier file.

Liste de paramètres

file

Le nom du fichier.

line

Le numéro de la ligne.

Valeurs de retour

Aucune valeur n'est retournée.

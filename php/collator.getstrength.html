Collator::getStrength

collator_get_strength

(PHP 5 >= 5.3.0, PHP 7, PECL intl >= 1.0.0)

Collator::getStrength -- collator_get_strength — Récupère la dureté de classement utilisé

Description

Style orienté objet

public Collator::getStrength ( void ) : int

Style procédural

collator_get_strength ( Collator $coll ) : int

Liste de paramètres

coll

Objet Collator.

Valeurs de retour

Retourne la dureté du classement, ou FALSE en cas d'erreur.

Exemples

Exemple #1 Exemple avec collator_get_strength()

<?php
$coll     = collator_create( 'en_US' );
$strength = collator_get_strength( $coll );
?>

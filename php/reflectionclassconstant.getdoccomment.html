ReflectionClassConstant::getDocComment

(PHP 7 >= 7.1.0)

ReflectionClassConstant::getDocComment — Récupère le commentaire de documentation

Description

public ReflectionClassConstant::getDocComment ( void ) : string

Récupère le commentaire de documentation d'une constante de classe.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Le commentaire de documentation s'il existe, sinon FALSE

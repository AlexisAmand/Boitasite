phpdbg_clear

(PHP 5 >= 5.6.0, PHP 7)

phpdbg_clear — Supprime tous les points d'arrêt

Description

phpdbg_clear ( void ) : void

Supprime tous les points d'arrêt qui ont été définis, soit via une des fonctions phpdbg_break_*(), ou interactivement via la console.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

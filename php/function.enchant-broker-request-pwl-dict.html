enchant_broker_request_pwl_dict

(PHP 5 >= 5.3.0, PHP 7, PECL enchant >= 0.1.0 )

enchant_broker_request_pwl_dict — Crée un dictionnaire en utilisant un fichier PWL

Description

enchant_broker_request_pwl_dict ( resource $broker , string $filename ) : resource

Crée un dictionnaire en utilisant un fichier PWL. Un fichier PWL est un fichier de mots personnels contenant un mot par ligne.

Liste de paramètres

broker

Ressource de sponsor

filename

Chemin vers le fichier PWL. Si le fichier n'existe pas, un nouveau sera créé, si possible.

Valeurs de retour

Retourne une ressource de dictionnaire en cas de succès ou FALSE si une erreur survient.

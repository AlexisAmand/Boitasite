Ev::backend

(PECL ev >= 0.2.0)

Ev::backend — Retourne un entier décrivant le backend utilisé par libev

Description

final public static Ev::backend ( void ) : int

Retourne un entier décrivant le backend utilisé par libev. Voir les drapeaux de Backend.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un entier (masque d'octets) décrivant le backend utilisé par libev.

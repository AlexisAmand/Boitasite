opcache_is_script_cached

(PHP 5 >= 5.5.11, PHP 7, PECL ZendOpcache >= 7.0.4)

opcache_is_script_cached — Indique si un script est dans le cache d'OPCache

Description

opcache_is_script_cached ( string $file ) : bool

Cette fonction vérifie si un script PHP a été mis dans le cache d'OPCache. Ceci peut être utilisé pour détecter facilement les "alertes" du cache pour un script particulier.

Liste de paramètres

file

Le chemin vers le script PHP à vérifier.

Valeurs de retour

Retourne TRUE si le script file est présent dans le cache d'OPCache, FALSE sinon.

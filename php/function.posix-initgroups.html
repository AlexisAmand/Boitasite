posix_initgroups

(PHP 5 >= 5.2.0, PHP 7)

posix_initgroups — Calcule la liste des accès de groupe

Description

posix_initgroups ( string $name , int $base_group_id ) : bool

Calcule la liste des accès de groupe pour l'utilisateur spécifié par son nom.

Liste de paramètres

name

L'utilisateur pour lequel on doit calculer la liste.

base_group_id

Typiquement, le numéro du groupe depuis le fichier de mot de passe.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

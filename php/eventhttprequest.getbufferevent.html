EventHttpRequest::getBufferEvent

(PECL event >= 1.8.0)

EventHttpRequest::getBufferEvent — Retourne l'objet EventBufferEvent

Description

public EventHttpRequest::closeConnection ( void ) : EventBufferEvent

Retourne l'objet EventBufferEvent qui représente l'événement de buffer que la connexion utilise.

Avertissement

Le compteur de référence de l'objet retourné sera incrémenté de un pour protéger les structures interne contre des destructions prématurés quand la méthode est appelé depuis une fonction de rappel. L'objet EventBufferEvent doit donc être libéré explicitement via la méthode EventBufferEvent::free(). Sinon il y aura une fuite de mémoire.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un objet EventBufferEvent.

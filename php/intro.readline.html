Introduction

Les fonctions readline implémentent une interface de la bibliothèque GNU Readline. Ce sont des fonctions qui fournissent des lignes de commandes éditables. Un exemple est fourni montrant comment Bash vous permet d'utiliser les flèches pour insérer des caractères ou scroller dans l'historique des commandes. De par la nature de cette bibliothèque, vous ne devriez pas en avoir besoin dans vos applications Web mais par contre, cela peut être très utile lors de l'écriture de scripts utilisés depuis la ligne de commande.

À partir de PHP 7.1.0 cette extension est supportée sur Windows.

Attention

L'extension readline n'est pas thread-safe ! Par conséquent l'usage de celle-ci avec n'importe quel SAPI qui est véritablement thread-safe (comme Apache mod_winnt) est fortement déconseillé.

Imagick::getImagePixelColor

(PECL imagick 2.0.0)

Imagick::getImagePixelColor — Retourne la couleur d'un pixel

Description

Imagick::getImagePixelColor ( int $x , int $y ) : ImagickPixel

Retourne la couleur d'un pixel.

Liste de paramètres

x

L'abscisse du pixel

y

L'ordonnée du pixel

Valeurs de retour

Retourne un objet ImagickPixel qui représente la couleur, pour ces coordonnées.

Erreurs / Exceptions

Lance une exception ImagickException si une erreur survient.

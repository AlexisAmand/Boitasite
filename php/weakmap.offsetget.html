WeakMap::offsetGet

(PECL weakref >= 0.2.0)

WeakMap::offsetGet — Retourne la valeur pointée par un certain objet

Description

public WeakMap::offsetGet ( object $object ) : mixed

Retourne la valeur pointée par un certain objet.

Liste de paramètres

object

L'objet clé.

Valeurs de retour

Retourne la valeur associée à l'objet clé passé en argument, ou NULL sinon.

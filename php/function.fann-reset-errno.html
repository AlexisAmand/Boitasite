fann_reset_errno

(PECL fann >= 1.0.0)

fann_reset_errno — Réinitialise le numéro de la dernière erreur

Description

fann_reset_errno ( resource $errdat ) : void

Réinitialise le numéro de la dernière erreur.

Liste de paramètres

errdat

Soit une ressource de réseau de neurones, soit une ressource de données d'entrainement d'un réseau de neurones.

Valeurs de retour

Aucune valeur retournée.

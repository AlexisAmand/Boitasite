MongoCollection::toIndexString

(PECL mongo >=0.9.0)

MongoCollection::toIndexString — Convertit les clés spécifiant un index en une chaîne de caractères

Description

static protected MongoCollection::toIndexString ( mixed $keys ) : string
Avertissement

Cette méthode est obsolète depuis la version 1.5.0.

Liste de paramètres

keys

Le ou les champs à convertir en chaîne(s) de caractères.

Valeurs de retour

Retourne une chaîne de caractères qui décrit l'index.

Exemples

Exemple #1 Exemple avec MongoCollection::toIndexString()

Cet exemple montre la façon dont vous pouvez créer un nom d'index. En raison du fait que ce soit une méthode protégée (statique), vous devez d'abord la surcharger en une classe fils.

<?php
// Crée une classe héritée pour rendre la méthode "publique".
class MyCollection extends MongoCollection
{
    static public function toIndexString($a)
    {
        return parent::toIndexString($a);
    }
}

echo MyCollection::toIndexString("foo"), "\n";
// Affiche : foo_1

echo MyCollection::toIndexString(array('name' => 1, 'age' => -1)), "\n";
// Affiche : name_1_age_-1
?>

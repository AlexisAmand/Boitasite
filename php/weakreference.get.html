WeakReference::get

(PHP 7 >= 7.4.0)

WeakReference::get — Récupère un objet faiblement référencé

Description

public WeakReference::get ( void ) : ?object

Récupère un objet faiblement référencé. Si l'objet a déjà été détruit, NULL est retourné.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne l'objet référencé, ou NULL si l'objet a été détruit.

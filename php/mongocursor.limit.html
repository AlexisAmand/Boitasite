MongoCursor::limit

(PECL mongo >=0.9.0)

MongoCursor::limit — Limite le nombre de résultats retournés

Description

public MongoCursor::limit ( int $num ) : MongoCursor

Liste de paramètres

num

Le nombre de résultats à retourner.

Valeurs de retour

Retourne ce curseur.

Erreurs / Exceptions

Émet une exception MongoCursorException si le curseur a commencé ses itérations.

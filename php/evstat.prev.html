EvStat::prev

(PECL ev >= 0.2.0)

EvStat::prev — Retourne le jeu précédent retourné par EvStat::attr

Description

public EvStat::prev ( void ) : void

Identique à la méthode EvStat::attr() mais retourne le jeu précédent de valeurs.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un tableau avec la même structure que le tableau retourné par la méthode EvStat::attr(). Le tableau contiendra les valeurs prédécemment détectées.

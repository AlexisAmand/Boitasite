Modification dans la gestion des fichiers INI

Intl

La directive de configuration intl.use_exceptions a été ajoutée, laquelle contrôle le comportement d'intl lorsque des erreurs globales surviennent, en conjonction avec la directive de configuration existante intl.error_level.

MySQLnd

La directive de configuration mysqlnd.sha256_server_public_key a été ajoutée pour permettre à mysqli d'utiliser le nouveau protocole d'authentification MySQL.

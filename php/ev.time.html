Ev::time

(PECL ev >= 0.2.0)

Ev::time — Retourne le temps courant depuis l'époque Unix

Description

final public static Ev::time ( void ) : float

Retourne le temps courant depuis l'époque Unix. Vous devriez plutôt utiliser la méthode Ev::now().

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne le temps courant, en nombre de secondes, depuis l'époque Unix.

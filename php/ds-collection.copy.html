Ds\Collection::copy

(PECL ds >= 1.0.0)

Ds\Collection::copy — Retourne une copie superficielle de la collection

Description

abstract public Ds\Collection::copy ( void ) : Ds\Collection

Retourne une copie superficielle de la collection.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne une copie superficielle de la collection.

Exemples

Exemple #1 Ds\Collection::copy() exemple

<?php
$a = new \Ds\Vector([1, 2, 3]);
$b = $a->copy();

$b->push(4);

print_r($a);
print_r($b);
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

Ds\Vector Object
(
    [0] => 1
    [1] => 2
    [2] => 3
)
Ds\Vector Object
(
    [0] => 1
    [1] => 2
    [2] => 3
    [3] => 4
)

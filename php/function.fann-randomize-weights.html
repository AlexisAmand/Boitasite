fann_randomize_weights

(PECL fann >= 1.0.0)

fann_randomize_weights — Donne à chaque connexion un poids aléatoire compris entre min_weight et max_weight

Description

fann_randomize_weights ( resource $ann , float $min_weight , float $max_weight ) : bool

Donne à chaque connexion un poids aléatoire compris entre min_weight et max_weight.

Depuis le début, les poids sont aléatoires, dans un intervalle -0.1 et 0.1.

Liste de paramètres

ann

Ressource de réseau de neurones.

min_weight

Valeur minimale du poids

max_weight

Valeur maximale du poids

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

GearmanClient::echo

(PECL gearman >= 0.5.0)

GearmanClient::echo — Envoi des données à tous les serveurs de travaux afin de vérifier qu'ils les affichent en retour (obsolète)

Description

public GearmanClient::echo ( string $workload ) : bool

La méthode GearmanClient::echo() est obsolète depuis pecl/gearman 1.0.0. Utilisez la méthode GearmanClient::ping() à la place.

Liste de paramètres

workload

Données arbitraires linéarisées

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

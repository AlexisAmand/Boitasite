Exception::getPrevious

(PHP 5 >= 5.3.0, PHP 7)

Exception::getPrevious — Retourne l'exception précédente

Description

final public Exception::getPrevious ( void ) : Throwable

Retourne l'exception précédente (le troisième paramètre de la méthode Exception::__construct()).

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la précédente Throwable si disponible, NULL sinon.

Historique

Version Description
7.0.0 La déclaration de type de retour a été modifié en Throwable.

Exemples

Exemple #1 Exemple avec Exception::getPrevious()

Une boucle, on affiche et on capture les exceptions.

<?php
class MyCustomException extends Exception {}

function doStuff() {
    try {
        throw new InvalidArgumentException("Vous avez fais une erreur !", 112);
    } catch(Exception $e) {
        throw new MyCustomException("Un problème est survenu", 911, $e);
    }
}


try {
    doStuff();
} catch(Exception $e) {
    do {
        printf("%s:%d %s (%d) [%s]\n", $e->getFile(), $e->getLine(), $e->getMessage(), $e->getCode(), get_class($e));
    } while($e = $e->getPrevious());
}
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

/home/bjori/ex.php:8 Un problème est survenu (911) [MyCustomException]
/home/bjori/ex.php:6 Vous avez fait une erreur ! (112) [InvalidArgumentException]

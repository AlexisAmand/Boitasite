PDO::pgsqlGetNotify

(PHP 5 >= 5.6.0, PHP 7)

PDO::pgsqlGetNotify — Récupère une notification asynchrone

Description

public PDO::pgsqlGetNotify ([ int $result_type = PDO::FETCH_USE_DEFAULT [, int $ms_timeout = 0 ]] ) : array

Retourne le jeu de résultats représentant une notification asynchrone en attente.

Liste de paramètres

result_type

Le format du résultat, représenté sous la forme d'une constante de type PDO::FETCH_*.

ms_timeout

La durée d'attente d'une réponse, en millisecondes.

Valeurs de retour

Si une ou plusieurs notifications sont en attente, retourne une seule ligne avec les champs message et pid, sinon, retourne FALSE.

EventBuffer::drain

(PECL event >= 1.2.6-beta)

EventBuffer::drain — Supprime un nombre spécifié d'octets depuis le début du tampon sans copier les données

Description

public EventBuffer::drain ( int $len ) : bool

Fonctionne de la même manière que la méthode EventBuffer::read() excepté le fait que les données supprimées ne sont pas copiées : elles sont seulement supprimer du début du tampon.

Liste de paramètres

len

Le nombre d'octets à supprimer depuis le tampon.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

Voir aussi

  • EventBuffer::read() - Lit les données d'un evbuffer, et vide les octets lus
  • EventBuffer::appendFrom() - Déplace le nombre d'octets fourni depuis un tampon source vers la fin du tampon courant

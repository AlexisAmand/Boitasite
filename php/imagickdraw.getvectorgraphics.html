ImagickDraw::getVectorGraphics

(PECL imagick 2.0.0)

ImagickDraw::getVectorGraphics — Retourne une chaîne contenant le vecteur graphique

Description

ImagickDraw::getVectorGraphics ( void ) : string
Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Retourne une chaîne contenant le vecteur graphique, généré par les appels graphiques depuis que l'objet ImagickDraw a été créé.

Valeurs de retour

Retourne une chaîne contenant les vecteurs graphiques.

pg_socket

(PHP 5 >= 5.6.0, PHP 7)

pg_socket — Récupère un gestionnaire de lecture seul sur le socket sous-jacent d'une connexion PostgreSQL

Description

pg_socket ( resource $connection ) : resource

La fonction pg_socket() retourne une ressource en lecture seule correspondant au socket sous-jacent de la connexion PostgreSQL fournie.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

connection

Ressource de connexion à la base de données PostgreSQL.

Valeurs de retour

Une ressource de socket en cas de succès ou FALSE si une erreur survient.

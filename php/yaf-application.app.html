Yaf_Application::app

(Yaf >=1.0.0)

Yaf_Application::app — Récupère une instance de l'application

Description

public staticYaf_Application::app ( void ) : mixed

Récupère l'instance Yaf_Application, que vous pouvez aussi récupérer en utilisant la méthode Yaf_Dispatcher::getApplication().

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Une instance Yaf_Application ; si aucune instante Yaf_Application n'avait été initialisé auparavant, NULL sera retourné.

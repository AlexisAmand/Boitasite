EvPrepare::createStopped

(PECL ev >= 0.2.0)

EvPrepare::createStopped — Crée une instance stoppée du watcher EvPrepare

Description

final public static EvPrepare::createStopped ( callable $callback [, mixed $data = NULL [, int $priority = 0 ]] ) : EvPrepare

Crée une instance stoppée du watcher EvPrepare. Contrairement à la méthode EvPrepare::__construct(), cette méthode ne démarre pas le watcher automatiquement.

Liste de paramètres

callback

Voir les fonctions de rappel des Watchers.

data

Données personnalisées à associer avec le watcher.

priority

Priorité du Watcher

Valeurs de retour

Retourne un objet EvPrepare en cas de succès.

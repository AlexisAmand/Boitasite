Yaf_Controller_Abstract::init

(Yaf >=1.0.0)

Yaf_Controller_Abstract::init — Initialisation du contrôleur

Description

public Yaf_Controller_Abstract::init ( void ) : void

La méthode Yaf_Controller_Abstract::__construct() est finale, ce qui signifie qu'elle ne peut être surchargée. Les utilisateurs peuvent définir la méthode Yaf_Controller_Abstract::init(), qui sera appelée après l'instanciation de l'objet contrôleur.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

xml_parser_free

(PHP 4, PHP 5, PHP 7)

xml_parser_free — Détruit un analyseur XML

Description

xml_parser_free ( resource $parser ) : bool

Détruit l'analyseur XML parser.

Attention

À partir de PHP 7.0.0, en plus d'appeler la fonction xml_parser_free() à la fin de l'analyse, il est nécessaire de supprimer explicitement la référence au paramètre parser pour éviter les fuites mémoires, si la ressource analysé est référencé depuis un objet, et cet objet fait reférence à la ressource analysé

Liste de paramètres

parser
Une référence sur un analyseur XML.

Valeurs de retour

Cette fonction retourne FALSE si parser n'est pas une référence valide ou, sinon, détruit l'analyseur et retourne TRUE.

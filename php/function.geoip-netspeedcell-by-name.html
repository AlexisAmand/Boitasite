geoip_netspeedcell_by_name

(PECL geoip >= 1.1.0)

geoip_netspeedcell_by_name — Récupère la vitesse de la connexion Internet

Description

geoip_netspeedcell_by_name ( string $hostname ) : string

La fonction geoip_netspeedcell_by_name() va retourner le type de connexion Internet ainsi que sa vitesse, suivant un nom d'hôte ou une adresse IP.

Cette fonction n'est disponible que lors de l'utilisation de la version 1.4.8 de la bibliothèque GeoIP, ou supérieures.

Cette fonction est actuellement uniquement disponible pour les utilisateurs qui ont acheté une édition commerciale GeoIP NetSpeedCell. Une alerte sera émise si la bonne base de données ne peut être trouvée.

La valeur retournée sera une chaîne de caractères dont les valeurs communues sont :

  • Cable/DSL
  • Dialup
  • Cellular
  • Corporate

Liste de paramètres

hostname

Le nom d'hôte, ou l'adresse IP.

Valeurs de retour

Retourne la vitesse de la connexion en cas de succès, ou FALSE si l'adresse ne peut être trouvée dans la base de données.

Exemples

Exemple #1 Exemple avec geoip_netspeedcell_by_name()

Cet exemple va afficher la vitesse de la connexion de l'hôte example.com.

<?php
$netspeed = geoip_netspeedcell_by_name('www.example.com');

if ($netspeed) {
    echo 'Le type de connexion est : '. $netspeed;
}
?>

L'exemple ci-dessus va afficher :

Le type de connexion est : Corporate

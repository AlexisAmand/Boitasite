Event::pending

(PECL event >= 1.2.6-beta)

Event::pending — Détecte si l'événement est en attente ou programmé

Description

public Event::pending ( int $flags ) : bool

Détecte si l'événement est en attente ou programmé.

Liste de paramètres

flags

Une ou plusieurs des constantes suivantes : Event::READ, Event::WRITE, Event::TIMEOUT, Event::SIGNAL.

Valeurs de retour

Retourne TRUE si l'événement est en attente ou programmé, FALSE sinon.

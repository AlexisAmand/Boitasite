EvIo::__construct

(PECL ev >= 0.2.0)

EvIo::__construct — Construit un nouvel objet EvIo

Description

public EvIo::__construct ( mixed $fd , int $events , callable $callback [, mixed $data [, int $priority ]] )

Construit un nouvel objet EvIo et démarre l'observateur automatiquement.

Liste de paramètres

fd

Peut être un flux ouvert avec la fonction fopen() ou toutes fonctions similaires, descripteurs de fichiers ou sockets.

events

Ev::READ et/ou Ev::WRITE. Voir les masques d'octets.

callback

Voir les fonctions de rappels des observateurs.

data

Données personalisées associées avec l'observateur.

priority

Priorité de l'observateur

Valeurs de retour

Retourne un objet EvIo en cas de succès.

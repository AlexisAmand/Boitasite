ps_closepath_stroke

(PECL ps >= 1.1.0)

ps_closepath_stroke — Ferme et trace un chemin

Description

ps_closepath_stroke ( resource $psdoc ) : bool

Connecte le dernier point avec le premier point du chemin et dessine le résultat de la ligne fermée.

Liste de paramètres

psdoc

Identifiant d'un fichier postscript retourné par ps_new().

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

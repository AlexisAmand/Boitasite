ldap_bind_ext

(PHP 7 >= 7.3.0)

ldap_bind_ext — Lie un dossier LDAP

Description

ldap_bind_ext ( resource $link_identifier [, string $bind_rdn = NULL [, string $bind_password = NULL [, array $serverctrls = array() ]]] ) : resource

Réalise la même chose que la fonction ldap_bind() mais retourne comme résultat une ressource LDAP à analyser avec la fonction ldap_parse_result().

Liste de paramètres

See ldap_bind()

Valeurs de retour

Retourne un identifiant LDAP ou FALSE si une erreur survient.

mqseries_set

(PECL mqseries >= 0.10.0)

mqseries_set — MQSeries MQSET

Description

mqseries_set ( resource $hConn , resource $hObj , int $selectorCount , array $selectors , int $intAttrCount , array $intAttrs , int $charAttrLength , array $charAttrs , resource &$compCode , resource &$reason ) : void

L'appel de mqseries_set() (MQSET) est utilisé pour modifier les attributs d'un objet représenté par un gestionnaire. L'objet doit être une file d'attente.

Liste de paramètres

hConn

Gestionnaire de connexion.

Ce gestionnaire représente la connexion au gestionaire de files d'attente.

hObj

Gestionnaire d'objet.

Ce gestionnaire représente l'objet à utiliser.

selectorCount

Compte des sélecteurs.

selectors

Tableau d'attributs des sélecteurs.

intAttrCount

Compte d'attributs entier.

intAttrs

Tableau d'attributs entier.

charAttrLength

Longueur du tampon d'attributs de caractères.

charAttrs

Attributs de caractère.

compCode

Code de complétion.

reason

La raison qui qualifie le compCode.

Valeurs de retour

Aucune valeur n'est retournée.

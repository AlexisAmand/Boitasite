EventBase::getMethod

(PECL event >= 1.2.6-beta)

EventBase::getMethod — Retourne la méthode d'événement utilisée

Description

public EventBase::getMethod ( void ) : string

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Une chaîne de caractères représentant la méthode d'événement utilisée (backend).

Exemples

Exemple #1 Exemple avec EventBase::getMethod()

<?php
$cfg = new EventConfig();
if ($cfg->avoidMethod("select")) {
    echo "Mise à l'écart de la méthode `select'\n";
}

// Crée un event_base associé avec la configuration
$base = new EventBase($cfg);
echo "Méthode d'événement utilisée : ", $base->getMethod(), PHP_EOL;

?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

Mise à l'écart de la méthode `select'
Méthode d'événement utilisée : epoll

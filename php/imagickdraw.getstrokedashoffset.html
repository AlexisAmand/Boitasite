ImagickDraw::getStrokeDashOffset

(PECL imagick 2.0.0)

ImagickDraw::getStrokeDashOffset — Retourne l'offset du pointillé dans le motif

Description

ImagickDraw::getStrokeDashOffset ( void ) : float
Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Retourne l'offset du pointillé dans le motif.

Valeurs de retour

Retourne un nombre décimal représentant, le début du pointillé dans le motif.

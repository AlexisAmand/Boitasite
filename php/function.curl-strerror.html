curl_strerror

(PHP 5 >= 5.5.0, PHP 7)

curl_strerror — Retourne la chaîne descriptive du code erreur fourni

Description

curl_strerror ( int $errornum ) : string

Retourne le message d'erreur décrivant le code erreur fourni.

Liste de paramètres

errornum

Une constante parmi les constantes » des codes erreurs cURL.

Valeurs de retour

Retourne la description de l'erreur ou NULL pour un code erreur invalide.

Exemples

Exemple #1 Exemple avec curl_errno()

<?php
// Crée un gestionnaire curl avec une erreur dans le protocole de l'URL utilisée
$ch = curl_init("htp://example.com/");

// Envoi la requête
curl_exec($ch);

// Vérifie les erreurs et affiche le message d'erreur
if($errno = curl_errno($ch)) {
    $error_message = curl_strerror($errno);
    echo "cURL error ({$errno}):\n {$error_message}";
}

// Ferme le gestionnaire
curl_close($ch);
?>

L'exemple ci-dessus va afficher :

cURL error (1):
 Unsupported protocol

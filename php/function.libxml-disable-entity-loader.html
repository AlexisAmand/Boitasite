libxml_disable_entity_loader

(PHP 5 >= 5.2.11, PHP 7)

libxml_disable_entity_loader — Désactive le chargement des entités externes

Description

libxml_disable_entity_loader ([ bool $disable = TRUE ] ) : bool

Active ou désactive le chargement des entités externes. Il est à noter que désactiver le chargement d'entités externes peut causer des problèmes lors du chargement de documents XML. Cependant à partir de libxml 2.9.0 la substitution d'entités est désactivé par défaut, ainsi il n'est pas nécéssaire de désactiver le chargement des entités externes.

Liste de paramètres

disable

Désactive (TRUE) ou active (FALSE) le chargement des entités externes par les extensions libxml (telles que DOM, XMLWriter et XMLReader).

Valeurs de retour

Retourne la configuration précédente.

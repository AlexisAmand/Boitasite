EvLoop::check

(PECL ev >= 0.2.0)

EvLoop::check — Crée un objet EvCheck associé avec l'instance de la boucle d'événements courante

Description

final public EvLoop::check ( string $callback [, string $data [, string $priority ]] ) : EvCheck

Crée un objet EvCheck associé avec l'instance de la boucle d'événements courante.

Liste de paramètres

Tous les paramètres ont la même signification que pour la méthode EvCheck::__construct().

Valeurs de retour

Retourne un objet EvCheck object en cas de succès.

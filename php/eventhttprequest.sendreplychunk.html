EventHttpRequest::sendReplyChunk

(PECL event >= 1.4.0-beta)

EventHttpRequest::sendReplyChunk — Envoi un autre bloc de données comme partie d'un bloc de réponse entrant

Description

public EventHttpRequest::sendReplyChunk ( EventBuffer $buf ) : void

Envoi un autre bloc de données comme partie d'un bloc de réponse entrant. Après l'appel de cette méthode, le paramètre buf sera vide.

Liste de paramètres

buf

Le bloc de données à envoyer comme partie de la réponse.

Valeurs de retour

Aucune valeur n'est retournée.

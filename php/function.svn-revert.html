svn_revert

(PECL svn >= 0.3.0)

svn_revert — Revenir en arrière sur les changements de la copie de travail

Description

svn_revert ( string $path [, bool $recursive = FALSE ] ) : bool

Revient en arrière sur tout changement local du chemin dans la copie de travail.

Liste de paramètres

path

Le chemin vers l'élément dans la copie de travail.

recursive

Optionnellement applique ceci récursivement.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

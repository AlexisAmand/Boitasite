mysqli_bind_result — Alias de mysqli_stmt_bind_result()

Description

Cette fonction est un alias de : mysqli_stmt_bind_result().

Avertissement

Cette fonction est OBSOLÈTE à partir de PHP 5.3.0 et a été SUPPRIMÉE à partir de PHP 5.4.0.

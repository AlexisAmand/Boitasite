Configuration à l'exécution

Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.

Options de configuration image
Nom Défaut Modifiable Historique
gd.jpeg_ignore_warning "1" PHP_INI_ALL Disponible à partir de PHP 5.1.3.
Pour plus de détails sur les modes PHP_INI_*, reportez-vous à Où une directive de configuration peut être modifiée.

Voici un éclaircissement sur l'utilisation des directives de configuration.

gd.jpeg_ignore_warning bool

Ignore les alertes (main pas les érreurs) créées par libjpeg(-turbo).

Historique pour gd.jpeg_ignore_warning
Version Description
7.1.0 La valeur par défaut de gd.jpeg_ignore_warning a été changé de 0 à 1.

Voir aussi les directives de configuration exif.

Avertissement

Les fonctions sur les images sont très gourmandes en terme de mémoire. Assurez-vous de définir une valeur assez importante pour memory_limit si vous utilisez la librairie GD intégrée.

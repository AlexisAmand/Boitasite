fann_set_cascade_activation_steepnesses

(PECL fann >= 1.0.0)

fann_set_cascade_activation_steepnesses — Défini le tableaux des raideurs d'activation candidate en cascade

Description

fann_set_cascade_activation_steepnesses ( resource $ann , array $cascade_activation_steepnesses_count ) : bool

Défini le tableaux des raideurs d'activation candidate en cascade.

Voir la fonction fann_get_cascade_num_candidates() pour savoir quels neurones candidats seront générés par ce tableau.

Liste de paramètres

ann

Ressource de réseau de neurones.

cascade_activation_steepnesses_count

Le tableaux des raideurs d'activation candidate en cascade.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

Voir aussi

  • fann_get_cascade_activation_steepnesses() - Returns the cascade activation steepnesses
  • fann_get_cascade_activation_steepnesses_count() - The number of activation steepnesses

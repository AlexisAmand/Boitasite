Incompatibilités

MySQL Native Driver est dans la plupart des cas compatible avec MySQL Client Library (libmysql). Cette section détaille les incompatibilités.

  • Les valeurs de type bit sont retournées comme des chaines binaires (e.g. "\0" or "\x1F") avec libmysql mais comme des chaines décimales (e.g. "0" or "31") avec mysqlnd. Si vous voulez être compatible avec les deux librairies, retournez toujours les champs bit comme des nombres, avec des requêtes MySQL comme: SELECT bit + 0 FROM table.

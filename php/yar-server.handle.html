Yar_Server::handle

(PECL yar >= 1.0.0)

Yar_Server::handle — Démarre un serveur RPC

Description

public Yar_Server::handle ( void ) : bool

Démarre un serveur RPC HTTP, et le rend prêt à accepter des requêtes RPC.

Note:

Les appels RPC sont envoyés par des requêtes HTTP POST. Si une requête HTTP GET est envoyé à l'URI, les informations du service (commenté dans la section ci-dessous) seront affichées sur la page.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Un booléen.

Exemples

Exemple #1 Exemple avec Yar_Server::handle()

<?php
class API {
    /**
     * les informations seront générées automatiquement sur la page d'informations du service.
     * @params 
     * @return
     */
    public function some_method($parameter, $option = "foo") {
    }

    protected function client_can_not_see() {
    }
}

$service = new Yar_Server(new API());
$service->handle();
?>

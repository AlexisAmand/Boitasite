imap_utf8_to_mutf7

(PHP 5 >= 5.3.0, PHP 7)

imap_utf8_to_mutf7 — Encode une chaîne de charactère UTF-8 en UTF-7 modifié

Description

imap_utf8_to_mutf7 ( string $in ) : string

Encode une chaîne de caractères UTF-8 en UTF-7 modifié (comme spécifié dans la RFC 2060, section 5.1.3).

Note:

Cette fonction est seulement disponible, si libcclient exporte utf8_to_mutf7().

Liste de paramètres

in

Une chaîne de caractères encodé en UTF-8.

Valeurs de retour

Retourne in convertit en UTF-7 modifié, ou FALSE si une erreur survient.

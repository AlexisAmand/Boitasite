imagegetclip

(PHP 7 >= 7.2.0)

imagegetclip — Récupère le rectangle de coupure

Description

imagegetclip ( resource $im ) : array

imagegetclip() récupère le rectangle de coupure actuel, c'est à dire la zone au-delà de laquelle aucun pixel ne sera dessiné.

Liste de paramètres

image

Une ressource d'image, retournée par une des fonctions de création d'images, comme imagecreatetruecolor().

Valeurs de retour

La fonction retourne un tableau indexé avec les coordonées du rectangle de coupure avec les entrées suivantes :

  • la coordonée x du coin supérieur gauche
  • la coordonée y du coin supérieur gauche
  • la coordonée x du coin inférieur droite
  • la coordonée y du coin inférieur droite

Exemples

Exemple #1 Exemple avec imagegetclip()

Définir et récupérer le rectangle de coupure.

<?php
$im = imagecreate(100, 100);
imagesetclip($im, 10,10, 89,89);
print_r(imagegetclip($im));

L'exemple ci-dessus va afficher :

Array
(
    [0] => 10
    [1] => 10
    [2] => 89
    [3] => 89
)

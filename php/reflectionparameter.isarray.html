ReflectionParameter::isArray

(PHP 5 >= 5.1.2, PHP 7)

ReflectionParameter::isArray — Vérifie si le paramètre attend un tableau

Description

public ReflectionParameter::isArray ( void ) : bool

Vérifie si le paramètre attend un tableau.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

TRUE si le paramètre attend un tableau, FALSE sinon.

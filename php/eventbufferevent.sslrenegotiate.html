EventBufferEvent::sslRenegotiate

(PECL event >= 1.2.6-beta)

EventBufferEvent::sslRenegotiate — Demande au tampon d'événement de commencer une renégociation SSL

Description

public EventBufferEvent::sslRenegotiate ( void ) : void

Demande au tampon d'événement de commencer une renégociation SSL.

Avertissement

L'appel à cette méthode demande à SSL une renégociation, et au tampon d'événement d'appeler la fonction de rappel propriée. C'est une méthode très avancée ; elle ne doit être appelée que si vous savez parfaitement ce que vous faîtes, tout spécialement depuis quelques versions SSL où des failles de sécurité ont été mises à jour lors de renégociation.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

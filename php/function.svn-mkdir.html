svn_mkdir

(PECL svn >= 0.4.0)

svn_mkdir — Crée un dossier dans la copie de travail ou dans le référentiel

Description

svn_mkdir ( string $path [, string $log_message ] ) : bool

Crée un dossier dans la copie de travail ou dans le référentiel.

Liste de paramètres

path

Le chemin vers la copie de travail ou le référentiel.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

EventHttpConnection::setLocalAddress

(PECL event >= 1.2.6-beta)

EventHttpConnection::setLocalAddress — Définit l'adresse IP depuis laquelle les connexions HTTP sont effectuées

Description

public EventHttpConnection::setLocalAddress ( string $address ) : void

Définit l'adresse IP depuis laquelle les connexions HTTP sont effectuées.

Liste de paramètres

address

L'adresse IIP depuis laquelle les connexions HTTP sont effectuées.

Valeurs de retour

Aucune valeur n'est retournée.

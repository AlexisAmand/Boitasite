Yaf_Request_Abstract::setModuleName

(Yaf >=1.0.0)

Yaf_Request_Abstract::setModuleName — Définit le nom du module

Description

public Yaf_Request_Abstract::setModuleName ( string $module [, bool $format_name = true ] ) : void

Définit le nom du module dans la requête ; habituellement utilisé par le routeur personnalisé pour définir le nom du module de la route résultante.

Liste de paramètres

module

chaîne de caractères ; nom du module, au style camel comme "Index" ou "Foo_Bar"

format_name

Introduit en Yaf 3.2.0 ; par défaut, Yaf formattera le nom en mode camel ; si définit à FALSE, Yaf définira le nom original dans la requête.

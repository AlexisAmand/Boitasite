fann_set_training_algorithm

(PECL fann >= 1.0.0)

fann_set_training_algorithm — Défini l'algorithme d'entrainement

Description

fann_set_training_algorithm ( resource $ann , int $training_algorithm ) : bool

Défini l'algorithme d'entrainement.

Plus d'informations disponible sur la page de la fonction fann_get_training_algorithm().

Liste de paramètres

ann

Ressource de réseau de neurones.

training_algorithm

Une constante d'algorithme d'entrainement

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

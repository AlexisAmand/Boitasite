CallbackFilterIterator::accept

(PHP 5 >= 5.4.0, PHP 7)

CallbackFilterIterator::accept — Appel la fonction de rappel avec la valeur courante, la clé courante, et l'itérateur interne comme arguments

Description

public CallbackFilterIterator::accept ( void ) : bool

Cette méthode appelle la fonction de rappel avec la valeur courante, la clé courante et l'itérateur interne.

La fonction de rappel doit retourner TRUE si l'élément courant est accepté, FALSE sinon.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE pour accepter l'élément courant, FALSE sinon.

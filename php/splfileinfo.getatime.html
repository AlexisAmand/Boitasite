SplFileInfo::getATime

(PHP 5 >= 5.1.2, PHP 7)

SplFileInfo::getATime — Lit la date de dernier accès au fichier

Description

public SplFileInfo::getATime ( void ) : int

Lit la date de dernier accès au fichier.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la date de dernier accès au fichier.

Erreurs / Exceptions

Émet une exception RunTimeException en cas d'erreur.

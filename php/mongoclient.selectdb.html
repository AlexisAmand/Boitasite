MongoClient::selectDB

(PECL mongo >=1.3.0)

MongoClient::selectDB — Récupère une base de données

Cette extension, qui définie cette méthode est obsolète. Veuillez utiliser l'extension MongoDB à la place. Il n'y a pas d'équivalent à cette méthode dans la nouvelle extension, mais il existe une alternative dans la bibliothèque PHP:

  • » MongoDB\Client::selectDatabase()

Description

public MongoClient::selectDB ( string $name ) : MongoDB

Liste de paramètres

name

Le nom de la base de données.

Valeurs de retour

Retourne un nouvel objet de base de données.

Erreurs / Exceptions

Emet une Exception si la base de données est invalide.

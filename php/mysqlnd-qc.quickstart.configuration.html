Configuration

Le plugin est implémenté comme une extension PHP. Voir aussi les instructions d'installation pour installer l'extension » PECL/mysqlnd_qc.

Vous devez compiler ou configurer l'extension PHP MySQL (mysqli, PDO_MYSQL, mysql) que vous prévoyez d'utiliser avec le support de la bibliothèque mysqlnd. PECL/mysqlnd_qc est un plugin pour la bibliothèque mysqlnd. Pour utiliser le plugin avec n'importe laquelle des extensions PHP MySQL (APIs), l'extension doit utiliser la bibliothèque mysqlnd.

Puis, vous devez charger l'extension dans PHP, et activer le plugin dans le fichier de configuration de PHP en utilisant la directive de configuration PHP nommée mysqlnd_qc.enable_qc.

Exemple #1 Activation du plugin (php.ini)

mysqlnd_qc.enable_qc=1

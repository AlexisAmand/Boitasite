ibase_delete_user

(PHP 5, PHP 7 < 7.4.0)

ibase_delete_user — Efface un utilisateur d'une base de données de sécurité

Description

ibase_delete_user ( resource $service_handle , string $user_name ) : bool

Liste de paramètres

service_handle

Le gestionnaire sur le service du serveur de base de données.

user_name

L'identifiant de l'utilisteur qui doit être supprimé de la base de donnée.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

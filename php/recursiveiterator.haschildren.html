RecursiveIterator::hasChildren

(PHP 5 >= 5.1.0, PHP 7)

RecursiveIterator::hasChildren — Vérifie si un itérateur peut être créé depuis l'entrée courante

Description

public RecursiveIterator::hasChildren ( void ) : bool

Vérifie si un itérateur peut être créé depuis l'entrée courante.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE si l'entrée courante peut être parcouru, FALSE sinon.

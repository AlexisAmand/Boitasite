EventDnsBase::parseResolvConf

(PECL event >= 1.2.6-beta)

EventDnsBase::parseResolvConf — Analyse le fichier resolv.conf

Description

public EventDnsBase::parseResolvConf ( int $flags , string $filename ) : bool

Analyse le fichier resolv.conf, et y lit toutes les options pprésentes.

Liste de paramètres

flags

Détermine les informations à analyser depuis le fichier resolv.conf. Voir la page du manuel système du fichier resolv.conf pour connaitre son format.

Les directives suivantes ne sont pas analysées dans le fichier : sortlist, rotate, no-check-names, inet6, debug.

Si cette fonction rencontre une erreur, les valeurs retournées possibles sont :

  • 1 = échec lors de l'ouverture du fichier
  • 2 = échec lors de la récupération des informations du fichier
  • 3 = fichier trop grand
  • 4 = dépassement mémoire
  • 5 = lecture trop courte du fichier
  • 6 = aucun serveur de noms listé dans le fichier

filename

Chemin vers le fichier resolv.conf.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

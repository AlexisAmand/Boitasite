RecursiveCallbackFilterIterator::getChildren

(PHP 5 >= 5.4.0, PHP 7)

RecursiveCallbackFilterIterator::getChildren — Retourne l'itérateur fils interne contenu dans un RecursiveCallbackFilterIterator

Description

public RecursiveCallbackFilterIterator::getChildren ( void ) : RecursiveCallbackFilterIterator

Récupère le fils filtré de l'itérateur interne.

RecursiveCallbackFilterIterator::hasChildren() doit être utilisé pour déterminer s'il y a des fils qui peuvent être récupérés.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un RecursiveCallbackFilterIterator contenant le fils.

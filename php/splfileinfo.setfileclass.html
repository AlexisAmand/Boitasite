SplFileInfo::setFileClass

(PHP 5 >= 5.1.2, PHP 7)

SplFileInfo::setFileClass — Configure la classe utilisée avec SplFileInfo::openFile()

Description

public SplFileInfo::setFileClass ([ string $class_name = "SplFileObject" ] ) : void

Utilisez cette méthode pour définir une classe personnalisée qui sera utilisée lorsque SplFileInfo::openFile() est appelé. Le nom de la classe passé à cette méthode doit être SplFileObject ou une classe dérivé de SplFileObject.

Liste de paramètres

class_name

Le nom de la classe utilisée par SplFileInfo::openFile().

Valeurs de retour

Aucune valeur n'est retournée.

Exemples

Exemple #1 Exemple avec SplFileInfo::setFileClass()

<?php
// Créée une classe qui étend SplFileObject
class MyFoo extends SplFileObject {}

$info = new SplFileInfo(__FILE__);
// Configure la classe à utiliser
$info->setFileClass('MyFoo');
var_dump($info->openFile());
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

object(MyFoo)#2 (0) { }

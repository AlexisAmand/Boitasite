Yaf_Request_Http::getRaw

(Yaf >=3.0.7)

Yaf_Request_Http::getRaw — Récupère le corps brut de la requête

Description

public Yaf_Request_Http::getRaw ( void ) : mixed

Récupère le corps brut de la requête.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne chaîne de caractères en cas de réussite, FALSE en cas d'échec.

SplFixedArray::offsetUnset

(PHP 5 >= 5.3.0, PHP 7)

SplFixedArray::offsetUnset — Détruit l'élément à un index donné

Description

public SplFixedArray::offsetUnset ( int $index ) : void

Détruit l'élément à l'index index.

Liste de paramètres

index

L'index de l'élément à détruire.

Valeurs de retour

Aucune valeur n'est retournée.

Erreurs / Exceptions

Lance une exception RuntimeException lorsque index est en dehors de la taille du tableau ou lorsque index n'a pu être analysé comme un entier.

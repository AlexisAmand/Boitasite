fann_get_training_algorithm

(PECL fann >= 1.0.0)

fann_get_training_algorithm — Retourne l'algorithme d'entrainement

Description

fann_get_training_algorithm ( resource $ann ) : int

Retourne l'algorithme d'entrainement. Cet algorithme d'entrainement est utilisé par fann_train_on_data() et les fonctions associées.

Notez que cet algorithme est également utilisé pendant fann_cascadetrain_on_data(), bien que seules les constantes FANN_TRAIN_RPROP et FANN_TRAIN_QUICKPROP sont autorisées pendant l'entrainement en cascade.

Liste de paramètres

ann

Ressource de réseau de neurones.

Valeurs de retour

La constante de l'algorithme d'entrainement ou FALSE si une erreur survient.

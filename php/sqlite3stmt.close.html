SQLite3Stmt::close

(PHP 5 >= 5.3.0, PHP 7)

SQLite3Stmt::close — Ferme une requête préparée

Description

public SQLite3Stmt::close ( void ) : bool

Ferme une requête préparée.

Note: Notez que tous les SQLite3Result qui ont été récupérés en exécutant cette instruction seront invalidés lorsque l'instruction sera fermée.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE

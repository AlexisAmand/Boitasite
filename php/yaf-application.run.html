Yaf_Application::run

(Yaf >=1.0.0)

Yaf_Application::run — Démarre Yaf_Application

Description

public Yaf_Application::run ( void ) : void

Exécute une Yaf_Application, permet à Yaf_Application d'accepter une requête, et route la requête, gère le contrôleur et l'action, et affiche la réponse. Retourne la réponse au client.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

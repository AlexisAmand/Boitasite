ReflectionClass::export

(PHP 5, PHP 7)

ReflectionClass::export — Exporte une classe

Avertissement

Cette fonction est OBSOLÈTE à partir de PHP 7.4.0. Dépendre de cette fonction est fortement déconseillé.

Description

public static ReflectionClass::export ( mixed $argument [, bool $return = FALSE ] ) : string

Exporte une classe reflétée.

Liste de paramètres

argument

La réflection à exporter.

return

Le fait de le définir à TRUE retournera l'export plutôt que l'émission. Le définir à FALSE (par défaut) fera l'inverse.

Valeurs de retour

Si le paramètre return est défini à TRUE, l'export sera retourné sous la forme d'une chaîne de caractères, sinon, NULL sera retourné.

Exemples

Exemple #1 Utilisation basique de ReflectionClass::export()

<?php
class Apple {
    public $var1;
    public $var2 = 'Orange';

    public function type() {
        return 'Apple';
    }
}
ReflectionClass::export('Apple');
?>

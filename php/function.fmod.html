fmod

(PHP 4 >= 4.2.0, PHP 5, PHP 7)

fmod — Retourne le reste de la division

Description

fmod ( float $x , float $y ) : float

Retourne le reste de la division de x par y. Ce reste est un nombre à virgule flottante. Le reste (r) est défini par : x = i * y + r, pour un entier i. Si y n'est pas nul, r a le même signe que x et une taille inférieure à celle de y.

Liste de paramètres

x

Le dividende

y

Le diviseur

Valeurs de retour

Le reste de la division de x par y.

Exemples

Exemple #1 Exemple avec fmod()

<?php
$x = 5.7;
$y = 1.3;
$r = fmod($x, $y);
// $r vaut 0.5, parce que 4 * 1.3 + 0.5 = 5.7
?>

SplFileInfo::getMTime

(PHP 5 >= 5.1.2, PHP 7)

SplFileInfo::getMTime — Lit la date de dernière modification

Description

public SplFileInfo::getMTime ( void ) : int

Lit la date de dernière modification du contenu du fichier. La date retournée est au format timestamp Unix.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne au format timestamp Unix.

Event::set

(PECL event >= 1.2.6-beta)

Event::set — Re-configure l'événement

Description

public Event::set ( EventBase $base , mixed $fd [, int $what [, callable $cb [, mixed $arg ]]] ) : bool

Re-configure l'événement. Notez que cette fonction n'appelle pas la fonction obsolète event_set de libevent. Elle appelle plutot la fonction event_assign.

Liste de paramètres

base

La base d'événement à associer.

fd

Une ressource de flux, une ressource de socket, ou un descripteur numérique de fichier. Pour les événements de type minuteur passez -1. Pour les événements de type signal passez le numéro du signal, par exemple SIGHUP.

what

Voir les drapeaux d'événements.

cb

La fonction de rappel de l'événement. Voir les fonctions de rappel d'événements.

arg

Données personnalisées à associer avec l'événement. Elles seront passées à la fonction de rappel lorsque l'événement deviendra actif.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

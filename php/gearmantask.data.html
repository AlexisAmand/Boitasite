GearmanTask::data

(PECL gearman >= 0.5.0)

GearmanTask::data — Récupère les données retournées par une tâche

Description

public GearmanTask::data ( void ) : string

Retourne les données retournées d'une tâche par un agent.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Les données linéarisées, ou FALSE s'il n'y a aucune donnée.

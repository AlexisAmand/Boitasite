Locale::setDefault

locale_set_default

(PHP 5 >= 5.3.0, PHP 7, PECL intl >= 1.0.0)

Locale::setDefault -- locale_set_default — Configure la locale par défaut

Description

Style orienté objet

public static Locale::setDefault ( string $locale ) : bool

Style procédural

locale_set_default ( string $locale ) : bool

Configure la locale par défaut à $locale. Cela change la valeur de la globale INTL 'default_locale'. Les extensions UAX #35 sont aussi acceptées.

Liste de paramètres

locale

Une langue au format BCP 47.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Exemples

Exemple #1 Exemple avec locale_set_default(), procédural

<?php
locale_set_default('de-DE');
echo locale_get_default();
?>

Exemple #2 Exemple avec locale_set_default(), POO

<?php
Locale::setDefault('de-DE');
echo Locale::getDefault();
?>

L'exemple ci-dessus va afficher :

de-DE

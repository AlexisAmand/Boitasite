MultipleIterator::key

(PHP 5 >= 5.3.0, PHP 7)

MultipleIterator::key — Récupère les instances des itérateurs enregistrés

Description

public MultipleIterator::key ( void ) : array

Récupère les clés des instances des itérateurs enregistrés.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Un tableau de tous les itérateurs enregistrés, ou FALSE si aucun sous-itérateur n'est attaché.

Erreurs / Exceptions

Une exception LogicException si le mode MIT_NEED_ALL est défini, et au moins un itérateur attaché invalide.

L'appel à cette méthode depuis foreach provoque une alerte de type "Illegal type returned".

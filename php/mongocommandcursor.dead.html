MongoCommandCursor::dead

(PECL mongo >=1.5.0)

MongoCommandCursor::dead — Vérifie s'il y a des documents qui n'ont pas encore été envoyés depuis la base de données

Description

public MongoCommandCursor::dead ( void ) : bool

Cette méthode vérifie si le curseur MongoCommandCursor s'est erminé, et que la base de données n'a plus de résultats à envoyer au client. Un curseur devenu "mort" ne signifie pas nécessairement qu'il n'y a plus de résultats disponibles pour son itération.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE s'il y a encore des résultats qui n'ont pas été envoyés au client, FALSE sinon.

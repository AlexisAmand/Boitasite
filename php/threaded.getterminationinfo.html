Threaded::getTerminationInfo

(PECL pthreads < 3.0.0)

Threaded::getTerminationInfo — Détection d'erreur

Avertissement

Cette méthode a été supprimée dans pthreads v3. Au lieu de cela, le corps de threaded::run() peut être enveloppé dans un bloc try...catch pour détecter les erreurs (car la plupart des erreurs dans PHP 7 ont été convertis en exceptions).

Description

public Threaded::getTerminationInfo ( void ) : array

Récupère les informations d'une erreur depuis l'objet référencé

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Un tableau contenant les informations de l'erreur de l'objet référencé.

Exemples

Exemple #1 Détection des erreurs fatales dans Threads

<?php
class My extends Thread {
    public function run() {
        @not_found();
    }
}

$my = new My();
$my->start();
$my->join();

var_dump($my->isTerminated(), $my->getTerminationInfo());
?>

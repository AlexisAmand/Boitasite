zlib_encode

(PHP 5 >= 5.4.0, PHP 7)

zlib_encode — Compresse des données avec l'encodage spécifié

Description

zlib_encode ( string $data , int $encoding [, int $level = -1 ] ) : string

Compresse des données avec l'encodage spécifié.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

data

Les données à compresser.

encoding

L'algorithme de compression. Soit ZLIB_ENCODING_RAW, ZLIB_ENCODING_DEFLATE ou ZLIB_ENCODING_GZIP.

level

Valeurs de retour

Exemples

Exemple #1 Exemple avec zlib_encode()

<?php
$str = 'hello world';
$enc = zlib_encode($str, ZLIB_ENCODING_DEFLATE);
echo bin2hex($enc);
?>

L'exemple ci-dessus va afficher :

789ccb48cdc9c95728cf2fca4901001a0b045d

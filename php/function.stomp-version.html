stomp_version

(PECL stomp >= 0.1.0)

stomp_version — Récupère la version de l'extension Stomp

Description

stomp_version ( void ) : string

Retourne une chaîne contenant la version de l'extension Stomp.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Version de l'extension Stomp.

Exemples

Exemple #1 Exemple avec stomp_version()

<?php

var_dump(stomp_version());

?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

string(5) "0.2.0"

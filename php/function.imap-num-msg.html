imap_num_msg

(PHP 4, PHP 5, PHP 7)

imap_num_msg — Retourne le nombre de messages dans la boîte aux lettres courante

Description

imap_num_msg ( resource $imap_stream ) : int

Retourne le nombre de messages dans la boîte aux lettres courante.

Liste de paramètres

imap_stream

Un flux IMAP retourné par la fonction imap_open().

Valeurs de retour

Retourne le nombre de messages dans la boîte aux lettres courante, sous la forme d'un entier, ou FALSE en cas d'erreur.

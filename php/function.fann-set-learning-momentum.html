fann_set_learning_momentum

(PECL fann >= 1.0.0)

fann_set_learning_momentum — Défini la dynamique d'apprentissage

Description

fann_set_learning_momentum ( resource $ann , float $learning_momentum ) : bool

Défini la dynamique d'apprentissage.

Plus d'informations disponible sur la page de documentation de la fonction fann_get_learning_momentum().

Liste de paramètres

ann

Ressource de réseau de neurones.

learning_momentum

La dynamique d'apprentissage.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

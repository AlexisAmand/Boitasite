maxdb_real_query

maxdb::real_query

(PECL maxdb >= 1.0)

maxdb_real_query -- maxdb::real_query — Exécute une requête SQL

Description

Style procédural

maxdb_real_query ( resource $link , string $query ) : bool

Style orienté objet

maxdb::real_query ( string $query ) : bool

maxdb_real_query() est identique au niveau fonctionnalité que la fonction maxdb_query().

Note:

Pour déterminer si une requête donnée doit retourner un jeu de résultats ou non, référez-vous à la fonction maxdb_field_count().

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

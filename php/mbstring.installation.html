Installation

mbstring est une extension PHP. L'extension doit être activé avec l'option configure. Voir la section installation pour plus de détails.

Les options de configurations suivantes sont liées à l'extension mbstring.

  • --enable-mbstring : Active les fonctions mbstring. Cette option est nécessaire pour utiliser les fonctions mbstring.

    libmbfl est nécessaire pour mbstring. libmbfl est inclus avec mbstring. Antérieur à PHP 7.3.0, si libmbfl est déjà installé sur le système, --with-libmbfl[=DIR] peut être spécifié pour utiliser la bibliothèque installée.

  • --disable-mbregex : Désactive les fonctions d'expression régulières avec le support de charactère multi-octets.

    Oniguruma est nécessaire pour les fonctions d'expression régulière avec le support de charactère multi-octets. Oniguruma est inclus avec mbstring. À partir de PHP 5.4.0, si Oniguruma est déjà installé sur le système, --with-onig[=DIR] peut être spécifié pour utiliser la bibliothèque installée. À partir de PHP 7.4.0 --with-onig a été supprimée et pkg-config est désormais utilisé pour détecter la bibliothèque libonig.

    À partir de PHP 5.4.0 il est possible de désactiver la vérification backtrack (retour sur ses pas) en spécifiant --disable-mbregex-backtrack.

SolrQuery::setStats

(PECL solr >= 0.9.2)

SolrQuery::setStats — Active ou désactive le composant statistique

Description

public SolrQuery::setStats ( bool $flag ) : SolrQuery

Active ou désactive le composant statistique.

Liste de paramètres

flag

TRUE pour activer le composant statistique, FALSE pour le désactiver.

Valeurs de retour

Retourne l'objet SolrQuery courant, si la valeur retournée est utilisée.

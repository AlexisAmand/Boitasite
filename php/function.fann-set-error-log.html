fann_set_error_log

(PECL fann >= 1.0.0)

fann_set_error_log — Défini l'endroit où les erreurs seront historisées

Description

fann_set_error_log ( resource $errdat , string $log_file ) : void

Défini l'endroit où les erreurs seront historisées.

Liste de paramètres

errdat

Soit une ressource de réseau de neurones, soit une ressource de données d'entrainement d'un réseau de neurones.

log_file

Le chemin du fichier d'historisation.

Valeurs de retour

Aucune valeur retournée.

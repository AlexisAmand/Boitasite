RecursiveCallbackFilterIterator::__construct

(PHP 5 >= 5.4.0, PHP 7)

RecursiveCallbackFilterIterator::__construct — Crée un objet RecursiveCallbackFilterIterator depuis une interface RecursiveIterator

Description

public RecursiveCallbackFilterIterator::__construct ( RecursiveIterator $iterator , string $callback )

Crée un itérateur filtré depuis une interface RecursiveIterator en utilisant la fonction de rappel callback pour déterminer les éléments acceptés ou rejetés.

Liste de paramètres

iterator

L'itérateur récursif à filtrer.

callback

La fonction de rappel, qui doit retourner TRUE pour accepter l'élément courant, FALSE sinon. Voir aussi les exemples.

Peut être n'importe quelle valeur de type callable.

Valeurs de retour

Aucune valeur n'est retournée.

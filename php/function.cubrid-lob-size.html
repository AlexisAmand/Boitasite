cubrid_lob_size

(PECL CUBRID >= 8.3.1)

cubrid_lob_size — Récupère la taille des données BLOB/CLOB

Description

cubrid_lob_size ( resource $lob_identifier ) : string

La fonction cubrid_lob_size() est utilisée pour récupérer la taille des données BLOB/CLOB.

Liste de paramètres

lob_identifier

Identifiant LOB.

Valeurs de retour

Une chaîne représentant la taille des données LOB, lorsque l'opération a réussi.

FALSE en cas d'échec.

Historique

Version Description
8.4.0 Le type de la valeur retourné à changer. Avant, c'était un entier, maintenant, c'est une chaîne de caractères.

Exemples

Exemple #1 Exemple avec cubrid_lob_size()

<?php
$lobs = cubrid_lob_get($con, "SELECT doc_content FROM doc WHERE doc_id=5");
echo "Taille de la documentation :".cubrid_lob_size($lobs[0]);
cubrid_lob_export($conn, $lobs[0], "doc_5.txt");
cubrid_lob_close($lobs);
?>

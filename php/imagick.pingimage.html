Imagick::pingImage

(PECL imagick 2.0.0)

Imagick::pingImage — Lit des attributs simples sans ouvrir une image

Description

Imagick::pingImage ( string $filename ) : bool

Cette méthode peut être utilisée pour obtenir les dimensions de l'image sans lire toute l'image en mémoire.

Liste de paramètres

filename

Le nom du fichier qui contient l'image.

Valeurs de retour

Retourne TRUE en cas de succès.

MongoDB\BSON\Javascript::unserialize

(mongodb >=1.2.0)

MongoDB\BSON\Javascript::unserialize — Désérialisé un Javascript

Description

final public MongoDB\BSON\Javascript::unserialize ( string $serialized ) : void

Liste de paramètres

serialized

Le MongoDB\BSON\Javascript sérialisé.

Valeurs de retour

Retourne le MongoDB\BSON\Javascript désérialisé.

Erreurs / Exceptions

  • Lance une exception MongoDB\Driver\InvalidArgumentException lors d'une erreur survenue pendant l'analyse d'un argument.
  • Lance MongoDB\Driver\Exception\UnexpectedValueException si les propriétés ne peuvent pas être désérialisées (par exemple si serialized a été mal formé).
  • Lance MongoDB\Driver\Exception\InvalidArgumentException si les propriétés sont incorrectes (par exemple des champs manquant ou des valeurs incorrectes).

LimitIterator::next

(PHP 5 >= 5.1.0, PHP 7)

LimitIterator::next — Déplace l'itérateur à la position suivante

Description

public LimitIterator::next ( void ) : void

Déplace l'itérateur à la position suivante.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

cubrid_lob2_new

(PECL CUBRID >= 8.4.1)

cubrid_lob2_new — Crée un nouvel objet LOB

Description

cubrid_lob2_new ([ resource $conn_identifier [, string $type = "BLOB" ]] ) : resource

La fonction cubrid_lob2_new() est utilisée pour créer un nouvel objet LOB (à la fois BLOB et CLOB). Cette fonction doit être utilisée avant de lier un objet LOB.

Liste de paramètres

conn_identifier

Un identifiant de connexion. S'il n'est pas spécifié, la dernière connexion ouverte avec la fonction cubrid_connect() ou la fonction cubrid_connect_with_url() sera utilisée.

type

Peut être "BLOB" ou "CLOB", et n'est pas sensible à la casse. La valeur par défaut est "BLOB".

Valeurs de retour

Un identifiant LOB en cas de succès.

FALSE si une erreur survient.

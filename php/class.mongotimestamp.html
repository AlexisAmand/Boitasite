La classe MongoTimestamp

(PECL mongo >=1.0.1)

Avertissement

Cette extension, qui définie cette classe est obsolète. Veuillez utiliser l'extension MongoDB à la place. Les alertnatives à cette classe sont :

  • MongoDB\BSON\Timestamp

Introduction

MongoTimestamp est un type interne utilisé par MongoDB pour la réplication et le partage. Il consiste en un timestamp sur 4 octets (i.e. secondes depuis l'époque Unix) et un incrément sur 4 octets. Ce type n'est pas conçu pour stocker une durée ou une valeur de date (i.e. champ "createdAt" d'un document).

Note: A moins que vous n'écriviez une application qui intéragit avec la réplication et le partage internes de MongoDB : STOP ! Allez directement à la méthode MongoDate, ne passer pas par la casse de départ, et toucher 20 000 euros ! Ce n'est pas la classe que vous cherchez ;)

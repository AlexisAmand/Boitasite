Memcached::getAllKeys

(PECL memcached >= 2.0.0)

Memcached::getAllKeys — Récupère toutes les clés stockés sur tous les serveurs

Description

public Memcached::getAllKeys ( void ) : array

Memcached::getAllKeys() interroge chaque serveur memcache et récupère un tableau contenant toutes les clés stockés sur chacun d'eux. Ce n'est pas une opération atomique, aussi, ce n'est pas une image réellement consistente des clés à l'instant donné. Sachant que memcache ne garantie pas de retourner tous les clés, vous ne pouvez pas être certain que toutes les clés ont bien été retournées.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne les clés stockées sur tous les serveurs en cas de succès ou FALSE si une erreur survient.

PDF_get_pdi_value

(PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0)

PDF_get_pdi_value — Lit des paramètres numériques dans le document PDF d'entrée (obsolète)

Description

PDF_get_pdi_value ( resource $p , string $key , int $doc , int $page , int $reserved ) : float

PDF_get_pdi_value() lit des paramètres numériques dans le document PDF d'entrée.

Cette fonction est obsolète depuis PDFlib version 7, utilisez plutôt la fonction PDF_pcos_get_number().

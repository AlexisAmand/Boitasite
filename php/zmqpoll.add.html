ZMQPoll::add

(PECL zmq >= 0.5.0)

ZMQPoll::add — Ajoute un élément dans le poll

Description

public ZMQPoll::add ( mixed $entry , int $type ) : string

Ajoute un nouvel élément dans le poll, et retourne l'ID interne de l'élément ajouté. L'élément peut être supprimé du poll en utilisant cet ID.

Liste de paramètres

entry

Objet ZMQSocket ou une ressource de flux PHP

type

Défini l'activité prévue du socket. Voir les constantes ZMQ::POLL_IN et ZMQ::POLL_OUT.

Valeurs de retour

Retourne l'ID de l'élément ajouté, qui pourra être utilisé par la suite pour supprimer l'élément. Lance une exception ZMQPollException en cas d'erreur.

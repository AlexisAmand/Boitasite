SolrQuery::getMltMinTermFrequency

(PECL solr >= 0.9.2)

SolrQuery::getMltMinTermFrequency — Retourne la fréquence en deçà duquel les termes doivent être ignorés dans le document source

Description

public SolrQuery::getMltMinTermFrequency ( void ) : int

Retourne la fréquence en deçà duquel les termes doivent être ignorés dans le document source.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un entier en cas de succès, et NULL si la fréquence n'est pas définie.

maxdb_next_result

maxdb::next_result

(PECL maxdb >= 1.0)

maxdb_next_result -- maxdb::next_result — Prépare le prochain résultat depuis multi_query

Description

maxdb_next_result ( resource $link ) : bool

Vu que les requêtes multiples ne sont pas encore implémentées, maxdb_next_result() retourne toujours FALSE.

Valeurs de retour

Retourne FALSE.

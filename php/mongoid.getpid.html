MongoId::getPID

(PECL mongo >= 1.0.11)

MongoId::getPID — Récupère l'identifiant du processus

Cette extension, qui définie cette méthode est obsolète. Veuillez utiliser l'extension MongoDB à la place. Il n'y a pas d'équivalent à cette méthode dans la nouvelle extension.

Description

public MongoId::getPID ( void ) : int

Extrait le PID depuis l'ID Mongo.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne le PID de MongoId.

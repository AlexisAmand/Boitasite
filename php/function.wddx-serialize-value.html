wddx_serialize_value

(PHP 4, PHP 5, PHP 7)

wddx_serialize_value — Enregistre une valeur dans un paquet WDDX

Avertissement

Cette fonction est a été SUPPRIMÉE à partir de PHP 7.4.0.

Description

wddx_serialize_value ( mixed $var [, string $comment ] ) : string

Enregistre une valeur donnée dans un paquet WDDX.

Liste de paramètres

var

La valeur à linéariser.

comment

Une chaîne de caractères optionnelle contenant un commentaire qui apparaît dans l'en-tête du paquet.

Valeurs de retour

Retourne le paquet WDDX, ou FALSE si une erreur survient.

Imagick::sampleImage

(PECL imagick 2.0.0)

Imagick::sampleImage — Met à l'échelle une image avec échantillonnage de pixels

Description

Imagick::sampleImage ( int $columns , int $rows ) : bool

Met à l'échelle une image aux dimensions désirée, à l'aide de la méthode d'échantillonnage de pixels. Contrairement aux autres méthodes, cette méthode n'introduit pas de nouvelle couleur dans l'image.

Liste de paramètres

columns

rows

Valeurs de retour

Retourne TRUE en cas de succès.

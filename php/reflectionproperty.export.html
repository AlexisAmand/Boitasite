ReflectionProperty::export

(PHP 5, PHP 7)

ReflectionProperty::export — Exporte

Description

public static ReflectionProperty::export ( mixed $class , string $name [, bool $return ] ) : string

Exporte une réflexion.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

argument

La réflection à exporter.

name

Le nom de la propriété.

return

Le fait de le définir à TRUE retournera l'export plutôt que l'émission. Le définir à FALSE (par défaut) fera l'inverse.

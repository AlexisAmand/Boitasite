MongoDB\Driver\Monitoring\removeSubscriber

(mongodb >=1.3.0)

MongoDB\Driver\Monitoring\removeSubscriber — Retire un abonnement de surveillance d'événement existant

Description

MongoDB\Driver\Monitoring\removeSubscriber ( MongoDB\Driver\Monitoring\Subscriber $subscriber ) : void

Retire un abonnement de surveillance d'événement existant du pilote. Les abonnements retiré ne seront plus notifié d'événements de surveillance.

Note: Si l'objet n'est pas enregistré, cette fontion est un no-op (pas d'opération).

Liste de paramètres

subscriber (MongoDB\Driver\Monitoring\Subscriber)

Un objet d'abonnement de surveillance d'événement à retirer.

Valeurs de retour

Aucune valeur n'est retournée.

Erreurs / Exceptions

  • Lance une exception MongoDB\Driver\InvalidArgumentException lors d'une erreur survenue pendant l'analyse d'un argument.

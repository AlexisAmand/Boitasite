SphinxClient::open

(PECL sphinx >= 1.0.3)

SphinxClient::open — Établit une connexion persistante avec le serveur

Description

public SphinxClient::open ( void ) : bool

Établit une connexion persistante avec le serveur.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Historique

Version PECL/sphinx Description
1.0.3 Ajout de SphinxClient::open(), uniquement disponible si compilé avec libsphinxclient >= 0.9.9.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

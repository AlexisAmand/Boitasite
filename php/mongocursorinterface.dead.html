MongoCursorInterface::dead

(PECL mongo >=1.5.0)

MongoCursorInterface::dead — Vérifie s'il y a des résultats qui n'ont pas encore été envoyés depuis la base de données

Description

abstract public MongoCursorInterface::dead ( void ) : bool

Cette méthode vérifie si le curseur est à la fin, et que la base de données n'a plus de résultats à envoyer au client. Un curseur devenu "mort" ne signifie pas obligatoirement qu'il n'y a plus de résultats disponibles pour l'itération.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE s'il y a encore des résultats qui n'ont pas encore été envoyés au client, FALSE sinon.

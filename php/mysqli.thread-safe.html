mysqli::thread_safe

mysqli_thread_safe

(PHP 5, PHP 7)

mysqli::thread_safe -- mysqli_thread_safe — Indique si le support des threads est activé ou pas

Description

Style orienté objet

public mysqli::thread_safe ( void ) : void

Style procédural

mysqli_thread_safe ( void ) : bool

Indique si la bibliothèque cliente est compilée avec la sécurité thread.

Valeurs de retour

TRUE si la bibliothèque cliente est thread-safe, FALSE sinon.

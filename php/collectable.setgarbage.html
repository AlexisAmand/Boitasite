Collectable::setGarbage

(PECL pthreads < 3.0.0)

Collectable::setGarbage — Marque un objet comme obsolète

Avertissement

Cette méthode a été supprimée dans pthreads v3.

Description

public Collectable::setGarbage ( void ) : void

Doit être appelé une fois par objet, lorsque l'objet a terminé son exécution ou son référencement.

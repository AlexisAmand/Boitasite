Mongo::getPoolSize

(PECL mongo >=1.2.0)

Mongo::getPoolSize — Récupère la taille de la file d'attente pour les connexions

Description

public static Mongo::getPoolSize ( void ) : int
Avertissement

Cette fonctionnalité est devenue OBSOLÈTE depuis la version 1.2.3. Nous vous encourageons vivement à ne plus l'utiliser. Utilisez la méthode MongoPool::getSize() à la place.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la taille de la file d'attente pour les connexions.

Historique

Version Description
1.2.11 Lance une alerte de niveau E_DEPRECATED lorsqu'utilisé.

Exemples

Exemple #1 Changer la taille de la file

Cet exemple retourne la taille de la file par défaut, change cette taille puis affiche la nouvelle taille de file et des informations de débogage. Notez que changer la taille de la file n'affecte que les nouvelles files de connections, pas les anciennes.

<?php

$connection = new Mongo("host1");

// Taille de la file : -1
echo "la taille de la file est de: ".Mongo::getPoolSize()."\n";

echo "Changement de la taille de la file à 200\n";

Mongo::setPoolSize(200);

// Taille de la file : 200
echo "la taille de la file est de: ".Mongo::getPoolSize()."\n";

$conn2 = new Mongo("host2");

// Il reste -2 pour host1
// Il reste 199 pour host2
var_dump(Mongo::poolDebug());

?>

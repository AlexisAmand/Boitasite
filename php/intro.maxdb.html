Introduction

L'extension MaxDB de PHP vous permet d'accéder aux fonctionnalités fournies par MaxDB 7.5.0 et suivant. Plus d'informations sur le serveur de base de données MaxDB peuvent être trouvées sur » http://www.sdn.sap.com/irj/sdn/maxdb.

L'extension MaxDB est compatible avec l'extension mysqli de MySQL. Il y a simplement des différences mineurs dans le comportement de certaines fonctions, liées au comportement des bibliothèques sous- jacentes.

Les différences notables par rapport à MySQLi sont :

  • maxdb_character_set_name() : retourne uniquement de l'ASCII ou unicode.
  • maxdb_get_client_info() : retourne une chaîne de version différente.
  • maxdb_get_client_version() : retourne une chaîne de version différente.
  • maxdb_get_host_info() : retourne localhost ou le nom de l'hôte.
  • maxdb_get_server_info() : retourne une chaîne de version différente.
  • maxdb_get_server_version() : retourne une chaîne de version différente.
  • maxdb_kill() : déconnecte uniquement la session.
  • maxdb_multi_query() : ne peut pas gérer les requêtes multiples SQL.
  • maxdb_next_result() : la fonction retourne toujours FALSE.
  • maxdb_options() : supporte un jeu d'options différentes.
  • maxdb_report() : supporte un mode de rapport différent.
  • maxdb_stat() : retourne une chaîne de statut système différente.
  • maxdb_stmt_store_result() : n'est pas nécessaire pour MaxDB.
  • maxdb_store_result() : n'est pas nécessaire pour MaxDB.

La documentation pour MaxDB peut être trouvée sur » http://maxdb.sap.com/documentation/.

ssh2_disconnect

(PECL ssh2 >= 1.0)

ssh2_disconnect — Ferme une connexion à un serveur SSH distant

Description

ssh2_disconnect ( resource $session ) : bool

Ferme une connexion à un serveur SSH distant.

Liste de paramètres

session

Un identifiant de lien de connexion SSH, obtenu depuis un appel à ssh2_connect().

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

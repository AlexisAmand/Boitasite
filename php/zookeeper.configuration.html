Configuration à l'exécution

Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.

Zookeeper Options de configuration
Nom Défaut Modifiable Changelog
zookeeper.recv_timeout 10000 PHP_INI_ALL  
zookeeper.session_lock 1 PHP_INI_SYSTEM  
zookeeper.sess_lock_wait 150000 PHP_INI_ALL  
Pour plus de détails sur les modes PHP_INI_*, reportez-vous à Où une directive de configuration peut être modifiée.

Voici un éclaircissement sur l'utilisation des directives de configuration.

zookeeper.recv_timeout integer

Délai maximum d'attente par defaut pour toutes les sessions de ZooKeeper.

zookeeper.session_lock integer

Autorise le verouillage des sessions PHP

zookeeper.sess_lock_wait integer

Temps d'attente en microsecondes pour les tentatives de vérouillage de la session PHP. Soyez prudent lors de la définition de cette valeur. Les valeurs valides sont des entiers, où 0 est interprété comme la valeur par défaut. Les valeurs négatives entraînent un verrouillage réduit à une tentative de verrouillage.

mysqli::reap_async_query

mysqli_reap_async_query

(PHP 5 >= 5.3.0, PHP 7)

mysqli::reap_async_query -- mysqli_reap_async_query — Lit un résultat pour une requête asynchrone

Description

Style orienté objet

public mysqli::reap_async_query ( void ) : mysqli_result

Style procédural

mysqli_reap_async_query ( mysqli $link ) : mysqli_result

Lit le résultat pour une requête asynchrone. Disponible uniquement avec mysqlnd.

Liste de paramètres

link

Seulement en style procédural : Un identifiant de lien retourné par la fonction mysqli_connect() ou par la fonction mysqli_init()

Valeurs de retour

Retourne un objet mysqli_result en cas de succès, et FALSE sinon.

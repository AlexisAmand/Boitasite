Exception::__construct

(PHP 5, PHP 7)

Exception::__construct — Construit l'exception

Description

public Exception::__construct ([ string $message = "" [, int $code = 0 [, Throwable $previous = NULL ]]] )

Construit l'exception.

Liste de paramètres

message

Le message de l'exception à lancer.

code

Le code de l'exception.

previous

L'exception précédente, utilisée pour le chaînage d'exception.

Note: L'appel du constructeur de l'exception de classe à partir d'une sous-classe ignore les arguments par défaut, si les propriétés $code et $message sont déjà définies.

Historique

Version Description
7.0.0 Le paramètre previous est maintenant de type Throwable.
5.3.0 Le paramètre previous a été ajouté.

Notes

Note:

Le message n'est PAS binary safe.

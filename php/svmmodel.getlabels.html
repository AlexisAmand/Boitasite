SVMModel::getLabels

(PECL svm >= 0.1.5)

SVMModel::getLabels — Récupère les labels utilisés pour entraîner le modèle

Description

public SVMModel::getLabels ( void ) : array

Retourne un tableau de labels utilisés pour entraîner le modèle. Pour les cas de régression et pour les classes mobèles seules, un tableau vide est retourné.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un tableau de labels.

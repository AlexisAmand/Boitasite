Pré-requis

L'extension ZMQ requiert 0MQ en version 2.0.7 ou supérieures. 0MQ peut être obtenu depuis » http://zeromq.org/area:download.

Installation

Des informations sur l'installation de ces extensions PECL peuvent être trouvées dans le chapitre du manuel intitulé Installation des extensions PECL. D'autres informations comme les notes sur les nouvelles versions, les téléchargements, les sources des fichiers, les informations concernant les mainteneurs ainsi qu'un CHANGELOG, peuvent être trouvées ici : » https://pecl.php.net/package/zmq

DirectoryIterator::getType

(PHP 5, PHP 7)

DirectoryIterator::getType — Récupère le type d'un fichier

Description

public DirectoryIterator::getType ( void ) : string

Détermine le type du fichier courant dans l'objet DirectoryIterator.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne une chaîne de caractères représentant le type de fichier. Cela peut être file (fichier normal), link (lien), ou dir (dossier).

Exemples

Exemple #1 Exemple avec DirectoryIterator::getType()

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
foreach ($iterator as $fileinfo) {
    echo $fileinfo->getFilename() . " " . $fileinfo->getType() . "\n";
}
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

. dir
.. dir
apple.jpg file
banana.jpg file
example.php file
pear.jpg file

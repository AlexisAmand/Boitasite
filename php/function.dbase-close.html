dbase_close

(PHP 5 < 5.3.0, dbase 5, dbase 7)

dbase_close — Ferme une base dBase

Description

dbase_close ( resource $database ) : bool

dbase_close() ferme la base de données correspondant à la ressource database.

Liste de paramètres

database

La ressource de base de données, retournée par dbase_open() ou dbase_create().

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Historique

Version Description
dbase 7.0.0 database est maintenant une resource au lieu d'un int.

Exemples

Exemple #1 Ferme un fichier de base de données dBase

<?php
                   
// Ouverture en mode lecture-écriture
$db = dbase_open('/tmp/test.dbf', 0);

if ($db) {
// Lecture des données ..

dbase_close($db);
}

?>

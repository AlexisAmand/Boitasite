phpdbg_break_next

(PHP 5 >= 5.6.3, PHP 7)

phpdbg_break_next — Insère un point d'arrêt au prochain opcode

Description

phpdbg_break_next ( void ) : void

Insère un point d'arrêt au prochain opcode.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

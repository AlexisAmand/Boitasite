Yaf_View_Simple::clear

(Yaf >=2.2.0)

Yaf_View_Simple::clear — Efface les valeurs assignées

Description

public Yaf_View_Simple::clear ([ string $name ] ) : bool

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

name

Nom de la variable assignée.

Si vide, toutes les variables assignées seront effacées.

Valeurs de retour

Exemples

Exemple #1 Exemple avec Yaf_View_Simple::clear()

<?php
class IndexController extends Yaf_Controller_Abstract {
    public function indexAction() {
        $this->getView()->clear("foo")->clear("bar"); // Efface "foo" et "bar"
        $this->_view->clear(); // Efface toutes les variables assignées
    }
?>

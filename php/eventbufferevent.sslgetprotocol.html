EventBufferEvent::sslGetProtocol

(PECL event >= 1.10.0)

EventBufferEvent::sslGetProtocol — Retourne le nom du protocole utilisé pour la connexion SSL courante

Description

public EventBufferEvent::sslGetProtocol ( void ) : string

Retourne le nom du protocole utilisé pour la connexion SSL courante.

Note:

Cette fonction n'est disponible que si Event a été compilé avec le support OpenSSL.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne le nom du protocole utilisé pour la connexion SSL courante.

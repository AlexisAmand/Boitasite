connection_aborted

(PHP 4, PHP 5, PHP 7)

connection_aborted — Indique si l'internaute a abandonné la connexion HTTP

Description

connection_aborted ( void ) : int

Indique si l'internaute a abandonné la connexion HTTP.

Valeurs de retour

Retourne 1 si le client est déconnecté, 0 sinon.

ReflectionClass::getTraits

(PHP 5 >= 5.4.0, PHP 7)

ReflectionClass::getTraits — Retourne un tableau des traits utilisés par cette classe

Description

public ReflectionClass::getTraits ( void ) : array

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un tableau avec le nom du trait en clé et des instances ReflectionClass des traits en question en valeurs. Retourne NULL en cas d'erreur.

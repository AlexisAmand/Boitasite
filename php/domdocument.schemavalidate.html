DOMDocument::schemaValidate

(PHP 5, PHP 7)

DOMDocument::schemaValidate — Valide un document selon un schéma

Description

public DOMDocument::schemaValidate ( string $filename [, int $flags = 0 ] ) : bool

Valide un document selon un schéma contenu dans un fichier donné.

Liste de paramètres

filename

Le chemin vers le fichier contenant le schéma.

flags

Un masque de drapeaux de validation du schéma Libxml. Actuellement, la seule valeur supportée est LIBXML_SCHEMA_CREATE. Disponible depuis PHP 5.5.2 et Libxml 2.6.14.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

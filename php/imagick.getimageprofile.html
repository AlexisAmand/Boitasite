Imagick::getImageProfile

(PECL imagick 2.0.0)

Imagick::getImageProfile — Retourne un profil d'une image

Description

Imagick::getImageProfile ( string $name ) : string

Retourne un profil d'une image, à partir de son nom.

Liste de paramètres

name

Le nom du profil à lire.

Valeurs de retour

Retourne une chaîne contenant le profile de l'image.

Erreurs / Exceptions

Lance une exception ImagickException si une erreur survient.

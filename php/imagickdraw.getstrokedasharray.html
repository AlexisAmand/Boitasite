ImagickDraw::getStrokeDashArray

(PECL imagick 2.0.0)

ImagickDraw::getStrokeDashArray — Retourne un tableau représentant le motif de pointillés

Description

ImagickDraw::getStrokeDashArray ( void ) : array
Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Retourne un tableau représentant le motif de pointillés, les vides et les pleins.

Valeurs de retour

Retourne un tableau en cas de succès, et un tableau vide sinon.

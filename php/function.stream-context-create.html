stream_context_create

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

stream_context_create — Crée un contexte de flux

Description

stream_context_create ([ array $options [, array $params ]] ) : resource

Crée et retourne un contexte de flux, avec les paramètres fournis par options.

Liste de paramètres

options

Doit être un tableau associatif, au format $arr['wrapper']['option'] = $value. Référez-vous aux options de contexte pour une liste des wrappers et des options disponibles.

Par défaut, c'est un tableau vide.

params

Doit être un tableau associatif de format $arr['parameter'] = $value. Référez-vous à la documentation sur les paramètres de contexte pour une liste des paramètres de flux standards.

Valeurs de retour

Une ressource représentant le contexte du flux.

Exemples

Exemple #1 Exemple avec stream_context_create()

<?php
$opts = array(
  'http'=>array(
    'method'=>"GET",
    'header'=>"Accept-language: en\r\n" .
              "Cookie: foo=bar\r\n"
  )
);

$context = stream_context_create($opts);

/* Envoi une requête HTTP vers www.example.com
   avec les en-têtes additionnels ci-dessus */
$fp = fopen('http://www.example.com', 'r', false, $context);
fpassthru($fp);
fclose($fp);
?>

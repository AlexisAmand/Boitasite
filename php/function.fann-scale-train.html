fann_scale_train

(PECL fann >= 1.0.0)

fann_scale_train — Met à l'échelle les données d'entrée et de sortie en se basant sur les paramètres précédemment calculés

Description

fann_scale_train ( resource $ann , resource $train_data ) : bool

Met à l'échelle les données d'entrée et de sortie en se basant sur les paramètres précédemment calculés.

Liste de paramètres

ann

Ressource de réseau de neurones.

train_data

Ressource de données d'entrainement du réseau de neurones.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

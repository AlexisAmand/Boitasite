symlink

(PHP 4, PHP 5, PHP 7)

symlink — Crée un lien symbolique

Description

symlink ( string $target , string $link ) : bool

symlink() crée un lien symbolique pour l'objet target avec le nom de link.

Liste de paramètres

target

La cible du lien.

link

Le nom du lien.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Exemples

Exemple #1 Création d'un lien symbolique

<?php
$target = 'uploads.php';
$link = 'uploads';
symlink($target, $link);

echo readlink($link);
?>

Notes

Note: Les utilisateurs de Windows doivent garder à l'esprit que cette fonction n'est disponible que pour Windows Vista/Windows Server 2008 ou supérieur. Pour les versions antérieures de Windows, cette fonction ne supporte pas les liens symboliques.

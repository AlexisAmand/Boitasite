EvIdle::__construct

(PECL ev >= 0.2.0)

EvIdle::__construct — Construit l'objet observateur EvIdle

Description

public EvIdle::__construct ( callable $callback [, mixed $data [, int $priority ]] )

Construit l'objet observateur EvIdle et démarre automatiquement l'observateur.

Liste de paramètres

callback

Voir les fonctions de rappel des observateurs.

data

Données personnalisées associées avec l'observateur.

priority

Priorité de l'observateur

Valeurs de retour

Returne un objet EvIdle en cas de succès.

  • EvLoop::idle() - Crée un objet EvIdle watcher associé avec l'instance de la boucle de l'événement courant

trader_set_compat

(PECL trader >= 0.2.2)

trader_set_compat — Définit le mode de compatibilité

Description

trader_set_compat ( int $compatId ) : void

Définit le mode de compatibilité qui va affecter la façon dont les calculs vont être effectués par toutes les fonctions de l'extension.

Liste de paramètres

compatId

Identifiant de compatibilité. Les constantes de la série TRADER_COMPATIBILITY_* doivent être utilisées.

Valeurs de retour

Aucune valeur n'est retournée.

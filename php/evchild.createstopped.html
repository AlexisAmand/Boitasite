EvChild::createStopped

(PECL ev >= 0.2.0)

EvChild::createStopped — Crée une instance de l'observateur stoppé EvCheck

Description

final public static EvChild::createStopped ( int $pid , bool $trace , callable $callback [, mixed $data [, int $priority ]] ) : object

Identique à EvChild::__construct(), mais ne démarre pas automatiquement l'observateur.

Liste de paramètres

pid

Identique à celui utilisé pour EvChild::__construct()

trace

Identique à celui utilisé pour EvChild::__construct()

callback

Voir les fonctions de rappel des observateurs.

data

Données personnalisées associées avec l'observateur.

priority

Priorité de l'observateur

Valeurs de retour

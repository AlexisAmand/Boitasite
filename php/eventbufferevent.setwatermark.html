EventBufferEvent::setWatermark

(PECL event >= 1.2.6-beta)

EventBufferEvent::setWatermark — Active la lecture, et/ou l'écriture des filigranes

Description

public EventBufferEvent::setWatermark ( int $events , int $lowmark , int $highmark ) : void

Active la lecture, l'écriture ou les deux pour les filigranes d'un tampon d'événement.

Un filigrane de tampon d'événement est une portion, une valeur spécifiant le nombre d'octets à lire ou à écrire avant d'appeler la fonction de rappel. Par défaut, chaque événement de lecture/écriture lance une fonction de rappel. Voir ausi la page suivante (en anglais) : » Fast portable non-blocking network programming with Libevent: Callbacks and watermarks

Liste de paramètres

events

Masque de constantes Event::READ, Event::WRITE, ou les deux.

lowmark

Valeur minimale du filigrane.

highmark

Valeur maximale du filigrane. La valeur 0 signifie "aucune limite".

Valeurs de retour

Aucune valeur n'est retournée.

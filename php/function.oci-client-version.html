oci_client_version

(PHP 5 >= 5.3.7, PHP 7, PECL OCI8 >= 1.4.6)

oci_client_version — Retourne la version de la bibliothèque cliente Oracle

Description

oci_client_version ( void ) : string

Retourne une chaîne de caractères contenant le numéro de la version de la bibliothèque cliente Oracle C, à laquelle PHP est liée.

Liste de paramètres

Aucun.

Valeurs de retour

Retourne le numéro de version, sous la forme d'une chaîne de caractères.

Exemples

Exemple #1 Exemple avec oci_client_version()

<?php
    echo "Client Version: " . oci_client_version(); // Version du client : 11.2.0.2
?>

Notes

Note:

Les bibliothèques Oracle avant la 10gR2 ne possédaient pas de fonctionnalité interne permettant de récupérer le numéro de version de la bibliothèque. La chaîne "Unknow" sera retournée dans ce cas.

PharFileInfo::hasMetadata

(PHP 5 >= 5.3.0, PHP 7, PECL phar >= 1.2.0)

PharFileInfo::hasMetadata — Retourne les méta-données de l'entrée

Description

public PharFileInfo::hasMetadata ( void ) : bool

Retourne les méta-données d'un fichier au sein d'une archive Phar.

Liste de paramètres

Aucun paramètre.

Valeurs de retour

Retourne FALSE si aucune méta-donnée n'est présente ou vaut NULL, TRUE si les méta-données ne valent pas NULL

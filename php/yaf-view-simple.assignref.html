Yaf_View_Simple::assignRef

(Yaf >=1.0.0)

Yaf_View_Simple::assignRef — Le but de assignRef

Description

public Yaf_View_Simple::assignRef ( string $name , mixed &$value ) : bool

Contrairement à la méthode Yaf_View_Simple::assign(), cette méthode assigne la référence à une valeur du moteur.

Liste de paramètres

name

Le nom sous forme de chaîne de caractères à utiliser pour accéder à la valeur dans le template.

value

Différents types de valeur.

Valeurs de retour

Exemples

Exemple #1 Exemple avec Yaf_View_Simple::assignRef()

<?php
class IndexController extends Yaf_Controller_Abstract {
    public function indexAction() {
        $value = "bar";
        $this->getView()->assign("foo", $value);

        /* Notez qu'il y avait un bogue avant la version 2.1.4 de Yaf,
         * qui faisait que l'affichage était "bar";
         */
        $dummy = $this->getView()->render("index/index.phtml");
        echo $value;

        // Ne pas utiliser l'auto-affichage
        Yaf_Dispatcher::getInstance()->autoRender(FALSE);
    }
?>

Exemple #2 Exemple avec template()

<html>
 <head>
  <title><?php echo $foo;  $foo = "changed"; ?></title>
 </head>  
<body>
</body>
</html>

L'exemple ci-dessus va afficher quelque chose de similaire à :

/* L'accès à l'index du contrôleur résultera en : */
changed

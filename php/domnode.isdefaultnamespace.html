DOMNode::isDefaultNamespace

(PHP 5, PHP 7)

DOMNode::isDefaultNamespace — Vérifie si l'espace de nom spécifié est l'espace de noms par défaut ou non

Description

public DOMNode::isDefaultNamespace ( string $namespaceURI ) : bool

Vérifie si namespaceURI est l'espace de noms par défaut.

Liste de paramètres

namespaceURI

L'URI de l'espace de noms à vérifier.

Valeurs de retour

Retourne TRUE si namespaceURI est l'espace de noms par défaut, FALSE sinon.

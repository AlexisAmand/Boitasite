DOMNode::C14NFile

(PHP 5 >= 5.2.0, PHP 7)

DOMNode::C14NFile — Canonise des noeuds en fichier

Description

public DOMNode::C14NFile ( string $uri [, bool $exclusive = FALSE [, bool $with_comments = FALSE [, array $xpath [, array $ns_prefixes ]]]] ) : int

Canonise des noeuds en fichier.

Liste de paramètres

uri

Chemin vers lequel le fichier sera créé.

exclusive

Active l'analyse des seuls noeuds correspondant à l'xpath ou préfixes d'espace de noms fournis.

with_comments

Conserve les commentaires dans la sortie.

xpath

Un tableau d'xpath utilisés pour filtrer les noeuds.

ns_prefixes

Un tableau de préfixes d'espace de noms utilisés pour filtrer les noeuds.

Valeurs de retour

Nombre d'octets écrits ou FALSE si une erreur survient

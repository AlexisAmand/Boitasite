ps_show2

(PECL ps >= 1.1.0)

ps_show2 — Affiche un texte à la position courante

Description

ps_show2 ( resource $psdoc , string $text , int $len ) : bool

Affiche le texte à la position courante. N'affiche pas plus de len caractères.

Liste de paramètres

psdoc

Identifiant de ressource d'un fichier postscript, tel que retourné par la fonction ps_new().

text

Le texte à afficher.

len

Le nombre maximal de caractères à afficher.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

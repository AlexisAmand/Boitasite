EventBuffer::unfreeze

(PECL event >= 1.2.6-beta)

EventBuffer::unfreeze — Ré-active les appels qui permettent de modifier un buffer d'événements

Description

public EventBuffer::unfreeze ( bool $at_front ) : bool

Ré-active les appels qui permettent de modifier un buffer d'événements.

Liste de paramètres

at_front

Si l'on doit activer les événements au début ou à la fin du buffer.

Valeurs de retour

Retourne TRUE en cas de succès. Sinon, FALSE.

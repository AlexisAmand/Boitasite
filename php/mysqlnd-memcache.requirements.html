Pré-requis

PHP : cette extension requiert PHP 5.4+, version 5.4.4 ou supérieure. Les extenions PHP requises sont PCRE (activée par défaut) et l'extension memcached version 2.0.x

Le plugin mysqlnd_memcache Memcache est compatible avec toutes les applications PHP et toutes les extensions PHP MySQL disponibles (mysqli, mysql, PDO_MYSQL). L'extension PHP MySQL doit être configurée avec le support de mysqlnd.

Pour accéder à des tables InnoDB, cette extension PHP requiert MySQL Server 5.6.6 ou une version plus récente avec le plugin InnoDB Memcache Daemon activé.

Pour accéder à des tables MySQL Cluster NDB, cette extension PHP requiert MySQL Cluster 7.2 ou une version plus récente avec NDB Memcache API nodes activé.

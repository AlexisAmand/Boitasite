Configuration à l'exécution

Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.

Yaconf Options de configuration
Nom Défaut Modifiable Historique
yaconf.check_delay 300 PHP_INI_SYSTEM
yaconf.directory /tmp/conf/ PHP_INI_SYSTEM

Voici un éclaircissement sur l'utilisation des directives de configuration.

yaconf.check_delay entier

Intervalle dans laquelle Yaconf détectera les modifications de fichier INI (grâce au mtime du dossier), si ceci est définie à zéro, vous devez redémarrer PHP pour recharger les configurations.

yaconf.directory chaîne de caractères

Chemin vers le dossier où tous les fichiers de configuration INI sont placés.

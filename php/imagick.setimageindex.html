Imagick::setImageIndex

(PECL imagick 2.0.0)

Imagick::setImageIndex — Modifie la position de l'itérateur

Avertissement

Cette fonction est OBSOLÈTE à partir de Imagick 3.4.4. Dépendre de cette fonctionnalité est fortement déconseillé.

Description

Imagick::setImageIndex ( int $index ) : bool

Modifie la position de l'itérateur à l'index index.

Cette méthode est obsolète. Voyez Imagick::setIteratorIndex()

Liste de paramètres

index

La position à laquelle mettre l'itérateur

Valeurs de retour

Retourne TRUE en cas de succès.

Erreurs / Exceptions

Lance une exception ImagickException si une erreur survient.

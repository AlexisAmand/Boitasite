dba_close

(PHP 4, PHP 5, PHP 7)

dba_close — Ferme une base DBA

Description

dba_close ( resource $handle ) : void

dba_close() ferme le lien établi avec la base de données et libère toutes les ressources de handle.

Liste de paramètres

handle

Le gestionnaire de la base de données, retourné par la fonction dba_open() ou la fonction dba_popen().

Valeurs de retour

Aucune valeur n'est retournée.

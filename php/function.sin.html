sin

(PHP 4, PHP 5, PHP 7)

sin — Sinus

Description

sin ( float $arg ) : float

sin() retourne le sinus de arg (arg in radians).

Liste de paramètres

arg

Une valeur, en radians

Valeurs de retour

Le sinus de arg.

Exemples

Exemple #1 Exemple avec sin()

<?php

// La précision dépend de la directive precision
echo sin(deg2rad(60));  //  0.866025403 ...
echo sin(60);           // -0.304810621 ...

?>

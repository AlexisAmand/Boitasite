SolrParams::toString

(PECL solr >= 0.9.2)

SolrParams::toString — Retourne tous les paramètres de l'objet

Description

final public SolrParams::toString ([ bool $url_encode = FALSE ] ) : string

Retourne tous les paramètres de l'objet, sous la forme de paire nom-valeur.

Liste de paramètres

url_encode

Si l'on doit retourner les valeurs encodées URL.

Valeurs de retour

Retourne une chaîne de caractères en cas de succès, et FALSE si une erreur survient.

Worker::unstack

(PECL pthreads >= 2.0.0)

Worker::unstack — Dépile une tâche

Description

public Worker::unstack ( void ) : int

Supprime la première tâche (la plus ancienne) de la pile.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

La nouvelle taille de la pile.

Historique

Version Description
v3 Le paramètre pour spécifier la tâche à dépiler a été supprimé. Maintenant, seule la première tâche dans la pile est supprimée.

Exemples

Exemple #1 Supprime l'objet depuis la pile d'attente des Workers

<?php
$my = new Worker();
$work = new class extends Threaded {};

var_dump($my->stack($work));
var_dump($my->unstack());

L'exemple ci-dessus va afficher :

int(1)
int(0)

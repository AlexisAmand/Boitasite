imap_headers

(PHP 4, PHP 5, PHP 7)

imap_headers — Retourne les en-têtes de tous les messages d'une boîte aux lettres

Description

imap_headers ( resource $imap_stream ) : array

Retourne les en-têtes de tous les messages d'une boîte aux lettres.

Liste de paramètres

imap_stream

Un flux IMAP retourné par la fonction imap_open().

Valeurs de retour

Retourne un tableau de chaînes contenant les en-têtes des messages : une chaîne par message.

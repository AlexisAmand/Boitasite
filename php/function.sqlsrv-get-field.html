sqlsrv_get_field — Récupère les données du champ depuis la ligne actuellement sélectionnée

Description

sqlsrv_get_field ( resource $stmt , int $fieldIndex [, int $getAsType ] ) : mixed

Récupère les données du champ depuis la ligne actuellement sélectionnée. Les champs doivent être lus dans l'ordre. Leurs indices commencent à 0.

Liste de paramètres

stmt

Une ressource de requête retournée par la fonction sqlsrv_query() ou la fonction sqlsrv_execute().

fieldIndex

L'index du champ à récupérer. Les indices des champs commencent à 0. Les champs doit être lus dans l'ordre, i.e. si vous accédez au champ d'indice 1, alors le champ d'indice 0 ne sera plus disponible.

getAsType

Le type de données PHP pour les données du champ retourné. Si ce paramètre n'est pas défini, les données du champ seront retournées sous la forme d'un type de données PHP par défaut. Pour plus d'informations sur les types de données PHP par défaut, reportez-vous à la section sur » les types de données PHP par défaut de la documentation Microsoft SQLSRV.

Valeurs de retour

Retourne les données depuis le champ spécifié en cas de succès. Retourne FALSE si une erreur survient.

Exemples

Exemple #1 sqlsrv_get_field() example

L'exemple suivant montre la façon de récupérer une ligne avec la fonction sqlsrv_fetch() et récupère les champs de la ligne avec la fonction sqlsrv_get_field().

<?php
$serverName = "serverName\sqlexpress";
$connectionInfo = array( "Database"=>"dbName", "UID"=>"username", "PWD"=>"password");
$conn = sqlsrv_connect( $serverName, $connectionInfo);
if( $conn === false ) {
     die( print_r( sqlsrv_errors(), true));
}

$sql = "SELECT Name, Comment 
        FROM Table_1
        WHERE ReviewID=1";
$stmt = sqlsrv_query( $conn, $sql);
if( $stmt === false ) {
     die( print_r( sqlsrv_errors(), true));
}

// Rend disponible la première (et dans ce cas, la seule) ligne du jeu de résultats pour lecture.
if( sqlsrv_fetch( $stmt ) === false) {
     die( print_r( sqlsrv_errors(), true));
}

// Récupère les champs de la ligne. Les indices commencent à 0 et doivent être récupérés dans l'ordre.
// La récupération des champs de la ligne par leurs noms n'est pas supportée par la fonction sqlsrv_get_field.
$name = sqlsrv_get_field( $stmt, 0);
echo "$name: ";

$comment = sqlsrv_get_field( $stmt, 1);
echo $comment;
?>

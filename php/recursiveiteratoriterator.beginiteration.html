RecursiveIteratorIterator::beginIteration

(PHP 5 >= 5.1.0, PHP 7)

RecursiveIteratorIterator::beginIteration — Commence une itération

Description

public RecursiveIteratorIterator::beginIteration ( void ) : void

Appelé lorsque l'itération commence (après le premier appel à la méthode RecursiveIteratorIterator::rewind()).

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

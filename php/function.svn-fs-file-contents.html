svn_fs_file_contents

(PECL svn >= 0.1.0)

svn_fs_file_contents — Retourne un flux afin d'accéder au contenu d'un fichier pour une révision donnée

Description

svn_fs_file_contents ( resource $fsroot , string $path ) : resource
Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Retourne un flux afin d'accéder au contenu d'un fichier pour une révision donnée.

Avertissement

Cette fonction est EXPERIMENTALE. Le comportement de cette fonction, son nom, et toute la documentation autour de cette fonction peut changer sans préavis dans une prochaine version de PHP. Cette fonction doit être utilisée à vos risques et périls.

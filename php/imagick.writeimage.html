Imagick::writeImage

(PECL imagick 0.9.0-0.9.9)

Imagick::writeImage — Écrit une image dans le fichier spécifié

Description

Imagick::writeImage ([ string $filename = NULL ] ) : bool

Écrit une image dans un fichier spécifié. Si le paramètre filename vaut NULL, l'image sera écrite dans le fichier défini par Imagick::readImage() ou Imagick::setImageFilename().

Liste de paramètres

filename

Nom du fichier dans lequel l'image sera écrite. L'extension du fichier définit le type de fichier. Le format peut être forcé malgré l'extension de fichier en utilisant format: préfixe, par exemple "jpg:test.png".

Valeurs de retour

Retourne TRUE en cas de succès.

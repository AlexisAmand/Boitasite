spl_object_id

(PHP 7 >= 7.2.0)

spl_object_id — Retourne le gestionnaire d'objet entier pour un objet donné

Description

spl_object_id ( object $obj ) : int

Cette fonction retourne un identifiant unique pour l'objet. L'id de l'objet est unique pour la durée de vie de l'objet. Une fois que l'objet est détruit, son id peut être réutilisé pour d'autres objets. Ce comportement est similaire à spl_object_hash().

Liste de paramètres

object

N'importe quel objet.

Valeurs de retour

Un identifiant entier qui est unique pour chaque objet actuellement existant et toujours le même pour chaque objet.

Exemples

Exemple #1 Un exemple de spl_object_id()

<?php
$id = spl_object_id($object);
$storage[$id] = $object;
?>

Notes

Note:

Quand un objet est détruit, son id peut être réutilisé pour d'autres objets.

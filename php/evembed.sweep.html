EvEmbed::sweep

(PECL ev >= 0.2.0)

EvEmbed::sweep — Balaye, une seule fois, et de façon non bloquante la boucle interne

Description

public EvEmbed::sweep ( void ) : void

Balaye, une seule fois, et de façon non bloquante la boucle interne. Fonctionne de la même façon que ce qui suit, mais dans une façon plus appropriée pour les boucles internes :

<?php
$other->start(Ev::RUN_NOWAIT);
?>

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

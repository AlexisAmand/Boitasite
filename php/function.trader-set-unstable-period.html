trader_set_unstable_period

(PECL trader >= 0.2.2)

trader_set_unstable_period — Définit une période instable

Description

trader_set_unstable_period ( int $functionId , int $timePeriod ) : void

Influence le facteur instable d'une période pour les fonctions qui en sont sensibles. Plus d'informations concernant les périodes instables peuvent être trouvées sur la page de la documentation de l'API » TA-Lib.

Liste de paramètres

functionId

Identifiant de la fonction du facteur à définir. Les constantes de la série TRADER_FUNC_UNST_* peuvent être utilisées pour affecter la fonction correspondante.

timePeriod

Valeur d'instabilité de la période.

Valeurs de retour

Aucune valeur n'est retournée.

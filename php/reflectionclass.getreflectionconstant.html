ReflectionClass::getReflectionConstant

(PHP 7 >= 7.1.0)

ReflectionClass::getReflectionConstant — Récupère un ReflectionClassConstant pour une constante d'une classe

Description

public ReflectionClass::getReflectionConstant ( string $name ) : ReflectionClassConstant

Récupère un ReflectionClassConstant pour une propriété de classe.

Liste de paramètres

name

Le nom de la constante de classe.

Valeurs de retour

Un ReflectionClassConstant.

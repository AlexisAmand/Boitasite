call_user_method_array

(PHP 4 >= 4.0.5, PHP 5)

call_user_method_array — Appelle une méthode utilisateur avec un tableau de paramètres

Avertissement

Cette fonction est OBSOLÈTE à partir de PHP 4.1.0 et a été SUPPRIMÉE à partir de PHP 7.0.0.

Les alternatives à cette fonction incluent :

  • call_user_func_array()

Description

call_user_method_array ( string $method_name , object &$obj , array $params ) : mixed

Liste de paramètres

method_name

Le nom de la méthode appelée.

obj

L'objet sur lequel method_name est appelée.

params

Un tableau de paramètres.

Exemples

Exemple #1 Alternative à la fonction call_user_method_array()

<?php
call_user_func_array(array($obj, $method_name), $params);
?>

imap_base64

(PHP 4, PHP 5, PHP 7)

imap_base64 — Décode un texte encodé en BASE64

Description

imap_base64 ( string $text ) : string

Décode le texte text encodé en BASE64.

Liste de paramètres

text

Le texte encodé.

Valeurs de retour

Retourne le texte décodé, sous la forme d'une chaîne de caractères.

ImagickPixel::isPixelSimilar — Vérifie la distance entre cette couleur et une autre

Description

ImagickPixel::isPixelSimilar ( ImagickPixel $color , float $fuzz ) : bool

Vérifie la distance entre la couleur décrite par cet objet ImagickPixel et celle de l'objet fourni. Si la distance entre les deux points est inférieure à la valeur du paramètre fuzz fourni, la couleur est similaire. Cette méthode remplace la méthode ImagickPixel::isSimilar() et normalise correctement la valeur fuzz d'ImageMagick QuantumRange.

Liste de paramètres

color

L'objet ImagickPixel à utiliser pour la comparaison.

fuzz

La distance maximale pour laquelle on considère les couleurs comme similaires. La valeur maximale théorique pour cette valeur est la racine quarré de trois (1.732).

Valeurs de retour

Retourne TRUE en cas de succès.

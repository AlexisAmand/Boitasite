wkhtmltox\Image\Converter::getVersion

(wkhtmltox >= 0.3.2)

wkhtmltox\Image\Converter::getVersion — Détermine la version du convertisseur

Description

public wkhtmltox\Image\Converter::getVersion ( void ) : string

Détermine la version du convertisseur, tel que rapporté par libwkhtmltox

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la version, sous la forme d'une chaîne de caractères

Avertissement

Ceci n'est actuellement pas implémenté par libwkhtmltox

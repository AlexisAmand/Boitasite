ArrayIterator::natcasesort

(PHP 5 >= 5.2.0, PHP 7)

ArrayIterator::natcasesort — Trie naturellement un tableau, en ne tenant pas compte de la casse

Description

public ArrayIterator::natcasesort ( void ) : void

Trie les entrées par les valeurs, en utilisant un algorithme insensible à la casse d'ordre naturel.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

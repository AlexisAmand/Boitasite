SolrClient::deleteByIds

(PECL solr >= 0.9.2)

SolrClient::deleteByIds — Effacement par Ids

Description

public SolrClient::deleteByIds ( array $ids ) : SolrUpdateResponse

Efface une collection de documents représentée par le jeu d'Ids spécifié.

Liste de paramètres

ids

Un tableau d'Ids, représentant le champ uniqueKey déclaré dans le schéma de chaque document.

Valeurs de retour

Retourne un objet SolrUpdateResponse en cas de succès, et lance une exception en cas d'échec.

Erreurs / Exceptions

Lance une exception SolrClientException si le client a échoué ou s'il y a eu un problème avec la connexion.

Lance une exception SolrServerException si le serveur Solr a échoué dans l'exécution de la requête.

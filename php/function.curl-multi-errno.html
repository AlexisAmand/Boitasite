curl_multi_errno

(PHP 7 >= 7.1.0)

curl_multi_errno — Retourne le dernier numéro d'erreur multiple cURL

Description

curl_multi_errno ( resource $mh ) : int

Retourne un entier contenant le dernier numéro d'erreur multiple cURL.

Liste de paramètres

mh

Un gestionnaire cURL multiple retourné par la fonction curl_multi_init().

Valeurs de retour

Retourne un entier contenant le dernier numéro d'erreur multiple cURL, ou FALSE si une erreur survient.

fann_get_total_neurons

(PECL fann >= 1.0.0)

fann_get_total_neurons — Récupère le nombre total de neurones dans la totalité du réseau

Description

fann_get_total_neurons ( resource $ann ) : int

Récupère le nombre total de neurones dans la totalité du réseau. Ce nombre n'inclut pas les neurones de polarisation - un réseau 2-4-2 a 2+4+2 +2(polarisants) = 10 neurones.

Liste de paramètres

ann

Ressource de réseau de neurones.

Valeurs de retour

Le nombre total de neurones dans la totalité du réseau, ou FALSE si une erreur survient.

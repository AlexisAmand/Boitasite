ibase_modify_user

(PHP 5, PHP 7 < 7.4.0)

ibase_modify_user — Modifie un utilisateur dans une base de données de sécurité

Description

ibase_modify_user ( resource $service_handle , string $user_name , string $password [, string $first_name [, string $middle_name [, string $last_name ]]] ) : bool

Liste de paramètres

service_handle

Le gestionnaire sur le service du serveur de base de données.

user_name

L'identifiant de l'utilisateur de base de donnée à modifier.

password

Le nouveau mot de passe de l'utilisateur.

first_name

Le nouveau prénom de l'utilisateur.

middle_name

Le nouveau deuxième prénom de l'utilisateur.

last_name

Le nouveau nom de famille de l'utilisateur.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

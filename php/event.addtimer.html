Event::addTimer

(PECL event >= 1.2.6-beta)

Event::addTimer — Alias de Event::add()

Description

public Event::addTimer ([ float $timeout ] ) : bool

La méthode Event::addTimer() est un alias de la méthode Event::add().

Valeurs de retour

Exemples

Exemple #1 Exemple avec Event::addTimer()

<?php
$base = new EventBase();
$n = 2;
$e = Event::timer($base, function($n) use (&$e) {
    echo "$n secondes passées\n";
    $e->delTimer();
}, $n);
$e->addTimer($n);
$base->loop();
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

2 secondes passées

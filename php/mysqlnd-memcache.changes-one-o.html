Séries PECL/mysqlnd_memcache 1.0

1.0.0-alpha

  • Date de sortie : TBD
  • Motto/thème : Mappage simple de SELECT SQL vers l'accès au plugin MySQL Memcache plugin access.

La version initiale fournit un mappage simple des requêtes SELECT SQL vers l'accès au plugin MySQL Memcache. Ceci a un bénéfice certain, sachant que l'accès direct clé-valeur au stockage MySQL en utilisant le protocole Memcache est plus rapide que l'utilisation SQL classique.

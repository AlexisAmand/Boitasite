image2wbmp

(PHP 4 >= 4.0.5, PHP 5, PHP 7)

image2wbmp — Affichage de l'image vers le navigateur ou dans un fichier

Avertissement

image2wbmp() est obsolète à partir de PHP 7.3.0, et sera supprimer dans la prochaine version majeure. Utilisez imagewbmp() à la place.

Description

image2wbmp ( resource $image [, string $filename [, int $foreground ]] ) : bool

image2wbmp() affiche ou enregistre une version WBMP de l'image image fournie.

Liste de paramètres

image

Une ressource d'image, retournée par une des fonctions de création d'images, comme imagecreatetruecolor().

filename

Chemin vers le fichier de sauvegarde. S'il n'est pas fourni, le flux de l'image sera directement affiché.

foreground

La couleur du premier plan peut être définit avec ce paramètre en fournissant un idantifiant obtenu avec imagecolorallocate(). La couleur du premier plan par défaut est noir.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Attention

Toutefois, si libgd ne parvient pas à produire l'image, cette fonction retourne TRUE.

Exemples

Exemple #1 Exemple avec image2wbmp()

<?php

$file = 'php.png';
$image = imagecreatefrompng($file);

header('Content-Type: ' . image_type_to_mime_type(IMAGETYPE_WBMP));
image2wbmp($image); // Affichage direct
imagedestroy($image);

?>

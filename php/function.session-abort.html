session_abort

(PHP 5 >= 5.6.0, PHP 7)

session_abort — Abandonne les changements sur le tableau de session et termine la session

Description

session_abort ( void ) : bool

session_abort() termine la session sans sauver les données. Les valeurs originales de la session sont conservées.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Historique

Version Description
7.2.0 Le type de retour de cette fonction est désormais booléen. Auparavant, c'était void.

pg_consume_input

(PHP 5 >= 5.6.0, PHP 7)

pg_consume_input — Lit l'entrée de la connexion

Description

pg_consume_input ( resource $connection ) : bool

La fonction pg_consume_input() lit toutes les entrées en attente d'être lues depuis le serveur de base de données.

Liste de paramètres

connection

Ressource de connexion à la base de données PostgreSQL.

Valeurs de retour

TRUE en cas de succès, ou FALSE si une erreur survient. Notez que TRUE n'indique pas forcément que l'entrée est en attente d'être lue.

ArrayObject::unserialize

(PHP 5 >= 5.3.0, PHP 7)

ArrayObject::unserialize — Délinéarisation d'un ArrayObject

Description

public ArrayObject::unserialize ( string $serialized ) : void

Délinéarise d'un objet ArrayObject linéarisé.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

serialized

L'objet ArrayObject linéarisé.

Valeurs de retour

The unserialized ArrayObject.

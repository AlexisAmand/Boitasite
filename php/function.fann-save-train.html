fann_save_train

(PECL fann >= 1.0.0)

fann_save_train — Sauvegarde la structure d'entrainement dans un fichier

Description

fann_save_train ( resource $data , string $file_name ) : bool

Sauvegarde la structure d'entrainement dans un fichier, dont le format est spécifié dans la fonction fann_read_train_from_file().

Liste de paramètres

data

Ressource de données d'entrainement du réseau de neurones.

file_name

Le nom du fichier où la sauvegarde à lieu.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

Constantes pré-définies

Ces constantes sont définies par cette extension, et ne sont disponibles que si cette extension a été compilée avec PHP, ou bien chargée au moment de l'exécution.

FORCE_GZIP (entier)
FORCE_DEFLATE (entier)
ZLIB_ENCODING_RAW (integer)
Algorithme DEFLATE tel que décrite dans la RFC 1951. Disponible depuis PHP 7.0.0.
ZLIB_ENCODING_DEFLATE (integer)
Algorithme de compression ZLIB tel que décrit dans la RFC 1950. Disponible depuis PHP 7.0.0.
ZLIB_ENCODING_GZIP (integer)
Algorithme GZIP tel que décrit par la RFC 1952. Disponible depuis PHP 7.0.0.
ZLIB_FILTERED (integer)
Disponible depuis PHP 7.0.0.
ZLIB_HUFFMAN_ONLY (integer)
Disponible depuis PHP 7.0.0.
ZLIB_FIXED (integer)
Disponible depuis PHP 7.0.0.
ZLIB_RLE (integer)
Disponible depuis PHP 7.0.0.
ZLIB_DEFAULT_STRATEGY (integer)
Disponible depuis PHP 7.0.0.
ZLIB_BLOCK (integer)
Disponible depuis PHP 7.0.0.
ZLIB_NO_FLUSH (integer)
Disponible depuis PHP 7.0.0.
ZLIB_PARTIAL_FLUSH (integer)
Disponible depuis PHP 7.0.0.
ZLIB_SYNC_FLUSH (integer)
Disponible depuis PHP 7.0.0.
ZLIB_FULL_FLUSH (integer)
Disponible depuis PHP 7.0.0.
ZLIB_FINISH (integer)
Disponible depuis PHP 7.0.0.

EventDnsBase::setSearchNdots

(PECL event >= 1.2.6-beta)

EventDnsBase::setSearchNdots — Définit le paramètre 'ndots' pour les recherches

Description

public EventDnsBase::setSearchNdots ( int $ndots ) : bool

Définit le paramètre 'ndots' pour les recherches. Définit le nombre de points qui, lorsqu'ils sont trouvés dans un nom, fait que la première requête se déroule sans recherche de domaine.

Liste de paramètres

ndots

Le nombre de points.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

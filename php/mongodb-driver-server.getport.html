MongoDB\Driver\Server::getPort

(mongodb >=1.0.0)

MongoDB\Driver\Server::getPort — Retourne le port sur lequel le serveur écoute

Description

final public MongoDB\Driver\Server::getPort ( void ) : int

Retourne le port sur lequel le serveur écoute.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne le port sur lequel le serveur écoute.

Erreurs / Exceptions

  • Lance une exception MongoDB\Driver\InvalidArgumentException lors d'une erreur survenue pendant l'analyse d'un argument.

Exemples

Exemple #1 Exemple avec MongoDB\Driver\Server::getPort()

<?php

$manager = new MongoDB\Driver\Manager("mongodb://localhost:27017/");

$rp = new MongoDB\Driver\ReadPreference(MongoDB\Driver\ReadPreference::RP_PRIMARY);
$server = $manager->selectServer($rp);

var_dump($server->getPort());

?>

L'exemple ci-dessus va afficher :

int(27017)

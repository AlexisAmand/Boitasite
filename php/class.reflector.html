L'interface Reflector

(PHP 5, PHP 7)

Introduction

Reflector est une interface implémentée par toutes les classes exportables Reflection.

Synopsis de la classe

/* Méthodes */
public static export ( void ) : string
public __toString ( void ) : string
}

fann_get_sarprop_step_error_threshold_factor

(PECL fann >= 1.0.0)

fann_get_sarprop_step_error_threshold_factor — Retourne le facteur de seuil d'erreur lors de l'étape sarprop

Description

fann_get_sarprop_step_error_threshold_factor ( resource $ann ) : float

Retourne le facteur de seuil d'erreur lors de l'étape sarprop.

Le facteur par défaut est 0.1.

Liste de paramètres

ann

Ressource de réseau de neurones.

Valeurs de retour

Le facteur de seuil d'erreur lors de l'étape sarprop, ou FALSE si une erreur survient.

Notes

Note:

Cette fonction est maintenant disponible si l'extension fann a été compilée avec libfann >= 2.2.

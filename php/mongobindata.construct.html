MongoBinData::__construct

(PECL mongo >= 0.8.1)

MongoBinData::__construct — Crée un nouvel objet de données binaires

Cette extension, qui définie cette méthode est obsolète. Veuillez utiliser l'extension MongoDB à la place. Les alertnatives à cette méthode sont :

  • MongoDB\BSON\Binary::__construct()

Description

public MongoBinData::__construct ( string $data [, int $type = 0 ] )

Crée un nouvel objet de données binaires.

Il y a sept types de données binaires actuellement reconnues par BSON, qui sont définis comme constantes de classe. Dans un souci de compatibilité ascendante, le driver PHP utilise MongoBinData::BYTE_ARRAY par défaut ; cependant, ceci peut changer en MongoBinData::GENERIC dans le futur. Les utilisateurs sont encouragés à spécifier un type plutôt que d'utiliser celui par défaut.

Liste de paramètres

data

Données binaires.

type

Le type de données.

Valeurs de retour

Retourne un nouvel objet de données binaires.

Historique

Version Description
1.5.0 La valeur par défaut passe de 2 (MongoBinData::BYTE_ARRAY) à 0 (MongoBinData::GENERIC).
1.2.11 Lance une alerte de niveau E_DEPRECATED lorsque le second argument n'est pas utilisé. La valeur par défaut du paramètre type peut changer dans de futures versions.

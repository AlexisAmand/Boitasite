ArrayAccess::offsetUnset

(PHP 5, PHP 7)

ArrayAccess::offsetUnset — Supprime un élément à une position donnée

Description

abstract public ArrayAccess::offsetUnset ( mixed $offset ) : void

Supprime un élément à une position donnée.

Note:

Cette méthode ne sera pas appelée lors du transtypage en (unset).

Liste de paramètres

offset

La position à supprimer.

Valeurs de retour

Aucune valeur n'est retournée.

Thread::kill

(PECL pthreads < 3.0.0)

Thread::kill — Exécution

Avertissement

Cette méthode a été supprimée dans pthreads v3.

Description

public Thread::kill ( void ) : void

Force le thread référencé à se terminer

Avertissement

Le développeur ne devrait pas avoir à forcer le thread à se terminer.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Un booléen indiquant le succès.

Exemples

Exemple #1 Termine le thread référencé

<?php
class T extends Thread {
    public function run() {
        $stdin = fopen("php://stdin", "r");
        while(($line = fgets($stdin))) {
            echo $line;
        }
    }
}

$t = new T();
$t->start();

var_dump($t->kill());
?>

L'exemple ci-dessus va afficher :

bool(true)

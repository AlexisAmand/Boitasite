EvIo::createStopped

(PECL ev >= 0.2.0)

EvIo::createStopped — Crée un objet d'observateur EvIo arrêté

Description

final public static EvIo::createStopped ( mixed $fd , int $events , callable $callback [, mixed $data = NULL [, int $priority = 0 ]] ) : EvIo

Identique à la méthode EvIo::__construct() mais ne démarre pas automatiquement l'observateur.

Liste de paramètres

fd

Identique à la méthode EvIo::__construct()

events

Identique à la méthode EvIo::__construct()

callback

Voir les fonctions de rappels des observateurs.

data

Données personalisées associées avec l'observateur.

priority

Priorité de l'observateur

Valeurs de retour

Retourne un objet EvIo en cas de succès.

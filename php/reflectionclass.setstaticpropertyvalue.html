ReflectionClass::setStaticPropertyValue

(PHP 5 >= 5.1.2, PHP 7)

ReflectionClass::setStaticPropertyValue — Définit la valeur d'une propriété statiques

Description

public ReflectionClass::setStaticPropertyValue ( string $name , mixed $value ) : void

Définit la valeur d'une propriété statiques.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

name

Le nom de la propriété.

value

La nouvelle valeur pour la propriété.

Valeurs de retour

Aucune valeur n'est retournée.

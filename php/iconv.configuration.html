Configuration à l'exécution

Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.

Options de configuration iconv
Nom Défaut Modifiable Historique
iconv.input_encoding "" PHP_INI_ALL Obsolète en PHP 5.6.0.
iconv.output_encoding "" PHP_INI_ALL Obsolète en PHP 5.6.0.
iconv.internal_encoding "" PHP_INI_ALL Obsolète en PHP 5.6.0.

Voici un éclaircissement sur l'utilisation des directives de configuration.

Avertissement

Quelques systèmes (comme IBM AIX) utilisent "ISO8859-1" au lieu de "ISO-8859-1", cette valeur doit donc être utilisée dans les options de configuration ainsi que dans les paramètres des fonctions.

iconv.input_encoding string
Avertissement

Cette fonctionnalité est OBSOLÈTE à partir de PHP 5.6.0. Dépendre de cette fonctionnalité est fortement déconseillé.

Les utilisateurs de PHP 5.6 et suivants doivent laisser cette option vide et définir plutôt l'option default_charset.

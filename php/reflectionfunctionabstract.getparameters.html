ReflectionFunctionAbstract::getParameters

(PHP 5 >= 5.2.0, PHP 7)

ReflectionFunctionAbstract::getParameters — Récupère les paramètres

Description

public ReflectionFunctionAbstract::getParameters ( void ) : array

Récupère les paramètres, sous la forme d'un tableau d'objets ReflectionParameter.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Les paramètres, sous la forme d'objets ReflectionParameter.

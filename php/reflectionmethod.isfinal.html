ReflectionMethod::isFinal

(PHP 5, PHP 7)

ReflectionMethod::isFinal — Vérifie si la méthode est finale

Description

public ReflectionMethod::isFinal ( void ) : bool

Vérifie si la méthode est finale.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

TRUE si la méthode est finale, FALSE sinon.

Configuration à l'exécution

Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.

Options de configuration Imagick
Nom Défaut Modifiable Historique
imagick.locale_fix FALSE PHP_INI_ALL Disponible depuis 2.1.0
imagick.progress_monitor FALSE PHP_INI_SYSTEM Disponible depuis Imagick 2.2.2
imagick.skip_version_check FALSE PHP_INI_SYSTEM Disponible depuis Imagick 3.3.0
Pour plus de détails sur les modes PHP_INI_*, reportez-vous à Où une directive de configuration peut être modifiée.

Voici un éclaircissement sur l'utilisation des directives de configuration.

imagick.locale_fix boolean

Fixes un bogue sur le dessin lorsque les locales utilisent le caractère ',' comme séparateurs.

imagick.progress_monitor boolean

Utilisé pour activer la surveillance de la progression de l'image.

imagick.skip_version_check boolean

Lorsque Imagick est chargé, le numéro de version d'ImageMagick utilisé pendant la compilation est vérifié, et est comparé avec le numéro de version actuellement utilisé ; une alerte est émise si les deux versions ne correspondent pas. Cette alerte peut être supprimée en activant cette option de configuration.

Le fait d'utiliser une version d'Imagick différente de la version d'ImageMagick n'est pas recommandé. Malgré le fait que cela peut fonctionner, il peut en résulter des erreurs aléatoires ou des comportements non définis.

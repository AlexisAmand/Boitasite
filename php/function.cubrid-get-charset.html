cubrid_get_charset

(PECL CUBRID >= 8.3.0)

cubrid_get_charset — Retourne le jeu de caractères utilisé sur la connexion CUBRID courante

Description

cubrid_get_charset ( resource $conn_identifier ) : string

Cette fonction retourne le jeu de caractères sur la connexion CUBRID courante et est similaire à la fonction CUBRID compatible MySQL cubrid_client_encoding().

Liste de paramètres

conn_identifier

La connexion CUBRID.

Valeurs de retour

Une chaîne de caractères représentant le jeu de caractères sur la connexion CUBRID en cas de succès.

FALSE si une erreur survient.

Exemples

Exemple #1 Exemple avec cubrid_get_charset()

<?php

$con = cubrid_connect("localhost", 33000, "demodb");
if (!$con)
{
    die('Connexion impossible.');
}

printf("Jeu de caractères CUBRID courant : %s\n", cubrid_get_charset($con));

?>

L'exemple ci-dessus va afficher :

Jeu de caractères CUBRID courant : iso8859-1

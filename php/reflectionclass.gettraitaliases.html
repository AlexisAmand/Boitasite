ReflectionClass::getTraitAliases

(PHP 5 >= 5.4.0, PHP 7)

ReflectionClass::getTraitAliases — Retourne un tableau des alias du trait

Description

public ReflectionClass::getTraitAliases ( void ) : array

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un tableau avec les nouveaux noms de méthodes en clés et les noms originaux en valeurs (au format "TraitName::original"). Retourne NULL en cas d'erreur.

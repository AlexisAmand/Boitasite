pg_flush

(PHP 5 >= 5.6.0, PHP 7)

pg_flush — Envoi les données de requête sortante sur la connexion

Description

pg_flush ( resource $connection ) : mixed

La fonction pg_flush() envoie toutes les données sortantes de requête en attente d'être envoyées sur la connexion.

Liste de paramètres

connection

Ressource de connexion à la base de données PostgreSQL.

Valeurs de retour

Retourne TRUE si l'envoi a réussi, ou si aucune donnée n'est en attente d'envoi, 0 si une partie des données en attente ont été envoyées mais qu'il en reste encore ou FALSE si une erreur survient.

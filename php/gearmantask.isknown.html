GearmanTask::isKnown

(PECL gearman >= 0.5.0)

GearmanTask::isKnown — Vérifie si une tâche est connue

Description

public GearmanTask::isKnown ( void ) : bool

Récupère les informations de statut si la tâche est connue sur le serveur de travaux.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

TRUE si la tâche est connue, FALSE sinon.

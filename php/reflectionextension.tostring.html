ReflectionExtension::__toString

(PHP 5, PHP 7)

ReflectionExtension::__toString — Récupère une représentation textuelle

Description

public ReflectionExtension::__toString ( void ) : string

Exporte une extension reflétée et retourne le résultat sous forme de chaîne de caractères. Cette méthode est identique à ReflectionExtension::export() avec return défini à TRUE.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne l'extension reflétée sous forme de chaîne de caractères, de la même façon que ReflectionExtension::export().

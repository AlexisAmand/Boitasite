libxml_get_last_error

(PHP 5 >= 5.1.0, PHP 7)

libxml_get_last_error — Lit la dernière erreur libxml

Description

libxml_get_last_error ( void ) : LibXMLError

libxml_get_last_error() lit la dernière erreur libxml.

Valeurs de retour

libxml_get_last_error() retourne un objet LibXMLError s'il y a une erreur, et FALSE sinon.

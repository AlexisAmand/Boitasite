ldap_dn2ufn

(PHP 4, PHP 5, PHP 7)

ldap_dn2ufn — Convertit un DN en format UFN (User Friendly Naming)

Description

ldap_dn2ufn ( string $dn ) : string

Convertit le DN dn dans un format plus lisible humainement, en supprimant les types des noms.

Liste de paramètres

dn

Le DN de l'entrée LDAP.

Valeurs de retour

Retourne l'UFN.

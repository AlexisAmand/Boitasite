ReflectionFunctionAbstract::getReturnType

(PHP 7)

ReflectionFunctionAbstract::getReturnType — Récupère le type de retour définit pour une fonction

Description

public ReflectionFunctionAbstract::getReturnType ( void ) : ReflectionType

Récupère le type de retour définit pour une fonction reflété.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un objet ReflectionType si un type de retour est définit, NULL sinon.

Exemples

Exemple #1 Exemple avec ReflectionFunctionAbstract::getReturnType()

<?php

function to_int($param) : int {
    return (int) $param;
}

$reflection1 = new ReflectionFunction('to_int');
echo $reflection1->getReturnType();

L'exemple ci-dessus va afficher :

int

Exemple #2 Utilisation sur les fonctions intégrés

<?php

$reflection2 = new ReflectionFunction('array_merge');

var_dump($reflection2->getReturnType());

L'exemple ci-dessus va afficher :

null

Ceci est le cas car beaucoup de fonction interne ne définissent pas un type pour leurs paramètres ou leurs valeur de retour. Il est donc conseillé d'éviter l'usage de cette méthode sur les fonctions intégrés.

SplFixedArray::__construct

(PHP 5 >= 5.3.0, PHP 7)

SplFixedArray::__construct — Construit un nouveau SplFixedArray

Description

public SplFixedArray::__construct ([ int $size = 0 ] )

Initialise un tableau fixe avec un nombre de valeurs NULL égales au paramètre size.

Liste de paramètres

size

La taille du tableau à taille fixe. Attends un nombre compris entre 0 et PHP_INT_MAX.

Valeurs de retour

Aucune valeur n'est retournée.

Erreurs / Exceptions

Lance une exception InvalidArgumentException lorsque size est négatif. number.

Émets une alerte de niveau E_WARNING lorsque size ne peut être analysé comme un nombre.

Exemples

Exemple #1 Exemple avec SplFixedArray::__construct()

<?php
$array = new SplFixedArray(5);

$array[1] = 2;
$array[4] = "foo";

foreach($array as $v) {
  var_dump($v);
}
?>

L'exemple ci-dessus va afficher :

NULL
int(2)
NULL
NULL
string(3) "foo"

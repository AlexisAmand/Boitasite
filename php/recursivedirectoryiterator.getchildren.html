RecursiveDirectoryIterator::getChildren

(PHP 5 >= 5.1.0, PHP 7)

RecursiveDirectoryIterator::getChildren — Retourne un itérateur pour l'entrée courante si c'est un dossier

Description

public RecursiveDirectoryIterator::getChildren ( void ) : mixed
Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Le nom du fichier, des informations sur le fichier ou $this suivant le drapeau utilisé. Voir les constantes FilesystemIterator.

ZMQSocket::isPersistent

(PECL zmq >= 0.5.0)

ZMQSocket::isPersistent — Vérifie si le socket est persistent

Description

public ZMQSocket::isPersistent ( void ) : bool

Vérifie si le socket est persistent.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un booléen suivant si le socket est persistent ou non.

MongoGridFSCursor::key

(PECL mongo >=0.9.0)

MongoGridFSCursor::key — Retourne l'identifiant courant

Description

public MongoGridFSCursor::key ( void ) : string

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

L'identifiant de résultat courant, sous la forme d'une chaîne de caractères.

Historique

Version Description
1.3.0 L'_id du document est retourné sous forme de chaîne de caractères, sachant que la clé doit être unique. Avant la version 1.3.0, filename était retourné.

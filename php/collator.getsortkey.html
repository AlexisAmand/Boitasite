Collator::getSortKey

collator_get_sort_key

(PHP 5 >= 5.3.2, PHP 7, PECL intl >= 1.0.3)

Collator::getSortKey -- collator_get_sort_key — Recupère la clé de tri pour une chaine

Description

Style orienté objet

public Collator::getSortKey ( string $str ) : string

Style procédural

collator_get_sort_key ( Collator $coll , string $str ) : string

Retourne la clé de la collation pour une chaîne. Les clés de collation peuvent être comparées directement au lieu de chaînes, néanmoins elles sont spécifique à l'implémentation et peuvent changer entre les versions de la bibliothéque ICU. Les clés de tries sont généralement seulement utile dans les bases de données ou autres circonstances où les appels de fonctions sont extrêment cher.

Liste de paramètres

coll

Objet Collator.

str

La chaine depuis laquelle produire la clé.

Valeurs de retour

Retourne la clé de collation pour la chaîne, ou FALSE si une erreur survient.

Avertissement

Cette fonction peut retourner FALSE, mais elle peut aussi retourner une valeur équivalent à FALSE. Veuillez lire la section sur les booléens pour plus d'informations. Utilisez l'opérateur === pour tester la valeur de retour exacte de cette fonction.

Historique

Version Description
5.3.15, 5.4.5 Les clés de tri ne contiennent plus d'octets NUL.

Exemples

Exemple #1 Exemple collator_get_sort_key()

<?php
$s1 = 'Hello';

$coll = collator_create('en_US');
$res  = collator_get_sort_key($coll, $s1);

echo bin2hex($res);
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :


3832404046010901dc08

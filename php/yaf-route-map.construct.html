Yaf_Route_Map::__construct

(Yaf >=1.0.0)

Yaf_Route_Map::__construct — Le but de __construct

Description

public Yaf_Route_Map::__construct ([ string $controller_prefer = FALSE [, string $delimiter = "" ]] )

Liste de paramètres

controller_prefer

Permet de considérer le résultat comme étant un contrôleur ou une action.

delimiter

Valeurs de retour

Exemples

Exemple #1 Exemple avec Yaf_Route_Map()

<?php
   /**
    * Ajoute une route de type carte à la pile de routes Yaf_Router
    */
    Yaf_Dispatcher::getInstance()->getRouter()->addRoute("name",
        new Yaf_Route_Map());
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

/* Pour http://yourdomain.com/product/foo/bar
 * la route contiendra les valeurs suivantes :
 */
array(
  "controller" => "product_foo_bar",
)

Exemple #2 Exemple avec Yaf_Route_Map()

<?php
   /**
    * Ajoute une route de type carte à la pile de routes Yaf_Router
    */
    Yaf_Dispatcher::getInstance()->getRouter()->addRoute("name",
        new Yaf_Route_Map(true, "_"));
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

/* Pour http://yourdomain.com/user/list/_/foo/22
 * la route contiendra les valeurs suivantes :
 */
array(
    "action" => "user_list",
)

/**
 * et les paramètres de la requête :
 */
array(
  "foo"   => 22,
)

Exemple #3 Exemple avec Yaf_Route_Map()

<?php
   /**
    * Ajoute une route de type carte à la pile de routes Yaf_Router en appelant addconfig
    */
    $config = array(
        "name" => array(
           "type"  => "map",         //Yaf_Route_Map route
           "controllerPrefer" => FALSE,
           "delimiter"        => "#!", 
           ),
    );
    Yaf_Dispatcher::getInstance()->getRouter()->addConfig(
        new Yaf_Config_Simple($config));
?>

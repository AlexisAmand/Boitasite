EvWatcher::start

(PECL ev >= 0.2.0)

EvWatcher::start — Démarre le Watcher

Description

public EvWatcher::start ( void ) : void

Marque le Watcher comme actif. Notez que seuls les Watchers actifs recevront les événements.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

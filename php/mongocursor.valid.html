MongoCursor::valid

(PECL mongo >=0.9.0)

MongoCursor::valid — Vérifie si le curseur lit un résultat valide

Description

public MongoCursor::valid ( void ) : bool

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE si le résultat courant n'est pas NULL, FALSE sinon.

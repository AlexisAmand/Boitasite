get_resources

(PHP 7)

get_resources — Retourne les ressources active

Description

get_resources ([ string $type ] ) : array

Retourne un tableau de toutes les ressources resource actuellement actives, optionellement filtré par le type de ressource.

Liste de paramètres

type

Si définis, ceci provoquera get_resources() de ne retourner seulement les ressources du type donnée. A list of resource types is available.

Si chaîne de caractères Unknown est fournis pour le type, dans ce cas là seulement les ressources dont le type est inconnue seront retournées.

Si omis, toutes les ressources seront retournées.

Valeurs de retour

Retourne un tableau des ressources actuellement actives, indexé par le numéro de la ressource.

Exemples

Exemple #1 get_resources() non-filtré

<?php
$fp = tmpfile();
var_dump(get_resources());
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

array(1) {
  [1]=>
  resource(1) of type (stream)
}

Exemple #2 get_resources() filtré

<?php
$fp = tmpfile();
var_dump(get_resources('stream'));
var_dump(get_resources('curl'));
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

array(1) {
  [1]=>
  resource(1) of type (stream)
}
array(0) {
}

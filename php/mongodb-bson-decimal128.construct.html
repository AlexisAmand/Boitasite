MongoDB\BSON\Decimal128::__construct

(mongodb >=1.2.0)

MongoDB\BSON\Decimal128::__construct — Construit un nouveau Decimal128

Description

final public MongoDB\BSON\Decimal128::__construct ([ string $value ] )

Note: MongoDB\BSON\Decimal128 n'est compatible qu'avec MongoDB 3.4+. Si vous tentez d'utiliser le type BSON avec une ancienne version de MongoDB, une erreur sera émise.

Liste de paramètres

value (chaîne de caractères)

Une chaîne décimale

Erreurs / Exceptions

  • Lance une exception MongoDB\Driver\InvalidArgumentException lors d'une erreur survenue pendant l'analyse d'un argument.
  • Lance MongoDB\Driver\Exception\InvalidArgumentException sit value n'est pas une chaîne décimale valide.

Exemples

Exemple #1 Exemple avec MongoDB\BSON\Decimal128::__construct()

<?php

var_dump(new MongoDB\BSON\Decimal128(1234.5678));
var_dump(new MongoDB\BSON\Decimal128(NAN));
var_dump(new MongoDB\BSON\Decimal128(INF));

?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

object(MongoDB\BSON\Decimal128)#1 (1) {
  ["dec"]=>
  string(9) "1234.5678"
}
object(MongoDB\BSON\Decimal128)#1 (1) {
  ["dec"]=>
  string(3) "NaN"
}
object(MongoDB\BSON\Decimal128)#1 (1) {
  ["dec"]=>
  string(8) "Infinity"
}

MongoDB\BSON\Timestamp::__toString

(mongodb >=1.0.0)

MongoDB\BSON\Timestamp::__toString — Retourne la représentation sous forme de chaîne de cet horodatage

Description

final public BSON\Timestamp::__toString ( void ) : ReturnType

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la représentation sous forme de chaîne de cet horodatage

Exemples

Exemple #1 Exemple avec MongoDB\BSON\Timestamp::__toString()

<?php

$timestamp = new MongoDB\BSON\Timestamp(1234, 5678);
var_dump((string) $timestamp);

?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

string(11) "[1234:5678]"

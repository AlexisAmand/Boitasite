EventBuffer::readLine

(PECL event >= 1.2.6-beta)

EventBuffer::readLine — Extrait une ligne depuis le début du tampon

Description

public EventBuffer::readLine ( int $eol_style ) : string

Extrait une ligne depuis le début du tampon et la retourne dans une nouvelle chaîne allouée. S'il n'y a pas de ligne complète à lire, la méthode retourne NULL. La terminaison de la ligne n'est pas incluse dans la chaîne copiée.

Liste de paramètres

eol_style

Une constante EventBuffer:EOL_*.

Valeurs de retour

En cas de succès, retourne la ligne lue depuis le tampon, NULL sinon.

phpdbg_color

(PHP 5 >= 5.6.0, PHP 7)

phpdbg_color — Définie la couleur de certaines éléments

Description

phpdbg_color ( int $element , string $color ) : void

Définie la couleur color pour l'élément element.

Liste de paramètres

element

Une des constantes PHPDBG_COLOR_*.

color

Le nom de la couleur. Peut être white, red, green, yellow, blue, purple, cyan ou black, et, optionnellement, avec les compléments -bold ou -underline ; par exemple white-bold ou green-underline.

Valeurs de retour

Aucune valeur n'est retournée.

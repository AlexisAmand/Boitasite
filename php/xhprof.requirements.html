Pré-requis

Bien qu'elle ne soit pas requise, xhprof inclut une interface écrite en PHP. Elle est utilisée pour sauvegarder et afficher les données de profilage de manière exploitable, où la lecture s'effectue directement via un navigateur web. Ainsi, un serveur web PHP rendra probablement l'utilisation de xhprof plus conviviale.

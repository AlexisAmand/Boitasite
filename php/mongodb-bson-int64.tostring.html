MongoDB\BSON\Int64::__toString

(mongodb >=1.5.0)

MongoDB\BSON\Int64::__toString — Retourne la représentation sous forme de chaîne de charactère de Int64

Description

final public MongoDB\BSON\Int64::__toString ( void ) : string

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la représentation sous forme de chaîne de charactère de Int64.

Exemples

Exemple #1 Exemple avec MongoDB\BSON\Int64::__toString()

<?php

$int64 = unserialize('C:18:"MongoDB\BSON\Int64":47:{a:1:{s:7:"integer";s:19:"9223372036854775807";}}');

var_dump((string) $int64);

?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

string(19) "9223372036854775807"

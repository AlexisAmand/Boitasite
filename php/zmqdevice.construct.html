ZMQDevice::__construct

(PECL zmq >= 0.5.0)

ZMQDevice::__construct — Construit un nouveau périphérique

Description

ZMQDevice::__construct ( ZMQSocket $frontend , ZMQSocket $backend [, ZMQSocket $listener ] )

"Les périphériques ØMQ peuvent effectuer une médiation entre les adresses, les services, les files d'attente ou toute autre abstraction pour définir le message et l'interface des sockets." -- zguide

Liste de paramètres

frontend

Paramètre Frontend pour le périphérique. Habituellement le lieu d'arrivée des messages.

backend

Paramètre Backend pour le périphérique. Habituellement le lieu de départ des messages.

listener

Socket d'écoute, qui recoit une copie de tous les messages entrants et sortants. Le type de ce socket doit être SUB, PULL ou DEALER.

Valeurs de retour

L'appel à cette méthode va préparer le périphérique. Habituellement, les périphériques sont des processus qui s'exécutent pendant beaucoup de temps, aussi l'exécution de cette méthode depuis un script interactiv n'est pas recommandé. Cette méthode lance une exception ZMQDeviceException si le périphérique ne peut être lancé.

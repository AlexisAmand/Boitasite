Insertion d'un document

Les tableaux associatifs sont les objets basiques qui peuvent être insérés dans une collection d'une base de données. Un document "document" peut être :

<?php
$doc = array(
    "name" => "MongoDB",
    "type" => "database",
    "count" => 1,
    "info" => (object)array( "x" => 203, "y" => 102),
    "versions" => array("0.9.7", "0.9.8", "0.9.9")
);
?>

Notez que vous pouvez avoir des tableaux et des objets imbriqués. Le driver va toujours stocker un tableau associatif sous forme d'objet dans la base de données. Un tableau indexé numériquement est stocké sous forme de tableau dans le cas où les clés commencent à 0, et ne sont pas interrompues, et sous forme d'objet si les clés du tableau ne commencent pas par 0 ou ont une interruption (e.g. : 0, 1, 4, 5).

Pour insérer ce document, utilisez la méthode MongoCollection::insert() :

<?php
$connection = new MongoClient();
$collection = $connection->database->collectionName;

$collection->insert( $doc );
?>

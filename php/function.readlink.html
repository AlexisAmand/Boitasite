readlink

(PHP 4, PHP 5, PHP 7)

readlink — Renvoie le contenu d'un lien symbolique

Description

readlink ( string $path ) : string

readlink() effectue la même chose que la fonction C readlink.

Liste de paramètres

path

Le chemin vers le lien symbolique.

Valeurs de retour

Retourne le contenu du lien symbolique ou FALSE si une erreur survient.

Note: La fonction échoue si le paramètre path n'est pas un lien symbolique, sauf sous Windows, où le chemin personnalisé sera retourné.

Exemples

Exemple #1 Exemple avec readlink()

<?php

// Affiche e.g. /boot/vmlinux-2.4.20-xfs
echo readlink('/vmlinuz');

?>

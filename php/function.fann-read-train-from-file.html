fann_read_train_from_file

(PECL fann >= 1.0.0)

fann_read_train_from_file — Lit un fichier contenant les données d'entrainement

Description

fann_read_train_from_file ( string $filename ) : resource

Lit un fichier contenant les données d'entrainement.

Liste de paramètres

filename

Le fichier d'entrée au format suivant : 

num_train_data num_input num_output
inputdata séparé par un espace
outputdata séparé par un espace

.
.
.

inputdata séparé par un espace
outputdata séparé par un espace

Valeurs de retour

Retourne une ressource de données d'entrainement en cas de succès, ou FALSE si une erreur survient.

Exemples

Exemple #1 Exemple avec fann_read_train_from_file()

<?php
$train_data = fann_read_train_from_file("xor.data");
if ($train_data) {
    // Faites quelque chose avec $train_data pour la fonction XOR
}
?>

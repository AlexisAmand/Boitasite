ReflectionParameter::isCallable

(PHP 5 >= 5.4.0, PHP 7)

ReflectionParameter::isCallable — Vérifie si le paramètre est de type callable

Description

public ReflectionParameter::isCallable ( void ) : bool

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE si le paramètre est de type callable, FALSE s'il ne l'est pas, ou NULL si une erreur survient.

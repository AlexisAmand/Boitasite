imap_undelete

(PHP 4, PHP 5, PHP 7)

imap_undelete — Enlève la marque d'effacement d'un message

Description

imap_undelete ( resource $imap_stream , int $msg_number [, int $flags = 0 ] ) : bool

Enlève la marque d'effacement du message msg_number, placée avec imap_delete() ou imap_mail_move().

Liste de paramètres

imap_stream

Un flux IMAP retourné par la fonction imap_open().

msg_number

Le numéro du message

flags

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Voir aussi

  • imap_delete() - Marque le fichier pour l'effacement, dans la boîte aux lettres courante
  • imap_mail_move() - Déplace des messages dans une boîte aux lettres

delete — Voir unlink() ou unset()

Description

Il n'y a pas de mot clé ou fonction delete dans le langage PHP. Si vous arrivez sur cette page en tentant de supprimer un fichier, essayer unlink(). Pour supprimer une variable de la portée locale regarder unset().

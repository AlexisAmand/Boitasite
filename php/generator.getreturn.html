Generator::getReturn

(PHP 7)

Generator::getReturn — Obtenir la valeur de retour d'un générateur

Description

public Generator::getReturn ( void ) : mixed

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la valeur de retour du générateur une fois qu'il a terminé l'exécution.

Exemples

Exemple #1 Generator::getReturn() exemple

<?php

$gen = (function() {
    yield 1;
    yield 2;

    return 3;
})();

foreach ($gen as $val) {
    echo $val, PHP_EOL;
}

echo $gen->getReturn(), PHP_EOL;

L'exemple ci-dessus va afficher :

1
2
3

ReflectionClass::getExtension

(PHP 5, PHP 7)

ReflectionClass::getExtension — Récupère un objet ReflectionExtension pour l'extension qui définit la classe

Description

public ReflectionClass::getExtension ( void ) : ReflectionExtension

Récupère un objet ReflectionExtension pour l'extension qui définit la classe.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Un objet ReflectionExtension représentant l'extension qui définit la classe, ou NULL pour les classes définies par l'utilisateur.

Exemples

Exemple #1 Utilisation simple de ReflectionClass::getExtension()

<?php
$class = new ReflectionClass('ReflectionClass');
$extension = $class->getExtension();
var_dump($extension);
?>

L'exemple ci-dessus va afficher :

object(ReflectionExtension)#2 (1) {
  ["name"]=>
  string(10) "Reflection"
}

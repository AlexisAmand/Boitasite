taint

(PECL taint >=0.1.0)

taint — Nettoie une chaîne

Description

taint ( string &$string [, string $... ] ) : bool

Rend une chaîne propre. Utilisé uniquement dans un but de test.

Liste de paramètres

string

...

Valeurs de retour

Retourne TRUE si la transformation a eu lieu, et retournera toujours TRUE si l'extension taint n'est pas activée.

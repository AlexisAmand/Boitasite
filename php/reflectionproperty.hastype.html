ReflectionProperty::hasType

(PHP 7 >= 7.4.0)

ReflectionProperty::hasType — Vérifie si la propriété à un type

Description

public ReflectionProperty::hasType ( void ) : bool

Vérifie si la propriété à un type associé avec.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

TRUE si un type est spécifié, FALSE sinon.

Exemples

Exemple #1 Exemple de ReflectionProperty::hasType()

<?php
class User
{
    public string $name;
}

$rp = new ReflectionProperty('User', 'name');
var_dump($rp->hasType());
?>

L'exemple ci-dessus va afficher :

bool(true)

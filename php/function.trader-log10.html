trader_log10

(PECL trader >= 0.2.0)

trader_log10 — Vecteur Log10

Description

trader_log10 ( array $real ) : array

Calcule le logarithme base-10 pour chaque valeur du paramètre real et retourne le tableau résultant.

Liste de paramètres

real

Tableau de valeurs réelles.

Valeurs de retour

Retourne un tableau avec les données calculées, ou FALSE si une erreur survient.

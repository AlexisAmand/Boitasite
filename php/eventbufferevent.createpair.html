EventBufferEvent::createPair

(PECL event >= 1.2.6-beta)

EventBufferEvent::createPair — Crée deux événements de tampon connectés l'un à l'autre

Description

public static EventBufferEvent::createPair ( EventBase $base [, int $options = 0 ] ) : array

Retourne un tableau de deux objets EventBufferEvent connectés l'un à l'autre. Toutes les options habituelles sont supportées, sauf pour EventBufferEvent::OPT_CLOSE_ON_FREE qui n'a aucun effet, et EventBufferEvent::OPT_DEFER_CALLBACKS qui est toujours activé.

Liste de paramètres

base

Evenement de base associé.

options

Constantes EventBufferEvent::OPT_* combinées avec l'opérateur OR.

Valeurs de retour

Retourne un tableau de deux objets EventBufferEvent connectés à l'un à l'autre.

Historique

1.9.0 La méthode est maintenant statique.

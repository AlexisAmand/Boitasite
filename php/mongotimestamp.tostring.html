MongoTimestamp::__toString

(PECL mongo >= 1.0.1)

MongoTimestamp::__toString — Retourne une représentation sous la forme d'une chaîne de caractères d'un timestamp

Cette extension, qui définie cette méthode est obsolète. Veuillez utiliser l'extension MongoDB à la place. Les alertnatives à cette méthode sont :

  • MongoDB\BSON\Timestamp::__toString()

Description

public MongoTimestamp::__toString ( void ) : string

Retourne le champ "sec" d'un timestamp.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Le nombre de seconde depuis l'époque Unix représenté par le timestamp.

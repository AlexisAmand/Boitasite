imageaffine

(PHP 5 >= 5.5.0, PHP 7)

imageaffine — Retourne une image contenant l'image source transformée, en utilisant optionnellement une zone de découpe

Description

imageaffine ( resource $image , array $affine [, array $clip ] ) : resource

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

image

Une ressource d'image, retournée par une des fonctions de création d'images, comme imagecreatetruecolor().

affine

Un tableau avec les clés de 0 à 5.

clip

Un tableau avec les clés "x", "y", "width" et "height".

Valeurs de retour

Retourne la ressource image liée en cas de succès ou FALSE si une erreur survient.

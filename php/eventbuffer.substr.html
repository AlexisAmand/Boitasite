EventBuffer::substr

(PECL event >= 1.6.0)

EventBuffer::substr — Soustrait une portion des données du tampon

Description

public EventBuffer::substr ( int $start [, int $length ] ) : string

Soustrait une taille de length octets du tampon, en commençant à la position start.

Liste de paramètres

start

La position de départ des données à soustraire.

length

Nombre maximal d'octets à soustraire.

Valeurs de retour

Retourne les données soustraites sous la forme d'une chaîne de caractères en cas de succès, ou bien FALSE si une erreur survient.

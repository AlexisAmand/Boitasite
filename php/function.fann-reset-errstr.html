fann_reset_errstr

(PECL fann >= 1.0.0)

fann_reset_errstr — Réinitialise le message de la dernière erreur

Description

fann_reset_errstr ( resource $errdat ) : void

Réinitialise le message de la dernière erreur.

Liste de paramètres

errdat

Soit une ressource de réseau de neurones, soit une ressource de données d'entrainement d'un réseau de neurones.

Valeurs de retour

Aucune valeur retournée.

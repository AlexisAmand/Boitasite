cubrid_get_autocommit

(PECL CUBRID >= 8.4.0)

cubrid_get_autocommit — Récupère le mode auto-commit de la connexion

Description

cubrid_get_autocommit ( resource $conn_identifier ) : bool

La fonction cubrid_get_autocommit() est utilisée pour récupérer le statut du mode auto-commit de la connexion à la base de données CUBRID.

En CUBRID 8.4.0, le mode auto-commit est désactivé par défault pour la gestion des transactions.

En CUBRID 8.4.1, le mode auto-commit est activé par défaut pour la gestion des transactions.

Liste de paramètres

conn_identifier

Identifiant de connexion.

Valeurs de retour

TRUE, lorsque le mode auto-commit est actif.

FALSE, lorsque le mode auto-commit est inactif.

NULL Si une erreur survient.

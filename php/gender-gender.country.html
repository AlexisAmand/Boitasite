Gender\Gender::country

(PECL gender >= 0.8.0)

Gender\Gender::country — Récupère la représentation textuelle d'un pays

Description

public Gender\Gender::country ( int $country ) : array

Récupère la représentation textuelle d'un pays depuis la constante de la classe Gender.

Liste de paramètres

country

Identifiant du pays (correspond à une constante de la classe Gender\Gender).

Valeurs de retour

Retourne un tableau contenant les noms courts et longs du pays en cas de succès ou FALSE si une erreur survient.

Exemples

Exemple #1 Exemple avec Gender\Gender::country()

$gender = new Gender\Gender;
var_dump($gender->country(Gender\Gender::BRITAIN));

L'exemple ci-dessus va afficher :

array(2) {
  'country_short' =>
  string(2) "UK"
  'country' =>
  string(13) "Great Britain"
}

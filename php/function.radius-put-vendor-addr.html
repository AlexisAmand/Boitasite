radius_put_vendor_addr

(PECL radius >= 1.1.0)

radius_put_vendor_addr — Attache une adresse IP spécifique à un vendeur

Description

radius_put_vendor_addr ( resource $radius_handle , int $vendor , int $type , string $addr ) : bool

Attache une adresse IP spécifique au vendeur pour la requête RADIUS courante.

Note:

Une requête doit être créée via la fonction radius_create_request() avant que cette fonction puisse être appelée.

Liste de paramètres

radius_handle

La ressource RADIUS.

vendor

L'identifiant du vendeur.

type

Le type d'attribut.

addr

Une adresse IPv4 ; par exemple 10.0.0.1.

options

Un masqe d'options d'attribut. Les options disponibles incluent RADIUS_OPTION_TAGGED et RADIUS_OPTION_SALT.

tag

L'attribut tag. Ce paramètre est ignoré tant que l'option RADIUS_OPTION_TAGGED est défini.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

Historique

Version Description
PECL radius 1.3.0 Les paramètres options et tag ont été ajoutés.

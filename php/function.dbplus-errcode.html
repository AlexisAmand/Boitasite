dbplus_errcode

(PHP 4 <= 4.3.0, PECL dbplus >= 0.9)

dbplus_errcode — Récupère le message d'erreur pour le code erreur fourni ou pour la dernière erreur survenue

Description

dbplus_errcode ([ int $errno ] ) : string

Retourne un message d'erreur sous forme de chaîne de caractères pour le code d'erreur fourni.

Liste de paramètres

errno

Le code d'erreur. S'il n'est pas fourni, le code de la dernière opération db++ sera utilisé.

Valeurs de retour

Retourne le message d'erreur.

Notes

Avertissement

Cette fonction est EXPERIMENTALE. Le comportement de cette fonction, son nom, et toute la documentation autour de cette fonction peut changer sans préavis dans une prochaine version de PHP. Cette fonction doit être utilisée à vos risques et périls.

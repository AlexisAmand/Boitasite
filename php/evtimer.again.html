EvTimer::again

(PECL ev >= 0.2.0)

EvTimer::again — Re-démarre le watcher Timer

Description

public EvTimer::again ( void ) : void

Cette méthode fonctionne comme si le Timer s'est terminé, et re-commence son cycle. La sémantique exacte est :

  1. si le Timer est en attente, son statut est nettoyé.

  2. si le Time est commencé, mais ne doit pas re-commencer, alors il sera stoppé (comme s'il s'était terminé normalement).

  3. si le Timer doit re-commencer, soit la méthode le démarre si nécessaire (avec la valeur repeat), ou ré-initialise le Time à la valeur de la variable repeat.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

ZipArchive::getStatusString

(PHP 5 >= 5.2.7, PHP 7)

ZipArchive::getStatusString — Retourne le message du statut de l'erreur, du système ou de zip

Description

public ZipArchive::getStatusString ( void ) : string

Retourne le message du statut de l'erreur, du système ou de zip.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne une chaîne de caractères contenant le message du statut en cas de succès ou FALSE si une erreur survient.

Historique

Version Description
8.0.0 / 1.18.0 Cette méthode peut être appéllé sur une archive fermée.

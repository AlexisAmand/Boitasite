Yaf_Request_Http::isXmlHttpRequest

(Yaf >=1.0.0)

Yaf_Request_Http::isXmlHttpRequest — Vérifie s'il s'agit d'une requête Ajax

Description

public Yaf_Request_Http::isXmlHttpRequest ( void ) : bool

Vérifie s'il s'agit d'une requête Ajax.

Note:

Cette méthode dépend de l'en-tête de la requête : HTTP_X_REQUESTED_WITH, quelques bibliothèques Javascript ne définissent pas cet en-tête lors d'une requête Ajax.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Un booléen

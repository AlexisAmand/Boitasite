Mise en place

Le plugin est implémenté sous la forme d'une extension PHP. Voir aussi les instructions d'installation pour installer cette extension.

Compilez ou configurez l'extension PHP MySQL (API) (mysqli, PDO_MYSQL, ou mysql). Cette extension doit utiliser la bibliothèque mysqlnd car mysqlnd_memcache est un plugin pour la bibliothèque mysqlnd. Pour plus d'informations, référez-vous aux instructions d'installation mysqlnd_memcache.

Ensuite, chargez cette extension dans PHP, et activez le plugin dans le fichier de configuration PHP, en utilisant la directive de configuration mysqlnd_memcache.enable.

Exemple #1 Activation du plugin (php.ini)

; Sous Windows, le nom du fichier est php_mysqnd_memcache.dll
; Charge l'extension
extension=mysqlnd_memcache.so
; L'active
mysqlnd_memcache.enable=1

Suivez les instructions fournies dans le »  manuel de référence MySQL sur l'installation des plugins Memcache pour le serveur MySQL. Activez le plugin et configure l'accès Memcache aux tables SQL.

Les exemples dans le guide de démarrage rapide assume que les tables suivantes existent, et que Memcache est configuré pour y accéder.

Exemple #2 Table SQL utilisée dans le démarrage rapide

CREATE TABLE test(
  id CHAR(16),
  f1 VARCHAR(255),
  f2 VARCHAR(255),
  f3 VARCHAR(255),
  flags INT NOT NULL,
  cas_column INT,
  expire_time_column INT,
  PRIMARY KEY(id)
  ) ENGINE=InnoDB;

INSERT INTO test (id, f1, f2, f3) VALUES (1, 'Hello', 'World', '!');
INSERT INTO test (id, f1, f2, f3) VALUES (2, 'Lady', 'and', 'the tramp');

INSERT INTO innodb_memcache.containers(
  name, db_schema, db_table, key_columns, value_columns, 
  flags, cas_column, expire_time_column, unique_idx_name_on_key)
VALUES (
  'plugin_test', 'test', 'test', 'id', 'f1,f2,f3',
  'flags', 'cas_column', 'expire_time_column', 'PRIMARY KEY');

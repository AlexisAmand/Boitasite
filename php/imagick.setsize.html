Imagick::setSize

(PECL imagick 2.0.0)

Imagick::setSize — Configure la taille de l'objet Imagick

Description

Imagick::setSize ( int $columns , int $rows ) : bool

Configure la taille de l'objet Imagick. Configurez cet attribut avant d'ouvrir une image brute, aux formats RGB, GRAY ou CMYK.

Liste de paramètres

columns

rows

Valeurs de retour

Retourne TRUE en cas de succès.

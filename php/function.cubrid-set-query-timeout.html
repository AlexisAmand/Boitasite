cubrid_set_query_timeout

(PECL CUBRID >= 8.4.1)

cubrid_set_query_timeout — Définit le délai maximal d'exécution d'une requête

Description

cubrid_set_query_timeout ( resource $req_identifier , int $timeout ) : bool

La fonction cubrid_set_query_timeout() est utilisée pour définir le délai maximal d'exécution d'une requête.

Liste de paramètres

req_identifier

Identifiant de requête.

timeout

Délai maximal, en millisecondes.

Valeurs de retour

TRUE en cas de succès.

FALSE lorsqu'une erreur survient.

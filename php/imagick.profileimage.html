Imagick::profileImage

(PECL imagick 2.0.0)

Imagick::profileImage — Ajoute ou retire un profil d'une image

Description

Imagick::profileImage ( string $name , string $profile ) : bool

Ajoute ou retire un profil ICC, IPTC ou générique d'une image. Si le profil est NULL il est retiré de l'image, et sinon, ajouté. Utilisez le nom de '*' et un profil NULL pour retirer tous les profils d'une image.

Liste de paramètres

name

profile

Valeurs de retour

Retourne TRUE en cas de succès.

Erreurs / Exceptions

Lance une exception ImagickException si une erreur survient.

Pré-requis

Cette extension requière » libzip. Version 1.1.2 était fournie avec PHP jusqu'à la version 7.3.

La version minimale supporté est 0.11, mais ultérieur est vivement recommandé.

  • Version 1.2 requis pour le support du chiffrement, voir ZipArchive::setEncryptionIndex()
  • Version 1.3 requis pour le support de l'avancement, voir ZipArchive::registerProgressCallback()
  • Version 1.6 requis pour le support d'annulation, voir ZipArchive::registerCancelCallback()

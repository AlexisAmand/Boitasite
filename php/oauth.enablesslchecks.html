OAuth::enableSSLChecks

(PECL OAuth >= 0.99.5)

OAuth::enableSSLChecks — Active la vérification SSL

Description

public OAuth::enableSSLChecks ( void ) : bool

Active les vérifications de certificats et d'hôtes SLL (activé par défaut).

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

TRUE

Historique

Version Description
0.99.8 La propriété debug a été ajoutée

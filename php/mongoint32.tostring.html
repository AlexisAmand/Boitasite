MongoInt32::__toString

(PECL mongo >= 1.0.9)

MongoInt32::__toString — Retourne la représentation sous forme de chaîne de caractères d'un entier 32-bit

Cette extension, qui définie cette classe est obsolète. Veuillez utiliser l'extension MongoDB à la place. Il n'y a pas d'équivalent à cette classe dans la nouvelle extension.

A la place, la nouvelle extension choisit le type approprié de base de données suivant la valeur entière.

Description

public MongoInt32::__toString ( void ) : string

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la représentation sous forme de chaîne de caractères de l'entier.

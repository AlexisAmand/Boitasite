getmyinode

(PHP 4, PHP 5, PHP 7)

getmyinode — Retourne l'inode du script

Description

getmyinode ( void ) : int

Retourne l'inode du script courant.

Valeurs de retour

Retourne l'inode du script courant, sous la forme d'un entier, ou FALSE si une erreur survient.

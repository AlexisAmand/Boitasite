rrdc_disconnect

(PECL rrd >= 1.1.2)

rrdc_disconnect — Ferme toutes les connexions vers le daemon de mise en cache rrd

Description

rrdc_disconnect ( void ) : void

Ferme toutes les connexions vers le daemon de mise en cache rrd.

Cette fonction est automatiquement appelée lorsque le processus global PHP se termine. Cela dépend de l'API utilisée. Par exemple, cette fonction est appelée automatiquement à la fin d'un script en ligne de commande.

Il en est de l'utilisateur que de vouloir appeler cette fonction à la fin de chaque requête ou non.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

ReflectionExtension::__construct

(PHP 5, PHP 7)

ReflectionExtension::__construct — Construit un nouvel objet ReflectionExtension

Description

public ReflectionExtension::__construct ( string $name )

Construit un objet ReflectionExtension.

Liste de paramètres

name

Nom de l'extension.

Valeurs de retour

Un objet ReflectionExtension.

Exemples

Exemple #1 Exemple avec ReflectionExtension

<?php
$ext = new ReflectionExtension('Reflection');

printf('Extension: %s (version: %s)', $ext->getName(), $ext->getVersion());
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

Extension: Reflection (version: $Revision: 346562 $)

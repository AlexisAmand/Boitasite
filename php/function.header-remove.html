header_remove

(PHP 5 >= 5.3.0, PHP 7)

header_remove — Supprime un en-tête HTTP

Description

header_remove ([ string $name ] ) : void

Supprime un en-tête HTTP précédemment ajouté avec header().

Liste de paramètres

name

Le nom de l'en-tête à supprimer.

Note: Ce paramètre est insensible à la casse.

Valeurs de retour

Aucune valeur n'est retournée.

Exemples

Exemple #1 Supprimer un en-tête HTTP avec header_remove()

<?php
header("X-Foo: Bar");
header("X-Bar: Baz");
header_remove("X-Foo"); 
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

X-Bar: Baz

Exemple #2 Supprimer tous les en-têtes HTTP avec header_remove()

<?php
header("X-Foo: Bar");
header("X-Bar: Baz");
header_remove(); 
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :


    

   

  

 



 

  
Notes

  
Attention
   

    Cette fonction va supprimer tous les en-têtes
    configurés par PHP, y compris les cookies, les sessions et les en-têtes
    X-Powered-By.
   

  

  
Note: 
 

  Les en-têtes ne seront accessibles et s'afficheront que lorsqu'un SAPI qui les supporte sera utilisé.

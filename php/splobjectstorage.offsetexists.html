SplObjectStorage::offsetExists

(PHP 5 >= 5.3.0, PHP 7)

SplObjectStorage::offsetExists — Vérifie si un objet existe dans le stockage

Description

public SplObjectStorage::offsetExists ( object $object ) : bool

Vérifie si un objet existe dans un stockage.

Note:

SplObjectStorage::offsetExists() est un alias de la méthode SplObjectStorage::contains().

Liste de paramètres

object

L'objet à rechercher.

Valeurs de retour

Retourne TRUE si l'objet existe, et FALSE sinon.

Exemples

Exemple #1 Exemple avec SplObjectStorage::offsetExists()

<?php
$s = new SplObjectStorage;
$o1 = new StdClass;
$o2 = new StdClass;

$s->attach($o1);

var_dump($s->offsetExists($o1)); // Similar to isset($s[$o1])
var_dump($s->offsetExists($o2)); // Similar to isset($s[$o2])
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

bool(true)
bool(false)

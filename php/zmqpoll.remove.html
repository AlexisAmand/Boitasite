ZMQPoll::remove

(PECL zmq >= 0.5.0)

ZMQPoll::remove — Supprime un élément du poll

Description

public ZMQPoll::remove ( mixed $item ) : bool

Supprime un élément du poll. Le paramètre item peut être un objet ZMQSocket object, une ressource de flux, ou l'ID retourné depuis la méthode ZMQPoll::add().

Liste de paramètres

item

L'objet ZMQSocket, le flux PHP ou une chaîne de caractères représentant l'id de l'élément.

Valeurs de retour

Retourne TRUE si l'élément a été supprimé et FALSE si l'objet avec l'ID fourni n'existe pas dans le poll.

curl_share_errno

(PHP 7 >= 7.1.0)

curl_share_errno — Retourne le dernier numéro d'erreur du gestionnaire partagé cURL

Description

curl_share_errno ( resource $sh ) : int

Retourne un entier contenant le dernier numéro d'erreur du gestionnaire partagé cURL.

Liste de paramètres

sh

Un gestionnaire partagé cURL retourné par curl_share_init().

Valeurs de retour

Retourne un entier contenant le dernier numéro d'erreur du gestionnaire partagé cURL, ou FALSE si une erreur survient.

XMLDiff\Base::__construct

(PECL xmldiff >= 0.8.0)

XMLDiff\Base::__construct — Constructeur

Description

public XMLDiff\Base::__construct ( string $nsname )

Constructeur de base pour toutes les classes de l'extension xmldiff.

Liste de paramètres

nsname

Espace de noms personnalisé pour le document de différence. L'espace de noms par défaut est http://www.locus.cz/diffmark et est suffisant pour éviter les conflits d'espaces de noms. Utilisez ce paramètre si vous voulez changer d'espace de noms.

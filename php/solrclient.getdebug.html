SolrClient::getDebug

(PECL solr >= 0.9.7)

SolrClient::getDebug — Récupère les données de débogage de la dernière tentative de connexion

Description

public SolrClient::getDebug ( void ) : string

Retourne les données de débogage de la dernière tentative de connexion.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne une chaîne de caractères en cas de succès, et NULL s'il n'y a aucune donnée à retourner.

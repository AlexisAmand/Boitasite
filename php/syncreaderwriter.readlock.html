SyncReaderWriter::readlock

(PECL sync >= 1.0.0)

SyncReaderWriter::readlock — Obtiens un verrou de lecture

Description

public SyncReaderWriter::readlock ([ int $wait = -1 ] ) : bool

Obtiens un verrou de lecture sur un objet SyncReaderWriter.

Liste de paramètres

wait

Le nombre de millisecondes à attente un verrou. Une valeur de -1 signifie que l'on attend indéfiniement.

Valeurs de retour

TRUE si le verrou a été obtenu, FALSE sinon.

Exemples

Exemple #1 Exemple avec SyncReaderWriter::readlock()

<?php
$readwrite = new SyncReaderWriter("FileCacheLock");
$readwrite->readlock();
/* ... */
$readwrite->readunlock();
?>

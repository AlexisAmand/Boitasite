EventBuffer::appendFrom

(PECL event >= 1.6.0)

EventBuffer::appendFrom — Déplace le nombre d'octets fourni depuis un tampon source vers la fin du tampon courant

Description

public EventBuffer::appendFrom ( EventBuffer $buf , int $len ) : int

Déplace le nombre d'octets fourni depuis un tampon source, vers la fin du tampon courant. S'il n'y a pas assez d'octets de disponible, tous les octets disponibles seront déplacés du tampon source.

Liste de paramètres

buf

Tampon source.

len

Valeurs de retour

Retourne le nombre d'octets lu.

Historique

Version Description
1.6.0 La méthode EventBuffer::appendFrom() a été renommée en EventBuffer::appendFrom().

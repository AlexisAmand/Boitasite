EventBufferEvent::writeBuffer

(PECL event >= 1.2.6-beta)

EventBufferEvent::writeBuffer — Ajoute le contenu entier d'un tampon dans un tampon d'événement de sortie

Description

public EventBufferEvent::writeBuffer ( EventBuffer $buf ) : bool

Ajoute le contenu entier d'un tampon dans un tampon d'événement de sortie.

Liste de paramètres

buf

Objet EventBuffer source.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

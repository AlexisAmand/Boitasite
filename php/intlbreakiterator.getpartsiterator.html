IntlBreakIterator::getPartsIterator

(PHP 5 >= 5.5.0, PHP 7)

IntlBreakIterator::getPartsIterator — Crée un itérateur pour naviguer dans les fragments entre les limites

Description

public IntlBreakIterator::getPartsIterator ([ int $key_type = IntlPartsIterator::KEY_SEQUENTIAL ] ) : IntlPartsIterator

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

key_type

Type de clé (optionnel). Les valeurs possibles sont :

  • IntlPartsIterator::KEY_SEQUENTIAL - La valeur par défaut. Entiers séquentiellement croissants utilisés comme clés.
  • IntlPartsIterator::KEY_LEFT - Décalage d'octet à gauche de la partie courante utilisée comme clé.
  • IntlPartsIterator::KEY_RIGHT - Octet décalage droit de la partie courante utilisée comme clé.

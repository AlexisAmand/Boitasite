dba_list

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

dba_list — Liste tous les fichiers de bases de données DBA ouverts

Description

dba_list ( void ) : array

dba_list() liste tous les fichiers de bases de données DBA ouverts.

Valeurs de retour

Un tableau associatif, sous la forme idressource => nomfichier.

MongoPool::getSize

(PECL mongo >= 1.2.3)

MongoPool::getSize — Récupère la taille de la file d'attente des connexions

Description

public static MongoPool::getSize ( void ) : int

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la taille de la file d'attente des connexions.

Historique

Version Description
1.2.11 Lance une alerte de niveau E_DEPRECATED lorsqu'utilisé.

Exemples

Exemple #1 Modification de la taille de la file d'attente

Cet exemple retourne la taille par défaut de la file d'attente, puis, définit une nouvelle taille, et enfin, affiche la nouvelle taille ainsi que les informations de débogage de la file d'attente. Notez que la modification de la taille de la file d'attente n'affecte que les nouvelles files d'attente de connexions, cela ne modifie pas les anciennes.

<?php

$connection = new Mongo("host1");

// La taille de la file est -1
echo "La taille de la file est : ".MongoPool::getSize()."\n";

echo "Modification de la taille à 200\n";

MongoPool::setSize(200);

// La taille de la file est 200
echo "La taille de la file est : ".MongoPool::getSize()."\n";

$conn2 = new Mongo("host2");

// Pour l'hôte host1, c'est -2
// Pour l'hôte host2, c'est 199
var_dump(Mongo::poolDebug());

?>

EventBase::dispatch

(PECL event >= 1.2.6-beta)

EventBase::dispatch — Distribut les événements en attente

Description

public EventBase::dispatch ( void ) : void

Attend que les événements deviennent actifs, et exécute leurs fonctions de rappel. Identique à la méthode EventBase::loop() mais sans drapeau.

Avertissement

Ne détruisez pas l'objet EventBase tant que les ressources associées aux objets Event ne sont pas libérées. Sinon, cela mènera à des résultats totalement indéfinis !

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

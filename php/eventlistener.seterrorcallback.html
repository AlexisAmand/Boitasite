EventListener::setErrorCallback

(PECL event >= 1.2.6-beta)

EventListener::setErrorCallback — Définit la fonction de rappel sur l'évènement d'erreur

Description

public EventListener::setErrorCallback ( string $cb ) : void

Définit la fonction de rappel sur l'évènement d'erreur.

Liste de paramètres

cb

La fonction de rappel pour l'erreur. Doit correspondre au prototype suivant :

callback ([ EventListener $listener = NULL [, mixed $data = NULL ]] ) : void

listener

L'objet EventListener.

data

Données personnalisées de l'utilisateur à associer à la fonction de rappel.

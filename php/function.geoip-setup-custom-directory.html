geoip_setup_custom_directory

(PECL geoip >= 1.1.0)

geoip_setup_custom_directory — Défini un dossier personnalisé pour la base de données GeoIP

Description

geoip_setup_custom_directory ( string $path ) : void

La fonction geoip_setup_custom_directory() va modifier le dossier par défaut de la base données GeoIP. Ceci est équivalent à modifier l'option de configuration geoip.custom_directory.

Liste de paramètres

path

Le chemin complet où se trouve la base de données GeoIP sur le disque.

Valeurs de retour

Aucune valeur n'est retournée.

Exemples

Exemple #1 Exemple avec geoip_setup_custom_directory()

Cet exemple va modifier le chemin par défaut vers la base de données GeoIP.

<?php

geoip_setup_custom_directory('/un/autre/chemin');

print geoip_db_filename(GEOIP_COUNTRY_EDITION);

?>

L'exemple ci-dessus va afficher :

/un/autre/chemin/GeoIP.dat

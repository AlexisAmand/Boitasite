Constantes pré-définies

Ces constantes sont définies par cette extension, et ne sont disponibles que si cette extension a été compilée avec PHP, ou bien chargée au moment de l'exécution.

En relation avec le plugin MySQL Memcache

MYSQLND_MEMCACHE_DEFAULT_REGEXP (chaîne de caractères)
Expression rationnelle par défaut (style PCRE) utilisée pour trouver les requêtes SELECT à mapper dans le point d'accès du plugin MySQL Memcache, si possible. Il est également possible d'utiliser la fonction mysqlnd_memcache_set(), mais l'approche par défaut est l'utilisation d'une expression rationnelle pour trouver le masque.

Assorted

Le numéro de version de ce plugin peut être obtenu en utilisant les constantes MYSQLND_MEMCACHE_VERSION ou MYSQLND_MEMCACHE_VERSION_ID. La constante MYSQLND_MEMCACHE_VERSION est une représentation sous forme de chaîne de caractères du numéro de version numérique contenu dans la constante MYSQLND_MEMCACHE_VERSION_ID, qui est un entier comme 10000. Les développeurs peuvent calculer le numéro de version comme ceci :

Version (partie) Exemple
Majeur*10000 1*10000 = 10000
Mineur*100 0*100 = 0
Patch 0 = 0
MYSQLND_MEMCACHE_VERSION_ID 10000

MYSQLND_MEMCACHE_VERSION (chaîne de caractères)
Chaîne de caractères représentant la version du plugin, par exemple, 1.0.0-alpha.
MYSQLND_MEMCACHE_VERSION_ID (entier)
Numéro de version du plugin, par exemple, 10000.

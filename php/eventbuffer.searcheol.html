EventBuffer::searchEol

(PECL event >= 1.5.0)

EventBuffer::searchEol — Cherche dans le tampon une occurrence de fin de ligne

Description

public EventBuffer::searchEol ([ int $start = -1 [, int $eol_style = EventBuffer::EOL_ANY ]] ) : mixed

Cherche dans le tampon une occurrence de fin de ligne spécifié par le paramètre eol_style. La méthode retourne la position numérique de la chaîne, ou FALSE si la chaîne n'a pas été trouvée.

Si l'argument start est fourni, il représente la position où la recherche commencera ; sinon, la recherche sera effectuée depuis le début de la chaîne. Si l'argument end est fourni, la recherche sera réalisée entre les positions de début et de fin du tampon.

Liste de paramètres

start

Position de début de recherche.

eol_style

Une constante EventBuffer:EOL_*.

Valeurs de retour

Retourne la position numérique de la première occurrence du symbole de fin de ligne dans le tampon, ou bien FALSE s'il n'a pas été trouvé.

Avertissement

Cette fonction peut retourner FALSE, mais elle peut aussi retourner une valeur équivalent à FALSE. Veuillez lire la section sur les booléens pour plus d'informations. Utilisez l'opérateur === pour tester la valeur de retour exacte de cette fonction.

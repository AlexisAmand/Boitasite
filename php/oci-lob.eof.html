OCI-Lob::eof

(PHP 5, PHP 7, PECL OCI8 >= 1.1.0)

OCI-Lob::eof — Test la fin du LOB Oracle

Description

OCI-Lob::eof ( void ) : bool

Vérifie si le pointeur interne d'un LOB est à la fin.

Valeurs de retour

Retourne TRUE si le pointeur interne de LOB a atteint la fin du LOB, et FALSE sinon.

Notes

Note:

Cette fonction retournera une erreur Oracle si OCI-Lob::setBuffering est actif sur le LOB.

mysqli_stmt::__construct

(PHP 5, PHP 7)

mysqli_stmt::__construct — Construit un nouvel objet mysqli_stmt

Description

public mysqli_stmt::__construct ( mysqli $link [, string $query ] )

Cette méthode construit un nouvel objet mysqli_stmt.

Note:

En général, vous devriez utiliser soit la fonction mysqli_prepare(), soit la fonction mysqli_stmt_init() pour créer un objet mysqli_stmt, plutôt que d'instancier directement l'objet avec new mysqli_stmt. Cette méthode (ainsi que la possibilité d'instancier directement des objets mysqli_stmt) peut devenir obsolète et être supprimée dans le futur.

Liste de paramètres

link

Seulement en style procédural : Un identifiant de lien retourné par la fonction mysqli_connect() ou par la fonction mysqli_init()

query

La requête, sous la forme d'une chaîne de caractères. Si ce paramètre est omis, alors le constructeur se comporte comme la fonction mysqli_stmt_init() ; s'il est fourni, alors il se comportera comme la fonction mysqli_prepare().

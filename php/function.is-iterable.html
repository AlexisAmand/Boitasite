is_iterable

(PHP 7 >= 7.1.0)

is_iterable — Détermine si le contenu de la variable est itérable.

Description

is_iterable ( mixed $var ) : bool

Vérifie que le contenu de la variable est accepté par le pseudo-type iterable, c'est-à-dire qu'il s'agit soit d'un array ou un objet qui implémente l'interface Traversable

Liste de paramètres

var

La valeur à vérifier

Valeurs de retour

Retourne TRUE si var est itérable, FALSE sinon.

Exemples

Exemple #1 Exemples avec is_iterable()

<?php

var_dump(is_iterable([1, 2, 3]));  // bool(true)
var_dump(is_iterable(new ArrayIterator([1, 2, 3])));  // bool(true)
var_dump(is_iterable((function () { yield 1; })()));  // bool(true)
var_dump(is_iterable(1));  // bool(false)
var_dump(is_iterable(new stdClass()));  // bool(false)

?>

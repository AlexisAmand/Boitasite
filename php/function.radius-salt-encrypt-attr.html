radius_salt_encrypt_attr

(PECL radius >= 1.3.0)

radius_salt_encrypt_attr — Crypte une valeur en utilisant un Salt

Description

radius_salt_encrypt_attr ( resource $radius_handle , string $data ) : string

Applique l'algorithme RADIUS salt à la valeur fournie.

En général, ceci est fait automatiquement en fournissant l'option RADIUS_OPTION_SALT à la fonction définissant l'attribut, mais cette fonction peut être utilisée si une construction de requête bas niveau est requise.

Liste de paramètres

data

Les données à crypter avec un salt.

Valeurs de retour

Retourne les données cryptées en utilisant un salt ou FALSE si une erreur survient.

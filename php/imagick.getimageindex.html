Imagick::getImageIndex

(PECL imagick 2.0.0)

Imagick::getImageIndex — Lit l'index de l'image courante

Avertissement

Cette fonction est OBSOLÈTE à partir de Imagick 3.4.4. Dépendre de cette fonctionnalité est fortement déconseillé.

Description

Imagick::getImageIndex ( void ) : int

Retourne l'index de l'image courante dans la séquence Imagick. Cette méthode est obsolète. Reportez-vous à la fonction Imagick::getIteratorIndex().

Valeurs de retour

Retourne un entier en cas de succès.

Erreurs / Exceptions

Lance une exception ImagickException si une erreur survient.

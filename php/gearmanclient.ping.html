GearmanClient::ping — Envoi des données à tous les serveurs de tâches pour s'assurer qu'ils sont toujours en fonctionnement

Description

public GearmanClient::ping ( string $workload ) : bool

Envoi de données arbitraires à tous les serveurs de tâches pour s'assurer qu'ils sont bien en fonctionnement. Les données envoyées ne sont en aucun cas utilisées ou analysées. L'utilisation primaire de cette fonction est lors des tests et du débogage.

Liste de paramètres

workload

Quelques données arbitraires qui seront retournées.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

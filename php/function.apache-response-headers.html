apache_response_headers

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

apache_response_headers — Récupère tous les en-têtes de réponse HTTP

Description

apache_response_headers ( void ) : array

Récupère tous les en-têtes de réponse HTTP. Fonctionne avec Apache, FastCGI, et les modules serveur NSAPI dans les serveurs web Netscape/iPlanet/SunONE.

Valeurs de retour

Un tableau de tous les en-têtes de réponse d'Apache en cas de réussite ou FALSE si une erreur survient.

Exemples

Exemple #1 Exemple avec apache_response_headers()

<?php
print_r(apache_response_headers());
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

Array
(
    [Accept-Ranges] => bytes
    [X-Powered-By] => PHP/4.3.8
)

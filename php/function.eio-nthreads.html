eio_nthreads

(PECL eio >= 0.0.1dev)

eio_nthreads — Retourne le nombre de threads actuellement utilisés

Description

eio_nthreads ( void ) : int

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

eio_nthreads() retourne le nombre de threads actuellement utilisés.

SessionIdInterface::create_sid

(PHP 5 >= 5.5.1, PHP 7)

SessionIdInterface::create_sid — Créer un ID de session

Description

abstract public SessionIdInterface::create_sid ( void ) : string

Crée un nouvel ID de session. Cette fonction est automatiquement exécuté quand un nouvel ID de session doit être crée.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Le nouvel ID de session. Il est à noter que cette valeur est retourné internellement à PHP pour du traitement.

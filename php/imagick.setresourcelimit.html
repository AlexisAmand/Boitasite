Imagick::setResourceLimit

(PECL imagick 2.0.0)

Imagick::setResourceLimit — Définit la limite pour une ressource particulière

Description

Imagick::setResourceLimit ( int $type , int $limit ) : bool

Cette méthode est utilisée pour modifier les limites de la ressource de la bibliothèque sous-jacente ImageMagick.

Liste de paramètres

type

Référez-vous à la liste des constantes de type de ressources.

limit

L'une des constantes de type de ressources. L'unité dépend du type de la ressource à limiter.

Valeurs de retour

Retourne TRUE en cas de succès.

Installation

Des informations sur l'installation de ces extensions PECL peuvent être trouvées dans le chapitre du manuel intitulé Installation des extensions PECL. D'autres informations comme les notes sur les nouvelles versions, les téléchargements, les sources des fichiers, les informations concernant les mainteneurs ainsi qu'un CHANGELOG, peuvent être trouvées ici : » https://pecl.php.net/package/sphinx

Si le script ./configure généré par phpize rencontre des problèmes pour trouver les fichiers libsphinxclient (par exemple, s'il a été installé dans un dossier particulier), utilisez ./configure --with-sphinx=$PREFIX, où $PREFIX est le préfixe de l'installation de libsphinxclient.

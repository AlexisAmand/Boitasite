trader_macd

(PECL trader >= 0.2.0)

trader_macd — Moyenne du déplacement Convergence/Divergence

Description

trader_macd ( array $real [, int $fastPeriod [, int $slowPeriod [, int $signalPeriod ]]] ) : array

Liste de paramètres

real

Tableau de valeurs réelles.

fastPeriod

Nombre de période pour le fast MA. Intervalle valide : 2 à 100000.

slowPeriod

Nombre de période pour le MA. Intervalle valide : 2 à 100000.

signalPeriod

Lissage de la ligne du signal (nombre de période). Intervalle valide : 1 à 100000.

Valeurs de retour

Retourne un tableau contenant les données calculées, ou FALSE si une erreur survient.

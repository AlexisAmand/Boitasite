ifxus_free_slob

(PHP 4, PHP 5 < 5.2.1)

ifxus_free_slob — Supprime un objet SLOB

Description

ifxus_free_slob ( int $bid ) : bool

Supprime un objet SLOB.

Liste de paramètres

bid

Un identifiant vers un slob existant.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

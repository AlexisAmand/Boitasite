ZMQDevice::getIdleTimeout — Récupère le délai d'attente maximal d'inactivité

Description

public ZMQDevice::getIdleTimeout ( void ) : ZMQDevice

Récupère le délai d'attente maximal d'inactivité. Ajouté dans l'extension ZMQ en version 1.1.0.

Valeurs de retour

Cette méthode retourne la valeur du délai d'attente maximal de la fonction de rappel d'inactivité.

TokyoTyrantIterator::current

(PECL tokyo_tyrant >= 0.2.0)

TokyoTyrantIterator::current — Récupère la valeur courante

Description

public TokyoTyrantIterator::current ( void ) : mixed

Retourne la valeur courante durant l'itération.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la valeur courante en cas de succès, FALSE si une erreur survient.

Yaf_Request_Abstract::setActionName

(Yaf >=1.0.0)

Yaf_Request_Abstract::setActionName — Définie le nom de l'action

Description

public Yaf_Request_Abstract::setActionName ( string $action [, bool $format_name = true ] ) : void

Définie le nom de l'action à requérir ; habituellement utilisé pour personnalisé le routeur pour définir le nom du contrôleur de la route résultante.

Liste de paramètres

action

chaîne de caractères ; le nom de l'action, en minuscule, comme "index" ou "foo_bar"

format_name

Ceci a été introduit en Yaf 3.2.0 ; par défaut, Yaf va formater le nom en minuscule, si définit à FALSE, Yaf va définir le nom original pour la requête.

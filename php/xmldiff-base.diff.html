XMLDiff\Base::diff

(PECL xmldiff >= 0.8.0)

XMLDiff\Base::diff — Produit la différence de deux documents XML

Description

abstract public XMLDiff\Base::diff ( mixed $from , mixed $to ) : mixed

Méthode abstraire de création de différence à implémenter par héritage de la classe.

Le but premier de cette méthode est de produire la différence de deux documents. L'ordre des paramètres est important, et va produire une sortie différente.

Liste de paramètres

from

Document XML source.

to

Document XML cible.

Valeurs de retour

Dépend de l'implémentation.

mysqli_warning::next

(PHP 5, PHP 7)

mysqli_warning::next — Récupère le prochain avertissement

Description

public mysqli_warning::next ( void ) : bool

Modifie les informations d'avertissement au prochain avertissement si possible.

Une fois que l'avertissement a été défini au prochain avertissement, de nouvelle valeurs pour les propriétés de message, sqlstate et errno de mysqli_warning sont disponible.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne TRUE si le prochain avertissement a été récupéré avec succès. S'il n'y a plus d'avertissement, retournera FALSE

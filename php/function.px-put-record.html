px_put_record

(PECL paradox >= 1.0.0)

px_put_record — Stocke un enregistrement dans une base de données Paradox

Description

px_put_record ( resource $pxdoc , array $record [, int $recpos = -1 ] ) : bool

Stocke un enregistrement dans une base de données Paradox. L'enregistrement sera toujours ajouté à la fin de la base de données, indépendamment des slots libres. Utilisez la fonction px_insert_record() pour ajouter un nouvel enregistrement dans le premier slot libre trouvé dans la base de données.

Liste de paramètres

pxdoc

Identifiant de ressource de base de données Paradox tel que retourné par la fonction px_new().

record

Tableau associatif ou indexé contenant les valeurs des champs tel que retourné par la fonction px_retrieve_record().

recpos

Ce paramètre optionnel peut être utilisé pour spécifier un nombre d'enregistrements supérieurs au nombre d'enregistrements actuellement en base de données. La fonction ajoutera autant d'enregistrements vides que nécessaires. Ce paramètre est inutile dans la plupart des cas.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

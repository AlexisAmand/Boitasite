EventHttpConnection::setMaxBodySize

(PECL event >= 1.2.6-beta)

EventHttpConnection::setMaxBodySize — Définit la taille maximale du corps pour la connexion

Description

public EventHttpConnection::setMaxBodySize ( string $max_size ) : void

Définit la taille maximale du corps pour la connexion.

Liste de paramètres

max_size

La taille maximale du cors, en octets.

Valeurs de retour

Aucune valeur n'est retournée.

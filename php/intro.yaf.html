Introduction

L'extension Yet Another Framework(Yaf) est un framework PHP, utilisé pour le développement d'application web.

Yaf requiert une version PHP 5.2.1 ou supérieur. Les versions plus anciennes peuvent ne pas fonctionner.

Un benchmark de Yaf peut être vu sur la page des » performances Yaf.

AppendIterator::current

(PHP 5 >= 5.1.0, PHP 7)

AppendIterator::current — Lit la valeur courante

Description

public AppendIterator::current ( void ) : mixed

Lit la valeur courante.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

La valeur courante si elle est valide, et NULL sinon.

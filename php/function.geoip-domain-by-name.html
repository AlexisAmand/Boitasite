geoip_domain_by_name

(PECL geoip >= 1.1.0)

geoip_domain_by_name — Récupère le second niveau du nom de domaine

Description

geoip_domain_by_name ( string $hostname ) : string

La fonction geoip_domain_by_name() va retourner le second niveau du domaine associé avec un nom d'hôte ou l'adresse IP.

Cette fonction n'est actuellement disponible que pour les utilisateurs qui ont acheté une édition commerciale GeoIP Domain. Une alerte sera émise si la bonne base de données ne peut être trouvée.

Liste de paramètres

hostname

Le nom d'hôte ou l'adresse IP.

Valeurs de retour

Retourne le nom de domaine en cas de succès, ou FALSE si l'adresse ne peut être trouvée dans la base de données.

Exemples

Exemple #1 Exemple avec geoip_domain_by_name()

Cet exemple va afficher le domaine associé avec l'IP 61.106.139.1.

<?php
$domain = geoip_domain_by_name('61.106.139.1');

if ($domain) {
    echo 'Le domaine est : '. $domain;
}

?>

L'exemple ci-dessus va afficher :

Le domaine est : von.co.kr

EventConfig::__construct

(PECL event >= 1.2.6-beta)

EventConfig::__construct — Construit un objet EventConfig

Description

public EventConfig::__construct ( void )

Construit un objet EventConfig qui pourra être passé au constructeur EventBase::__construct().

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne un objet EventConfig.

Exemples

Exemple #1 Exemple avec EventConfig::__construct()

<?php
// On ignore la méthode "select"
$cfg = new EventConfig();
if ($cfg->avoidMethod("select")) {
    echo "Méthode `select' ignorée\n";
}

// Crée un event_base associé avec la configuration
$base = new EventBase($cfg);

/* Maintenant, $base est configuré pour ignorer la méthode select */
?>

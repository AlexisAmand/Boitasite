SyncEvent::wait

(PECL sync >= 1.0.0)

SyncEvent::wait — Attend que l'objet SyncEvent ne soit lancé

Description

public SyncEvent::wait ([ int $wait = -1 ] ) : bool

Attend que l'objet SyncEvent ne soit lancé.

Liste de paramètres

wait

Le nombre de millisecondes à attendre que l'événement ne soit lancé. Une valeur de -1 signifie que l'on attend indéfiniement.

Valeurs de retour

Un booléen valant TRUE signifie que l'événement a été émis, FALSE sinon.

Exemples

Exemple #1 Exemple avec SyncEvent::wait()

<?php
// Dans une application web :
$event = new SyncEvent("GetAppReport");
$event->fire();

// Dans un cron :
$event = new SyncEvent("GetAppReport");
$event->wait();
?>

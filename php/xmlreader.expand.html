XMLReader::expand

(PHP 5 >= 5.1.0, PHP 7)

XMLReader::expand — Retourne une copie du noeud courant comme un noeud d'objet DOM

Description

public XMLReader::expand ([ DOMNode $basenode ] ) : DOMNode

Cette méthode copie le noeud courant et retourne l'objet approprié DOM.

Liste de paramètres

basenode

Un DOMNode définissant la cible DOMDocument pour l'objet DOM créé.

Valeurs de retour

L'objet DOMNode résultant ou FALSE en cas d'erreur.

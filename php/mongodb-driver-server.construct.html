MongoDB\Driver\Server::__construct

(mongodb >=1.0.0)

MongoDB\Driver\Server::__construct — Créer un nouveau serveur (non utilisé)

Description

final private MongoDB\Driver\Server::__construct ( void )

Les objets MongoDB\Driver\Server sont créés en interne par MongoDB\Driver\Manager lorsqu'une connexion de base de données est établie et peut être retournée par MongoDB\Driver\Manager::getServers() et MongoDB\Driver\Manager::selectServer().

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Voir aussi

  • MongoDB\Driver\Manager::getServers() - Retourner les serveurs auxquels ce gestionnaire est connecté
  • MongoDB\Driver\Manager::selectServer() - Select a server matching a read preference

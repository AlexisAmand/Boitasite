ZMQSocket::sendmulti

(PECL zmq >= 0.8.0)

ZMQSocket::sendmulti — Envoi un message multipart

Description

public ZMQSocket::sendmulti ( array $message [, int $mode = 0 ] ) : ZMQSocket

Envoi un message multipart en utilisant le socket. L'opération sera bloquante à moins que ZMQ::MODE_NOBLOCK ne soit utilisé.

Liste de paramètres

message

Le message à envoyer - un tableau de chaînes de caractères

mode

Drapeaux pour le mode pour recevoir les messages multipart ou faire que l'opération ne soit pas bloquante. Voir les constantes ZMQ::MODE_*.

Valeurs de retour

Retourne l'objet courant. Lance une exception ZMQSocketException en cas d'erreur. Si ZMQ::MODE_NOBLOCK est utilisé, et que l'opération est bloquante, FALSE sera retourné.

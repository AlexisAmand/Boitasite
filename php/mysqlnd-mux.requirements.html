Pré-requis

PHP 5.5.0 ou supérieur. Quelques fonctionalités avancées nécessitent PHP 5.5.0 ou supérieur.

Le plugin de multiplexage mysqlnd_mux supporte toutes les applications PHP et toutes les extensions PHP MySQL (mysqli, mysql, PDO_MYSQL). L'extension PHP MySQL doit être configurée pour utiliser mysqlnd afin de pouvoir utiliser le plugin mysqlnd_mux pour mysqlnd.

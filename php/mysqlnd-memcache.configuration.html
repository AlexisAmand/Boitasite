Configuration à l'exécution

Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.

Options de configuration Mysqlnd_memcache
Nom Défaut Modifiable Historique
mysqlnd_memcache.enable 1 PHP_INI_SYSTEM Disponible depuis la version 1.0.0

Voici un éclaircissement sur l'utilisation des directives de configuration.

mysqlnd_memcache.enable integer

Active ou désactive le plugin. Si désactivé, l'extension ne se raccordera pas à mysqlnd pour se comporter comme proxy des appels API C mysqlnd.

Note:

Cette option est principalement utilisée par les développeurs pour intégrer cette extension statiquement dans PHP. Les utilisateurs sont encouragés à utiliser cette extension comme un objet partagé et de le décharger complètement quand elle n'est pas nécessaire.

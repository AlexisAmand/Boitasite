cubrid_set_autocommit

(PECL CUBRID >= 8.4.0)

cubrid_set_autocommit — Définit le mode auto-commit pour la connexion

Description

cubrid_set_autocommit ( resource $conn_identifier , bool $mode ) : bool

La fonction cubrid_set_autocommit() est utilisée pour définir le mode auto-commit pour la connexion à la base de données CUBRID.

En CUBRID PHP, le mode auto-commit est désactivé par défaut pour la gestion des transactions. Lorsque le mode auto-commit passe de Off à On, tous les travaux en attente sont automatiquement validés.

Liste de paramètres

conn_identifier

Identifiant de connexion.

mode

Le mode auto-commit. Les constantes suivantes peuvent être utilisées :

  • CUBRID_AUTOCOMMIT_FALSE
  • CUBRID_AUTOCOMMIT_TRUE

Valeurs de retour

TRUE lorsque l'opération est un succès.

FALSE lorsqu'une erreur survient.

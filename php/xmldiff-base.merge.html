XMLDiff\Base::merge

(PECL xmldiff >= 0.8.0)

XMLDiff\Base::merge — Produit un nouveau document XML basé sur une différence

Description

abstract public XMLDiff\Base::merge ( mixed $src , mixed $diff ) : mixed

Méthode abstraite de fusion à implémenter par héritage de la classe.

Basiquement, le but de cette méthode est de produire un nouveau document XML basé sur les informations de différence.

Liste de paramètres

src

Document XML source.

diff

Document produit par la méthode de création de différence.

Valeurs de retour

Dépend de l'implémentation.

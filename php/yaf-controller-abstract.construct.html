Yaf_Controller_Abstract::__construct

(Yaf >=1.0.0)

Yaf_Controller_Abstract::__construct — Constructeur de Yaf_Controller_Abstract

Description

final private Yaf_Controller_Abstract::__construct ( void )

Yaf_Controller_Abstract::__construct() est final, ce qui signifie qu'il ne peut pas être surchargé. Pour cela, reportez-vous plutôt à la méthode Yaf_Controller_Abstract::init().

Liste de paramètres

Cette fonction ne contient aucun paramètre.

fbsql_num_fields

(PHP 4 >= 4.0.6, PHP 5 < 5.3.0)

fbsql_num_fields — Lit le nombre de champs dans un résultat

Description

fbsql_num_fields ( resource $result ) : int

Retourne le nombre de champs dans le résultat result.

Liste de paramètres

result

Un identifiant de résultat retourné par la fonction fbsql_query() ou la fonction fbsql_db_query().

Valeurs de retour

Retourne le nombre de champs, sous la forme d'un entier.

Voir aussi

  • fbsql_db_query() - Envoie une requête à la base FrontBase
  • fbsql_query() - Exécute une requête sur un serveur FrontBase
  • fbsql_fetch_field() - Lit des informations sur une colonne dans un résultat, et retourne un objet
  • fbsql_num_rows() - Lit le nombre de lignes dans un résultat

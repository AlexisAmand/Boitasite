Qu'est-ce qui a changé en PHP 5.5.x

La plupart des améliorations de PHP 5.5.x n'ont aucun impact sur le code existant. Il y a quelques incompatibilités ainsi que quelques nouvelles fonctionalités qui devraient être prises en compte, et votre code devrait être testé avant de changer de version de PHP en environnement de production.

Pour les systèmes mis à jour depuis une version de PHP plus ancienne, la documentation correspondante est disponible :

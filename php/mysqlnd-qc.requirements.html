Pré-requis

PHP 5.3.3 ou une versions plus récentes de PHP.

PECL/mysqlnd_qc est un plugin mysqlnd. Il se connecte directement à la bibliothèque mysqlnd. Pour utiliser ce plugin avec une extension PHP MySQL, l'extension (mysqli, mysql, ou PDO_MYSQL) doit être activée dans la bibliothèque mysqlnd.

Pour l'utilisation du gestionnaire de stockage MEMCACHE : Utilisez libmemcache 0.38 ou suivants. PECL/mysqlnd_qc 1.2 a été testé avec libmemcache 1.4.0.

Pour l'utilisation du gestionnaire de stockage sqlite : Utilisez l'extension sqlite3 fourni avec PHP.

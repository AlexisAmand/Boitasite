GearmanTask::isRunning

(PECL gearman >= 0.5.0)

GearmanTask::isRunning — Vérifie si la tâche est actuellement en fonctionnement

Description

public GearmanTask::isRunning ( void ) : bool

Indique si la tâche est actuellement en fonctionnement.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

TRUE si la tâche est en fonctionnement, FALSE sinon.

MongoCursor::current

(PECL mongo >=0.9.0)

MongoCursor::current — Retourne l'élément courant

Description

public MongoCursor::current ( void ) : array

Retourne NULL tant que la méthode MongoCursor::next() est appelé.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Le document résultant, sous la forme d'un tableau associatif. NULL sera retourné s'il n'y a aucun résultat.

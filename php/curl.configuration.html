Configuration à l'exécution

Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.

Options de configuration cURL
Nom Défaut Modifiable Historique
curl.cainfo NULL PHP_INI_SYSTEM Disponible depuis PHP 5.3.7.
Pour plus de détails sur les modes PHP_INI_*, reportez-vous à Où une directive de configuration peut être modifiée.

Voici un éclaircissement sur l'utilisation des directives de configuration.

curl.cainfo string

Une valeur par défaut pour l'option CURLOPT_CAINFO. Ce doit être un chemin absolu.

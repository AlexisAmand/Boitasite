fann_set_sarprop_step_error_shift

(PECL fann >= 1.0.0)

fann_set_sarprop_step_error_shift — Défini le changement de l'étape d'erreur sarprop

Description

fann_set_sarprop_step_error_shift ( resource $ann , float $sarprop_step_error_shift ) : bool

Défini le changement de l'étape d'erreur sarprop.

Liste de paramètres

ann

Ressource de réseau de neurones.

sarprop_step_error_shift

Le changement de l'étape d'erreur sarprop.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

Notes

Note:

Cette fonction est maintenant disponible si l'extension fann a été compilée avec libfann >= 2.2.

SplObjectStorage::offsetGet

(PHP 5 >= 5.3.0, PHP 7)

SplObjectStorage::offsetGet — Retourne les données associées à un objet

Description

public SplObjectStorage::offsetGet ( object $object ) : mixed

Retourne les données associées à un objet dans le stockage.

Liste de paramètres

object

L'objet à rechercher.

Valeurs de retour

Les données précédemment associées avec l'objet.

Erreurs / Exceptions

Lance une exception UnexpectedValueException lorsque object n'a pû être trouvé.

Exemples

Exemple #1 Exemple avec SplObjectStorage::offsetGet()

<?php
$s = new SplObjectStorage;

$o1 = new StdClass;
$o2 = new StdClass;

$s[$o1] = "hello";
$s->attach($o2);


var_dump($s->offsetGet($o1)); // Similaire à $s[$o1]
var_dump($s->offsetGet($o2)); // Similaire à $s[$o2]
?>

L'exemple ci-dessus va afficher quelque chose de similaire à :

string(5) "hello"
NULL

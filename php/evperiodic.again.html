EvPeriodic::again

(PECL ev >= 0.2.0)

EvPeriodic::again — Arrête et re-démarre le watcher périodique

Description

public EvPeriodic::again ( void ) : void

Arrête et re-démarre le watcher périodique. Ceci n'est utile que lorsque les attributs ont changé.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Aucune valeur n'est retournée.

sqlsrv_server_info — Retourne des informations sur le serveur

Description

sqlsrv_server_info ( resource $conn ) : array

Retourne des informations sur le serveur.

Liste de paramètres

conn

La ressource de connexion qui connecte le client et le serveur.

Valeurs de retour

Retourne un tableau tel que décrit dans la table suivante :

Tableau retourné
CurrentDatabase La base de données connectée
SQLServerVersion La version du serveur SQL
SQLServerName Le nom du serveur

Exemples

Exemple #1 Exemple avec sqlsrv_server_info()

<?php
$serverName = "serverName\sqlexpress";
$conn = sqlsrv_connect( $serverName);
if( $conn === false ) {
     die( print_r( sqlsrv_errors(), true));
}

$server_info = sqlsrv_server_info( $conn);
if( $server_info )
{
    foreach( $server_info as $key => $value) {
       echo $key.": ".$value."<br />";
    }
} else {
      die( print_r( sqlsrv_errors(), true));
}
?>

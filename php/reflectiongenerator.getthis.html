ReflectionGenerator::getThis

(PHP 7)

ReflectionGenerator::getThis — Récupère la valeur de $this du générateur

Description

public ReflectionGenerator::getThis ( void ) : object

Récupère la valeur de $this du générateur auquel il a accès.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la valeur de $this, ou NULL si le générateur n'a pas été crée dans un contexte de classe.

Exemples

Exemple #1 Exemple avec ReflectionGenerator::getThis()

<?php

class GenExample
{
    public function gen()
    {
        yield 1;
    }
}

$gen = (new GenExample)->gen();

$reflectionGen = new ReflectionGenerator($gen);

var_dump($reflectionGen->getThis());

L'exemple ci-dessus va afficher quelque chose de similaire à :

object(GenExample)#3 (0) {
}

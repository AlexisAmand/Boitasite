mysqlnd_ms_fabric_select_shard

(No version information available, might only be in Git)

mysqlnd_ms_fabric_select_shard — Passe en mode partagé

Description

mysqlnd_ms_fabric_select_shard ( mixed $connection , mixed $table_name , mixed $shard_key ) : array
Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Relatif à MySQL Fabric.

Passe la connexion en mode partagé pour le nom de table fourni, et la clé partagée.

Liste de paramètres

connection

Un gestionnaire de connexion MySQL obtenu depuis n'importe quelles fonctions de connexion des extensions mysqli, mysql ou PDO_MYSQL.

table_name

Le nom de la table à demander à Fabric.

shard_key

La clé partagée à demander à Fabric.

Valeurs de retour

FALSE en cas d'erreur. Sinon, TRUE

Notes

Note:

mysqlnd_ms_fabric_select_shard() requiert PECL mysqlnd_ms >> 1.6.0.

fann_test_data

(PECL fann >= 1.0.0)

fann_test_data — Effectue un test sur un jeu de données d'entrainement et calcule le MSE pour ces données

Description

fann_test_data ( resource $ann , resource $data ) : float

Effectue un test sur un jeu de données d'entrainement et calcule le MSE pour ces données.

Cette fonction met à jour le MSE à sa valeur la plus sûre.

Liste de paramètres

ann

Ressource de réseau de neurones.

data

Ressource de données d'entrainement du réseau de neurones.

Valeurs de retour

Le MSE mis à jour, ou FALSE si une erreur survient.

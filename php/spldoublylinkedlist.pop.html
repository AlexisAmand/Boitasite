SplDoublyLinkedList::pop

(PHP 5 >= 5.3.0, PHP 7)

SplDoublyLinkedList::pop — Dépile et retourne le dernier élément de la liste

Description

public SplDoublyLinkedList::pop ( void ) : mixed

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

Retourne la valeur du dernier élément de la liste.

Erreurs / Exceptions

Lance une exception RuntimeException lorsque la structure de données est vide.

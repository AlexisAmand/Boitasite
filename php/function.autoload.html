__autoload

(PHP 5, PHP 7)

__autoload — Tente de charger une classe indéfinie

Avertissement

Cette fonctionnalité est OBSOLÈTE à partir de PHP 7.2.0. Dépendre de cette fonctionnalité est fortement déconseillé.

Description

__autoload ( string $class ) : void

Vous pouvez définir cette fonction pour activer le chargement automatique des classes.

Liste de paramètres

class

Nom de la classe à charger.

Valeurs de retour

Aucune valeur n'est retournée.

Configuration à l'exécution

Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.

memcached.use_sasl 0 PHP_INI_SYSTEM

Voici un éclaircissement sur l'utilisation des directives de configuration.

memcached.sess_locking integer

Utilise le verrouillage de session. Les valeurs valides sont : On, Off ; par défaut, c'est On.

memcached.sess_consistent_hash integer

Mode de hachage de consistence de la session Memcache. Si défini à On, le hachage de consistence est utilisé par le gestionnaire de session. Lorsque le hachage de consistence est utilisé, il est possible d'ajouter ou de supprimer un ou des noeuds du cache sans trop se soucier des clés existantes. Par défaut, vaut Off.

memcached.sess_binary integer

Utilise le mode binaire de session memcache. La réplication Libmemcache ne fonctionne que si le mode binaire est activé. Par défaut, vaut Off.

memcached.sess_lock_wait integer

Délai d'attente entre plusieurs essais de tentative de déverrouillage des verrous de session, en microseconde. Soyez prudent en définissant cette valeur. Les valeurs valides sont des entiers, où 0 est la valeur par défaut. Les valeurs négatives seront interprétées comme une tentative de verrouillage. La valeur par défaut est 150000.

memcached.sess_prefix string

Préfixe pour les clés de session Memcache. Les valeurs valides sont des chaînes de caractères dont la longueur ne devra pas excéder 219 octets. La valeur par défaut est "memc.sess.key."

memcached.sess_number_of_replicas integer

Le nombre de réplication de sessions Memcache.

memcached.sess_randomize_replica_read integer

La session de réplication Memcache lit de façon aléatoire.

memcached.sess_remove_failed integer

Permet la suppression automatique des serveurs Memcache en échec.

memcached.compression_type string

Définit le type de compression ; les valeurs valides sont : fastlz, zlib. Par défaut, vaut fastlz.

memcached.compression_factor float

Facteur de compression. Le stockage ne compresse les valeurs que si le facteur de compression (sauvegarde) dépasse la limite définie. Le stockage se fait de façon compressé si : plain_len > comp_len * factor. La valeur par défaut est 1.3 (23% d'espace épargné).

memcached.compression_threshold integer

Le seuil de compression. Ne compressez pas les valeurs linéarisées en dessous de ce seuil. Par défaut, vaut 2000 octets.

memcached.serializer string

Définit le linéariseur par défaut pour les nouveaux objets memcache. Les valeurs valides sont : php, igbinary, json, json_array.

json

Encodage standard php JSON. Ce linéariseur est rapide, compacte, mais ne fonctionne que sur des données encodées UTF-8, et n'implémente pas complètement la linéarisation. Voir l'extension JSON.

json_array

Identique à JSON, mais le décodage se faire dans des tableaux.

php

Le linéariseur PHP standard.

igbinary

Un linéariseur binaire.

Par défaut, ce sera igbinary s'il est disponible, sinon PHP.

memcached.use_sasl integer

Utilisation de l'authentification SASL pour les connexions. Les valeurs valides sont : On, Off. Par défaut, vaut Off.

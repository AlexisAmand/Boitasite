La classe MongoDB\BSON\UTCDatetime

(mongodb >=1.0.0)

Introduction

Représente une » date BSON. La valeur est un entier de 64 bits qui représente le nombre de millisecondes depuis l'époque UNIX (1 janvier 1970). Les valeurs négatives représentent des dates avant le 1970.

Imagick::setImageMatteColor

(PECL imagick 2.0.0)

Imagick::setImageMatteColor — Configure la couleur mate

Avertissement

Cette fonction est OBSOLÈTE à partir de Imagick 3.4.4. Dépendre de cette fonctionnalité est fortement déconseillé.

Description

Imagick::setImageMatteColor ( mixed $matte ) : bool

Configure la couleur mate.

Liste de paramètres

matte

Valeurs de retour

Retourne TRUE en cas de succès.

Erreurs / Exceptions

Lance une exception ImagickException si une erreur survient.

Historique

Version Description
2.1.0 Permet désormais l'utilisation d'une chaîne pour représenter la couleur. Les versions précédentes ne permettaient que les objets ImagickPixel.

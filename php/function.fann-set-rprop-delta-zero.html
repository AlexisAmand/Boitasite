fann_set_rprop_delta_zero

(PECL fann >= 1.0.0)

fann_set_rprop_delta_zero — Défini la taille de l'étape initiale

Description

fann_set_rprop_delta_zero ( resource $ann , float $rprop_delta_zero ) : bool

Défini la taille de l'étape initiale ; doit être un nombre positif.

Liste de paramètres

ann

Ressource de réseau de neurones.

rprop_delta_zero

La taille de l'étape initiale.

Valeurs de retour

Retourne TRUE en cas de succès, FALSE sinon.

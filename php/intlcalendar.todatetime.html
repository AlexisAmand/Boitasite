IntlCalendar::toDateTime

(PHP 5 >= 5.5.0, PHP 7, PECL >= 3.0.0a2)

IntlCalendar::toDateTime — Converti un objet IntlCalendar en un objet DateTime

Description

Style orienté objet

public IntlCalendar::toDateTime ( void ) : DateTime

Style procédural

intlcal_to_date_time ( IntlCalendar $cal ) : DateTime

Crée un objet DateTime qui représente le même instant (avec une précison à la seconde) et avec un fuseau horaire analogue à l'objet initial (la seule différence est que le fuseau horaire de l'objet DateTime sera compris de PHP, alors que le fuseau horaire de l'objet IntlCalendar sera compris d'ICU).

Liste de paramètres

cal

La ressource IntlCalendar.

Valeurs de retour

Un objet DateTime avec le même fuseau horaire que l'objet d'origine (en utilisant la base de données PHP au lieu de celle d'ICU) et représentant le même moment, mis à part la précision (précision à la seconde au lieu de milliseconde). Retourne FALSE en cas d'échec.

Exemples

Exemple #1 Exemple avec IntlCalendar::toDateTime()

<?php
ini_set('date.timezone', 'UTC');
ini_set('intl.default_locale', 'pt_PT');

$cal = IntlCalendar::createInstance('Europe/Lisbon'); //temps courant

$dt = $cal->toDateTime();
print_r($dt);

L'exemple ci-dessus va afficher :

DateTime Object
(
    [date] => 2013-07-02 00:29:13
    [timezone_type] => 3
    [timezone] => Europe/Lisbon
)

Types de ressources

Cette extension définit les types de ressources suivantes :

Liste des ressource dans GD
Nom Description Notes
gd Ressource Image, utilisé par des fonctions comme imagecreatefrompng()  
gd font Ressource Police créé en interne par imageloadfont()  
gd PS font Ressource police PostScript Type 1 retournée par imagepsloadfont() Supprimé à partir de PHP 7.0.0.
gd PS encoding PostScript Type 1 encoding resource, returned by imagepsencodefont() Supprimé à partir de PHP 7.0.0.

DOMDocument::schemaValidateSource

(PHP 5, PHP 7)

DOMDocument::schemaValidateSource — Valide un document selon un schéma

Description

public DOMDocument::schemaValidateSource ( string $source [, int $flags ] ) : bool

Valide un document selon le schéma défini dans la chaîne donnée.

Liste de paramètres

source

Une chaîne contenant le schéma.

flags

Un masque de drapeau de validation de schéma Libxml. Actuellement, la seule valeur supportée est LIBXML_SCHEMA_CREATE. Disponible depuis PHP 5.5.2 et Libxml 2.6.14.

Valeurs de retour

Cette fonction retourne TRUE en cas de succès ou FALSE si une erreur survient.

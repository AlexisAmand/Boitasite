DateTimeImmutable::modify

(PHP 5 >= 5.5.0, PHP 7)

DateTimeImmutable::modify — Créé un nouvel objet dont le timestamp a été modifié

Description

public DateTimeImmutable::modify ( string $modifier ) : DateTimeImmutable

Crée un nouvel objet DateTimeImmutable dont le timestamp a été modifié. L'objet original ne sera pas modifié.

Liste de paramètres

object

Style procédural uniquement : Un objet DateTime retourné par la fonction date_create(). Cette fonction modifie cet objet.

modifier

Une chaîne date/heure. Les formats valides sont expliqués dans la documentation sur les formats Date et Heure.

Valeurs de retour

Retourne le nouvel objet créé ou FALSE si une erreur survient.

mysqli_get_links_stats

(PHP 5 >= 5.6.0, PHP 7)

mysqli_get_links_stats — Retourne des informations sur les liens ouverts et mis en cache

Description

mysqli_get_links_stats ( void ) : array

La fonction mysqli_get_links_stats() retourne des informations sur les liens MySQL ouverts et mis en cache.

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

La fonction mysqli_get_links_stats() retourne un tableau associatif contenant trois éléments, dont les clés sont les suivantes :

total

Un entier indiquant le nombre total de liens ouverts, quelques soient leurs statuts.

active_plinks

Un entier représentant le nombre de connexions persistantes actives.

cached_plinks

Un entier représentant le nombre de connexions persistantes inactives.

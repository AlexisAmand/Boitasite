MongoCommandCursor::next

(PECL mongo >=1.5.0)

MongoCommandCursor::next — Avance le curseur au prochain résultat

Description

public MongoCommandCursor::next ( void ) : void

Liste de paramètres

Cette fonction ne contient aucun paramètre.

Valeurs de retour

NULL.

Erreurs / Exceptions

Lance une exception MongoConnectionException si la base de données ne peut être jointe, et une exception MongoCursorTimeoutException si le délai d'attente maximal est atteint.

EvFork::createStopped

(PECL ev >= 0.2.0)

EvFork::createStopped — Crée un instance arrêté de la classe de l'observateur EvFork

Description

final public static EvFork::createStopped ( string $callback [, string $data [, string $priority ]] ) : object

Identique à la méthode EvFork::__construct() mais ne démarre pas automatiquement l'observateur.

Liste de paramètres

callback

Voir les fonctions de rappel d'observateurs.

data

Données personalisées associées avec l'observateur.

priority

Priorité d'observateur

Valeurs de retour

Retourne un objet EvFork (arrêté) en cas de succès.

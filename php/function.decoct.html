decoct

(PHP 4, PHP 5, PHP 7)

decoct — Convertit de décimal en octal

Description

decoct ( int $number ) : string

Retourne une chaîne contenant la représentation octale du nombre donné number. Le nombre le plus grand qui puisse être converti est 4294967295 en décimal, ce qui donnera 37777777777. Pour les plates-formes 64 bits, il s'agit généralement de 9223372036854775807 en décimal résultant en 777777777777777777777.

Liste de paramètres

number

Valeur décimale à convertir

Valeurs de retour

Une représentation octale de number.

Exemples

Exemple #1 Exemple avec decoct()

<?php
echo decoct(15) . "\n";
echo decoct(264);
?>

L'exemple ci-dessus va afficher :

17
410

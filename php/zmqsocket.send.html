ZMQSocket::send

(PECL zmq >= 0.5.0)

ZMQSocket::send — Envoi un message

Description

public ZMQSocket::send ( string $message [, int $mode = 0 ] ) : ZMQSocket

Envoi un message en utilisant le socket. L'opération peut être bloquante à moins que ZMQ::MODE_NOBLOCK ne soit utilisé.

Liste de paramètres

message

Le message à envoyer.

mode

Drapeaux du mode pour recevoir des messages multipart ou rendre l'opération non-bloquante. Voir les constantes ZMQ::MODE_*.

Valeurs de retour

Retourne l'objet courant. Lance une exception ZMQSocketException en cas d'erreur. Si ZMQ::MODE_NOBLOCK est utilisé, et que l'opération est bloquante, FALSE sera retourné.

XMLReader::getAttributeNo

(PHP 5 >= 5.1.0, PHP 7)

XMLReader::getAttributeNo — Récupère la valeur d'un attribut par index

Description

public XMLReader::getAttributeNo ( int $index ) : string

Retourne la valeur d'un attribut basé sur sa position ou d'une chaîne de caractères vide si l'attribut n'existe pas ou n'est pas positionné sur le noeud.

Liste de paramètres

index

La position de l'attribut.

Valeurs de retour

La valeur de l'attribut, une chaîne de caractères vide (avant PHP 5.6) ou NULL (depuis PHP 5.6) si aucun attribut n'existe à index ou n'est pas positionné sur l'élément.

Historique

Version Description
5.6.0 XMLReader::getAttributeNo() retourne maintenant NULL si l'attribut n'existe pas.

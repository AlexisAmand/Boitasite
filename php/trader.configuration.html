Configuration à l'exécution

Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.

Options de configuration Trader
Nom Défaut Modifiable Historique
trader.real_precision 3 PHP_INI_ALL Depuis trader 0.2.1
trader.real_round_mode HALF_DOWN PHP_INI_ALL Depuis trader 0.3.0

Voici un éclaircissement sur l'utilisation des directives de configuration.

trader.real_precision integer

Toutes les valeurs des tableaux retournés seront arrondies à cette précision. Cependant, les calculs dans la TA-Lib sont effectués avec des valeurs non-arrondies.

trader.real_round_mode string

Contrôle la politique d'arrondissement des réels trader. Les valeurs valides sont HALF_UP, HALF_DOWN, HALF_EVEN et HALF_ODD. Le comportement est identique à la fonction round() utilisée avec l'argument mode.

RecursiveIteratorIterator::setMaxDepth

(PHP 5 >= 5.1.0, PHP 7)

RecursiveIteratorIterator::setMaxDepth — Définit la profondeur maximale

Description

public RecursiveIteratorIterator::setMaxDepth ([ int $max_depth = -1 ] ) : void

Définit la profondeur maximale autorisée.

Avertissement

Cette fonction est actuellement non documentée ; seule la liste des arguments est disponible.

Liste de paramètres

max_depth

La profondeur maximale autorisée. -1 sera utilisé pour une profondeur infinie.

Valeurs de retour

Aucune valeur n'est retournée.

Erreurs / Exceptions

Émet une Exception si max_depth est inférieur à -1.
